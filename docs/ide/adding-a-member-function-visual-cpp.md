# 멤버 함수 추가(Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Class View**에서 모든 클래스에 멤버 함수를 추가할 수 있습니다.  이것을 사용하면, 헤더 파일에 선언이 추가되고, 클래스의 구현 파일에 스텁 멤버 함수 본문이 추가되므로 함수 본문을 수정할 수 있습니다.  
  
### 클래스에 멤버 함수를 추가하려면  
  
1.  **클래스 뷰**에서 프로젝트 노드를 확장하여 프로젝트의 클래스를 표시합니다. \(**클래스 뷰**를 열기 위해, 메뉴 표시줄에서 **보기**,  **클래스 뷰**를 선택합니다\)  
  
2.  클래스의 멤버 함수를 추가하려면 원하는 클래스의 바로 가기 메뉴를 열고  **추가**,  **함수 추가**를 선택합니다.  
  
3.  멤버 함수에 대한 적절 한 정보를 제공합니다.  자세한 내용은 [멤버 함수 추가 마법사](../ide/add-member-function-wizard.md)을 참조하십시오.  
  
4.  **완료**버튼을 선택하여,  멤버 함수 코드를 생성합니다.
