# 리소스 속성 페이지
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

리소스 컴파일러 사용에 대한 자세한 내용은 [Using RC\(The RC Command Line\)](http://msdn.microsoft.com/library/windows/desktop/aa381055)를 참조하십시오.  
  
 **리소스** 속성 페이지에 액세스하는 방법에 대한 내용은 [방법: 속성 페이지를 사용하여 프로젝트 속성 지정](../misc/how-to-specify-project-properties-with-property-pages.md)을 참조하십시오.  
  
 프로그래밍 방식으로 이러한 속성에 액세스하려면 <xref:Microsoft.VisualStudio.VCProjectEngine.VCResourceCompilerTool>을 참조하십시오.
