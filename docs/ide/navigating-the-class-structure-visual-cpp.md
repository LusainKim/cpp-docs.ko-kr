# 클래스 구조 탐색(Visual C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[클래스 뷰](http://msdn.microsoft.com/ko-kr/8d7430a9-3e33-454c-a9e1-a85e3d2db925) 또는 [개체 브라우저](http://msdn.microsoft.com/ko-kr/f89acfc5-1152-413d-9f56-3dc16e3f0470)를 사용하여 함수, 상속, 인터페이스 등을 포함하는 프로젝트의 클래스 구조를 탐색할 수 있습니다.  
  
 클래스 뷰에서 함수 이름을 두 번 클릭하여 함수 선언으로 이동할 수 있습니다.  클래스 뷰에서 클래스 이름을 두 번 클릭하여 클래스 정의로 이동할 수 있습니다.
