# 웹 참조 속성 페이지
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**웹 참조** 속성 페이지에서는 XML Web services 프록시 클래스가 생성되는 방법을 지정합니다.  프로젝트에 웹 참조를 추가하면 XML Web services 프록시 클래스가 생성됩니다.  자세한 내용은 [웹 참조 추가 대화 상자](http://msdn.microsoft.com/ko-kr/bdf05776-c591-40af-bfd7-e1e2aa1e87b5)를 참조하십시오.  
  
 **웹 참조** 속성 페이지에는 다음 속성이 포함되어 있습니다.  
  
 **출력 파일**  
 XML Web services 프록시 클래스를 포함시킬 파일 이름입니다.  
  
 **시작 배너 표시 안 함**  
 웹 서비스 설명 언어 도구\(Wsdl.exe\)에 대한 배너를 표시하지 않습니다.  
  
 **Namespace**  
 생성된 웹 프록시의 이름을 지정합니다.  
  
 **추가 참조**  
 프록시 DLL에서 참조하는 추가 DLL을 지정합니다.  
  
 **웹 참조** 속성 페이지에 액세스하는 방법에 대한 내용은 [방법: 속성 페이지를 사용하여 프로젝트 속성 지정](../misc/how-to-specify-project-properties-with-property-pages.md)을 참조하십시오.
