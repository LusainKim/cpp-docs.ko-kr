# ordered (OpenMP Clauses)
[!INCLUDE[vs2017banner](../../../assembler/inline/includes/vs2017banner.md)]

병렬에 필요한 [for](../../../parallel/openmp/reference/for-openmp.md) 문 경우는 [ordered](../../../parallel/openmp/reference/ordered-openmp-directives.md) 지시문이 루프에서 사용 될 수 있습니다.  
  
## 구문  
  
```  
ordered  
```  
  
## 설명  
 `ordered`적용 되는 [for](../../../parallel/openmp/reference/for-openmp.md) 지시문입니다.  
  
 자세한 내용은 [2.4.1 for Construct](../../../parallel/openmp/2-4-1-for-construct.md)를 참조하십시오.  
  
## 예제  
 참조 하십시오 [ordered](../../../parallel/openmp/reference/ordered-openmp-directives.md) 를 사용 하는 예에 대 한 `ordered` 절.
