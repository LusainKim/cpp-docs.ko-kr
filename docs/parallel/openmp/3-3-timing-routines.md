# <a name="33-timing-routines"></a>3.3 타이밍 루틴
이 섹션에서 설명 하는 함수는 휴대용 벽 시계 타이머를 지원 합니다.  
  
-   `omp_get_wtime` 함수 경과 된 벽 시계 시간을 반환 합니다.  
  
-   `omp_get_wtick` 함수 연속 클록 틱 간 시간 (초)을 반환 합니다.
