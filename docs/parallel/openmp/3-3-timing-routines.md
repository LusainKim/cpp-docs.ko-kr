# 3.3 Timing Routines
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 섹션에서 설명 하는 기능은 휴대용 시계 타이머를 지원 합니다.  
  
-   `omp_get_wtime` 함수 벽 시계 경과 된 시간을 반환 합니다.  
  
-   `omp_get_wtick` 함수를 연속 클록 틱 사이의 시간 \(초\) 반환 합니다.
