# 2.7 Data Environment
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 섹션에서는 지시문 및 데이터 환경의 병렬 영역을 실행 하는 동안 다음과 같은 제어 하기 위한 여러 개의 절을 제공 합니다.  
  
-   A  **threadprivate** 지시문 파일 범위, 네임 스페이스 범위, 또는 정적 블록 범위 변수를 스레드에 로컬 하 제공 됩니다 \(다음 섹션 참조\).  
  
-   병렬 또는 작업 공유 생성 기간 동안 공유 특성 변수를 제어 하는 지시문을 지정할 수 있습니다 절에 설명 되어  [섹션 2.7.2](../../parallel/openmp/2-7-2-data-sharing-attribute-clauses.md) 25 페이지입니다.
