# 2.7.2.4 shared
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 절에 나타나는 변수 공유의  *변수 목록* 팀에서 모든 스레드 간에.  모든 스레드는 팀 내에서 동일한 저장소 영역에 대 한 액세스  **공유** 변수입니다.  
  
 구문에는  **공유** 절은 다음과 같습니다:  
  
```  
shared(variable-list)  
```
