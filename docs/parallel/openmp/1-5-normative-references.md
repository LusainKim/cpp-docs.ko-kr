# <a name="15-normative-references"></a>1.5 표준 참조
-   ISO/IEC 9899: 1999, *정보 기술-프로그래밍 언어-C*합니다. 이 OpenMP API 사양에서는 ISO/IEC 9899: 1999 C99 라고 합니다.  
  
-   ISO/IEC 9899:1990 *정보 기술-프로그래밍 언어-C*합니다. 이 OpenMP API 사양에서는 ISO/IEC 9899:1990 C90 라고 합니다.  
  
-   ISO/IEC 14882:1998 *정보 기술-프로그래밍 언어-c + +*합니다. C + +는이 OpenMP API 사양 ISO/IEC 14882:1998를 나타냅니다.  
  
 이 OpenMP API 사양 C를 가리키는 경우 구현에서 지원 되는 기본 언어 참조가 이루어집니다.
