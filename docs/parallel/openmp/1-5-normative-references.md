# 1.5 Normative References
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

-   ISO\/IEC 9899:1999  *정보 기술\-언어 프로그래밍\-C*.  이 OpenMP API 사양에 ISO\/IEC 9899:1999 c99로 참조합니다.  
  
-   ISO\/IEC 9899:1990  *정보 기술\-언어 프로그래밍\-C*.  이 OpenMP API 사양에 ISO\/IEC 9899:1990 c90로 참조합니다.  
  
-   ISO\/IEC 14882  *정보 기술\-언어 프로그래밍\-C\+\+*.  이 OpenMP API 사양에 ISO\/IEC 14882 C\+\+ 이름으로 참조합니다.  
  
 OpenMP API 사양은이 C를 참조 하는 경우 참조 구현에서 지원 되는 기본 언어로 만들어집니다.
