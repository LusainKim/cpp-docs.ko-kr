# StringReference::Data 메서드
이 `StringReference`의 콘텐츠를 char16 값의 배열로 반환합니다.  
  
## 구문  
  
```cpp  
const ::default::char16 * Data() const  
```  
  
## 반환 값  
 char16 유니코드 텍스트 문자의 배열입니다.
