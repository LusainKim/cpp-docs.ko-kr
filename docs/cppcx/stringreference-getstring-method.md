# StringReference::GetString 메서드
문자열의 내용을 `Platform::String^`로 반환합니다.  
  
## 구문  
  
```scr  
__declspec(no_release_return) __declspec(no_refcount)  
    ::Platform::String^ GetString() const  
```  
  
## 반환 값  
 문자열 데이터가 포함된 `Platform::String^`입니다.
