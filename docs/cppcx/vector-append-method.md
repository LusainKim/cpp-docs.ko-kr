# Vector::Append 메서드
현재 Vector의 마지막 항목 다음에 지정된 항목을 삽입합니다.  
  
## 구문  
  
```  
  
virtual void Append(  
   T item  
);  
```  
  
#### 매개 변수  
 `index`  
 Vector에 삽입할 항목입니다.`item` 형식은 *T* 형식 이름으로 정의됩니다.
