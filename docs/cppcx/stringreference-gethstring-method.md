# StringReference::GetHSTRING 메서드
문자열의 내용을 `__abi_HSTRING`으로 반환합니다.  
  
## 구문  
  
```scr  
__abi_HSTRING GetHSTRING() const  
  
```  
  
## 반환 값  
 문자열 데이터가 포함된 `__abi_HSTRING`입니다.
