# Platform::Box 클래스
`Windows::Foundation::DateTime` 등의 값 형식 또는 `int` 등의 스칼라 형식을 `Platform::Object` 형식에 저장할 수 있습니다. 값 형식을 `Object^`으로 캐스팅할 때 boxing이 암시적으로 발생하므로 일반적으로 `Box`를 명시적으로 사용할 필요가 없습니다.  
  
## 구문  
  
```cpp  
ref class Box abstract;  
```
