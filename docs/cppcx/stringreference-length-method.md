# StringReference::Length 메서드
문자열의 문자 수를 반환합니다.  
  
## 구문  
  
```cpp  
unsigned int Length() const  
```  
  
## 반환 값  
 문자열에 있는 문자의 수를 지정하는 부호 없는 정수입니다.
