# Agile::~Agile 소멸자
Agile 클래스의 현재 인스턴스를 제거합니다.  
  
## 구문  
  
```cpp  
  
~Agile();  
```  
  
## 설명  
 이 소멸자는 현재 Agile 개체가 나타내는 개체도 해제합니다.
