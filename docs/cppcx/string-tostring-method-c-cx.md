# String::ToString 메서드(C++/CX)
현재 문자열과 같은 값을 갖는 String 개체를 반환합니다.  
  
## 구문  
  
```cpp  
  
String^ String::ToString()  
```  
  
## 반환 값  
 현재 문자열과 같은 값을 갖는 String 개체입니다.
