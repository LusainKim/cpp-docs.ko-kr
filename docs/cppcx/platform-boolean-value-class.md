# Platform::Boolean 값 클래스
부울 값을 나타냅니다.`bool`에 해당합니다.  
  
## 구문  
  
```cpp  
public value struct Boolean  
```  
  
## 멤버  
 부울에는 [Platform::Object 클래스](../cppcx/platform-object-class.md)에서 파생된 Equals\(\), GetHashCode\(\) 및 ToString\(\) 메서드와 [Platform::Type 클래스](../cppcx/platform-type-class.md)에서 파생된 GetTypeCode\(\) 메서드가 있습니다.
