# WriteOnlyArray::Length 속성
호출자가 할당한 배열의 요소 수를 반환합니다.  
  
## 구문  
  
```cpp  
property unsigned int Length{  
   unsigned int get() const;  
}  
```  
  
## 반환 값  
 배열의 요소 수입니다.  
  
## 요구 사항  
 **헤더:** vccorlib.h  
  
 **네임스페이스:** Platform
