# String::Begin 메서드
현재 문자열의 시작 부분에 대한 포인터를 반환합니다.  
  
## 구문  
  
```cpp  
  
char16* Begin()  
```  
  
## 반환 값  
 현재 문자열의 시작 부분에 대한 포인터입니다.
