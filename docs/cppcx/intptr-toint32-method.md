# IntPtr::ToInt32 메서드
현재 IntPtr 값을 32비트 정수로 변환합니다.  
  
## 구문  
  
```cpp  
int32 IntPtr::ToInt32();  
```  
  
## 반환 값  
 32비트 정수입니다.
