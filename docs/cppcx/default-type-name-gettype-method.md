# default::(type_name)::GetType 메서드
현재 형식을 나타내는 Platform::Type^을 반환합니다.  
  
## 구문  
  
```cpp  
  
Platform::Type^ GetType()  
```  
  
## 반환 값  
 현재 개체를 나타내는 [Platform::Type](../cppcx/platform-type-class.md)^ 개체입니다.
