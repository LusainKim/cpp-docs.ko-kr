# COMException::COMException 생성자
COMException 클래스의 새 인스턴스를 초기화합니다.  
  
## 구문  
  
```cpp  
COMException( int hresult )  
```  
  
#### 매개 변수  
 hresult  
 예외로 표시되는 오류 HRESULT입니다.
