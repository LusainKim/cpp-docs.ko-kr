# Map::Remove 메서드
지정한 키\/값 쌍을 현재 Map 개체에서 삭제합니다.  
  
## 구문  
  
```  
  
virtual void Remove(  
   K key);  
```  
  
#### 매개 변수  
 `key`  
 키\-값 쌍의 키 부분입니다.`key`의 형식은 형식 이름 *K*입니다.
