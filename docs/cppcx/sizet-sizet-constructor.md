# SizeT::SizeT 생성자
지정된 값을 사용하여 SizeT의 새 인스턴스를 초기화합니다.  
  
## 구문  
  
```cpp  
SizeT( uint32 value1 );   SizeT( void* value2 );  
```  
  
## 매개 변수  
 value1  
 부호 없는 32비트 값입니다.  
  
 value2  
 부호 없는 32비트 값에 대한 포인터입니다.
