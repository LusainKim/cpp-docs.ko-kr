# Platform::Metadata::DefaultMemberAttribute 특성
몇 가지 가능한 오버로드된 함수 중에서 호출할 기본 설정 함수를 나타냅니다.  
  
## 구문  
  
```cpp  
  
public ref class DefaultMember abstract : Attribute  
```  
  
## 상속  
 [Platform::Object](../cppcx/object-object-constructor.md)  
  
 [Platform::Metadata::Attribute](../cppcx/platform-metadata-attribute-attribute.md)  
  
## 설명  
 JavaScript 응용 프로그램에서 사용할 메서드에 DefaultMember 특성을 적용합니다.
