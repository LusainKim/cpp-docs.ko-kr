# Windows::Foundation::Collections 네임스페이스(C++/CX)
[!INCLUDE[cppwrt_short](../cppcx/includes/cppwrt-short-md.md)]는 Vector, VectorView, Map 및 MapView 컬렉션 클래스 사용을 간소화하는 함수를 통해 Windows::Foundation::Collections 네임스페이스를 보완합니다.  
  
## 구문  
  
```  
  
namespace Windows {  
    namespace Foundation {  
        namespace Collections;  
    }  
}  
```  
  
### 함수  
  
|이름|설명|  
|--------|--------|  
|[back\_inserter 함수](../cppcx/back-inserter-function.md)|컬렉션 끝에 값을 삽입하는 데 사용될 수 있는 반복기를 반환합니다.|  
|[begin 함수](../cppcx/begin-function.md)|컬렉션의 시작 부분을 가리키는 반복기를 반환합니다.|  
|[end 함수](../cppcx/end-function.md)|컬렉션 끝 너머를 가리키는 반복자를 반환합니다.|  
|[to\_vector 함수](../cppcx/to-vector-function.md)|컬렉션을 std::vector로 반환합니다.|  
  
## 요구 사항  
 **헤더:** collection.h  
  
 **네임스페이스:** Windows::Foundation::Collections
