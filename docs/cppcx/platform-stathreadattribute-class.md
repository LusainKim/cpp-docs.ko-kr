# Platform::STAThreadAttribute 클래스
응용 프로그램에 대한 스레딩 모델이 STA\(단일 스레드 아파트\)임을 나타냅니다.  
  
## 구문  
  
```  
public ref class STAThreadAttribute sealed : Attribute  
```  
  
## 멤버  
  
### Public 생성자  
  
|이름|설명|  
|--------|--------|  
|[STAThreadAttribute 생성자](../cppcx/stathreadattribute-constructor-1.md)|클래스의 새 인스턴스를 초기화합니다.|  
  
### Public 메서드  
 STAThreadAttribute 특성은 [Platform::Object 클래스](../cppcx/platform-object-class.md)에서 상속됩니다. STAThreadAttribute도 다음 멤버를 오버로드하거나 포함합니다.  
  
|이름|설명|  
|--------|--------|  
|[STAThreadAttribute::Equals](../cppcx/stathreadattribute-equals.md)|지정한 개체와 현재 개체가 같은지 여부를 확인합니다.|  
|[STAThreadAttribute::GetHashCode](../cppcx/stathreadattribute-gethashcode.md)|이 인스턴스의 해시 코드를 반환합니다.|  
|[STAThreadAttribute::ToString](../cppcx/stathreadattribute-tostring.md)|현재 개체를 나타내는 문자열을 반환합니다.|  
  
## 상속 계층  
 `Platform`
