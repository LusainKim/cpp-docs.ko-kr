# UnorderedMapView::Size 메서드
UnorderedMapView에 있는 [Windows::Foundation::Collections::IKeyValuePair\<K,V\>](http://msdn.microsoft.com/library/windows/apps/br226031.aspx) 요소의 수를 반환합니다.  
  
## 구문  
  
```cpp  
  
virtual property unsigned int Size;  
```  
  
## 반환 값  
 UnorderedMapView의 요소 수입니다.  
  
## 요구 사항  
 **헤더:** collection.h  
  
 **네임스페이스:** Platform::Collections
