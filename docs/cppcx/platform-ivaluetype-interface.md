# Platform::IValueType 인터페이스
`Platform::IValueType`은 값 클래스 및 값 구조체로 구현되는 인프라 인터페이스입니다. 코드에서 명시적으로 사용되지 않아야 합니다.  
  
## 구문  
  
```cpp  
interface class IValueType  
```
