# try_to_lock_t 구조체
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Represents a `struct` that defines a [try\_to\_lock](../Topic/try_to_lock%20Variable.md) object.  Used to select one of the overloaded constructors of [unique\_lock](../standard-library/unique-lock-class.md).  
  
## 구문  
  
```cpp  
struct try_to_lock_t;  
```  
  
## 요구 사항  
 **헤더:** mutex  
  
 **네임스페이스:** std
