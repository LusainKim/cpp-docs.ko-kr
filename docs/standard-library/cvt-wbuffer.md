# &lt;cvt/wbuffer&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

이전 버전의 Visual Studio에서는 헤더 `<cvt/wstring>`이 stdext::cvt 네임스페이스의 템플릿 클래스 [wbuffer\_convert 클래스](../standard-library/wbuffer-convert-class.md)를 정의했습니다.  이 헤더는 이전 버전과 호환성을 위해 유지됩니다.  새 코드는 `std` 네임스페이스의 [\<locale\>](../standard-library/locale.md)에 정의된 버전의 클래스를 사용해야 합니다.  
  
## 구문  
  
```  
  
#include <cvt/wbuffer>  
  
```
