# defer_lock_t 구조체
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Represents a type that defines a [defer\_lock](../Topic/defer_lock%20Variable.md) object that is used to select one of the overloaded constructors of [unique\_lock](../standard-library/unique-lock-class.md).  
  
## 구문  
  
```cpp  
struct defer_lock_t;  
```
