# <a name="lttypeinfogt"></a>&lt;typeinfo&gt;
형식 식별 연산자 [typeid 연산자](../cpp/typeid-operator.md)와 연결된 여러 형식을 정의하기 위한 표준 헤더 `<typeinfo>`를 포함합니다. 이 연산자는 정적 및 동적 형식에 대한 정보를 생성합니다.  
  
## <a name="syntax"></a>구문  
  
```  
#include <typeinfo>  
  
```  
  
## <a name="remarks"></a>설명  
 \<typeinfo>에서 정의되는 클래스에 대한 자세한 내용은 다음 항목을 참조하세요.  
  
- [bad_cast 예외](../cpp/bad-cast-exception.md)  
  
- [bad_typeid 예외](../cpp/bad-typeid-exception.md)  
  
- [type_info 클래스](../cpp/type-info-class.md)
