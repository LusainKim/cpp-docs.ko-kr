##  <a name="generic_errno"></a>  generic_errno  
 `<errno.h>`에서 Posix에 의해 정의되는 모든 오류 코드 매크로의 심볼 이름을 제공하는 열거형을 나타내는 형식입니다.  
  
```
typedef errc generic_error;
```  
  
### <a name="remarks"></a>설명  
 이 형식은 [errc](../standard-library/system-error-enums.md#errc)와 동일한 의미입니다.
