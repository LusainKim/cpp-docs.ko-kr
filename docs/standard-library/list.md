# &lt; 목록 &gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

컨테이너 템플릿 클래스 목록 및 다양한 지원 템플릿을 정의합니다.  
  
## <a name="syntax"></a>구문  
  
```  
#include <list>  
```  
  
### <a name="operators"></a>연산자  
  
|||  
|-|-|  
|[연산자! =](../Topic/%3Clist%3E%20operators.md#operator_neq)|연산자의 좌변에 있는 목록 개체가 우변에 있는 목록 개체와 같지 않은지 테스트합니다.|  
|[연산자 <](../Topic/%3Clist%3E%20operators.md#operator_lt_)|연산자의 좌변에 있는 목록 개체가 우변에 있는 목록 개체보다 작은지 테스트합니다.|  
|[연산자 \< =](../Topic/%3Clist%3E%20operators.md#operator_lt__eq)|연산자의 좌변에 있는 목록 개체가 우변에 있는 목록 개체보다 작거나 같은지 테스트합니다.|  
|[연산자 = =](../Topic/%3Clist%3E%20operators.md#operator_eq_eq)|연산자의 좌변에 있는 목록 개체가 우변에 있는 목록 개체와 같은지 테스트합니다.|  
|[연산자 >](../Topic/%3Clist%3E%20operators.md#operator_gt_)|연산자의 좌변에 있는 목록 개체가 우변에 있는 목록 개체보다 큰지 테스트합니다.|  
|[연산자 > =](../Topic/%3Clist%3E%20operators.md#operator_gt__eq)|연산자의 좌변에 있는 목록 개체가 우변에 있는 목록 개체보다 크거나 같은지 테스트합니다.|  
  
### <a name="classes"></a>클래스  
  
|||  
|-|-|  
|[list 클래스](../standard-library/list-class.md)|선형 배열에서 해당 요소를 유지 관리하고 시퀀스 내 모든 위치에서 효율적인 삽입과 삭제를 수행할 수 있도록 하는 시퀀스 컨테이너의 템플릿 클래스입니다.|
