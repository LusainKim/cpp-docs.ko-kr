# &lt;cinttypes&gt;
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

표준 C 라이브러리 헤더 \<inttypes.h\>를 포함합니다.  이 헤더를 포함하면 \<cstdint\>도 포함됩니다.  
  
## 구문  
  
```  
  
#include <cinttypes>  
  
```  
  
## 설명  
 이 헤더를 포함하는 경우 표준 C 라이브러리 헤더의 외부 링크를 사용하여 선언한 이름이 `std` 네임스페이스에 선언됩니다.
