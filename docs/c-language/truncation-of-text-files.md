# 텍스트 파일 잘라내기
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.9.3** 텍스트 스트림에서 쓰기가 해당 지점을 벗어난 관련 파일을 잘라내는지 여부  
  
 텍스트 스트림에서 쓰기는 해당 지점을 벗어난 파일을 잘라내지 않습니다.
