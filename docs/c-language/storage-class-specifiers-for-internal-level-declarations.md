# 내부 수준 선언에 대한 저장소 클래스 지정자
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

내부 수준에서 변수 선언에 대한 네 가지 *storage\-class\-specifier* 비단말 중 하나를 사용할 수 있습니다.  이러한 선언에서 *storage\-class\-specifier*를 생략하는 경우 기본 저장소 클래스는 **auto**입니다.  따라서 **auto** 키워드는 C 프로그램에서 거의 표시되지 않습니다.
