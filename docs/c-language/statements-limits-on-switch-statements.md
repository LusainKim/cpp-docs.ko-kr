# 문: Switch 문의 제한
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.6.4.2** `switch` 문에 있는 **case** 값의 최대 수  
  
 Microsoft C에서는 **switch** 문에 있는 **case** 값의 수를 제한하지 않습니다.  이 수는 사용 가능한 메모리에 의해서만 제한됩니다.
