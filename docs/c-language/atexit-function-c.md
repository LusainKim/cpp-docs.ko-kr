# atexit 함수 (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** 인수의 값이 0이 아니거나 EXIT\_SUCCESS이거나 EXIT\_FAILURE인 경우 `atexit` 함수에서 반환하는 상태입니다.  
  
 `atexit` 함수는 성공하면 0을 반환하고 실패하면 0이 아닌 값을 반환합니다.
