# 가장 큰 배열 크기
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 3.3.3.4, 4.1.1** 배열의 최대 크기를 수용하는 데 필요한 정수 형식, 즉 **size\_t**의 크기  
  
 이 `size_t` 형식 정의는 32 비트 x86 플랫폼에서 `unsigned int` 입니다.  64 비트 플랫폼에서의 `size_t` typedef는 **부호 없는 \_\_int64**입니다.
