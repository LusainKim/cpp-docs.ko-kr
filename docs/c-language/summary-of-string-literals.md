# <a name="summary-of-string-literals"></a>문자열 리터럴 요약
*string-literal*:  
 **'** *s-char-sequence* opt**'**  
  
 **L'** *s-char-sequence* opt**'**  
  
 *s-char-sequence*:  
 *s-char*  
  
 *s-char-sequence s-char*  
  
 *s-char*:  
 큰따옴표("), 백슬래시(\\) 또는 줄 바꿈 문자를 제외한 소스 문자 집합의 멤버 *escape-sequence*
