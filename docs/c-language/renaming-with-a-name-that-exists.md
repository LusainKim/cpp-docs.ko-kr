# <a name="renaming-with-a-name-that-exists"></a>이미 있는 이름으로 바꾸기
**ANSI 4.9.4.2** **rename** 함수를 호출하기 전에 새 이름의 파일이 존재하는 경우의 효과  
  
 존재하는 이름을 사용하여 파일 이름을 바꾸려고 하면 **rename** 함수가 실패하고 오류 코드를 반환합니다.
