# <a name="filenames"></a>파일 이름
**ANSI 4.9.3** 유효한 파일 이름을 작성하는 규칙  
  
 파일 지정에는 선택적 드라이브 문자(항상 뒤에 콜론이 붙음), 일련의 선택적 디렉터리 이름(백슬래시로 구분됨) 및 파일 이름이 포함될 수 있습니다.  
  
 자세한 내용은 [파일 이름 지정](http://msdn.microsoft.com/library/windows/desktop/aa365247)을 참조하세요.
