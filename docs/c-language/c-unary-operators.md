# 단항 연산자
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

단항 연산자는 피연산자 앞에 나타나고 오른쪽에서 왼쪽으로 연결됩니다.  
  
## 구문  
 *unary\-expression*:  
 *postfix\-expression*  
  
 **\+\+**  *unary\-expression*  
  
 `--`  *unary\-expression*  
  
 *unary\-operator cast\-expression*  
  
 **sizeof**  *unary\-expression*  
  
 **sizeof \(**  *type\-name*  **\)**  
  
 *unary\-operator*: 다음 중 하나  
 **&**  
 **\* \+ –** `~` **\!**
