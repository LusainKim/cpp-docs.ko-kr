# 문자당 비트
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** 문자의 비트 수  
  
 문자의 비트 수는 매니페스트 상수 **CHAR\_BIT**로 표현됩니다.  LIMITS.H 파일에서는 **CHAR\_BIT**를 8로 정의합니다.
