# 예외 문제 해결: System.InvalidProgramException
프로그램에 잘못된 MSIL\(Microsoft Intermediate Language\) 또는 메타데이터가 있을 경우 throw되는 예외입니다. 이 예외는 일반적으로 프로그램을 생성한 컴파일러의 오류를 나타냅니다.
