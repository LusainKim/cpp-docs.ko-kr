# &#39;&lt;modulename&gt;&#39; 모듈은 형식으로 사용할 수 없습니다.
모듈을 형식으로 사용하려고 했습니다. 모듈은 `Get` 형식으로가 아니면 사용할 수 없습니다.  
  
 **오류 ID:** BC30371  
  
### 이 오류를 해결하려면  
  
-   모듈을 올바른 형식으로 사용할 수 있는 항목으로 바꿉니다.
