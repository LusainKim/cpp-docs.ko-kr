# 직접 실행 창에서는 &#39;RaiseEvent&#39; 문을 사용할 수 없습니다.
**직접 실행** 창에서는 `RaiseEvent` 문을 사용할 수 없습니다.  
  
 **오류 ID:** BC30739  
  
### 이 오류를 해결하려면  
  
-   **코드 편집기**의 코드에서 `RaiseEvent` 문을 테스트합니다.
