# Error getting Codezone Community sites.
이 오류는 온라인 웹 서비스와의 연결을 찾을 수 없는 경우 발생합니다.  웹 서비스는 **옵션** 대화 상자의 **환경**, **도움말**에서 **온라인** 페이지의 정보를 채웁니다.  
  
### 이 오류를 해결하려면  
  
1.  활성 인터넷 연결이 있는지 확인합니다.  
  
     — 또는 —  
  
     프록시 서버를 통해 인터넷에 연결하는 경우 프록시 서버 설정을 확인합니다.  
  
2.  온라인 페이지에 다시 액세스해 봅니다.
