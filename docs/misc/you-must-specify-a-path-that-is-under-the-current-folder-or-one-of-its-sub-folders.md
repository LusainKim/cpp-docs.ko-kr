# 현재 폴더 또는 하위 폴더 중 하나에 있는 경로를 지정해야 합니다.
지정한 경로가 현재 디렉터리에 없습니다. 상대 경로를 제공할 때 파일 경로가 올바르게 구문 분석되지 않은 것 같습니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 경로를 제공했는지 확인합니다.
