# XML 특성 값에 짝이 되는 닫는 작은따옴표가 필요합니다.
작은따옴표로 구분되는 특성 값에 닫는 인용 부호 문자가 없습니다.  
  
 **오류 ID:** BC31163  
  
### 이 오류를 해결하려면  
  
-   닫는 작은따옴표 문자\('\)를 추가하여 특성 값을 구분합니다.
