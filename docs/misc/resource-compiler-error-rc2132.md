# 리소스 컴파일러 오류 RC2132
VALUE, BLOCK 또는 END 키워드가 필요합니다.  
  
 **VERSION** 리소스의 블록이 올바르게 종료되지 않았거나 새 블록이 올바르게 시작되지 않았습니다. 따라서 **VERSION** 문이 유효한 블록이 아닙니다.
