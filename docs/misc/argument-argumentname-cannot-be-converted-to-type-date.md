# &#39;&lt;argumentname&gt;&#39; 인수를 &#39;Date&#39; 형식으로 변환할 수 없습니다.
변환에서 `Date` 형식으로 변수를 변경하려고 했지만 값을 유효한 날짜로 인식하지 못합니다.  
  
### 이 오류를 해결하려면  
  
-   의미 있는 변환을 찾습니다.
