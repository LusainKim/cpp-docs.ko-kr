# &#39;&lt;filename&gt;&#39; DLL에서 &#39;&lt;name&gt;&#39; 진입점을 찾을 수 없습니다. &lt;error&gt;
지정된 DLL에서 프로시저 이름을 찾을 수 없습니다.  
  
 **오류 ID:** BC31014  
  
### 이 오류를 해결하려면  
  
1.  프로시저 이름의 철자가 정확하고 대소문자가 DLL의 함수와 일치하는지 확인합니다.
