# &#39;&lt;typename&gt;&#39;이 정의되지 않았거나 이를 포함하는 모듈이 디버깅 세션에서 로드되지 않았습니다.
사용할 수 없거나 정의되지 않은 형식을 참조했습니다.  
  
 **오류 ID:** BC30743  
  
### 이 오류를 해결하려면  
  
1.  형식 이름의 철자가 올바른지 확인합니다.  
  
2.  현재 클래스 또는 모듈 내에서 형식을 정의합니다.
