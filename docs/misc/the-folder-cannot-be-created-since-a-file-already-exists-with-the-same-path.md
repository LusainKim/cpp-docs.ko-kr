# 동일한 경로에 파일이 이미 있으므로 폴더를 만들 수 없습니다.
해당 경로에 파일이 이미 있는데 디렉터리를 만들려고 했습니다. 특히 상대 경로를 사용할 때 경로가 잘못 구문 분석된 것 같습니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 경로를 제공했는지 확인합니다.  
  
-   더 이상 필요하지 않은 경우 중복 파일을 제거합니다.
