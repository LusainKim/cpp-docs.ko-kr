# &#39;Sub New&#39;는 &#39;&lt;modifier&gt;&#39;로 선언할 수 없습니다.
`Sub New`를 `Overrides`와 같은 한정자로 선언하려고 했으며, 유효하지 않습니다.  
  
 **오류 ID:** BC30364  
  
### 이 오류를 해결하려면  
  
-   한정자를 제거합니다.
