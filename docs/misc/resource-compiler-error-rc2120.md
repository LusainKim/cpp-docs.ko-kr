# 리소스 컴파일러 오류 RC2120
대화 상자 클래스에 따옴표 붙은 문자열이 필요합니다.  
  
 **DIALOG** 문의 CLASS 옵션에 있는 `class` 필드는 큰따옴표로 묶인 정수 또는 문자열이어야 합니다.
