# The project file is missing the &#39;subsection&#39; sub-section of the &#39;section&#39; section
하위 섹션\(subsection\)이 Include이고 섹션\(section\)이 Files입니다.  
  
 이 오류는 주로 프로젝트 파일을 직접 편집했을 때 발생합니다.  
  
 **이 오류를 해결하려면**  
  
-   솔루션 탐색기 도구 모음에서 해당 단추를 클릭하여 **모든 파일 표시** 모드에서 작업하고 관련 파일을 포함합니다.  
  
     이 오류가 발생하면 프로젝트가 열리기만 하고 파일이 추가되지 않습니다.
