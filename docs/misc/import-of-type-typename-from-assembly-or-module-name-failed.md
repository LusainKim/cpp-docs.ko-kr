# &#39;&lt;name&gt;&#39; 어셈블리 또는 모듈에서 &#39;&lt;typename&gt;&#39; 형식을 가져오지 못했습니다.
참조된 모든 라이브러리가 서로 호환되는지 확인합니다.  
  
 **오류 ID:** BC31091  
  
### 이 오류를 해결하려면  
  
-   참조된 형식을 포함하는 파일이 손상되지 않았는지 확인합니다.
