# &lt;specifier1&gt; &lt;type&gt;은 기본 &lt;type&gt;의 액세스를 확장하므로 &lt;specifier2&gt; &lt;type&gt;에서 상속할 수 없습니다.
`Private` 또는 `Friend`로 표시된 기본 클래스에서 공용 클래스를 상속하려고 했습니다.  
  
 **오류 ID:** BC30509  
  
### 이 오류를 해결하려면  
  
-   기본 클래스 `Public`을 선언하거나 상속하는 클래스 `Private` 또는 `Friend`를 선언합니다.
