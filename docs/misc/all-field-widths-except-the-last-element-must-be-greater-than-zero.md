# 마지막 요소를 제외한 모든 필드 너비는 0보다 커야 합니다.
마지막 요소를 제외한 모든 필드 너비는 0보다 커야 합니다. 마지막 요소의 0보다 작거나 같은 필드 너비는 마지막 필드가 가변 길이임을 나타냅니다.  
  
 마지막 요소 이외의 다른 필드 너비가 0 이하로 설정되었으므로 작업이 실패했습니다. 0보다 작거나 같은 필드 너비를 마지막 요소로 사용하여 마지막 필드가 가변 길이임을 나타낼 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   필드 너비를 올바른 길이로 설정합니다.
