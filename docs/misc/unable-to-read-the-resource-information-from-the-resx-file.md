# Unable to read the resource information from the resx file
프로젝트 시스템에서 .resx 파일을 읽을 수 없습니다.  
  
 운영 체제와 관련된 특정 원인이 있는 경우에는 이 메시지의 끝에 추가됩니다.  읽을 수 없는 파일의 이름도 추가됩니다.  
  
 이 오류가 발생하는 몇 가지 이유로는 파일을 읽기 위한 사용 권한 부족이나 웹 프로젝트가 있는 웹 서버와의 통신 두절이 있습니다.  
  
 이 오류가 발생하면 빌드 프로세스는 실패합니다.
