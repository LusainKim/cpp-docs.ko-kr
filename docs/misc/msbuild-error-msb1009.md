# MSBuild 오류 MSB1009
**프로젝트 파일이 없습니다.**  
  
 지정된 디렉터리에 이 프로젝트 파일이 없거나 디렉터리를 지정하지 않았거나 현재 작업 디렉터리에 프로젝트 파일이 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  파일 이름 및 경로의 맞춤법을 확인하십시오.
