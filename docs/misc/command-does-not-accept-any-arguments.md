# Command does not accept any arguments.
이 오류는 스위치만 사용할 수 있는 명령에 인수를 입력하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 명령 구문에 대한 설명서를 확인한 다음 다시 시도하십시오.
