# 이 시스템에서는 로캘 ID &#39;&lt;name&gt;&#39;이 지원되지 않습니다.
`VbStrConv`를 사용하려고 시도하지만 시스템이 지정된 로캘을 지원하지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  로캘을 지원하도록 컴퓨터를 업그레이드하거나 지정된 로캘을 지원하는 컴퓨터에서 응용 프로그램을 실행합니다.
