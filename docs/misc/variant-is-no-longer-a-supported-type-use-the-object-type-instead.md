# &#39;Variant&#39; 형식이 더 이상 지원되지 않으므로 &#39;Object&#39; 형식을 대신 사용하세요.
`Object` 데이터 형식은 `Variant`를 대체합니다.  
  
 **오류 ID:** BC30804  
  
### 이 오류를 해결하려면  
  
-   항목을 `Object`로 선언합니다.
