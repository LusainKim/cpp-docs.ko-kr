# Argument is missing in the pattern.
이 오류는 검색 문자열에 완전하지 않은 정규식 구문을 입력한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  문자열을 다시 입력합니다.  구문에 대한 자세한 내용은 [정규식](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)을 참조하십시오.
