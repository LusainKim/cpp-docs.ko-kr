# Property value is not valid. (Visual Studio) (2)
이 오류는 잘못된 속성 값을 입력하는 경우 발생합니다.  예를 들어 정수 값에 잘못된 형식을 입력하거나 short 값에 너무 큰 값을 입력하면 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 속성 값을 입력하십시오.  
  
    > [!TIP]
    >  온라인 설명서를 검색하여 해당 속성에 사용 가능한 값을 확인할 수 있습니다.
