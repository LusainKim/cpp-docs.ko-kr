# &#39;&lt;namespace&gt;&#39;에 &#39;Sub Main&#39;을 두 번 이상 선언했습니다. &lt;message&gt;
`Sub Main`은 네임스페이스 내에서 한 번만 선언할 수 있습니다.  
  
 **오류 ID:** BC30738  
  
### 이 오류를 해결하려면  
  
-   프로젝트에 하나의 `Sub Main` 프로시저만 있어야 합니다.
