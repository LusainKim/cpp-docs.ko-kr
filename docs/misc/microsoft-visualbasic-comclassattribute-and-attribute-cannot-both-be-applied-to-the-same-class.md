# &#39;Microsoft.VisualBasic.ComClassAttribute&#39;와 &#39;&lt;attribute&gt;&#39;를 같은 클래스에 적용할 수 없습니다.
`COMClassAttribute` 특성 블록이 COM 개체에 적용되지 않는 특성과 함께 사용됩니다.[!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)]와 COM 특성을 함께 사용하고 있기 때문일 수 있습니다.  
  
 **오류 ID:** BC32501  
  
### 이 오류를 해결하려면  
  
-   COM에 적용되지 않는 `COMClassAttribute` 특성 블록 또는 특성을 제거합니다.
