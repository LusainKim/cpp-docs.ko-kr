# 변수와 그 형식 모두에 배열 한정자를 지정할 수 없습니다.
배열 한정자가 변수와 해당 형식 둘 다에 있으므로 배열의 배열을 나타냅니다.  
  
 **오류 ID:** BC31087  
  
### 이 오류를 해결하려면  
  
-   형식 지정자에서 배열 한정자를 제거합니다.
