# 인터페이스에는 문을 사용할 수 없습니다.
인터페이스에서 잘못된 문을 발견했습니다.  
  
 **오류 ID:** BC31041  
  
### 이 오류를 해결하려면  
  
-   인터페이스의 문에서 요소를 정의하지만 구현을 제공하지 않는지 확인합니다.
