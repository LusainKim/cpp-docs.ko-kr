# File is not a valid .NET Framework or COM component
개체 탐색기에서 올바른 개체가 포함되어 있지 않은 파일을 보려고 시도했습니다.  개체 탐색기에서는 [!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)] 및 COM 구성 요소만 볼 수 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  잘못된 파일을 선택한 경우 올바른 파일을 선택하고 다시 시도합니다.  
  
2.  **연결 프로그램** 대화 상자에서 실수로 개체 브라우저를 선택한 경우 다른 프로그램을 선택하여 연결할 파일을 엽니다.
