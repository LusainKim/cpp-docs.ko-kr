# &#39;&lt;attributename&gt;&#39; 어셈블리 특성이 잘못되었습니다. &lt;error&gt;
공용 언어 런타임은 어셈블리에 대한 특성을 사용할 수 없습니다.  
  
 **오류 ID:** BC30129  
  
### 이 오류를 해결하려면  
  
1.  선언에서 특성을 제거하거나 상황에 맞는 유효한 특성으로 바꿉니다.  
  
2.  따옴표 붙은 오류 메시지를 검사하고 적절한 조치를 수행합니다.
