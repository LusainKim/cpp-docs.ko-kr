# Wizard file &lt;name&gt; format is not recognized.
해당 마법사를 열 수 없습니다.  이 오류는 마법사 파일이 손상되었거나 마법사가 Visual Studio의 버전 번호와 일치하지 않는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  마법사 파일의 버전 번호가 `VSWIZARD 7.0`과 같이 VSWIZARD 값으로 표시된 Visual Studio의 버전 번호와 같은지 확인합니다.  
  
     — 또는 —  
  
     Visual Studio를 다시 설치하여 손상된 파일을 복구합니다.
