# &#39;&lt;modifier&gt;&#39;는 Delegate 선언에서 사용할 수 없습니다.
대리자 선언에서 `Shared`과 같은 한정자를 사용하려고 했습니다. 대리자는 개체 메서드 지정이 전달되는 생성자를 정의하여 다른 개체의 메서드를 호출하는 데 사용되는 개체를 위임합니다. 대리자 선언에서 한정자를 지정할 수 없습니다.  
  
 **오류 ID:** BC30385  
  
### 이 오류를 해결하려면  
  
1.  한정자를 제거합니다.
