# Most recently used list can contain &lt;number&gt; items.
MRU 목록에는 항목을 24개까지 포함할 수 있습니다.  이 오류는 옵션 대화 상자의 환경 옵션에 있는 일반 창에서 **가장 최근에 사용한 목록의 항목 \<number\>개**에 1보다 작은 수나 24보다 큰 수를 입력하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  상자에 1에서 24 사이의 숫자를 입력하십시오.
