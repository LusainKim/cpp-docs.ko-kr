# 예외 문제 해결: Microsoft.Office.Tools.ControlNameAlreadyExistsException
<xref:Microsoft.Office.Tools.ControlNameAlreadyExistsException> 예외는 프로젝트의 다른 컨트롤과 동일한 이름의 컨트롤을 프로그래밍 방식으로 만들려는 경우에 throw됩니다.  
  
## 관련 팁  
 **컨트롤 중 하나에 다른 이름을 사용하십시오.**  
 컨트롤 이름은 고유해야 합니다.
