# 그림이 잘못되었습니다.
폼 또는 컨트롤의 `Picture` 속성에 비트맵, 아이콘 또는 Windows 메타파일 이외의 그래픽 형식을 할당하려고 했습니다.  
  
### 이 오류를 해결하려면  
  
1.  `Picture` 속성에 로드하려는 파일이 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]에서 지원되는 유효한 그래픽 파일인지 확인합니다.
