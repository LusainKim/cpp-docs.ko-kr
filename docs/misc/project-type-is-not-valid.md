# Project type is not valid.
이 오류는 **새 프로젝트 추가** 명령의 *templatename* 인수에 잘못된 값을 입력한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  설명서에서 **새 프로젝트 추가** 명령의 올바른 구문을 확인하고 다시 시도합니다.
