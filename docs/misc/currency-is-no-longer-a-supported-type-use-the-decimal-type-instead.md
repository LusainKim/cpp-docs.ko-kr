# &#39;Currency&#39;가 더 이상 지원되지 않으므로 &#39;Decimal&#39; 형식을 대신 사용합니다.
`Currency` 데이터 형식이 `Decimal` 데이터 형식으로 대체되었습니다.  
  
 **오류 ID:** BC30803  
  
### 이 오류를 해결하려면  
  
-   `Currency` 대신 `Decimal` 데이터 형식을 사용합니다.
