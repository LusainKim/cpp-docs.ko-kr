# 최대 오류 개수를 초과했습니다.
컴파일 오류의 최대 개수는 100개입니다. 컴파일러는 오류 보고서의 크기를 줄이기 위해 이 제한을 적용합니다.  
  
 **오류 ID:** BC30041  
  
### 이 오류를 해결하려면  
  
-   이 컴파일에서 생성된 다른 오류를 검토하고 가능한 한 많이 해결합니다. 소스 코드에 100개가 넘는 오류가 있을 경우 다음 컴파일 시 더 많은 오류가 표시될 수도 있습니다.
