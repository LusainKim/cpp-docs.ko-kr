# 지정한 DLL 함수를 찾을 수 없습니다.
사용자 라이브러리 참조에서 DLL\(동적 연결 라이브러리\)을 찾았지만 지정된 DLL 함수를 DLL 내에서 찾지 못했습니다.  
  
### 이 오류를 해결하려면  
  
1.  함수 선언에서 올바른 서수를 지정합니다.  
  
2.  DLL의 이름 및 별칭이 올바른지 확인합니다.
