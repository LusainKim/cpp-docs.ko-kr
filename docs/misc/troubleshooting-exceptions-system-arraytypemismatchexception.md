# 예외 문제 해결: System.ArrayTypeMismatchException
<xref:System.ArrayTypeMismatchException> 예외는 시스템에서 배열에 대해 선언된 형식으로 요소를 변환할 수 없는 경우에 throw됩니다.  
  
## 관련 팁  
 **개체 형식이 배열 형식으로 변환될 수 있는지 확인하세요.**  
 두 형식 사이에는 변환이 가능해야 합니다.  
  
 Visual Basic에 관련된 자세한 내용은 [배열](../Topic/Arrays%20in%20Visual%20Basic.md)을 참조하세요.
