# 예외 문제 해결: System.Data.SqlServerCe.SqlCeInvalidDatabaseFormatException
이전 버전의 제품에서 만든 데이터베이스 파일을 SQL Server Compact에서 여는 경우 `System.Data.SqlServerCe.SqlCeInvalidDatabaseFormatException` 예외가 발생합니다.  
  
## 설명  
 [!INCLUDE[vs_orcas_long](../atl/reference/includes/vs_orcas_long_md.md)]을 사용하거나 API `System.Data.SqlServerCe.SqlCeEngine.Upgrade`를 사용하여 데이터베이스 파일을 업데이트할 수 있습니다.  
  
 자세한 내용은 [SQL Server Compact 3.5 클래스 라이브러리](http://go.microsoft.com/fwlink/?LinkID=102595)를 참조하세요.
