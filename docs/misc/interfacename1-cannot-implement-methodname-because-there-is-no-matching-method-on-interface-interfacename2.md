# &#39;&lt;interfacename2&gt;&#39; 인터페이스에 일치하는 &lt;method&gt;가 없기 때문에 &#39;&lt;interfacename1&gt;&#39;이 &#39;&lt;methodname&gt;&#39;을 구현할 수 없습니다.
첫 번째 인터페이스가 파생된 인터페이스에 일치하는 메서드가 없을 때 인터페이스에서 메서드를 구현하려고 했습니다.  
  
 **오류 ID:** BC30401  
  
### 이 오류를 해결하려면  
  
-   메서드의 이름을 잘못 입력하지 않았는지 확인하거나 누락된 메서드를 제공합니다.
