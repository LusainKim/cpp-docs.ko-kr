# MSBuild 오류 MSB3178
**MSB3178: '\<assembly\>' 어셈블리가 파일로 잘못 지정되었습니다.**  
  
 이 경고는 응용 프로그램 매니페스트를 생성하는 과정에서 파일 참조가 실제로는 관리되는 어셈블리나 네이티브 어셈블리라는 사실을 빌드 프로세스를 통해 발견한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
