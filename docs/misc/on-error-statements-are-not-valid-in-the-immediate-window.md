# 직접 실행 창에서는 &#39;On Error&#39; 문을 사용할 수 없습니다.
`On Error` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30720  
  
### 이 오류를 해결하려면  
  
-   디버깅 코드에서 `On Error` 문을 제거합니다.
