# 필요한 &#39;version&#39; 특성이 XML 선언에 없습니다.
XML 문서 리터럴에 대한 `version` 특성이 누락되었습니다.  
  
 **오류 ID:** BC31153  
  
### 이 오류를 해결하려면  
  
-   다음 예제와 같이 `version` 특성을 XML 문서 선언에 추가합니다. `<?xml version="1.0"?>`
