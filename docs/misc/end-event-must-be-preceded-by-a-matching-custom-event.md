# &#39;End Event&#39;는 짝이 되는 &#39;Custom Event&#39; 뒤에 와야 합니다.
`End Event` 문이 해당 `Custom Event` 문 없이 발생합니다.`End Event`는 해당 `Custom Event` 문 뒤에 와야 합니다.  
  
 **오류 ID:** BC31123  
  
### 이 오류를 해결하려면  
  
-   앞의 `Custom Event` 문이 유효하고 철자가 올바른지 확인합니다.
