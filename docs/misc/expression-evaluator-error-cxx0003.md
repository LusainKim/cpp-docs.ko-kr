# 식 계산기 오류 CXX0003
식 계산기에 내부 오류가 있습니다.  
  
 식 계산기에서 내부 오류가 발생했습니다.  
  
 오류가 발생한 환경을 기록하고 Microsoft Corporation에 알립니다. 온라인\([http:\/\/support.microsoft.com\/support\/visualc\/report\/](http://support.microsoft.com/support/visualc/report/)\)에서 오류를 보고하거나 Microsoft 기술 지원 서비스를 호출할 수 있습니다. 자세한 내용을 보려면 Visual C\+\+의 **도움말** 메뉴에서 **기술 지원**을 클릭합니다.  
  
 이 오류는 CAN0003과 동일합니다.
