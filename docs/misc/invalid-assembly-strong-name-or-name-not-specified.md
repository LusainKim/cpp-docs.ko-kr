# Invalid assembly strong name or name not specified
강력한 이름이 있는 어셈블리로 빌드할 프로젝트의 키 이름이나 키 컨테이너를 지정하는 데 사용하는 프로젝트 시스템의 개체 모델에 있는 프로젝트 속성이 비어 있습니다.  이 메시지가 표시되면 빌드 프로세스는 실패합니다.  
  
### 이 오류를 해결하려면  
  
1.  코드에서 키 컨테이너 이름이나 키 파일을 지정합니다.
