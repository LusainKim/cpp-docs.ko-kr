# 예외 문제 해결: System.Workflow.Runtime.WorkflowOwnershipException
워크플로 런타임 엔진이 현재 다른 워크플로 런타임 엔진 인스턴스에서 로드한 워크플로 인스턴스를 로드하려고 하는 경우 <xref:System.Workflow.Runtime.WorkflowOwnershipException> 예외가 throw됩니다. 또한, 워크플로를 로드하는 동안 지정된 소유권 시간 제한이 만료된 후 워크플로 런타임 엔진이 워크플로를 저장하려고 하는 경우에도 이 예외가 throw됩니다.
