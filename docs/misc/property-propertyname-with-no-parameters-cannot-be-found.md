# 매개 변수가 없는 &#39;&lt;propertyname&gt;&#39; 속성을 찾을 수 없습니다.
사용 중인 특성에 하나 이상의 매개 변수가 필요한 속성이 있습니다.  
  
 **오류 ID:** BC30658  
  
### 이 오류를 해결하려면  
  
1.  속성에 대한 매개 변수를 지정합니다.
