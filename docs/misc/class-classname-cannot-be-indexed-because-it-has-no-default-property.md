# &#39;&lt;classname&gt;&#39;에 기본 속성이 없으므로 인덱싱할 수 없습니다.
클래스에 대한 기본 속성을 제공하지 못했습니다.  
  
 **오류 ID:** BC30367  
  
### 이 오류를 해결하려면  
  
-   `Default` 키워드로 선언을 시작하여 속성을 기본 속성으로 지정합니다.
