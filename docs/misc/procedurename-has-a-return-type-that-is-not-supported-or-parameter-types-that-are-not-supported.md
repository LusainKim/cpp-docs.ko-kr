# &#39;&lt;procedurename&gt;&#39;에 지원되지 않는 반환 형식 또는 지원되지 않는 매개 변수 형식이 있습니다.
프로시저가 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]\(으\)로 표현할 수 없는 반환 형식 또는 매개 변수를 사용하여 선언되었습니다.  
  
 **오류 ID:** BC30657  
  
### 이 오류를 해결하려면  
  
-   지원되는 형식으로 반환 형식 또는 매개 변수를 정의합니다.  
  
-   이 프로시저에서 사용되는 데이터를 설명하는 새 형식을 개발합니다.
