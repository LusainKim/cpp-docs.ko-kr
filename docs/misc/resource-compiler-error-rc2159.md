# 리소스 컴파일러 오류 RC2159
잘못된 액셀러레이터 형식\[ASCII 또는 VIRTKEY\]  
  
 **ACCELERATORS** 문의 `type` 필드는 ASCII 또는 VIRTKEY 값을 포함해야 합니다.
