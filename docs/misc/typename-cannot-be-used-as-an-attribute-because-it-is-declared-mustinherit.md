# &#39;&lt;typename&gt;&#39;은 &#39;MustInherit&#39;으로 선언되었으므로 특성으로 사용할 수 없습니다.
사용자 지정 특성 클래스는 `MustInherit`로 선언될 수 없습니다.  
  
 **오류 ID:** BC31506  
  
### 이 오류를 해결하려면  
  
-   사용자 지정 특성에서 `MustInherit` 한정자를 제거합니다.
