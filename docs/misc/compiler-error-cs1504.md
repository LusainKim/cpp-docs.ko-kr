# 컴파일러 오류 CS1504
'file' 원본 파일을 열 수 없습니다\('reason'\).  
  
 원본 파일을 열거나 컴파일러에서 읽을 수 없습니다. 파일이 다른 응용 프로그램에 의해 잠겨 있거나 다른 운영 체제 문제가 있을 수 있습니다. 메시지에는 컴파일러에서 파일을 열거나 읽을 수 없는 운영 체제 이유가 포함됩니다.
