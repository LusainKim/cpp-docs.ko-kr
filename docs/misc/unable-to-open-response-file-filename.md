# &#39;&lt;filename&gt;&#39; 지시 파일을 열 수 없습니다.
액세스하려는 파일을 사용할 수 없습니다.  
  
 **오류 ID:** BC2011  
  
### 이 오류를 해결하려면  
  
-   네트워크 연결이 여전히 활성 상태인지 확인합니다.  
  
-   파일에 대한 사용 권한을 확인합니다.
