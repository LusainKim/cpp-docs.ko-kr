# MSBuild 오류 MSB3147
**MSB3147: '\<path\>'에서 필요한 파일 '\<file\>'을\(를\) 찾을 수 없습니다.**  
  
 이 오류는 부트스트래퍼가 디스크에서 setup.bin을 찾을 수 없는 경우 발생합니다.
