# 메서드에는 &#39;Try&#39; 문과 함께 &#39;On Error&#39; 또는 &#39;Resume&#39; 문을 사용할 수 없습니다.
`Try` 문을 `On Error` 또는 `Resume`과 결합했습니다.  
  
 **오류 ID:** BC30544  
  
### 이 오류를 해결하려면  
  
1.  `On Error` 또는 `Resume`을 제거합니다.
