# 리소스 컴파일러 경고 RC4003
'identifier' 매크로의 실제 매개 변수가 부족합니다.  
  
 특정 식별자로 지정된 실제 매개 변수 수가 식별자의 매크로 정의에 지정된 정식 매개 변수 수보다 적습니다.  
  
 정식 매개 변수가 정의에서 참조되고 해당 실제 매개 변수가 제공되지 않은 경우 매크로 확장에서 빈 텍스트가 대체됩니다.
