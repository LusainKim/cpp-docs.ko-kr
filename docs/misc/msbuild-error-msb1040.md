# MSBuild 오류 MSB1040
**MSB1040: ToolsVersion이 잘못되었습니다.**  
  
 **\/ToolsVersion** 스위치를 사용하여 지정한 `ToolsVersion` 값이 잘못되었습니다.  
  
### 이 오류를 해결하려면  
  
-   명령줄에서 **\/ToolsVersion** 스위치를 사용하여 빌드할 때 올바른 `ToolsVersion` 값을 지정합니다.
