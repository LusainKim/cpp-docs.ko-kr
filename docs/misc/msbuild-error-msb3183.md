# MSBuild 오류 MSB3183
**MSB3183: '\<assembly\>' 참조는 완전 신뢰가 필요한 interop 어셈블리입니다.**  
  
 이 경고는 부분적으로 신뢰할 수 있는 응용 프로그램에 대한 매니페스트를 생성할 때 참조 중에서 interop 어셈블리를 발견한 경우 발생합니다.
