# 모듈의 형식은 &#39;&lt;specifier&gt;&#39;로 선언할 수 없습니다.
잘못된 한정자가 포함된 형식이 모듈 내에서 선언되었습니다.  
  
 **오류 ID:** BC30735  
  
### 이 오류를 해결하려면  
  
-   모듈 내에서 선언된 형식에서 `Protected` 또는 `Protected Friend` 한정자를 사용하지 마세요.
