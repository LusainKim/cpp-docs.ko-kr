# XML 네임스페이스 접두사 &#39;prefix&#39;는 XML에서 사용하도록 예약되었으므로 네임스페이스 URI를 변경할 수 없습니다.
`xmlns`와 같은 예약된 XML 네임스페이스 접두사가 XML 리터럴에서 사용되었습니다.  
  
 **오류 ID:** BC31152  
  
### 이 오류를 해결하려면  
  
-   XML 네임스페이스 접두사를 예약되지 않은 값으로 변경합니다.
