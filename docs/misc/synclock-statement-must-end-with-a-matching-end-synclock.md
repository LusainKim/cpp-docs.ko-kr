# &#39;SyncLock&#39; 문은 짝이 되는 &#39;End SyncLock&#39;으로 끝나야 합니다.
`SyncLock` 블록이 `SyncLock` 키워드로 시작하고 `End SyncLock` 구문으로 끝납니다.  
  
 **오류 ID:** BC30675  
  
### 이 오류를 해결하려면  
  
-   `SyncLock` 블록이 `End SyncLock` 구문으로 끝나는지 확인합니다.
