# 컴파일러 오류 CS2006
명령줄 구문 오류: 'option' 옵션에 대한 ':text'가 없습니다.  
  
 *옵션* 구문은 추가 텍스트가 필요합니다. 자세한 내용은 [컴파일러 옵션](../Topic/C%23%20Compiler%20Options.md)을 참조하세요.
