# An invalid file was specified for /resetsettings: &lt;filename&gt;.  /resetsettings will be ignored.
이 오류는 설정 파일을 읽을 수 없거나 설정 파일이 없거나 설정 파일에 잘못된 형식의 XML이 포함되어 있는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  파일 이름이 올바른지 확인합니다.  
  
2.  파일 형식이 올바른지 확인합니다.  
  
3.  파일에 올바른 형식의 XML이 포함되어 있는지 확인합니다.
