# 예외 문제 해결: System.Workflow.Activities.Rules.RuleSetValidationException
규칙 집합의 유효성을 확인할 수 없어 처리를 계속할 수 없는 경우 <xref:System.Workflow.Activities.Rules.RuleSetValidationException> 예외가 throw됩니다. 사용된 <xref:System.Workflow.Activities.Rules.RuleEngine>에 유효성 검사 오류가 있는 경우 <xref:System.Workflow.Activities.Rules.RuleSet> 생성자에 의해 이 예외가 throw됩니다.
