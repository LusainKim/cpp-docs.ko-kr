# 줄임표(...)
줄임표 연산자\(`...`\)는 C와 C\+\+에서 컨텍스트에 따라 여러 가지로 사용됩니다.  원래 C에서 변수 함수 매개 변수 목록에 대한 abstract declarator로 사용되었으며 variadic 매크로의 형태로 전처리기 매크로 프로그래밍에도 사용됩니다.  C\+\+에서는 `...`는 예외 처리 `catch` 블록에 사용되고 C\+\+11에서는 variadic 템플릿에 사용됩니다.
