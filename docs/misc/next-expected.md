# &#39;Next&#39;가 필요합니다.
`On Error Resume` 구문이 종료 `Next` 없이 발생합니다.  
  
 **오류 ID:** BC30003  
  
### 이 오류를 해결하려면  
  
-   `On Error` 문의 끝에 `Next` 키워드를 추가합니다.
