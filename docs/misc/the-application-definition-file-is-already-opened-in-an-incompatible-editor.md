# 응용 프로그램 정의 파일이 호환되지 않는 편집기에 이미 열려 있음
**호환되지 않는 편집기에 응용 프로그램 정의 파일이 이미 열려 있습니다.  다른 편집기를 닫고 프로젝트 속성 페이지를 다시 로드하십시오.**  
  
 이 오류는 다른 편집기에 이미 열려 있는 응용 프로그램 정의 파일을 열려고 할 때 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  다른 편집기를 닫습니다.  
  
2.  프로젝트 디자이너를 닫습니다.  
  
3.  **프로젝트** 메뉴에서 **속성**을 클릭하여 프로젝트 디자이너를 다시 엽니다.
