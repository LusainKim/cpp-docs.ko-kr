# TargetFilePath가 기존 폴더를 지정합니다.
매개 변수 `TargetFilePath`가 기존 디렉터리를 지정하지만 새 디렉터리의 이름을 지정해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   `TargetFilePath`를 변경하여 기존 디렉터리 대신 새 디렉터리를 지정합니다.
