# &lt;optionname&gt; 옵션 뒤에는 &#39;+&#39; 또는 &#39;-&#39;만 올 수 있습니다.
옵션 뒤에 오는 텍스트가 더하기 기호\(\+\) 또는 빼기 기호\(\-\)가 아닙니다.  
  
 **오류 ID:** BC2009  
  
### 이 오류를 해결하려면  
  
-   옵션을 설정하려면 옵션 뒤에 오는 텍스트를 더하기 기호\(\+\)로 바꿉니다.  
  
     — 또는 —  
  
-   옵션을 해제하려면 옵션 뒤에 오는 텍스트를 빼기 기호\(\-\)로 바꿉니다.
