# 컴파일러 오류 CS1577
어셈블리를 생성하지 못했습니다. reason  
  
 C\# 컴파일러가 소스 코드를 성공적으로 컴파일한 후 컴파일의 어셈블리 생성 부분이 실패했습니다. 자세한 내용은 ALink 유틸리티에 대한 오류 설명서\([Al.exe 도구 오류 및 경고](http://msdn.microsoft.com/ko-kr/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)\)를 참조하세요.
