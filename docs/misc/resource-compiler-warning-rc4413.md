# 리소스 컴파일러 경고 RC4413
'filename'의 파일 형식을 확인할 수 없습니다. 8비트 ASCII로 간주합니다.  
  
 리소스 컴파일러가 지정된 파일의 형식을 알 수 없습니다. 8비트 ASCII 파일로 컴파일되었습니다.
