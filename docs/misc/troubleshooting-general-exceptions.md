# 문제 해결: 일반 예외
죄송합니다. 이 예외에 대한 도움말 페이지를 조회할 수 없습니다. MSDN에서 예외 클래스에 대한 참조 항목을 검색하거나 일반적인 인터넷 검색을 수행하는 것이 좋습니다. 또한 이 환경을 개선할 수 있도록 피드백 링크를 사용하여 찾고 있는 내용을 알려주세요.
