# MSBuild 오류 MSB3484
**MSB3484: 서명 대상 '\<file\>'을\(를\) 찾을 수 없습니다.**  
  
 이 오류는 서명하려는 파일을 찾을 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   유효한 파일을 서명하도록 지정했는지 확인합니다.
