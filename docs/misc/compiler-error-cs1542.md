# 컴파일러 오류 CS1542
'dll'은 이미 어셈블리이므로 이 어셈블리에 추가할 수 없습니다. 대신 '\/R' 옵션을 사용합니다.  
  
 [\/addmodule](../Topic/-addmodule%20\(C%23%20Compiler%20Options\).md) 컴파일러 옵션으로 참조된 파일이 [\/target:module](../Topic/-target:module%20\(C%23%20Compiler%20Options\).md)로 작성되지 않았습니다. [\/reference](../Topic/-reference%20\(C%23%20Compiler%20Options\).md)를 사용하여 이 컴파일의 파일을 참조하세요.
