# Property value is not valid. (Visual Studio) (1)
이 오류는 잘못된 속성 값을 입력하는 경우 발생합니다.  예를 들어 속성이 TRUE 및 FALSE의 유효한 값을 가진 부울 속성인 경우 값 MAYBE는 이 부울 속성에 유효하지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 속성 값을 입력하십시오.  
  
    > [!TIP]
    >  온라인 설명서에서 속성 이름을 검색하여 해당 속성에 사용 가능한 값을 확인할 수 있습니다.
