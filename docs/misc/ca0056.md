# CA0056
프로젝트 파일에 있는 분석 도구의 버전이 잘못되었습니다.  
  
 이 경고는 잘못된 코드 분석 도구 버전이 있는 프로젝트 파일을 사용하여 FxCopCmd.exe를 수동으로 호출한 경우에 발생합니다.
