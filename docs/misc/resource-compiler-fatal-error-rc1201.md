# 리소스 컴파일러 심각한 오류 RC1201
잘못된 스위치 \- \/i 뒤에 포함 경로가 누락되었습니다.  
  
 경로 뒤에 포함 경로 지정\(\/i\) 옵션이 나오지 않았습니다.
