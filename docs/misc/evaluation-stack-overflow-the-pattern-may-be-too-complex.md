# Evaluation stack overflow. The pattern may be too complex.
이 오류는 IDE\(통합 개발 환경\)에서 처리할 수 없는 복잡한 정규식을 검색 문자열에 사용한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  절과 클로저의 수를 줄이고 다시 시도합니다.
