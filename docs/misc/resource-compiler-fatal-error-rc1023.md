# 리소스 컴파일러 심각한 오류 RC1023
'filename' 소스 파일을 열 수 없습니다.  
  
 지정된 파일이 없거나, 열 수 없거나, 찾을 수 없습니다.  
  
 환경 설정이 유효하고, 올바른 파일 경로를 지정했는지 확인합니다.  
  
 이 오류가 오류 메시지 없이 표시되는 경우 리소스 컴파일러에 파일 핸들이 부족합니다.
