# BaudRate는 0보다 커야 합니다.
`My.Computer.Ports.OpenSerialPort` 메서드에 지정된 `BaudRate` 인수는 0보다 커야 합니다.  
  
### 이 오류를 해결하려면  
  
-   `BaudRate` 인수 값을 양수로 변경합니다.
