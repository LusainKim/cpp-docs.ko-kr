# &#39;End Structure&#39;는 짝이 되는 &#39;Structure&#39; 뒤에 와야 합니다.
`End Structure` 문은 `Structure` 문으로 시작하는 구조체 블록 끝에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30621  
  
### 이 오류를 해결하려면  
  
-   현재 구조체 블록은 `Structure` 문으로 시작해야 합니다.
