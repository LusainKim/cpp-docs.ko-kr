# &#39;Compare&#39;가 필요합니다.
`Option` 문에서 `Text` 또는 `Binary`를 지정하지만 `Compare`를 지정하지 않습니다.  
  
 **오류 ID:** BC30208  
  
### 이 오류를 해결하려면  
  
-   `Option` 문에 `Compare` 키워드를 추가합니다\(예: `Option Compare Binary`\).
