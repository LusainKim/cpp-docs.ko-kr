# Unable to retrieve resource information
.resx 파일의 계층 구조를 검색하지 못했습니다.  
  
 **출력** 창에 표시되는 `Preparing resources` 빌드 단계의 일부로 프로젝트 시스템에서는 프로젝트 계층 구조에서 확장명이 .resx인 모든 파일을 검색합니다.  그런 다음 프로젝트 출력에 리소스로 포함하기 전에 .resources 이진 파일로 변환해야 할 .resx XML 파일 목록이 만들어집니다.  
  
 **이 오류를 해결하려면**  
  
-   [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]를 다시 시작합니다.  그래도 문제가 해결되지 않으면 Microsoft 기술 지원 서비스에 문제를 보고하십시오.  
  
     이 오류가 발생하면 빌드는 실패합니다.
