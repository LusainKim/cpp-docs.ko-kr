# &lt;name&gt; is not a valid project name.
이 오류는 **명령** 창 또는 **찾기\/명령** 상자에서 `File.NewProject` 명령을 실행하고 *projectname*에 잘못된 프로젝트 이름을 입력하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  프로젝트 이름에 불필요한 공백이 없고 해당 이름이 NUL, CON 또는 AUX와 같은 예약어가 아닌지 확인하십시오.  
  
     — 또는 —  
  
     프로젝트 이름을 생략하고 명령을 다시 입력하십시오.
