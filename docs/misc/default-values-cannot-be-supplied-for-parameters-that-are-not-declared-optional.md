# &#39;Optional&#39;로 선언하지 않은 매개 변수에 대해 기본값을 지정할 수 없습니다.
프로시저를 호출할 때 필요한 인수를 제공해야 하며 선언의 기본값을 사용하지 않습니다.  
  
 **오류 ID:** BC32024  
  
### 이 오류를 해결하려면  
  
-   인수를 `Optional`로 선언하거나 기본값을 제거합니다.
