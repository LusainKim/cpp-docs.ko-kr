# This project cannot be added to the solution because it already exists in the solution in a hidden Solution Folder.
프로젝트는 한 위치의 솔루션에만 있을 수 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  숨겨진 솔루션 폴더를 표시하려면 해당 솔루션을 선택하거나 숨겨진 솔루션 폴더를 포함하는 솔루션 폴더를 선택한 다음 **프로젝트** 메뉴에서 **폴더 숨기기 취소**를 선택합니다.
