# 식 &#39;&lt;name&gt;&#39;은 프로시저가 아니지만 프로시저 호출 대상인 것처럼 나타납니다.
프로시저가 아닌 식에 대한 프로시저 호출을 만들었습니다.  
  
### 이 오류를 해결하려면  
  
1.  프로시저 이름의 철자가 틀리지 않았는지 확인합니다.  
  
2.  \<`name`\>이 호출하는 프로시저를 제공합니다.
