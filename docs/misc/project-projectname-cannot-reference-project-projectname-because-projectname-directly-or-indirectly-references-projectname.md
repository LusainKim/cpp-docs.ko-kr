# &#39;&lt;projectname&gt;&#39;이 직접 또는 간접적으로 &#39;&lt;projectname&gt;&#39;을 참조하기 때문에 &#39;&lt;projectname&gt;&#39; 프로젝트는 &#39;&lt;projectname&gt;&#39; 프로젝트를 참조할 수 없습니다.
프로젝트에 두 번째 프로젝트에 대한 참조가 포함되어 있습니다. 또한 두 번째 프로젝트에는 해당 프로젝트를 참조하는 프로젝트에 대한 참조가 포함되어 있습니다.  
  
 **오류 ID:** BC30761  
  
### 이 오류를 해결하려면  
  
-   참조 중 하나를 제거합니다.
