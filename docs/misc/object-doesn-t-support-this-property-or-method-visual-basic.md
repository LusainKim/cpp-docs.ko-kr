# 개체에서 이 속성 또는 메서드를 지원하지 않습니다(Visual Basic).
이 자동화 개체에 대해 존재하지 않는 메서드 또는 속성을 지정했습니다.  
  
### 이 오류를 해결하려면  
  
-   개체 설명서에서 개체에 대한 자세한 내용을 읽고 속성 및 메서드의 철자를 확인합니다.
