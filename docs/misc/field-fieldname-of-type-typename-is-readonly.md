# &#39;&lt;typename&gt;&#39; 형식의 &#39;&lt;fieldname&gt;&#39; 필드가 ‘ReadOnly&#39;입니다.
`ReadOnly`로 선언된 필드에 쓰려고 했습니다.  
  
### 이 오류를 해결하려면  
  
-   필드를 변경하여 `ReadOnly`로 더 이상 선언되지 않도록 합니다.  
  
-   또는  
  
-   이 필드에 쓰면 안 됩니다.
