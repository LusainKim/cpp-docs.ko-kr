# &#39;&lt;argumentname&gt;&#39; 인수는 다차원 배열이 아니어야 합니다.
배열 인수의 차수는 1보다 크지만 1차원 배열만 가능합니다.  
  
### 이 오류를 해결하려면  
  
-   인수의 차수를 1로 변경합니다.
