# &#39;&lt;parametername&gt;&#39; 매개 변수에 인수를 지정하지 않았습니다.
필수 매개 변수에 대한 인수가 제공되지 않았습니다.  
  
 **오류 ID:** BC36569  
  
### 이 오류를 해결하려면  
  
-   지정된 매개 변수에 대한 인수를 제공합니다.
