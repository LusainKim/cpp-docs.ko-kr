# File &lt;filename&gt; cannot be created.
이 오류는 코드 주석 웹 보고서를 만들려고 했지만 현재 사용자에게 쓰기 권한이 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  쓰기 권한이 있는지 확인합니다.  
  
2.  페이지를 저장할 수 있는 충분한 공간이 있는지 확인합니다.
