# 예외 문제 해결: System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException
세그먼트 디렉터리가 파이프라인 디렉터리 구조에 없는 경우 <xref:System.AddIn.Hosting.AddInSegmentDirectoryNotFoundException> 예외가 throw됩니다.  
  
## 설명  
 추가 기능이 활성화되면 호스트에 대한 해당 파이프라인이 생성됩니다. 추가 기능 시스템을 사용하려면 [파이프라인 개발 요구 사항](http://msdn.microsoft.com/ko-kr/ef9fa986-e80b-43e1-868b-247f4c1d9da5)에 설명되어 있는 것처럼 지정된 디렉터리 이름의 지정된 디렉터리 구조체에 해당 파이프라인 세그먼트를 배치해야 합니다.
