# MSBuild 오류 MSB1039
**MSB1039: 도구 집합의 버전을 지정하십시오.**  
  
 **\/ToolsVersion** 스위치 사용 시 값이 지정되지 않았습니다.  
  
### 이 오류를 해결하려면  
  
-   **\/ToolsVersion** 스위치 사용 시 `ToolsVersion`의 비어 있지 않은 올바른 값을 지정합니다.
