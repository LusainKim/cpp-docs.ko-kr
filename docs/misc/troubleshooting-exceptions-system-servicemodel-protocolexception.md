# 예외 문제 해결: System.ServiceModel.ProtocolException
데이터 전송 프로토콜이 일치하지 않기 때문에 원격 사용자와 통신할 수 없는 경우 클라이언트에서 <xref:System.ServiceModel.ProtocolException> 예외가 발생합니다.  
  
## 설명  
 끝점 간의 호환되지 않는 바인딩, 잘못된 원격 끝점 주소 사용 또는 이와 유사한 잘못된 구성에 의해 <xref:System.ServiceModel.ProtocolException>이 발생할 수 있습니다.
