# 포함 식은 여기에 사용할 수 없습니다.
포함 식이 XML 리터럴의 지원되지 않는 위치에 포함되었습니다. 포함 식은 XML 네임스페이스 값을 지정하는 데 사용할 수 없습니다.  
  
 **오류 ID:** BC31172  
  
### 이 오류를 해결하려면  
  
-   포함 식을 지원되지 않는 위치에서 제거합니다.
