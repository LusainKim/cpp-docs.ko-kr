# 파일이 이미 있습니다.
이 오류는 기존 파일의 이름으로 새 파일의 이름을 지정하거나 **다른 이름으로 저장** 명령을 사용하여 현재 로드된 프로젝트를 이미 있는 이름으로 저장하는 경우에 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   이름을 확인하고 충돌하지 않는 이름으로 변경하거나, 더 이상 필요하지 않은 경우 기존 파일이나 프로젝트를 삭제합니다.
