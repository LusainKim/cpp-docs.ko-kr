# 예외 문제 해결: System.Net.NetworkInformation.NetworkInformationException
네트워크 정보를 검색하는 동안 오류가 발생한 경우 throw되는 예외입니다.  
  
## 설명  
 <xref:System.Net.NetworkInformation> 네임스페이스에 있는 형식은 Win32 함수 호출에 실패한 경우 이 예외를 throw합니다.<xref:System.Net.NetworkInformation.NetworkInformationException.ErrorCode%2A> 속성에는 함수에서 반환된 결과가 포함됩니다.
