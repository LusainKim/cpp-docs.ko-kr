# &#39;&lt;variablename&gt;&#39;은 지역 변수 또는 매개 변수가 아니므로 &#39;Catch&#39; 변수로 사용할 수 없습니다.
`Try...Catch...Finally` 문의 변수는 로컬로 선언되어야 하며 현재 프로시저의 매개 변수여야 합니다.  
  
 **오류 ID:** BC31082  
  
### 이 오류를 해결하려면  
  
1.  `Try...Catch...Finally` 문의 지역 변수 또는 매개 변수를 선언합니다.
