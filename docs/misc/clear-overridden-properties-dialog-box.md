# 재정의된 속성 지우기 대화 상자
> [!NOTE]
>  이 대화 상자는 [!INCLUDE[vs_dev10_long](../build/includes/vs_dev10_long_md.md)]에서 지원되지 않습니다.  
  
 사용자 정의 속성 시트\(.props 파일\)를 프로젝트 구성에 추가하면 해당 프로젝트 구성은 기본적으로 이 속성 시트에 정의된 모든 속성을 상속합니다.  이 대화 상자를 사용하여 프로젝트 구성에 필요하지 않은 상속된 속성을 삭제할 수 있습니다.
