# File is currently in use and cannot be overwritten.
현재 파일을 저장할 수 없습니다.  이 오류는 열려 있는 파일 이름을 사용하여 동시에 파일을 저장하려는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  같은 이름의 파일을 닫고 다시 시도하십시오.  
  
     — 또는 —  
  
     파일 이름을 바꾸어 저장하십시오.
