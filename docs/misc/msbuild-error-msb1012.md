# MSBuild 오류 MSB1012
**지시 파일을 지정하십시오.**  
  
 @ 스위치에 대해 지시 파일을 지정해야 합니다.  
  
### 이 오류를 해결하려면  
  
-   지시 파일을 지정하십시오.  구문은 @\<file name\>입니다\(예: `@BuildHW.txt` \).  
  
-   명령줄에 @ 스위치를 포함하지 마십시오.
