# MSBuild 오류 MSB2005
**\<{0}\> 요소는 특성을 포함할 수 없습니다.**  
  
 이 메시지에서 지적한 요소는 특성을 포함할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   언급된 요소에서 특성을 삭제합니다.  
  
-   프로젝트 파일이 수정되었거나 손상되었는지 확인합니다.  프로젝트가 수정되거나 손상되었으면 프로젝트를 만든 버전의 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]에서 프로젝트를 열고 저장한 다음 다시 변환해 봅니다.
