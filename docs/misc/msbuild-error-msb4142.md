# MSBuild 오류 MSB4142
**MSB4142: MSBuildToolsPath가 ToolsVersion "\<x.x\>"의 MSBuildBinPath와 다릅니다.**  
  
 도구 집합 정의에서 `MSBuildBinPath` 및 `MSBuildToolsPath` 값이 서로 다르게 지정되어 있습니다.  
  
### 이 오류를 해결하려면  
  
-   도구 집합 정의에서 `MSBuildBinPath` 및 `MSBuildToolsPath` 값이 같은지 확인합니다.
