# Miscellaneous Files projects can contain &lt;number&gt; items.
기타 파일 프로젝트에는 항목을 256개까지 저장할 수 있습니다.  이 오류는 옵션 대화 상자의 환경 옵션에 있는 문서 창에서 **프로젝트에서 마지막으로 저장한 기타 파일 \<number\>개**에 1보다 작은 수나 256보다 큰 수를 입력하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  상자에 1에서 256 사이의 숫자를 입력하십시오.
