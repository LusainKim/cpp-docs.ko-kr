# 구조체의 멤버는 &#39;&lt;specifier&gt;&#39;로 선언할 수 없습니다.
`Structure` 문 내에서 멤버에 유효하지 않은 지정자를 사용했습니다.  
  
 **오류 ID:** BC30435  
  
### 이 오류를 해결하려면  
  
-   지정자를 제거합니다.
