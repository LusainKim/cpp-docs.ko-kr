# 리소스 컴파일러 심각한 오류 RC1005
파일을 찾을 수 없습니다. filename  
  
 리소스 컴파일러 명령줄에 지정된 파일을 찾을 수 없습니다. 파일이 다른 디렉터리로 이동되고 파일 이름 또는 경로가 올바르게 입력되었는지 확인합니다.  
  
 **INCLUDE** 환경 변수 또는 Visual C\+\+ 디렉터리 설정을 사용하기 위해 파일을 검색합니다.
