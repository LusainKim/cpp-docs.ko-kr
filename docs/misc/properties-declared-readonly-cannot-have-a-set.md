# &#39;ReadOnly&#39;로 선언된 속성에는 &#39;Set&#39;을 사용할 수 없습니다.
`Set` 프로시저가 속성의 값을 씁니다.`ReadOnly` 속성은 쓸 수 없습니다.  
  
 **오류 ID:** BC30022  
  
### 이 오류를 해결하려면  
  
-   `Property` 문에서 `ReadOnly` 키워드를 제거하거나 속성 본문에서 `Set` 프로시저를 제거합니다.
