# &#39;&lt;filename&gt;&#39; 참조 라이브러리를 로드할 수 없습니다. &lt;error&gt;
이 프로젝트에서 참조되는 라이브러리를 로드할 수 없습니다.  
  
 **오류 ID:** BC31011  
  
### 이 오류를 해결하려면  
  
1.  파일이 지정된 위치에 있고 다른 응용 프로그램에 의해 잠기지 않았는지 확인합니다.
