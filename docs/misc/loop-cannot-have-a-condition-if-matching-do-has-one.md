# 짝이 되는 &#39;Do&#39;에 조건이 있으면 &#39;Loop&#39;에는 조건을 사용할 수 없습니다.
`Loop` 문에 `While` 또는 `Until` 절이 포함되어 있으며, 해당 `Do` 문에도 이러한 절이 포함되어 있습니다. 루프의 `Do` 및 `Loop` 문 중 하나에서만 조건을 지정할 수 있습니다.  
  
 **오류 ID:** BC30238  
  
### 이 오류를 해결하려면  
  
-   `Do` 문 또는 `Loop` 문에서 `While` 또는 `Until` 절을 제거합니다.
