# Visual Basic 솔루션의 모든 프로젝트는 동일한 플랫폼을 대상으로 해야 하지만 추가하려는 프로젝트는 솔루션에서 지정된 플랫폼이 아닌 플랫폼을 대상으로 해야 합니다.
[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 솔루션의 모든 프로젝트는 동일한 플랫폼을 대상으로 지정해야 하지만 추가하는 프로젝트는 솔루션에서 지정된 플랫폼이 아닌 플랫폼을 대상으로 지정해야 합니다.  
  
 **오류 ID:** BC30765  
  
### 이 오류를 해결하려면  
  
-   솔루션에서 지정된 동일한 플랫폼을 대상으로 하는 프로젝트만 추가합니다.
