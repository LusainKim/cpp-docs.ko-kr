# 구현된 &#39;&lt;interfacename&gt;&#39; 인터페이스를 포함하는 &#39;&lt;modulename&gt;&#39; 모듈에 필요한 참조입니다.
구현된 '\<interfacename\>' 인터페이스를 포함하는 '\<modulename\>' 모듈에 필요한 참조입니다. 하나를 프로젝트에 추가합니다.  
  
 인터페이스는 프로젝트에서 직접 참조되지 않은 모듈에서 정의됩니다. 인터페이스가 둘 이상의 모듈에서 정의되는 경우 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 컴파일러가 참조하는 데 모호성이 없어야 합니다.  
  
 **오류 ID:** BC30010  
  
### 이 오류를 해결하려면  
  
-   프로젝트 참조에 참조되지 않은 모듈의 이름을 포함합니다.
