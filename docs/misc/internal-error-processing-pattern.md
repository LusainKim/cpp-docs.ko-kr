# Internal error processing pattern.
이 오류는 검색 문자열에 정규식이 포함되었지만 이 정규식 구문에 오류가 있는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 정규식 구문을 사용하여 검색 문자열을 다시 입력합니다.  자세한 내용은 [정규식](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)을 참조하십시오.
