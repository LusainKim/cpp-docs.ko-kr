# 특성 상수 &#39;&lt;constantname&gt;&#39;은 할당 대상일 수 없습니다.
특성에 선언된 상수에 값을 할당하려고 했습니다.  
  
 **오류 ID:** BC31510  
  
### 이 오류를 해결하려면  
  
-   잘못된 할당을 제거합니다.  
  
-   직접 개발한 사용자 지정 특성을 사용 중인 경우 특성의 멤버를 필드로 다시 정의합니다.
