# 예외 문제 해결: System.Transactions.TransactionInDoubtException
의심스러운 트랜잭션에 대해 작업을 수행하려고 할 때 throw되는 예외입니다.  
  
## 설명  
 등록된 리소스 관리자에 연결할 수 없는 경우에는 트랜잭션이 안전하지 않습니다.
