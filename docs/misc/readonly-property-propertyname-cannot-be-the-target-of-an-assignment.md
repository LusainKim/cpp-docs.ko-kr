# &#39;ReadOnly&#39; 속성 &#39;&lt;propertyname&gt;&#39;은 할당 대상일 수 없습니다.
`ReadOnly` 속성은 값을 할당하는 컨텍스트에서 발생합니다. 쓰기 가능한 변수, 속성 및 배열 요소만 실행 중 할당되는 값을 가질 수 있습니다.  
  
 **오류 ID:** BC30098  
  
### 이 오류를 해결하려면  
  
-   변수 선언 중 `Property` 문에서 `ReadOnly` 키워드를 제거하거나 값을 할당하는 문을 제거합니다.
