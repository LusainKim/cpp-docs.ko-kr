# MSBuild 오류 MSB3173
**MSB3173: '\<file\>' 매니페스트에 쓸 수 없습니다. '  \<error\>'**  
  
 이 오류는 빌드 프로세스에서 매니페스트 파일에 쓰는 동안 일반적인 문제를 발견한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
