# 개체에서 지원하는 최대 개수의 이벤트 수신자로 이벤트를 발생시키고 있으므로 개체의 이벤트를 싱크할 수 없습니다.
개체에서 지원하는 최대 이벤트 수신기 수가 초과되었습니다.  
  
### 오류를 수정하려면  
  
-   이벤트 수신기 수를 줄입니다.
