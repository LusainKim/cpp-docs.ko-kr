# 모듈에는 &#39;Implements&#39;를 사용할 수 없습니다.
`Implements` 문 또는 절이 모듈 내에서 발생합니다. 모듈은 인터페이스를 구현할 수 없습니다.  
  
 **오류 ID:** BC30231  
  
### 이 오류를 해결하려면  
  
-   `Implements` 문 또는 절을 제거하거나 모듈을 클래스로 다시 입력합니다.
