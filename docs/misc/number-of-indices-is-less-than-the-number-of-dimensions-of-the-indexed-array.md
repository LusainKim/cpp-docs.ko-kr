# 인덱스 수가 인덱싱된 배열의 차수보다 적습니다.
배열 요소에 액세스하는 데 사용하는 인덱스 수는 정확하게 배열의 순위, 즉 배열에 선언된 차원 수와 같아야 합니다.  
  
 **오류 ID:** BC30105  
  
### 이 오류를 해결하려면  
  
-   아래 첨자의 총 수가 배열의 차원과 같아질 때까지 배열 참조에 아래 첨자를 추가합니다.
