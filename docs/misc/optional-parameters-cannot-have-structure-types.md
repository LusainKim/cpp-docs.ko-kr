# 선택적 매개 변수에는 구조체 형식을 사용할 수 없습니다.
선택적 매개 변수는 구조체 형식일 수 없습니다.  
  
 **오류 ID:** BC31405  
  
### 이 오류를 해결하려면  
  
1.  필요에 따라 구조체 형식 매개 변수를 선언합니다.  
  
2.  구조체의 위치를 사용하도록 정적 클래스 멤버를 선언합니다.
