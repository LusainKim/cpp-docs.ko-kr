# For 루프 제어 변수 &#39;&lt;variablename&gt;&#39;이 바깥쪽 For 루프에 이미 사용되고 있습니다.
중첩된 `For` 루프는 모두 고유한 제어 변수를 사용해야 합니다.  
  
 **오류 ID:** BC30069  
  
### 이 오류를 해결하려면  
  
-   제어 변수 이름 중 하나를 변경합니다.
