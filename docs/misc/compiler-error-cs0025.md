# 컴파일러 오류 CS0025
'file' 표준 라이브러리 파일을 찾을 수 없습니다.  
  
 컴파일러에서 필요한 파일이 없습니다. 경로가 올바르고 파일이 있는지 확인합니다.  
  
 파일이 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 시스템 파일인 경우 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 설치를 복구하거나 완전히 다시 설치해야 할 수 있습니다.
