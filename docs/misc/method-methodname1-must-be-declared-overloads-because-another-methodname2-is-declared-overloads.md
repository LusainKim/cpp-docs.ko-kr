# 다른 &#39;&lt;methodname2&gt;&#39;가 &#39;Overloads&#39;로 선언되어 있으므로 &lt;method&gt; &#39;&lt;methodname1&gt;&#39;은 &#39;Overloads&#39;로 선언되어야 합니다.
동일한 이름의 다른 속성 또는 메서드가 `Overloads` 한정자로 선언되었습니다.  
  
 **오류 ID:** BC31409  
  
### 이 오류를 해결하려면  
  
1.  `Overloads` 한정자를 속성 또는 메서드에 추가합니다.  
  
2.  `Overloads` 한정자를 동일한 이름의 모든 속성 또는 메서드에서 제거합니다.
