# &lt;type1&gt; &#39;&lt;typename&gt;&#39;은 기본 &lt;type2&gt;의 &lt;type1&gt;을 재정의하지 않으므로 &#39;Overrides&#39;로 선언할 수 없습니다.
기본 클래스에 동일한 이름을 가진 형식이 없을 때 `Sub`, `Function` 또는 `Property` 문에서 `Overrides`를 지정합니다.  
  
 **오류 ID:** BC30284  
  
### 이 오류를 해결하려면  
  
1.  형식 이름 철자가 올바른지 확인합니다.  
  
2.  불필요한 `Overrides` 키워드를 제거합니다.
