# &#39;Inherits&#39; 문은 클래스의 모든 선언 앞에 와야 합니다.
`Inherits` 문은 클래스 정의의 시작 부분에 있어야 합니다.  
  
 **오류 ID:** BC30683  
  
### 이 오류를 해결하려면  
  
-   `Inherits` 문을 클래스의 맨 위로 이동합니다.
