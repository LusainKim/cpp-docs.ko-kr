# &#39;End Enum&#39;은 짝이 되는 &#39;Enum&#39; 뒤에 와야 합니다.
`End Enum` 문이 해당 `Enum` 문 없이 발생합니다.`End Enum`은 해당 `Enum` 문 뒤에 와야 합니다.  
  
 **오류 ID:** BC30184  
  
### 이 오류를 해결하려면  
  
1.  앞에 있는 `Enum` 문의 철자가 잘못되거나 유효하지 않은지 확인합니다.  
  
2.  `Enum` 멤버의 형식이 올바르게 지정되었는지 확인합니다.
