# Some Unicode characters in this file will not be saved.
일반적으로, 이 오류는 유니코드 코드 페이지를 적용해야만 저장할 수 있는 문자가 파일 이름에 포함된 경우에 발생합니다.  
  
### 유니코드 문자를 사용하여 파일을 저장하려면  
  
1.  **Cancel**을 클릭합니다.  
  
2.  **파일** 메뉴에서 **저장 고급 옵션**을 선택합니다.  
  
3.  **저장 고급 옵션** 대화 상자에서 파일에 적합한 인코딩을 선택합니다.
