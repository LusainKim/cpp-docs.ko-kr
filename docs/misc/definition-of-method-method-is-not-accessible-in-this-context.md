# 이 컨텍스트에서는 &lt;method&gt; 메서드의 정의에 액세스할 수 없습니다.
존재하지 않는 집계 함수가 `Into` 연산자에 포함되었습니다.  
  
 **오류 ID:** BC36594  
  
### 이 오류를 해결하려면  
  
-   집계 함수 이름의 철자를 확인하고 필요한 경우 수정합니다.  
  
-   메서드를 사용자 지정 집계 함수로 추가합니다\(확장 메서드\). 사용자 지정 집계 함수를 만드는 방법에 대한 자세한 내용은 [Aggregate Clause](../Topic/Aggregate%20Clause%20\(Visual%20Basic\).md)를 참조하세요.
