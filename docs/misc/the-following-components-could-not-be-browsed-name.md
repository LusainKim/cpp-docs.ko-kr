# The following components could not be browsed: &lt;name&gt;
이 오류는 구성 요소 선택기 대화 상자에서 개체 검색을 지원하지 않는 .htm 및 .txt 같은 파일 형식을 추가하려는 경우 발생합니다.  개체 브라우저는 .bsc, .olb, .tlb, .dll, .exe, .ocx 등과 같은 파일 형식을 지원합니다.  
  
### 이 오류를 해결하려면  
  
1.  선택한 구성 요소 목록에서 지원되지 않는 파일 형식을 제거하고 다른 파일을 선택하십시오.
