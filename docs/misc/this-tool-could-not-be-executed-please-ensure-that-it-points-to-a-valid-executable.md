# This tool could not be executed. Please ensure that it points to a valid executable.
이 오류는 외부 도구를 실행할 수 없는 경우 발생합니다.  지정한 파일이 올바른 실행 파일 또는 배치 파일이 아니기 때문에 발생할 수 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  **도구** 메뉴에서 액세스할 수 있는 **외부 도구** 대화 상자를 사용하여 해당 외부 도구가 올바른 실행 파일 또는 배치 파일인지 확인합니다.
