# 지정된 버전의 .NET Compact Framework에서 &#39;End&#39; 문을 지원하지 않습니다.
작업하고 있는 .NET Compact Framework 버전에서 `End` 문을 지원하지 않습니다.  
  
 **오류 ID:** BC30769  
  
### 이 오류를 해결하려면  
  
-   문을 제거합니다.
