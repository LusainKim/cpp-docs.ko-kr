# XML 주석 형식 매개 변수 &lt;parameter&gt;가 해당 &lt;keyword&gt; 문의 형식 매개 변수와 일치하지 않습니다.
XML 주석 매개 변수 \<parameter\>가 해당 \<keyword\> 문의 형식 매개 변수와 일치하지 않습니다. XML 주석이 무시됩니다.  
  
 XML 주석 태그가 메서드의 일부 매개 변수\(모든 매개 변수 아님\)에 대해 지정되었습니다.  
  
 **오류 ID:** BC42317  
  
### 이 오류를 해결하려면  
  
-   메서드의 모든 형식 매개 변수에 대해 XML 주석 태그를 제공합니다.
