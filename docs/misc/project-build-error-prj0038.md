# 프로젝트 빌드 오류 PRJ0038
IIS 루트 디렉터리를 찾지 못했습니다. 웹 배포 도구를 작동하려면 IIS를 설치해야 합니다.  
  
 웹 배포에서는 컴퓨터에 IIS가 필요합니다.
