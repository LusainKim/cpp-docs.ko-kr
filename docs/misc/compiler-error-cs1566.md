# 컴파일러 오류 CS1566
'file' 리소스 파일을 읽는 동안 오류가 발생했습니다. 'reason'  
  
 컴파일러에 [\/resource](../Topic/-resource%20\(C%23%20Compiler%20Options\).md) 컴파일러 옵션에 전달된 파일 이름 문제가 있습니다.
