# Character &lt;character&gt; is not valid for alias names.
이 오류는 별칭 이름의 일부로 잘못된 문자를 입력하는 경우 발생합니다.  별칭에는 공백, \=, \[, \], \/, \- 또는 "이 포함될 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  잘못된 문자를 빼고 별칭 이름을 다시 입력하십시오.
