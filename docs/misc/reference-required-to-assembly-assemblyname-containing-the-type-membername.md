# &#39;&lt;membername&gt;&#39; 형식을 포함하는 &#39;&lt;assemblyname&gt;&#39; 어셈블리에 필요한 참조입니다.
'\<membername\>' 형식을 포함하는 '\<assemblyname\>' 어셈블리에 필요한 참조입니다. 하나를 프로젝트에 추가합니다.  
  
 프로젝트에는 프로젝트 참조에 포함되지 않은 어셈블리와 다른 형식에 종속되는 형식이 있습니다.  
  
 **오류 ID:** BC30652  
  
### 이 오류를 해결하려면  
  
-   프로젝트의 종속 형식에 대한 프로젝트 참조를 추가합니다.
