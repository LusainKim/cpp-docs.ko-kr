# 컴파일러 경고(수준 1) CS2029
'\/define'의 값이 잘못되었습니다. 'identifier'는 올바른 식별자가 아닙니다.  
  
 이 경고는 [\/define](../Topic/-define%20\(C%23%20Compiler%20Options\).md) 옵션에 사용되는 값에 일부 잘못된 문자가 있는 경우 발생합니다.  
  
 이 경고는 [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md) 옵션으로 표시하지 않을 수 없습니다.
