# Command &lt;name&gt; is not valid.
이 오류는 **찾기\/명령** 상자 또는 **명령** 창에 Visual Studio에서 인식할 수 없는 명령이나 명령 별칭을 입력한 경우 발생합니다.  별칭을 입력하면 오류 메시지에 별칭 이름 대신 전체 명령 이름이 나타납니다.  
  
### 이 오류를 해결하려면  
  
1.  명령 이름을 다시 입력하거나 올바른 명령 이름을 선택하십시오.
