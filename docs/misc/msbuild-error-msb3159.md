# MSBuild 오류 MSB3159
**MSB3159: '\<file\>' 파일에서 XML 유효성 검사 오류가 발생했습니다. '\<error\>'**  
  
 이 오류는 제품이나 package.xml 파일이 XML 스키마와 일치하지 않는 경우 발생합니다.  이 오류의 뒷부분에는 특정 오류 메시지가 표시됩니다.  
  
### 이 오류를 해결하려면  
  
-   XML 파일을 업데이트하여 유효성 검사 오류를 해결합니다.
