# Visual Studio has encountered an unexpected error.
이 오류는 일반적으로 Visual Studio에서 예기치 않은 형식의 데이터를 수신하거나 예기치 않은 소스로부터 데이터를 수신하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   Visual Studio를 닫았다가 다시 시작합니다.
