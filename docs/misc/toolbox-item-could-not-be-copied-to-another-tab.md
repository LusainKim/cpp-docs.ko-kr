# Toolbox item could not be copied to another tab.
이 오류는 해당 항목이 손상되었거나 컴퓨터에 메모리가 부족한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  열려 있는 다른 응용 프로그램을 닫고 항목을 다시 이동합니다.  
  
     — 또는 —  
  
2.  이동할 다른 항목을 선택합니다.
