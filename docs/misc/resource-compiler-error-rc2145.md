# 리소스 컴파일러 오류 RC2145
주 언어 ID가 너무 큽니다.  
  
 주 언어 ID는 유효한 16진수 언어 ID여야 합니다. 유효한 언어 ID 목록은 [런타임 라이브러리 참조](../c-runtime-library/locale-names-languages-and-country-region-strings.md)에서 *언어 및 국가\/지역 문자열*을 참조하세요.
