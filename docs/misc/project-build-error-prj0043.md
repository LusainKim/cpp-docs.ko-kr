# 프로젝트 빌드 오류 PRJ0043
'rule' 사용자 지정 빌드 규칙\('file' 파일에 할당됨\)에 대한 '출력' 속성이 잘못되었습니다. 이 속성에 'string'이 포함되어 있으며 이것은 'value'로 계산됩니다.  
  
 출력 속성이 빈 문자열로 계산되었습니다. 사용자 지정 빌드 규칙이 빌드 작업의 출력을 필요로 합니다.
