# MSBuild 오류 MSB8020
*\<플랫폼\>*\(플랫폼 도구 집합 \= '*\<버전\>*'\)용 빌드 도구를 찾을 수 없습니다.*\<버전\>* 빌드 도구를 사용하여 빌드하려면 *\<플랫폼\>* 빌드 도구를 설치하세요.  또는 프로젝트 메뉴를 선택하고 솔루션을 마우스 오른쪽 단추로 클릭한 다음 “솔루션 대상 변경”을 선택하여 최신 Visual Studio 도구로 업그레이드할 수 있습니다.
