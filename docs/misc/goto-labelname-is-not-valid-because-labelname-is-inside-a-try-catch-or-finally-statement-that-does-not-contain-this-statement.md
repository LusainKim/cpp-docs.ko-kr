# &#39;&lt;labelname&gt;&#39;이 이 문을 포함하지 않는 &#39;Try&#39;, &#39;Catch&#39; 또는 &#39;Finally&#39; 문 내부에 있으므로 &#39;GoTo &lt;labelname&gt;&#39;은 사용할 수 없습니다.
`Try...Catch...Finally` 블록으로 분기할 수 없습니다.  
  
 **오류 ID:** BC30754  
  
### 이 오류를 해결하려면  
  
-   레이블이 `Try...Catch...Finally` 블록 앞에 오도록 코드 구조를 바꿉니다.
