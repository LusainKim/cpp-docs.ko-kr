# 리소스 컴파일러 심각한 오류 RC1049
'string' 숫자 인수가 잘못되었습니다.  
  
 리소스 컴파일러에 숫자 인수가 필요한데 지정된 문자열을 받았습니다.  
  
 이 오류는 필요한 \\x 접두사 없이 16진수 숫자를 지정하거나 잘못된 형식의 부동 소수점 숫자에 의해 발생할 수 있습니다.
