# Clipboard cannot be emptied.
이 오류는 클립보드에 이미 제거할 수 없는 정보가 들어 있어 항목을 복사했으나 클립보드에 배치할 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  클립보드를 사용 중인 다른 프로그램을 닫고 다시 시도하십시오.
