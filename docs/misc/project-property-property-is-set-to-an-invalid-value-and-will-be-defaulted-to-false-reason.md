# Project property &#39;property&#39; is set to an invalid value and will be defaulted to FALSE. &lt;reason&gt;
`Boolean` 속성에 대한 지속성 값이 다음 중 하나가 아닙니다.  
  
-   1 또는 0  
  
-   On 또는 Off  
  
-   true 또는 false  
  
 이 오류는 주로 프로젝트 파일을 직접 편집했을 때 발생합니다.  
  
 **이 오류를 해결하려면**  
  
-   프로젝트 파일을 다시 편집하여 적절한 조건으로 복원합니다.  
  
     해당 속성 값은 false로 설정됩니다.
