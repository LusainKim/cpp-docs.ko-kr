# TextFieldParser에 전달된 스트림을 읽을 수 없습니다.
[TextFieldParser Object](../Topic/TextFieldParser%20Object.md)는 전달된 스트림을 읽을 수 없습니다. 텍스트 파일이 아닌 파일을 읽으려고 하는 경우 이러한 문제가 발생할 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   파일이 텍스트 파일인지 확인합니다.
