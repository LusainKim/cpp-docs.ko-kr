# &#39;&lt;modulename&gt;&#39; 모듈의 인스턴스를 만들 수 없습니다.
모듈은 단일 공유 인스턴스로만 존재하며 추가 인스턴스를 만들 수 없습니다.  
  
 **오류 ID:** BC30166  
  
### 이 오류를 해결하려면  
  
-   모듈을 클래스로 변경하거나 `New` 절에서 클래스 이름으로 바꿉니다.
