# 리소스 컴파일러 심각한 오류 RC1014
포함 파일이 너무 많습니다. 수준 \= 'level'  
  
 `#include` 지시문의 중첩 한도가 너무 컸습니다.  
  
 열려 있는 파일을 포함하려면 중첩된 지시문을 사용합니다. 지시문을 포함하는 원본 파일은 파일 중 하나로 계산됩니다.
