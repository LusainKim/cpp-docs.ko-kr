# 이벤트 이름 길이는 1011자를 초과할 수 없습니다.
`Event` 문으로 선언된 모든 이벤트의 이름 또는 식별자는 1011자로 제한됩니다.  
  
 **오류 ID:** BC32204  
  
### 이 오류를 해결하려면  
  
-   식별자의 길이를 줄입니다.
