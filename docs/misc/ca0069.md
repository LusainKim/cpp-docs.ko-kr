# CA0069
대체 플랫폼을 사용하십시오.  *FrameworkVersion1*을\(를\) 찾을 수 없습니다.  대신 *FrameworkVersion2*을\(를\) 사용하십시오.  최상의 분석 결과를 얻으려면 올바른 .NET Framework가 설치되어 있어야 합니다.
