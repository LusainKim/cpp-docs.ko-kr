# &#39;Default&#39;는 &#39;&lt;specifier&gt;&#39;와 함께 사용할 수 없습니다.
이 컨텍스트에서 유효하지 않은 `Shared` 또는 `Private`과 같은 지정자와 `Default` 키워드를 결합하려고 했습니다.  
  
 **오류 ID:** BC30490  
  
### 이 오류를 해결하려면  
  
-   `Default`를 제거합니다.  
  
     또는  
  
-   충돌하는 지정자를 제거합니다.
