# 마우스 휠이 없습니다.
`My.Computer.Mouse.WheelScrollLines` 속성을 호출했지만 마우스에 스크롤 휠이 없습니다.  
  
### 이 오류를 해결하려면  
  
-   `My.Computer.Mouse.WheelExists` 속성을 확인하여 `My.Computer.Mouse.WheelScrollLines` 속성을 호출하기 전에 마우스에 스크롤 휠이 있는지 확인합니다.  
  
     또는  
  
-   컴퓨터에 스크롤 휠이 있는 마우스를 설치합니다.
