# MSBuild 오류 MSB3110
**MSB3110: '\<assembly\>' 어셈블리가 ID '\<AssemblyIdentity\>'과\(와\) 일치하지 않습니다. 파일 이름이 '\<file\>'이어야 합니다.**  
  
 이 오류는 어셈블리 ID에서 가져온 어셈블리 파일 이름이 실제 파일 이름과 일치하지 않는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   어셈블리 ID와 일치하도록 어셈블리 파일 이름을 수정합니다.
