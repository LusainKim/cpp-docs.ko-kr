# &#39;#If&#39; 블록은 짝이 되는 &#39;#End If&#39;로 끝나야 합니다.
`#If`는 조건부 컴파일 지시문입니다.`#If` 블록이 `#End If` 지시문으로 종료되지 않았습니다.  
  
 **오류 ID:** BC30012  
  
### 이 오류를 해결하려면  
  
-   `#End If` 지시문을 조건부 컴파일 블록의 끝에 추가합니다.
