# 리소스 컴파일러 오류 RC2119
대화 상자 제목에 따옴표 붙은 문자열이 필요합니다.  
  
 **DIALOG** 문의 CAPTION 옵션에 있는 `captiontext` 필드는 큰따옴표로 묶인 ASCII 문자열이어야 합니다.
