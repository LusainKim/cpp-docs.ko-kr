# 리소스 컴파일러 오류 RC2020
기본 기수에 대한 숫자가 잘못되었습니다.  
  
 지정된 `digit`가 `radix`에서 지정한 기준에 유효한 숫자가 아닙니다. 숫자나 기수가 잘못된 것 같습니다.  
  
 8진수는 0에서 7까지의 숫자, 16진수는 0에서 9까지의 숫자 또는 A부터 E까지의 문자여야 합니다.
