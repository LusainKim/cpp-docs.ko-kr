# &lt;filename&gt; 지시 파일을 여러 번 포함했습니다.
지시 파일을 두 번 이상 지정했습니다. 지시 파일에는 컴파일할 소스 코드 파일 및 컴파일러 옵션이 포함되어 있습니다.  
  
 **오류 ID:** BC2003  
  
### 이 오류를 해결하려면  
  
-   중복된 지시 파일 지정자를 제거합니다.
