# CA0053
규칙 어셈블리를 로드할 수 없습니다.  
  
 리플렉션으로 규칙 어셈블리를 로드할 수 없음을 나타냅니다.  사용자가 올바르게 빌드되지 않은 사용자 지정 규칙을 가지고 있거나 분석 도구에서 파일 시스템 액세스 거부 등과 같은 다른 이유로 어셈블리를 로드하지 못한 경우에 발생할 수 있습니다.
