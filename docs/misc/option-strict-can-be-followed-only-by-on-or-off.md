# &#39;Option Strict&#39; 뒤에는 &#39;On&#39; 또는 &#39;Off&#39;만 올 수 있습니다.
`Option Strict` 문은 `On` 또는 `Off`를 지정해야 합니다. 다른 모든 값이 올바르지 않습니다.  
  
 **오류 ID:** BC30620  
  
### 이 오류를 해결하려면  
  
-   `Option Strict` 문에 대해 `On` 또는 `Off`를 지정합니다.
