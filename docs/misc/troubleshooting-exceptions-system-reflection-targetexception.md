# 예외 문제 해결: System.Reflection.TargetException
잘못된 대상을 호출하려고 할 때 throw되는 예외입니다.  
  
## 설명  
 <xref:System.Reflection.TargetException>은 null 개체에 대해 비정적 메서드를 호출하면 throw됩니다. 멤버에 대한 액세스 권한이 호출자에게 없거나 대상의 멤버가 정의되어 있지 않거나 이와 유사한 상황에서 이 예외가 발생할 수 있습니다.
