# &#39;&lt;filename&gt;&#39; 키 파일을 열 수 없습니다. &lt;error&gt;
이 프로젝트에 대해 지정된 키 파일을 열 수 없습니다.  
  
 **오류 ID:** BC31025  
  
### 이 오류를 해결하려면  
  
-   키 파일이 있고, 다른 응용 프로그램에 의해 잠기지 않았는지 확인합니다.
