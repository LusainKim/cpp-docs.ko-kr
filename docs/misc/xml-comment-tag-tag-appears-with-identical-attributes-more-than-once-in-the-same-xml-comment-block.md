# 특성이 같은 XML 주석 태그 &#39;&lt;tag&gt;&#39;가 동일한 XML 주석 블록에서 두 번 이상 나타납니다.
특성이 같은 XML 주석 태그 \<tag\>가 동일한 XML 주석 블록에서 두 번 이상 나타납니다. XML 주석이 무시됩니다.  
  
 XML 주석이 특성이 같은 태그를 두 개 이상 포함합니다.  
  
 **오류 ID:** BC42305  
  
### 이 오류를 해결하려면  
  
-   중복된 태그를 제거합니다.
