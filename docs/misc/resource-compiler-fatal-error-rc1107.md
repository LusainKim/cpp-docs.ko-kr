# 리소스 컴파일러 심각한 오류 RC1107
사용이 잘못되었습니다. RC \/?를 사용합니다. 도움말에 사용됩니다.  
  
 리소스 컴파일러 옵션이 잘못 지정되었습니다. 유효한 옵션은 \/r, \/v, \/d, \/fo, \/l, \/i \/x 및 \/c입니다.
