# File name &lt;name&gt; cannot contain character device names.
이 오류는 실제 문자 장치 이름을 파일 이름에 입력하는 경우 발생합니다.  문자 장치 이름에는 CON, PRN, AUX, CLOCK$, NUL, LPT1, LPT2, COM1, COM2, COM3, COM4 등이 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  다른 파일 이름을 입력하십시오.
