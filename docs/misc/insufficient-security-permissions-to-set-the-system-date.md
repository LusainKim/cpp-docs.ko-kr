# 시스템 날짜를 설정할 수 있는 보안 권한이 없습니다.
시스템 날짜를 설정할 수 있는 보안 권한이 없습니다.  
  
### 이 오류를 해결하려면  
  
-   시스템 날짜를 설정할 수 있도록 보안 권한을 변경합니다.
