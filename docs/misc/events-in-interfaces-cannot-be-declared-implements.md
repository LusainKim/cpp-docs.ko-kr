# 인터페이스의 이벤트는 &#39;&lt;implements&gt;&#39;로 선언할 수 없습니다.
인터페이스에 선언된 이벤트는 다른 인터페이스의 이벤트를 구현할 수 없습니다.  
  
 **오류 ID:** BC30688  
  
### 이 오류를 해결하려면  
  
1.  `Implements` 문을 제거합니다.  
  
2.  클래스 또는 구조체 내의 이벤트를 구현합니다.
