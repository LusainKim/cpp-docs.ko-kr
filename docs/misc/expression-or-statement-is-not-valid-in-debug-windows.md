# 디버그 창에서는 식 또는 문을 사용할 수 없습니다.
`Try...Catch...Finally` 문을 디버깅 컨텍스트에서 사용할 수 없습니다.  
  
 **오류 ID:** BC30706  
  
### 이 오류를 해결하려면  
  
1.  디버깅 코드에서 `Try...Catch...Finally` 문을 제거합니다.
