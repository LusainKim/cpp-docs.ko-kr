# 편집하며 계속하기 작업으로 생성한 개체는 아직 평가할 수 없습니다.
Visual Studio 디버거에서 새 개체를 만들고 있지만 아직 액세스할 수 없습니다.  
  
 개체를 사용할 수 없는 원인으로 메모리 할당 실패, 개체 상태가 동기화되지 않음 등이 있습니다. 이는 일시적인 내부 상태이며, 일반적으로 디버거에 대한 입력의 결과가 아닙니다.  
  
 **오류 ID:** BC30958  
  
### 이 오류를 해결하려면  
  
-   종료 후 디버깅을 다시 시작합니다.
