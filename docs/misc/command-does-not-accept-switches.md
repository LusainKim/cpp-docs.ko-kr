# Command does not accept switches.
이 오류는 인수만 인식하는 명령에 스위치를 제공하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  명령의 구문 요구 사항을 검토합니다.  자세한 내용은 [인수를 사용하는 Visual Studio 명령](../Topic/Visual%20Studio%20Commands.md)을 참조하십시오.
