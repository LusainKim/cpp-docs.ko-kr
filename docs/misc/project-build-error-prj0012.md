# 프로젝트 빌드 오류 PRJ0012
시스템 리소스가 심각하게 부족할 수 있습니다. 빌드를 시작하는 데 필요한 스레드를 만들 수 없습니다.  
  
 시스템 리소스가 부족한 경우  
  
-   실행 중인 응용 프로그램의 일부를 닫습니다.  
  
-   Visual Studio를 다시 시작하여 리소스를 해제합니다.  
  
-   다시 부팅합니다.
