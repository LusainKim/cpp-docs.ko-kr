# XML 네임스페이스 접두사 &#39;prefix&#39;가 이미 선언되었습니다.
`Imports` 문에서 정의된 XML 네임스페이스 접두사가 이전에 정의된 XML 네임스페이스 접두사와 일치합니다.  
  
 **오류 ID:** BC30573  
  
### 이 오류를 해결하려면  
  
-   XML 네임스페이스 접두사에 대해 고유한 이름을 사용합니다.
