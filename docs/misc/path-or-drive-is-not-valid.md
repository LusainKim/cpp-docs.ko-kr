# Path or drive is not valid.
이 오류는 찾는 위치 대화 상자에서 찾아보기 상자에 위치 정보를 지정하지 않고 Enter 키를 누를 때 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  경로 또는 디렉터리를 입력하고 Enter 키를 누릅니다.
