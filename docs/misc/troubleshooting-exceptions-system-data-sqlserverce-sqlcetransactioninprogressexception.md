# 예외 문제 해결: System.Data.SqlServerCe.SqlCeTransactionInProgressException
응용 프로그램에서 다른 트랜잭션이 처리되는 동안 데이터베이스를 수정하려고 시도하는 경우 `System.Data.SqlServerCe.SqlCeTransactionInProgressException` 예외가 발생합니다. 자세한 내용은 [SQL Server Compact 3.5 클래스 라이브러리](http://go.microsoft.com/fwlink/?LinkID=102595)를 참조하세요.
