# 예외 문제 해결: System.MulticastNotSupportedException
결합할 수 없는 대리자 형식의 두 인스턴스를 결합할 때 throw되는 예외입니다. 피연산자 중 하나가 null 참조\(Visual Basic의 경우 `Nothing`\)이면 예외가 throw되지 않습니다.
