# 파일 이름으로 지정된 파일이 FileEncoding으로 지정된 인코딩을 사용하지 않습니다.
파일에서 읽으려고 했지만 지정된 인코딩이 해당 파일에서 사용되지 않습니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 파일 인코딩을 제공했는지 확인합니다.
