# &#39;WithEvents&#39; 변수에는 &#39;As&#39; 절이 있어야 합니다.
`As` 절에 `WithEvents` 키워드를 제공하지 않았습니다. 이벤트를 발생시킬 수 있는 특정 클래스로 변수를 선언합니다.  
  
 **오류 ID:** BC30412  
  
### 이 오류를 해결하려면  
  
1.  이벤트를 발생시킬 수 있는 클래스를 지정하는 `As` 절을 추가합니다.
