# String contains too many tagged expressions.
이 오류는 검색 문자열에 태그가 지정된 식을 10개 이상 지정하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  태그가 지정된 식의 개수를 줄이십시오.
