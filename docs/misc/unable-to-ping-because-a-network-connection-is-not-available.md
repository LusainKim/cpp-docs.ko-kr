# 네트워크 연결을 사용할 수 없어 ping을 실행할 수 없습니다.
네트워크 연결을 사용할 수 없으므로 작업을 수행할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   네트워크에 연결하고 작업을 다시 시도합니다.
