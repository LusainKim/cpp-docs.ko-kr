# &#39;&lt;membername&gt;&#39;은 이 &lt;containername&gt;에 대해 생성된 &#39;&lt;procedurename&gt;&#39;에 의해 이미 선언되었습니다.
공용 언어 런타임에 의해 선언된 멤버와 동일한 이름을 가진 멤버가 선언되었습니다.  
  
 **오류 ID:** BC30733  
  
### 이 오류를 해결하려면  
  
-   멤버의 이름을 변경하여 충돌을 제거합니다.
