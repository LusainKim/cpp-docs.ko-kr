# 메서드 인수는 괄호로 묶어야 합니다.
프로시저 호출을 제어하는 규칙은 최신 버전의 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]에서 더 간단합니다. 모든 인수는 괄호로 묶어야 합니다.  
  
 **오류 ID:** BC30800  
  
### 이 오류를 해결하려면  
  
-   인수 목록을 괄호로 묶습니다. 예:  
  
    ```  
    MySub(ArrayIndex, NewValue)  
    ```
