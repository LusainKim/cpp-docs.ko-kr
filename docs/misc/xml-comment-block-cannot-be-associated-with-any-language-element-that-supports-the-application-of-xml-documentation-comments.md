# XML 주석 블록을 XML 문서 주석의 응용이 지원되는 언어 요소와 연결할 수 없습니다.
XML 주석 블록을 유효한 언어 요소와 연결할 수 없습니다. XML 주석이 무시됩니다. 주석 블록의 XML 태그에 일치하는 언어 요소가 없는 경우에도 이 오류가 발생합니다.  
  
 **오류 ID:** BC42312  
  
### 이 오류를 해결하려면  
  
-   잘못된 XML 태그를 찾아서 제거하거나 XML 태그를 설명하는 언어 요소를 제공합니다.
