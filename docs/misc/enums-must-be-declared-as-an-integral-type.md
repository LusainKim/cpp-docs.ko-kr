# 열거형은 정수 형식으로 선언해야 합니다.
열거형에서 사용할 수 있는 올바른 유형은 `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long` 및 `ULong`뿐입니다. 다른 데이터 형식은 사용할 수 없습니다.  
  
 **오류 ID:** BC30650  
  
### 이 오류를 해결하려면  
  
-   `SByte`, `Byte`, `Short`, `UShort`, `Integer`, `UInteger`, `Long` 또는 `ULong` 데이터 형식을 지정합니다.
