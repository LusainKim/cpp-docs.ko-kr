# MSBuild 오류 MSB3176
**MSB3176: 지정한 최소 버전이 현재 게시 버전보다 높습니다.  현재 게시 버전보다 낮거나 같은 버전을 지정하십시오.**  
  
 이 오류는 필요한 최소 응용 프로그램 버전이 배포 매니페스트의 현재 버전보다 높은 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   필요한 최소 버전을 현재 게시 버전보다 낮거나 같게 지정합니다.
