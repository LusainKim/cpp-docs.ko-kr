# File &lt;name&gt; size is incorrect.
파일이 잘렸거나 손상되었기 때문에 예상한 것보다 적은 바이트를 포함합니다.  
  
### 이 오류를 해결하려면  
  
1.  Visual Studio를 닫은 후 다시 시작합니다.  
  
2.  오류가 계속되면 Visual Studio를 다시 설치하여 파일을 복구합니다.
