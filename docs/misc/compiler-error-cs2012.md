# 컴파일러 오류 CS2012
'file'을 쓰기용으로 열 수 없습니다.  
  
 [\/bugreport:](../Topic/-bugreport%20\(C%23%20Compiler%20Options\).md) `file` 컴파일러 옵션을 사용하는 경우 파일을 쓰기용으로 열 수 없습니다. 올바른 파일 이름을 지정했는지, 파일이 읽기 전용이 아닌지 확인합니다.
