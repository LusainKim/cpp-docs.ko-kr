# MSBuild 오류 MSB3168
**MSB3168: 중복 항목 '\<package\>'는 무시됩니다.**  
  
 이 경고는 부트스트래퍼에서 동일한 두 개의 부트스트래퍼 패키지를 설치하도록 지정한 경우 발생합니다.  이 경우 부트스트래퍼는 인스턴스를 하나만 설치합니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
