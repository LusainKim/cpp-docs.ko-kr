# &#39;Set&#39; 메서드에는 둘 이상의 매개 변수를 사용할 수 없습니다.
둘 이상의 매개 변수가 속성 `Set` 프로시저에 대해 정의되어 있습니다.  
  
 **오류 ID:** BC31063  
  
### 이 오류를 해결하려면  
  
-   속성 `Set` 프로시저에서 불필요한 매개 변수를 모두 제거합니다.
