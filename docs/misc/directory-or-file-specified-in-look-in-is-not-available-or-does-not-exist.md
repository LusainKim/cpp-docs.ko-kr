# Directory or file specified in Look In is not available or does not exist.
이 오류는 브라우저 목록에 지정된 경로 또는 파일 이름을 찾을 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  경로 및 파일 이름을 다시 입력하십시오.  
  
     — 또는 —  
  
2.  브라우저 목록 및 **사용 가능한 항목** 상자를 사용하여 올바른 파일 또는 폴더를 탐색하십시오.
