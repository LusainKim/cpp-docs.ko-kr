# 네임스페이스에서 형식에 대한 정규화된 이름은 &lt;number&gt;자를 초과할 수 없습니다.
형식의 정규화된 이름은 지정한 문자 수를 초과할 수 없습니다.  
  
 **오류 ID:** BC30031  
  
### 이 오류를 해결하려면  
  
-   형식의 이름을 줄입니다.
