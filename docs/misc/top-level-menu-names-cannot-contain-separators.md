# Top-level menu names cannot contain separators.
이 오류는 대시\(\-\)와 같은 구분 문자를 포함하는 텍스트를 메뉴 이름으로 끄는 경우 발생합니다.  최상위 메뉴에는 메뉴 항목들을 구분하도록 예약된 이러한 문자를 포함할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  구분 문자를 사용하지 않고 텍스트를 입력하십시오.
