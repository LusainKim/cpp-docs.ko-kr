# &#39;&lt;argumentname&gt;&#39; 인수의 범위는 -32768에서 65535 사이여야 합니다.
인수 값이 \-32,768에서 65,535 사이의 허용 범위를 벗어났습니다.  
  
### 이 오류를 해결하려면  
  
1.  인수 값이 유효한 범위 내에 있는지 확인합니다.  
  
2.  인수를 계산에서 제공하는 경우 계산에서 처리하는 값을 확인합니다.
