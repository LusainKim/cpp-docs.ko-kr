# 참조된 &#39;&lt;membername&gt;&#39;에 &#39;Nothing&#39; 값이 있습니다.
참조되는 항목에 값이 할당되지 않았거나 `Nothing` 값이 할당되었습니다.  
  
 **오류 ID:** BC30759  
  
### 이 오류를 해결하려면  
  
-   항목 이름의 철자가 올바르고 제대로 설정되었는지 확인합니다.
