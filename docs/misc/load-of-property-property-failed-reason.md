# Load of property &#39;property&#39; failed. &lt;reason&gt;
속성 값이 범위를 벗어납니다.  
  
 이 오류는 주로 프로젝트 파일을 직접 편집했을 때 발생합니다.  
  
 **이 오류를 해결하려면**  
  
-   프로젝트 파일을 다시 편집하고 오류를 수동으로 해결합니다.  
  
     해당 속성은 프로젝트를 만들었을 때의 속성 값인 기본값으로 설정됩니다.
