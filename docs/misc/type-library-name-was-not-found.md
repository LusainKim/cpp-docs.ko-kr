# Type library &lt;name&gt; was not found.
DTE.olb 및 vsext.tlb 형식 라이브러리가 없으면 Visual Studio를 실행할 수 없습니다.  이 오류는 이 두 개의 파일이 제거되었거나 삭제된 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  Visual Studio를 다시 설치하십시오.
