# Option is not currently valid.
이 오류는 현재 사용할 수 없는 인수 또는 스위치를 명령에 지정한 경우 발생합니다.  예를 들어 `>Edit.Find test /sel`을 입력하고 파일에서 아무 것도 선택하지 않으면 \/sel 스위치를 사용할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  입력한 인수 및 스위치를 현재 사용할 수 있는지 확인합니다.  자세한 내용은 [인수를 사용하는 Visual Studio 명령](../Topic/Visual%20Studio%20Commands.md)을 참조하십시오.
