# 예외 문제 해결: System.IO.PathTooLongException
<xref:System.IO.PathTooLongException> 예외는 경로 이름이나 파일 이름이 시스템에서 정의한 최대 길이보다 더 긴 경우에 throw됩니다.  
  
## 관련 팁  
 **경로의 길이는 시스템에서 정의한 최대 길이보다 길지 않아야 합니다.**  
 Windows 기반 플랫폼에서 경로는 248자 미만이어야 하고 파일 이름은 260자 미만이어야 합니다.
