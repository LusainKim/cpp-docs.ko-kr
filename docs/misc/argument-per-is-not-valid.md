# &#39;Per&#39; 인수가 잘못되었습니다.
1에서 `NPer`에 이르는 지불 기간을 지정하는 `Double`여야 하는 `Per`에 제공된 인수가 올바르지 않습니다.  
  
### 이 오류를 해결하려면  
  
-   인수가 유효한 범위 내에 들어가는지 확인합니다.
