# 대상 경로에 하나 이상의 폴더가 없는 경우
대상 경로에 포함되는 하나 이상의 디렉터리가 없는 경우입니다. 경로가 올바르게 구문 분석되지 않은 것 같습니다. 특히 상대 경로를 지정하는 경우 그럴 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 대상 경로를 지정했는지 확인합니다.
