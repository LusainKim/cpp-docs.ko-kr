# Command was not specified.
이 오류는 **명령** 창 또는 **찾기\/명령** 상자에 빈 문자열을 입력하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 명령을 입력합니다.  자세한 내용은 [Visual Studio 명령](../Topic/Visual%20Studio%20Commands.md)을 참조하십시오.
