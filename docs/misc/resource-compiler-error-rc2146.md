# 리소스 컴파일러 오류 RC2146
LANGUAGE 문에 COMMA가 없습니다.  
  
 기본 언어 및 보조 언어를 분리하는 쉼표가 없습니다.  
  
 **LANGUAGE** 문에서는 다음 구문을 사용해야 합니다.  
  
 **LANGUAGE** *primary\_language\_ID*,*secondary\_language\_ID*
