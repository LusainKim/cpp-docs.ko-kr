# &#39;&lt;typename&gt;&#39;에 대한 형식 정보를 가져올 수 없습니다.
공용 언어 런타임에 의해 로드되지 않은 형식이 참조되었습니다.  
  
 **오류 ID:** BC30751  
  
### 이 오류를 해결하려면  
  
-   현재 범위 내에서 형식이 정의되도록 코드를 재구성합니다.
