# &#39;&lt;typename&gt;&#39; 형식에 대한 기본 멤버를 찾을 수 없습니다.
런타임에 바인딩된 인덱스 `Get` 또는 `Set`를 호출하려고 했습니다.  
  
### 이 오류를 해결하려면  
  
1.  만들려는 대상에 따라 개체에 `Get` 또는 `Set`를 사용하는 기본 속성이 있는지 확인합니다.
