# &#39;Resume&#39; 또는 &#39;GoTo&#39;가 필요합니다.
`On Error` 문이 `Resume` 절 또는 `GoTo` 절을 포함하지 않습니다. 두 절 중 하나를 사용해야 합니다.  
  
 **오류 ID:** BC32019  
  
### 이 오류를 해결하려면  
  
-   `On Error` 문에 `Resume` 절 또는 `GoTo` 절을 추가합니다.
