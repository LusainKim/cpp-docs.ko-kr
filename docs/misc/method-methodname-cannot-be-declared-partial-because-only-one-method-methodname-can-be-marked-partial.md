# &#39;&lt;methodname&gt;&#39; 메서드 하나만 &#39;Partial&#39;로 표시할 수 있으므로 &#39;&lt;methodname&gt;&#39; 메서드는 &#39;Partial&#39;로 선언할 수 없습니다.
이름과 서명이 같은 두 개의 부분 메서드 선언을 사용할 수 없습니다.  
  
 **오류 ID:** BC31433  
  
### 이 오류를 해결하려면  
  
1.  부분 메서드 선언 중 하나의 이름이나 서명을 변경합니다.
