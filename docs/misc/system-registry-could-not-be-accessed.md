# System registry could not be accessed.
이 오류는 운영 체제가 레지스트리 파일을 찾을 수 없거나, 파일이 손상되었거나 또는 파일을 읽거나 파일에 쓸 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  Visual Studio를 다시 설치하십시오.
