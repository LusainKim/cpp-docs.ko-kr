# 컴파일러 오류 CS1503
'number' 인수를 TypeA에서 TypeB로 변환할 수 없습니다.  
  
 메서드에 포함된 한 인수의 형식이 클래스를 인스턴스화할 때 전달된 형식과 일치하지 않습니다. 이 오류는 일반적으로 CS1502와 함께 나타납니다. 이 오류를 해결하는 방법에 대한 자세한 내용은 [CS1502](../Topic/Compiler%20Error%20CS1502.md)를 참조하세요.
