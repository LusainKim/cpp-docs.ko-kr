# &#39;&lt;methodname&gt;&#39; 메서드를 &lt;number&gt; 인수로 호출할 수 없습니다.
메서드 호출에 잘못된 인수 개수가 포함되어 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  제공한 인수를 계산합니다.  
  
2.  호출하려는 메서드의 매개 변수의 수와 숫자가 일치하는지 확인합니다.  
  
3.  두 수가 일치하지 않는 경우 인수의 수를 조정합니다.
