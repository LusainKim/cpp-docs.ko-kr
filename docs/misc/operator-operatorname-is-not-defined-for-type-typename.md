# &#39;&lt;typename&gt;&#39; 형식에 대한 &#39;&lt;operatorname&gt;&#39; 연산자가 정의되지 않았습니다.
정의되지 않은 `String` 등의 형식에 단항 연산자\(`+`, `-` 또는 `Not`\)를 사용하려고 했습니다.  
  
 **오류 ID:** BC30487  
  
### 이 오류를 해결하려면  
  
-   단항 연산자를 식에 적절한 연산자로 바꿉니다.
