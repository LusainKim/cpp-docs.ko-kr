# 파일 형식이 잘못되었습니다.
파일을 구성 요소에 로드하려고 했지만 파일의 데이터 형식이 구성 요소와 호환되지 않거나, 구성 요소 데이터를 파일에 저장하려고 했지만 데이터 형식이 파일의 형식과 호환되지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  디스크 파일 데이터에 적합한 형식을 결정하려면 해당 구성 요소 설명서를 참조하세요.  
  
2.  형식 간의 변환을 지원하는지 여부를 확인하려면 해당 구성 요소 설명서를 참조하세요.
