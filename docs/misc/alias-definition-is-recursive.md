# Alias definition is recursive.
이 오류는 자신을 직접 또는 간접적으로 참조하는 별칭을 정의하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  별칭 정의를 변경하고 다시 시도하십시오.  
  
     — 또는 —  
  
2.  **명령** 창에 `>alias`를 입력하여 현재 별칭 목록 및 해당 정의를 검토한 후 다시 시도합니다.
