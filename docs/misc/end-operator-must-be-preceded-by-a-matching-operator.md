# &#39;End Operator&#39;는 짝이 되는 &#39;Operator&#39; 뒤에 와야 합니다.
`End Operator` 문이 앞에서 일치하는 `Operator` 선언 없이 나타납니다.  
  
 **오류 ID:** BC33007  
  
### 이 오류를 해결하려면  
  
-   `End Operator` 문이 중복되는 경우 제거합니다.  
  
-   `Operator` 프로시저가 누락된 경우 제공합니다.  
  
-   `End Operator` 문을 코드의 적절한 위치로 이동합니다.
