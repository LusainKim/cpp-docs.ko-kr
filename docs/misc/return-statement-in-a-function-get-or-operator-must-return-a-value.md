# Function, Get 또는 Operator의 &#39;Return&#39; 문은 값을 반환해야 합니다.
`Return` 문을 사용하여 호출하는 프로시저에 값을 반환해야 합니다.`Return` 문을 단독으로 사용하여 프로그램 흐름을 제어할 수 없습니다.  
  
 **오류 ID:** BC30654  
  
### 이 오류를 해결하려면  
  
1.  함수 또는 프로시저에서 반환할 수 있는 값을 지정합니다.  
  
2.  `End` 문을 사용하여 프로그램이 현재 프로시저를 종료하게 합니다.
