# &#39;&lt;name&gt;&#39;이 &#39;&lt;projectname2&gt;&#39; 프로젝트에서 참조하지 않는 &#39;&lt;projectname1&gt;&#39; 프로젝트에서 선언되었습니다.
현재 프로젝트에 참조가 없는 프로그래밍 요소가 프로젝트에 정의되어 있습니다.  
  
 **오류 ID:** BC32004  
  
### 이 오류를 해결하려면  
  
-   현재 프로젝트에서 원하는 요소를 포함하는 프로젝트에 대한 참조를 추가합니다.
