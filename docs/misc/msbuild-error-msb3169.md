# MSBuild 오류 MSB3169
**MSB3169: 부트스트래퍼를 생성하는 동안 오류가 발생했습니다. '\<error\>'**  
  
 이 메시지에는 부트스트래퍼 빌드에서 catch할 수 없는 오류가 표시됩니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
