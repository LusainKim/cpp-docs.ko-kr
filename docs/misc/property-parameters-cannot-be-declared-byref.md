# &lt;Property&gt; 매개 변수는 &#39;ByRef&#39;로 선언할 수 없습니다.
속성에 대한 매개 변수는 `ByVal` 한정자로 선언해야 합니다.  
  
 **오류 ID:** BC30651  
  
### 이 오류를 해결하려면  
  
-   속성 매개 변수를 `ByVal`로 정의합니다.
