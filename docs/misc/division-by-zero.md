# Division by zero.
0으로 나누기는 수학적으로 불가능합니다.  이 오류는 제수로 사용하는 식의 값이 0인 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  식에 맞춤법 오류, 논리적 오류 또는 대\/소문자 오류가 있는지 확인하십시오.
