# MSBuild 오류 MSB2007
**Visual Studio에서 "{0}" 프로젝트 파일을 찾을 수 없습니다.**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]에서 지정된 프로젝트 파일을 찾을 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  프로젝트 파일 이름의 맞춤법을 확인하십시오.
