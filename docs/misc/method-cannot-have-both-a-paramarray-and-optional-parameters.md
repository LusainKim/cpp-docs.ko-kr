# 메서드에는 ParamArray 매개 변수와 Optional 매개 변수를 함께 사용할 수 없습니다.
`ParamArray` 인수는 자동으로 선택 사항이 되며 프로시저 선언에서 유일한 선택적 인수여야 합니다. 앞의 모든 인수는 필수여야 합니다.  
  
 **오류 ID:** BC30046  
  
### 이 오류를 해결하려면  
  
-   선언에서 선택적 인수를 제거합니다.
