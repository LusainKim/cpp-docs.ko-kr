# &#39;&lt;charactername&gt;&#39; 형식 문자가 선언된 &#39;&lt;type&gt;&#39; 데이터 형식과 일치하지 않습니다.
변수가 특정 데이터 형식으로 선언되었지만 호환되지 않는 데이터 형식을 나타내는 형식 문자로 참조되었습니다. 예를 들어 `K`가 `Integer`로 선언된 경우의 `K$ = 10`입니다.  
  
 **오류 ID:** BC30277  
  
### 이 오류를 해결하려면  
  
-   변수의 선언된 데이터 형식을 변경하거나, 참조에서 형식 문자를 변경 또는 제거합니다.
