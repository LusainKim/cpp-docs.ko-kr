# &#39;End AddHandler&#39;는 짝이 맞는 &#39;AddHandler&#39; 선언 뒤에 와야 합니다.
`End AddHandler` 문이 해당 `AddHandler` 문 없이 발생합니다.`End AddHandler`는 해당 `AddHandler` 문 뒤에 와야 합니다.  
  
 **오류 ID:** BC31124  
  
### 이 오류를 해결하려면  
  
-   앞의 `AddHandler` 문이 유효하고 철자가 올바른지 확인합니다.
