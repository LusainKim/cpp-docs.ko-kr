# The application cannot shut down.
이 오류 메시지는 Visual Studio에 모달 대화 상자가 열려 있는 동안 외부 프로그램\(예: 작업 관리자\)이 Visual Studio를 닫으려고 할 때 나타납니다.  
  
### 이 오류를 해결하려면  
  
1.  모달 대화 상자를 닫고 다시 시도하십시오.
