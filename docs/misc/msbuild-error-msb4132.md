# MSBuild 오류 MSB4132
**MSB4132: 도구 버전 "'\<버전\>'"을\(를\) 인식할 수 없습니다.**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]에서는 지정된 `ToolsVersion` 값에 해당하는 도구 집합을 찾을 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild **\/ToolsVersion** 스위치를 사용할 때 프로젝트 태그나 명령줄에서 올바른 `ToolsVersion` 값을 지정합니다.
