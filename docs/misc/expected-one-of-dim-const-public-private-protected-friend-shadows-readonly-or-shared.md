# &#39;Dim&#39;, &#39;Const&#39;, &#39;Public&#39;, &#39;Private&#39;, &#39;Protected&#39;, &#39;Friend&#39;, &#39;Shadows&#39;, &#39;ReadOnly&#39; 또는 &#39;Shared&#39; 중 하나가 필요합니다.
선언문에 선언 키워드가 없습니다. 한 가지 가능한 원인은 특성 선언이 메서드를 호출하기 때문입니다.  
  
 **오류 ID:** BC30195  
  
### 이 오류를 해결하려면  
  
1.  메서드가 특성 선언 내에서 선언되었는지 확인합니다.  
  
2.  적절한 선언 키워드로 문을 시작합니다.
