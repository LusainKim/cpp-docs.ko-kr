# An add-in caused an access violation while sinking on &lt;item&gt;. Do you wish to terminate the application?
이 오류는 이벤트 처리기를 호출하는 동안 추가 기능에서 오류를 일으키는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  **예**를 클릭하여 응용 프로그램을 닫고 다시 시도합니다.
