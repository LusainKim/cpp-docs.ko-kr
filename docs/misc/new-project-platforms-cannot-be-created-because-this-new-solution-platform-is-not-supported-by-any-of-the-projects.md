# New project platforms cannot be created because this new solution platform is not supported by any of the projects.
이 오류는 **새 프로젝트 플랫폼 만들기**를 선택했지만 새 솔루션 플랫폼을 지원하는 프로젝트가 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   **새 프로젝트 플랫폼 만들기**의 선택을 취소하고 **확인**을 클릭합니다.  
  
     — 또는 —  
  
     **새 프로젝트 플랫폼 만들기**의 선택을 취소하고 드롭다운 목록에서 다른 플랫폼 이름을 선택한 다음 **확인**을 클릭합니다.
