# 문에서 &#39;Get&#39; 또는 &#39;Set&#39; 메서드를 선언하지 않았습니다.
문이 `Property` 프로시저에서 `Get` 또는 `Set` 선언문을 제공하지 않습니다. 속성은 `Property` 및 `End Property` 문으로 둘러싸인 코드 블록으로 정의됩니다. 이 블록에서 각 `Property` 프로시저는 선언문과 end 문으로 둘러싸인 내부 블록으로 표시됩니다.  
  
 **오류 ID:** BC30576  
  
### 이 오류를 해결하려면  
  
-   `Get` 또는 `Set` 선언문을 제공합니다.
