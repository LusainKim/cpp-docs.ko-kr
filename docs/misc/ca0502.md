# CA0502
지원되지 않는 언어입니다.  
  
 이 오류는 사용자가 C\#이나 Visual Basic이 아닌 언어로 작성된 하나 이상의 페이지나 클래스가 포함된 ASP.NET 프로젝트를 분석하려고 할 때 발생합니다.  이 오류를 해결하려면 프로젝트에서 파일을 제거하거나 코드 분석을 비활성화하십시오.
