# An invalid cookie was passed to UnregisterExtenderProvider().
더 이상 유효하지 않은 쿠키를 지정했습니다.  
  
### 이 오류를 해결하려면  
  
1.  RegisterExtenderProvider\(\)를 호출했을 때와 같은 쿠키 값을 반환합니다.
