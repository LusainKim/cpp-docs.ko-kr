# &#39;Option Compare&#39; 뒤에는 &#39;Text&#39; 또는 &#39;Binary&#39;가 와야 합니다.
`Option Compare` 문에 잘못된 설정이 있거나 설정이 없습니다.`Option Compare`에서는 `Text` 및 `Binary` 설정만 허용됩니다.  
  
 **오류 ID:** BC30207  
  
### 이 오류를 해결하려면  
  
1.  설정 지정자의 철자가 맞는지 확인합니다.  
  
2.  `Option Compare` 문에 `Text` 또는 `Binary`를 추가합니다\(예: `Option Compare Text`\).
