# 리소스 컴파일러 오류 RC2157
액셀러레이터 키 테이블에 쉼표가 필요합니다.  
  
 리소스 컴파일러에서는 **ACCELERATORS** 문의 `event` 및 *idvalue* 필드 사이에 쉼표가 있어야 합니다.
