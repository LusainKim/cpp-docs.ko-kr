# 모듈은 &#39;&lt;specifier&gt;&#39;로 선언할 수 없습니다.
모듈이 잘못된 액세스 한정자로 선언되었습니다.  
  
 **오류 ID:** BC31052  
  
### 이 오류를 해결하려면  
  
-   액세스 한정자를 `Public` 또는 `Friend`와 같은 유효한 액세스 한정자로 바꿉니다.
