# 리소스 컴파일러 오류 RC2149
문자열 테이블에 숫자 상수가 필요합니다.  
  
 `#define` 문에 정의된 숫자 상수는 **STRINGTABLE** 문의 **BEGIN** 키워드 바로 뒤에 와야 합니다.
