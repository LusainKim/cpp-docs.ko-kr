# 예외 문제 해결: System.Drawing.Printing.InvalidPrinterException
<xref:System.Drawing.Printing.InvalidPrinterException> 예외는 잘못된 프린터 설정을 사용하여 프린터에 액세스하려 할 때 throw됩니다.  
  
## 관련 팁  
 **프린터가 있는지 확인하십시오.**  
 프린터 설정에서 가장 일반적으로 발생하는 문제는 존재하지 않는 프린터를 참조하는 경우입니다. 자세한 내용은 <xref:System.Drawing.Printing>을 참조하세요.  
  
 **기본 프린터가 설치되어 있는지 확인하십시오.**  
 프린터가 지정되지 않았으면 기본 프린터가 설치되어 있는지 확인합니다. 자세한 내용은 <xref:System.Drawing.Printing.PrintDocument.PrinterSettings%2A>을 참조하세요.
