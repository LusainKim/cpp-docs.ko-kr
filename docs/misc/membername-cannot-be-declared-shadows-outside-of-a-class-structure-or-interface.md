# 클래스, 구조체 또는 인터페이스 외부에서는 &#39;&lt;membername&gt;&#39;을 &#39;Shadows&#39;로 선언할 수 없습니다.
`Shadows` 키워드가 네임스페이스, 모듈 또는 파일 수준에서 선언에 표시됩니다. 섀도잉은 기본 요소에서 상속할 수 있는 프로그래밍 요소 내에서만 의미가 있습니다.  
  
 **오류 ID:** BC32200  
  
### 이 오류를 해결하려면  
  
-   `Shadows` 키워드를 제거하거나 선언을 클래스, 구조체 또는 인터페이스 내부로 이동합니다.
