# The application cannot undo.
이 오류는 다른 파일에 연결된 항목을 실행 취소하려는 경우 발생합니다.  연결된 항목을 수정한 후 이 작업을 수행하면 Visual Studio에서 변경 내용을 롤백할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  변경 내용을 수동으로 바꾸십시오.
