# &#39;&lt;typename&gt;&#39; 형식의 &#39;&lt;fieldname&gt;&#39; 필드가 포함된 구조체의 파일 I/O는 유효하지 않습니다.
파일에 쓰려는 구조체에 지원되지 않는 형식이 포함되어 있습니다.  
  
### 이 오류를 해결하려면  
  
-   구조체에서 지원되지 않는 형식이 발생하는 위치를 결정하는지 확인하고 유효한 형식으로 변경합니다.
