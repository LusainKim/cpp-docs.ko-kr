# System information cannot be displayed.
레지스트리 값이 없거나 msinfo32.exe를 실행할 수 없으므로 운영 체제 정보를 표시할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  Visual Studio를 다시 설치하십시오.
