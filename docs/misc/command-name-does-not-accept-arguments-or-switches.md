# Command &lt;name&gt; does not accept arguments or switches.
이 오류는 인수나 스위치를 사용하지 않는 명령 또는 별칭과 함께 스위치나 인수를 입력하는 경우 발생합니다.  별칭을 입력하면 오류 메시지 텍스트에 별칭 이름이 아닌 전체 명령 이름이 나타납니다.  스위치와 인수를 사용하는 유효한 명령 목록은 [Visual Studio 명령](../Topic/Visual%20Studio%20Commands.md)을 참조하십시오.  
  
### 이 오류를 해결하려면  
  
1.  명령과 인수를 다시 입력하거나 새 명령을 입력하십시오.
