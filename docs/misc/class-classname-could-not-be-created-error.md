# &#39;&lt;classname&gt;&#39; 클래스를 만들 수 없습니다. &lt;error&gt;
클래스를 초기화하지 못했습니다.[!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)] 설치가 손상되었기 때문일 수 있습니다.  
  
 **오류 ID:** BC32400  
  
### 이 오류를 해결하려면  
  
1.  따옴표 붙은 오류 메시지를 검사하고 적절한 조치를 수행합니다.  
  
2.  [!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)]를 다시 설치하고 프로그램을 다시 컴파일합니다.  
  
3.  그래도 오류가 다시 발생하면 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 컴파일러를 다시 설치합니다.  
  
4.  다시 설치 이후에도 오류가 계속 발생하면 해당 상황에 대한 정보를 수집하여 Microsoft 기술 지원 서비스에 알립니다.
