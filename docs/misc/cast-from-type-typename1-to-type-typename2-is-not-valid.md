# &#39;&lt;typename1&gt;&#39; 형식에서 &#39;&lt;typename2&gt;&#39; 형식으로의 캐스팅이 잘못되었습니다.
한 형식에서 다른 형식으로 변환할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  변환과 관련된 형식을 확인하고 의미 있는 형식으로 변환하려고 하는지 확인합니다.
