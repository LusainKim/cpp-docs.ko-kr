# 지정자는 선언의 시작 부분에서만 사용할 수 있습니다.
접근성 또는 다른 지정을 지정하는 키워드는 `Dim` 또는 `Function`과 같이 선언 키워드 앞에 나와야 합니다.  
  
 **오류 ID:** BC30181  
  
### 이 오류를 해결하려면  
  
-   특성 목록\(있는 경우\)과 선언 키워드 사이에 있는 모든 지정자를 이동합니다.
