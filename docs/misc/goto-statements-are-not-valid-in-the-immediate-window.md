# 직접 실행 창에서는 &#39;GoTo&#39; 문을 사용할 수 없습니다.
`GoTo` 문은 분기를 수행하며 디버깅 컨텍스트에서는 허용되지 않습니다.  
  
 **오류 ID:** BC30133  
  
### 이 오류를 해결하려면  
  
-   **직접 실행** 창에서 `GoTo` 문을 실행하지 않아야 합니다.
