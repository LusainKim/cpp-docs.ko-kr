# &#39;&lt;typename&gt;&#39;에 정의된 &#39;&lt;methodname&gt;&#39; 확장 메서드의 &#39;&lt;parametername&gt;&#39; 매개 변수에는 인수가 지정되어 있지 않습니다.
확장 메서드에서 필요한 매개 변수에 대해 인수를 제공하지 않았습니다.  
  
 **오류 ID:** BC36586  
  
### 이 오류를 해결하려면  
  
-   지정된 매개 변수에 대한 인수를 제공합니다.
