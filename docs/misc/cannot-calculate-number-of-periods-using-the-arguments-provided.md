# 지정한 인수를 사용하여 기간을 계산할 수 없습니다.
`NPer` 함수 호출에 필요한 인수가 모두 포함되지 않았습니다.  
  
### 이 오류를 해결하려면  
  
-   `Rate`, `Prnt` 및 `PV` 값이 함수 호출에 포함되었는지 확인합니다.
