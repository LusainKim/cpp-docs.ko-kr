# 프로젝트 빌드 오류 PRJ0048
다른 빌드가 이미 진행 중이므로 빌드를 시작할 수 없습니다.  
  
 다른 빌드가 진행 중인 경우 빌드를 시작할 수 없습니다. 현재 빌드가 완료될 때까지 기다리거나 빌드를 취소합니다.
