# &#39;Custom&#39; 한정자는 &#39;Event&#39; 선언 바로 앞에만 사용할 수 있습니다.
`Custom` 키워드는 사용자 지정 이벤트 선언에서 `Event` 키워드 앞에만 와야 합니다.  
  
 **오류 ID:** BC31140  
  
### 이 오류를 해결하려면  
  
-   `Custom` 키워드가 `Event` 키워드 앞에 오도록 이동합니다.  
  
     또는  
  
-   `Custom` 키워드를 제거합니다.
