# &#39;&lt;methodname&gt;&#39;의 &#39;&lt;parametername&gt;&#39; 매개 변수에 인수를 지정하지 않았습니다.
필수 매개 변수에 대해 인수를 제공하지 않았습니다.  
  
 **오류 ID:** BC30455  
  
### 이 오류를 해결하려면  
  
-   지정된 매개 변수에 대한 인수를 제공합니다.
