# &#39;&lt;name&gt;&#39;은 &#39;&lt;classname&gt;&#39;의 멤버가 아닙니다.
제공한 멤버가 클래스의 멤버가 아닙니다.  
  
 **오류 ID:** BC30456  
  
### 이 오류를 해결하려면  
  
1.  멤버의 이름이 정확한지 확인합니다.  
  
2.  클래스의 실제 멤버를 사용합니다.
