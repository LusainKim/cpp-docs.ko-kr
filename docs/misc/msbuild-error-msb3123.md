# MSBuild 오류 MSB3123
**MSB3123: 파일 연결 수가 한계인 \<limit\>를 초과합니다.**  
  
 각 파일에는 고정된 개수의 파일 확장명만 연결할 수 있습니다.  이 오류는 대상 운영 체제에 허용되는 수보다 많은 파일 확장명을 연결하려고 할 때 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   파일 연결 수를 지정된 수로 제한합니다.
