# The application cannot run.
메모리가 부족하여 Visual Studio를 실행할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  컴퓨터 사양이 Visual Studio를 실행하는 데 필요한 최소 요구 사항에 맞는지 확인합니다.  
  
2.  컴퓨터에서 실행 중인 다른 프로그램을 닫아 메모리를 확보합니다.  
  
     — 또는 —  
  
     Visual Studio를 다시 설치하십시오.
