# Component is not correctly registered.
이 오류는 레지스트리가 손상되었거나, 시스템이 레지스트리에 액세스할 수 없거나 또는 항목이 잘못된 형식으로 레지스트리에 쓰인 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  구성 요소를 다시 설치하십시오.
