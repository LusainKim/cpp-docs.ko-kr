# 도구에 제목이 있어야 합니다. 이 도구를 저장하지 않으려면 외부 도구 대화 상자에서 삭제를 클릭하세요.
이 오류는 제목이 지정되지 않은 새 도구를 외부 도구 대화 상자에 추가한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  이 도구를 저장하려면 **확인**을 클릭하고 **외부 도구** 대화 상자에 도구 제목을 입력합니다.  
  
2.  이 도구를 저장하지 않으려면 **확인**을 클릭하고 **외부 도구** 대화 상자에서 **삭제**를 클릭합니다.
