# Add-in &lt;name&gt; cannot be loaded. Remove it from the Add-in Manager?
선택된 추가 기능을 찾을 수 없습니다.  이 오류는 선택된 추가 기능에 .dll 또는 필요한 다른 구성 요소가 없을 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  추가 기능을 다시 설치하십시오.
