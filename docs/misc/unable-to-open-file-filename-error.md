# &#39;&lt;filename&gt;&#39; 파일을 열 수 없습니다. &lt;error&gt;
이 프로젝트에서 파일을 여는 동안 오류가 발생했습니다.  
  
 **오류 ID:** BC31027  
  
### 이 오류를 해결하려면  
  
1.  파일이 있고, 손상되었거나 다른 응용 프로그램에 의해 잠기지 않았는지 확인합니다.
