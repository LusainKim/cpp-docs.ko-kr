# 예외 문제 해결: System.Data.ReadOnlyException
<xref:System.Data.ReadOnlyException> 예외는 읽기 전용 데이터 열의 값을 변경할 때 throw됩니다.  
  
## 관련 팁  
 **열을 읽기\/쓰기로 변경하십시오.**  
 이 예외는 읽기 전용인 데이터 열의 값을 변경하는 경우에 throw됩니다. 자세한 내용은 <xref:System.Data.DataColumn>을 참조하세요.  
  
 **올바른 열의 데이터를 수정하고 있는지 확인하십시오.**  
 이 예외는 읽기 전용인 데이터 열의 값을 변경하는 경우에 throw됩니다. 자세한 내용은 <xref:System.Data.DataColumn>을 참조하십시오.
