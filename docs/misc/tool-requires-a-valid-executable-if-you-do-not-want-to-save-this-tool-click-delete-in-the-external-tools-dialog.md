# 도구에 올바른 실행 파일이 있어야 합니다. 이 도구를 저장하지 않으려면 외부 도구 대화 상자에서 삭제를 클릭하세요.
이 오류는 실행 파일이 지정되지 않은 도구를 외부 도구 대화 상자에 추가한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  이 도구를 저장하려면 **확인**을 클릭하고 도구의 **명령** 상자에 파일 이름을 입력하거나 줄임표 단추를 선택하여 파일을 찾아봅니다.  
  
2.  이 도구를 저장하지 않으려면 **확인**을 클릭하고 **외부 도구** 대화 상자에서 **삭제**를 클릭합니다.
