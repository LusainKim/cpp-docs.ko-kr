# &lt;propertyname&gt; 속성은 빈 문자열이나 Nothing으로 설정할 수 없습니다.
속성에 빈 문자열\(""\) 또는 `Nothing` 이외의 값이 필요합니다. 이는 값을 잘못 계산한 결과일 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   올바른 값을 제공합니다.
