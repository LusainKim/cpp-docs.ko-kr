# 리소스 컴파일러 오류 RC2014
전처리기 명령은 공백 아닌 문자로 시작해야 합니다.  
  
 비공백 문자가 같은 줄의 전처리기 지시문의 숫자 기호\(**\#**\) 앞에 나타납니다.
