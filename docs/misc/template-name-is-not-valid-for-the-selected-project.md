# Template &lt;name&gt; is not valid for the selected project.
이 오류는 **새 파일** 명령인 `File.NewFile` 또는 **새 솔루션 항목 추가** 명령인 `File.AddNewSolutionItem`에 알 수 없는 템플릿 이름을 입력한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  해당 명령과 인수를 다시 입력하십시오.  
  
     — 또는 —  
  
2.  "새 파일 명령" 또는 "새 솔루션 항목 추가 명령" 항목에서 올바른 구문을 확인하십시오.
