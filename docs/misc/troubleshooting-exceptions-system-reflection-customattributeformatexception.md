# 예외 문제 해결: System.Reflection.CustomAttributeFormatException
사용자 지정 이진형식이 잘못된 경우 throw되는 예외입니다.  
  
## 설명  
 이 예외는 오류의 원인이 된 데이터 형식을 만드는 데 사용한 도구에 버그가 있는 경우에 발생할 수 있습니다.
