# 여러 기본 인터페이스에 선언된 메서드를 오버로드할 수 없습니다.
여러 개의 상속된 인터페이스가 암시적으로 동일한 메서드를 오버로드합니다.  
  
 **오류 ID:** BC31410  
  
### 이 오류를 해결하려면  
  
-   `Overloads` 한정자 대신 `Shadows` 한정자를 사용합니다.
