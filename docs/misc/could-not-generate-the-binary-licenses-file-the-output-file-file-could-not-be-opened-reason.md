# 이진 라이선스 파일을 생성하지 못했습니다. 출력 파일 &#39;file&#39;을 열 수 없습니다. &lt;이유&gt;
프로젝트 시스템이 오래된 라이선스 파일 복사본을 삭제하지 못했습니다. 라이선스를 준비할 때 프로젝트 시스템에서는 텍스트 파일 .licx를 .NET 라이선싱 시스템에서 인식할 수 있는 이진 라이선스 파일로 변환합니다.  
  
 **이 오류를 해결하려면**  
  
-   [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]를 다시 시작합니다.  
  
     이 오류가 발생하면 빌드 프로세스가 실패합니다.
