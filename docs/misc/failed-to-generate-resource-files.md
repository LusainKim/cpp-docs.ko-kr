# Failed to generate resource files
.NET 리소스 관리자가 인식할 수 있는 이진 .resources 파일을 만드는 프로세스의 일부로 .resx 파일을 읽을 때 일반 오류가 발생했습니다.  
  
 이 오류는 일반적으로 잘못된 .resx 파일로 인해 발생합니다.  
  
 이 경고가 나타나더라도 프로젝트가 빌드됩니다.  그러나 이 오류가 표시된 .resx 파일은 프로젝트에 어셈블리 매니페스트 리소스로 포함되지 않습니다.
