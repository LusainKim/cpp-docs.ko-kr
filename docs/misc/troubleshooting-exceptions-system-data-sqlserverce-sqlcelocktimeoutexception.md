# 예외 문제 해결: System.Data.SqlServerCe.SqlCeLockTimeoutException
잠금 제한 시간에 도달하고 잠금을 대기하는 동안 SQL Server Compact 제한 시간이 초과되는 경우 `System.Data.SqlServerCe.SqlCeLockTimeoutException` 예외가 throw됩니다. 기본 잠금 제한 시간은 2000밀리초입니다. 자세한 내용은 [SQL Server Compact 3.5 클래스 라이브러리](http://go.microsoft.com/fwlink/?LinkID=102595)를 참조하세요.
