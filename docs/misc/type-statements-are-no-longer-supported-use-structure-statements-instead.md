# &#39;Type&#39; 문이 더 이상 지원되지 않으므로 &#39;Structure&#39; 문을 대신 사용합니다.
사용자 정의 형식을 정의하기 위해 더 이상 `Type` 키워드를 사용할 수 없습니다.  
  
 **오류 ID:** BC30802  
  
### 이 오류를 해결하려면  
  
-   `Structure` 문을 사용하여 새 형식을 정의합니다.
