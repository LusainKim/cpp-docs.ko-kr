# Window menu can contain &lt;number&gt; items.
창 메뉴의 Windows 목록에는 창 이름을 24개까지만 표시할 수 있습니다.  이 오류는 옵션 대화 상자의 환경 옵션에서 일반 창에 있는 **\<number\> 창 메뉴에 표시되는 항목** 상자에 1보다 작거나 24보다 큰 수를 입력한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  상자에 1에서 24 사이의 숫자를 입력하십시오.
