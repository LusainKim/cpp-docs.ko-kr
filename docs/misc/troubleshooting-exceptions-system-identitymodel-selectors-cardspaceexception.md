# 예외 문제 해결: System.IdentityModel.Selectors.CardSpaceException
CardSpace 서비스 수준에서 하나 이상의 예외가 발생한 경우 <xref:System.IdentityModel.Selectors.CardSpaceException> 예외가 throw됩니다. 오류 원인은 이벤트 로그에 기록됩니다.  
  
## 설명  
 저장소 또는 장부를 로드하는 동안 오류가 발생하는 경우와 같은 조건에서 이 예외가 발생합니다.
