# &lt;number&gt; power closure argument &#39;n&#39; must be from 1 to 256.
이 오류는 숫자 값이 허용 범위\(1 ~ 256\)를 초과하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  1에서 256 사이의 숫자를 입력하고 다시 시도합니다.  구문에 대한 자세한 내용은 [정규식](../Topic/Using%20Regular%20Expressions%20in%20Visual%20Studio.md)을 참조하십시오.
