# 특성 멤버 &#39;&lt;membername&gt;&#39;은 &#39;Public&#39;으로 선언되지 않았으므로 할당 대상일 수 없습니다.
특성의 전용 멤버에 값을 할당하려고 했습니다.  
  
 **오류 ID:** BC31511  
  
### 이 오류를 해결하려면  
  
1.  할당을 제거합니다.  
  
2.  개발한 사용자 지정 특성을 사용하는 경우 특성 멤버의 액세스 한정자를 `Public`으로 변경합니다.
