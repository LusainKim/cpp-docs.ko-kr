# File &lt;name&gt; cannot be registered as an COM component.
파일을 프로젝트의 참조에 추가할 수 없습니다.  이 오류는 파일이 손상되었거나, 형식 라이브러리를 포함하지 않거나, 호환되지 않는 파일 형식을 사용하거나 또는 시스템에 리소스가 부족하여 작업을 완료할 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  파일이 올바른 형식 라이브러리를 포함하는지 확인하고 다시 시도하십시오.  
  
     — 또는 —  
  
     해당 파일을 설치한 제품을 다시 설치하십시오.  
  
    > [!NOTE]
    >  파일을 설치한 제품이 Visual Studio가 아닐 수도 있습니다.
