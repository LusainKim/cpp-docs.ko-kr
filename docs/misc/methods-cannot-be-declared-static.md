# 메서드는 &#39;Static&#39;으로 선언할 수 없습니다.
`Static` 한정자는 전체 프로시저에 적용할 수 없습니다.  
  
 **오류 ID:** BC30810  
  
### 이 오류를 해결하려면  
  
-   `Static` 한정자를 사용하여 프로시저 내의 개별 항목을 static으로 선언합니다.
