# Visual Studio requires Windows 2000 or above.
Visual Studio는 Windows NT 4.0 서비스 팩 3이 설치된 빌드 1381 또는 Windows 95 빌드 708 이상 버전에서 실행됩니다.  
  
### 이 오류를 해결하려면  
  
1.  현재 Windows NT 또는 Windows 버전을 위에서 설명한 버전으로 업그레이드하십시오.
