# Unable to Open Linked File
파일이 링크된 파일로 프로젝트에 추가되었는데  프로젝트를 열 때 프로젝트 시스템에서 이 파일을 찾을 수 없습니다.  
  
 **이 오류를 해결하려면**  
  
-   솔루션 탐색기에서 링크된 파일을 삭제합니다.  
  
     \-또는\-  
  
-   파일을 프로젝트 시스템에서 찾을 수 있는 위치에 둡니다.
