# discrete_distribution::probabilities
분포에 대해 저장된 확률을 반환합니다.  
  
## 구문  
  
```  
std::vector<double> probabilities() const;  
```  
  
## 설명  
 이 멤버 함수는 저장된 확률 값을 반환합니다.
