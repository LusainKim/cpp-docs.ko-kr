# The user options settings file for this project is missing the &#39;section&#39; section
.vbproj.user 또는 .csproj.user 파일에 Build 노드가 없습니다.  
  
 Build 섹션은 디버깅 설정을 포함합니다.  이 섹션이 없으면 사용자의 디버깅 설정이 로드되지 않고 기본값이 사용됩니다.  
  
 이러한 상황은 주로 프로젝트 파일을 직접 편집했을 때 발생합니다.  
  
 프로젝트를 닫을 때 사용자 단위 프로젝트 파일이 자동으로 다시 쓰여집니다.  
  
 이 메시지는 정보를 제공하기 위한 경고입니다.
