# 컴파일러 오류 CS1719
Win32 리소스 파일 'File Name'을 읽는 동안 오류가 발생했습니다. 'reason'  
  
 "파일을 찾을 수 없습니다." 또는 "액세스가 거부되었습니다." 등 오류에서 제공하는 이유로 인해 Win32 리소스 파일을 읽지 못했습니다. 이 오류는 이유에 설명된 문제를 수정하여 해결할 수 있습니다.
