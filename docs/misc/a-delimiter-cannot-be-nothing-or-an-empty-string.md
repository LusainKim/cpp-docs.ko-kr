# 구분 기호는 Nothing이나 빈 문자열일 수 없습니다.
`Delimiters` 속성이 `Nothing`으로 설정되거나 빈 `String`\(""\)이므로 파일에서 `TextFieldParser`를 읽을 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   `Delimiters`에 유효한 값을 지원합니다.
