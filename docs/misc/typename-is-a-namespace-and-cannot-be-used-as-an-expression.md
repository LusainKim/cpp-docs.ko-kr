# &#39;&lt;typename&gt;&#39;은 네임스페이스이므로 식으로 사용할 수 없습니다.
형식 이름은 식이 필요한 경우 발생합니다. 식은 변수, 상수, 리터럴, 속성 및 `Function` 프로시저 호출로 구성되어야 합니다.  
  
 **오류 ID:** BC30112  
  
### 이 오류를 해결하려면  
  
-   형식 이름을 제거하고 유효한 요소를 사용하여 식을 생성합니다.
