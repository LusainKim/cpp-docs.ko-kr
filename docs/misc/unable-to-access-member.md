# 멤버에 액세스할 수 없습니다.
지정된 멤버에 액세스하지 못했습니다.  
  
 **오류 ID:** BC30749  
  
### 이 오류를 해결하려면  
  
1.  멤버가 정의되고 멤버 이름을 올바르게 입력했는지 확인합니다.  
  
2.  모듈에 선언된 멤버 중 하나에 액세스해 봅니다. 선언된 모듈이 아직 로드되지 않아 디버깅 환경에서 멤버를 찾을 수 없는 경우도 있습니다.
