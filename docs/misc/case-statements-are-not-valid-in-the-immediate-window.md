# 직접 실행 창에서는 &#39;Case&#39; 문을 사용할 수 없습니다.
`Case` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30719  
  
### 이 오류를 해결하려면  
  
-   **직접 실행** 창에서 `Case` 문을 제거합니다.
