# You must supply a title for this tool.
이 오류는 **외부 도구** 대화 상자의 **제목** 필드가 비어 있는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  **제목** 입력란에 새 외부 도구의 이름을 입력하십시오.
