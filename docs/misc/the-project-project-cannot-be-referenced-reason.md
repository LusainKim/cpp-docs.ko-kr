# The project &#39;project&#39; cannot be referenced. &lt;reason&gt;
특정 프로젝트에 대해 프로젝트 대 프로젝트 참조를 만들 수 없습니다.  
  
 이 오류는 대부분 .exe를 빌드하는 프로젝트를 참조할 때 발생합니다.  뿐만 아니라 이 오류는 프로젝트가 출력이 어셈블리가 아닌\(예: 네이티브 C\+\+ 응용 프로그램\) 프로젝트를 참조함을 나타냅니다.
