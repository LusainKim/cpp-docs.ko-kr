# 리소스 컴파일러 오류 RC2138
닫는 주석 '\/\*'가 일치하지 않습니다.  
  
 주석 열기 시퀀스\(`/*`\)가 주석 닫기 시퀀스\(`*/`\)와 짝이 맞지 않습니다. 이 오류는 중첩된 주석에 의해 발생할 수 있습니다.
