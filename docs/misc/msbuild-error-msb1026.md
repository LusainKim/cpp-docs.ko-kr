# MSBuild 오류 MSB1026
**스키마 파일이 없습니다.**  
  
 지정된 위치에 스키마 파일이 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  스키마 파일의 절대 위치나 상대 위치가 지정된 경우 스키마 파일이 지정된 디렉터리에 있는지 확인합니다.
