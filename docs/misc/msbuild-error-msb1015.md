# MSBuild 오류 MSB1015
**이 운영 체제 버전에서는 MSBuild를 실행할 수 없습니다.  MSBuild는 Windows 2000 및 Windows XP 이상 버전에서만 지원됩니다.**  
  
 이 버전의 운영 체제에서는 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]가 지원되지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  지원되는 버전의 운영 체제 중 하나에서 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)]를 설치하고 실행합니다.
