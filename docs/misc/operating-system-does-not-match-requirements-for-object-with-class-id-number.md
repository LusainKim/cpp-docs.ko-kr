# Operating system does not match requirements for object with class ID &lt;number&gt;.
이 오류는 호환되지 않는 운영 체제나 호환되지 않는 운영 체제 버전이 검색된 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  응용 프로그램의 시스템 요구 사항을 확인합니다.  
  
2.  호환되는 운영 체제를 설치하고 다시 시도합니다.
