# MSBuild 오류 MSB3112
**MSB3112: 둘 이상의 어셈블리에서 같은 ID '\<assembly\>'을\(를\) 사용합니다.**  
  
 이 경고는 둘 이상의 참조된 어셈블리에 동일한 ID가 사용되는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   참조된 어셈블리 중 하나를 프로젝트에서 제거합니다.
