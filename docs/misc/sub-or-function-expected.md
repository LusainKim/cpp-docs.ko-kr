# &#39;Sub&#39; 또는 &#39;Function&#39;이 필요합니다.
`Declare` 문이 `Sub` 또는 `Function`을 지정하지 않습니다.`Declare`는 외부 프로시저를 식별하며 프로시저 종류를 지정해야 합니다.  
  
 **오류 ID:** BC30215  
  
### 이 오류를 해결하려면  
  
-   `Declare` 문에 `Sub` 또는 `Function` 키워드를 추가합니다.
