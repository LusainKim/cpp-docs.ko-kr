# 컴파일러 오류 CS0148
'delegate' 대리자에 올바른 생성자가 없습니다.  
  
 다른 컴파일러로 만든 관리되는 프로그램\(.NET Framework 공용 언어 런타임을 사용하는 프로그램\)을 가져와서 사용했습니다. 해당 컴파일러에서 잘못된 형식의 [대리자](../Topic/delegate%20\(C%23%20Reference\).md) 생성자를 허용했습니다. 자세한 내용은 [대리자](../Topic/Delegates%20\(C%23%20Programming%20Guide\).md)을 참조하세요.
