# CA0055
\<path\> 파일을 로드할 수 없습니다.  
  
 코드 분석 도구에서 분석을 위해 지정된 대상 파일을 로드하지 못했습니다.  파일이 없거나 파일에 대한 액세스가 거부된 경우에 이 경고가 발생합니다.  일반적으로 사용자가 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]에서 입력 어셈블리 속성을 재정의하거나 fxcopcmd.exe를 수동으로 호출한 경우에 발생합니다.
