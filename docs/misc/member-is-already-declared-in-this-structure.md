# &#39;&lt;member&gt;&#39;가 이 structure에 이미 선언되어 있습니다.
열거형 멤버가 이 구조체로 여러 번 선언되었습니다.  
  
 **오류 ID:** BC31421  
  
### 이 오류를 해결하려면  
  
-   불필요한 선언을 제거합니다.
