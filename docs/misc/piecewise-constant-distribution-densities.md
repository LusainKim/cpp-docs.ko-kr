# piecewise_constant_distribution::densities
분포에 대해 저장된 밀도를 반환합니다.  
  
## 구문  
  
```  
std::vector<double> densities() const;  
```  
  
## 설명  
 이 멤버 함수는 가중치 매개 변수에서 계산되어 저장된 밀도를 반환합니다.  
  
## 요구 사항  
 **헤더:** \<random\>  
  
 **네임스페이스:** std
