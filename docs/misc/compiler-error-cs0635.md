# 컴파일러 오류 CS0635
'attribute': System.Interop.UnmanagedType.CustomMarshaller는 명명된 인수 ComType 및 Marshal이 필요합니다.  
  
 마샬링 서식이 **System.Interop.UnmanagedType.CustomMarshaller**일 때 **ComType** 및 **Marshal** 인수가 지정되어야 합니다.
