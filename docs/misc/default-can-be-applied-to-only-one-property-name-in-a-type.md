# &lt;type&gt;에서 &#39;Default&#39;를 하나의 속성 이름에만 적용할 수 있습니다.
`Default`를 한 클래스, 인터페이스 또는 구조체 내에서 둘 이상의 속성 이름에 적용하려고 했습니다.  
  
 **오류 ID:** BC30359  
  
### 이 오류를 해결하려면  
  
-   `Default` 선언 중 하나만 남기고 모두 제거합니다.
