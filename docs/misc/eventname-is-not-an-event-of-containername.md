# &#39;&lt;eventname&gt;&#39;은 &#39;&lt;containername&gt;&#39;의 이벤트가 아닙니다.
지정된 이벤트가 개체에 대해 선언되지 않았습니다.  
  
 **오류 ID:** BC30676  
  
### 이 오류를 해결하려면  
  
1.  이벤트 이름의 철자가 정확한지 확인합니다.  
  
2.  올바른 개체에 액세스하고 있는지 확인합니다. 개체에 대한 정규화된 참조를 사용하거나 `Imports` 문을 사용하여 필요에 따라 적절한 네임스페이스를 가져옵니다.
