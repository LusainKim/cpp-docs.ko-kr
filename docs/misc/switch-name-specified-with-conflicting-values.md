# Switch &lt;name&gt; specified with conflicting values.
이 오류는 명령 창에 같은 스위치를 두 번 입력했지만 각 인스턴스에서 스위치에 지정한 값이 다를 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   각 스위치를 한 번씩만 사용하여 명령을 다시 입력합니다.
