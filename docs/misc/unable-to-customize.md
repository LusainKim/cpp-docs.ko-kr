# Unable to customize.
이 오류는 응용 프로그램을 디버깅하는 동안 메뉴나 도구 모음을 사용자 지정하려고 할 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   디버깅을 완료할 때까지 기다렸다가 도구 모음이나 메뉴를 사용자 지정합니다.  
  
     — 또는 —  
  
     디버깅을 중지한 다음 도구 모음이나 메뉴를 사용자 지정합니다.
