# 루트 폴더의 이름을 바꿀 수 없습니다.
루트 디렉터리의 이름을 바꾸려고 했습니다. 파일 경로가 올바르게 구문 분석되지 않은 것 같습니다.  
  
### 이 오류를 해결하려면  
  
-   정확한지 확인하려면 파일 경로를 확인합니다.
