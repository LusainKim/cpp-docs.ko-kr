# &#39;Is&#39;가 필요합니다.
`TypeOf` 연산자가 `Is` 절 없이 발생합니다.  
  
 **오류 ID:** BC30224  
  
### 이 오류를 해결하려면  
  
-   `TypeOf` 연산자 뒤에 `Is` 절을 추가합니다\(예: `TypeOf MyControl Is ComboBox`\).
