# MSBuild 오류 MSB3121
**MSB3121: 응용 프로그램 매니페스트의 파일 연결 요소에 필요한 특성인 확장명, 설명, progid 또는 기본 아이콘 중 하나 이상이 없습니다.**  
  
 [ClickOnce 배포 매니페스트](../Topic/ClickOnce%20Deployment%20Manifest.md)에는 네 특성에 대한 값이 모두 있어야 합니다.  
  
### 이 오류를 해결하려면  
  
-   각 [ClickOnce 배포 매니페스트](../Topic/ClickOnce%20Deployment%20Manifest.md) 특성을 올바른 값으로 설정합니다.
