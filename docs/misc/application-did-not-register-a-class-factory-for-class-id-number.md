# Application did not register a class factory for class ID &lt;number&gt;.
응용 프로그램의 IClassFactory 인터페이스를 구현하는 COM 개체가 등록되지 않았습니다.  이것은 운영 체제 오류입니다.  
  
### 이 오류를 해결하려면  
  
1.  Visual Studio 또는 영향을 받는 응용 프로그램을 다시 설치한 다음 다시 시도해 보십시오.
