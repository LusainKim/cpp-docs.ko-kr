# &#39;Private&#39;으로 선언된 형식은 다른 형식의 내부에 있어야 합니다.
`Private` 한정자가 다른 형식에 포함되지 않은 형식에서 사용되었습니다.  
  
 **오류 ID:** BC31089  
  
### 이 오류를 해결하려면  
  
1.  `Friend`와 같은 덜 제한적인 액세스 한정자를 사용합니다.  
  
2.  다른 형식 내에서 형식을 선언합니다.
