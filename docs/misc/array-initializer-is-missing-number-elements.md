# 배열 이니셜라이저에 요소가 &lt;number&gt;개 없습니다.
배열 이니셜라이저에 충분한 요소를 제공하지 않았습니다.  
  
 **오류 ID:** BC30567  
  
### 이 오류를 해결하려면  
  
-   누락된 인수 또는 인수를 제공합니다.  
  
-   `ReDim`을 사용하여 배열 크기를 변경합니다.
