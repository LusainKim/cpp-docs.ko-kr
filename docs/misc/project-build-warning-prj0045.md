# 프로젝트 빌드 경고 PRJ0045
'rule' 사용자 지정 빌드 규칙\('%file' 파일에 할당됨\)에 대한 'Outputs' 속성이 설정되어 있지 않았습니다. 이 규칙은 실행하지 않고 건너뜁니다.  
  
 Outputs 속성이 사용자 지정 빌드 규칙에 대해 정의되지 않았습니다. 사용자 지정 빌드 규칙이 빌드 작업의 출력을 필요로 합니다.
