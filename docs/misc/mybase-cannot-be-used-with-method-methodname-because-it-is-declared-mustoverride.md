# &#39;MyBase&#39;는 &#39;MustOverride&#39;로 선언된 &#39;&lt;methodname&gt;&#39; 메서드와 함께 사용할 수 없습니다.
`MustOverride`로 선언된 메서드와 함께 `MyBase`를 사용하려고 했습니다.  
  
 **오류 ID:** BC30399  
  
### 이 오류를 해결하려면  
  
-   `MustOverride` 선언을 제거합니다.
