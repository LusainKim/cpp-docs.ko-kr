# &#39;&lt;structurename&gt;&#39; 구조체에 기본 속성이 없으므로 구조체를 인덱싱할 수 없습니다.
기본 속성이 없는 구조체와 함께 기본 속성 구문을 사용하려고 했습니다.  
  
 **오류 ID:** BC30690  
  
### 이 오류를 해결하려면  
  
-   표준 속성 구문을 사용하여 구조체의 속성에 액세스합니다.
