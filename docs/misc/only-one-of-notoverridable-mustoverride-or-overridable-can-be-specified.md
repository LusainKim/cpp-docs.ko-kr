# &#39;NotOverridable&#39;, &#39;MustOverride&#39; 또는 &#39;Overridable&#39; 중 하나만 지정할 수 있습니다.
요소 선언에서 충돌하는 재정의 지정이 발생합니다. 재정의 키워드는 함께 사용할 수 없습니다.  
  
 **오류 ID:** BC30177  
  
### 이 오류를 해결하려면  
  
-   하나의 재정의 지정자를 선택하고 선언에서 다른 키워드를 제거합니다.
