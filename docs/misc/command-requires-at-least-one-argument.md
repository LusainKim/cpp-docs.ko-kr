# Command requires at least one argument.
이 오류는 적어도 하나 이상의 인수가 필요한 명령에 인수를 입력하지 않은 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 명령 구문에 대한 설명서를 확인한 다음 다시 시도하십시오.
