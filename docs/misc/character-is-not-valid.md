# 문자가 잘못되었습니다.
현재 컨텍스트는 이 문자를 허용하지 않습니다. 예를 들어 물결표\(`~`\) 또는 세로 막대\(`|`\)는 선언된 요소 이름의 일부가 될 수 없습니다.  
  
 **오류 ID:** BC30037  
  
### 이 오류를 해결하려면  
  
1.  프로그래밍 요소에 불필요한 문자가 실수로 삽입되었는지 확인합니다.  
  
2.  프로그래밍 요소의 철자 또는 프로그래밍 요소가 해당 종류의 요소에 대한 올바른 문자만 사용하여 생성되었는지 확인합니다.
