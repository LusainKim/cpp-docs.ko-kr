# 프로젝트 빌드 오류 PRJ0051
'파일'을 대상 디렉터리에 복사하지 못했습니다.  대상 파일이 사용 중이 아닌지 확인하세요.  
  
 빌드 시스템에서 파일을 복사하지 못했습니다.  
  
 파일이 컴퓨터에서 열려 있거나 다른 프로그램에서 사용하고 있어서는 안 됩니다.
