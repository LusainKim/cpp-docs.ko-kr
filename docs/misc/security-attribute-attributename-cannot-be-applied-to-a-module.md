# &#39;&lt;attributename&gt;&#39; 보안 특성을 모듈에 적용할 수 없습니다.
공용 언어 런타임은 모듈에 대한 보안 특성을 사용할 수 없습니다.  
  
 **오류 ID:** BC30131  
  
### 이 오류를 해결하려면  
  
1.  선언에서 특성을 제거합니다.
