# 예외 문제 해결: System.IdentityModel.Selectors.StsCommunicationException
보안 토큰 서비스와 관련하여 통신 문제가 있는 경우 <xref:System.IdentityModel.Selectors.StsCommunicationException> 예외가 throw됩니다.  
  
 다음과 같은 경우에 이 문제가 발생할 수 있습니다.  
  
-   네트워크 오류  
  
-   MEX 끝점 도달 실패  
  
-   보안 토큰 서비스에 유효한 끝점 ID가 없는 경우
