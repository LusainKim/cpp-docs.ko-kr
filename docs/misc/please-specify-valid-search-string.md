# Please specify valid search string.
이 오류는 **기호 찾기** 대화 상자의 **찾을 내용**에 검색할 문자열을 입력하지 않고 검색하려는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  **찾을 내용** 상자에 올바른 텍스트를 입력하고 다시 **찾기**를 선택하십시오.
