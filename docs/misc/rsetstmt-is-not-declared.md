# &#39;&lt;rsetstmt&gt;&#39;가 선언되지 않았습니다.
'\<rsetstmt\>'가 선언되지 않았습니다. RSet 문은 더 이상 지원되지 않습니다. 대신 Microsoft.VisualBasic.RSet를 사용합니다.  
  
 이전 버전의 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]에 내장된 여러 함수가 <xref:Microsoft.VisualBasic?displayProperty=fullName> 네임스페이스로 이동되었습니다. 그 결과 모든 프로그래밍 언어에서 더 일반적으로 기능을 사용할 수 있게 되었습니다.  
  
 **오류 ID:** BC30821  
  
### 이 오류를 해결하려면  
  
-   대신 `Microsoft.VisualBasic` 네임스페이스의 `RSet` 함수를 사용합니다.
