# Error getting information on languages supported by online search providers.
이 오류는 [Help, Environment, Options Dialog Box](http://msdn.microsoft.com/ko-kr/a2a5dff5-70a5-4f14-bf4c-66793894de6a)에 있는 **온라인 항목 언어** 목록에서 언어를 선택하려고 했으나 온라인 서비스에 액세스할 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  활성 인터넷 연결이 있는지 확인합니다.  
  
2.  프록시 서버를 사용하는 경우 프록시 서버가 제대로 구성되었는지 확인합니다.  
  
3.  **온라인 항목 언어** 드롭다운을 다시 선택합니다.
