# Tagged expression number is out of range.
이 오류는 바꾸기 문자열에 태그가 지정된 식에 대한 참조를 사용했지만 지정한 숫자가 0에서 9까지의 범위를 벗어나는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  0 에서 9 사이의 숫자를 입력하십시오.
