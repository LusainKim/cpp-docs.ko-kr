# &#39;Conditional&#39; 특성은 &#39;Sub&#39; 선언에서만 유효함
`Sub` 프로시저 이외의 메서드 선언에 <xref:System.Diagnostics.ConditionalAttribute> 특성을 적용했습니다.  
  
 **오류 ID:** BC41007  
  
### 이 오류를 해결하려면  
  
1.  잘못된 선언에서 <xref:System.Diagnostics.ConditionalAttribute>를 제거합니다.
