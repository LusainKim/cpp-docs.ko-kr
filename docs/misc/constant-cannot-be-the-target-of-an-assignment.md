# 상수는 할당 대상일 수 없습니다.
상수는 값을 할당하는 컨텍스트에서 발생합니다. 쓰기 가능한 변수, 속성 및 배열 요소만 실행 중 할당되는 값을 가질 수 있습니다.  
  
 **오류 ID:** BC30074  
  
### 이 오류를 해결하려면  
  
-   상수를 쓰기 가능한 단일 변수, 속성 또는 배열 요소로 바꿉니다.
