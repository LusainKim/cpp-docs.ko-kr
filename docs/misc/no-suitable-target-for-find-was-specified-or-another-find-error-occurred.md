# No suitable target for find was specified or another find error occurred.
이 오류는 검색 범위를 현재 문서로 설정하고 검색을 시작했지만 파일이 열려 있지 않은 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  파일을 열고 다시 검색하거나 검색 범위를 변경합니다.
