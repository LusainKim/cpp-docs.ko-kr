# &#39;WithEvents&#39; 변수는 &#39;&lt;containername&gt;&#39;에서 액세스할 수 있는 인스턴스 이벤트를 발생시키지 않습니다.
참조하는 개체가 액세스할 수 있는 이벤트를 발생시키지 않습니다.  
  
 **오류 ID:** BC30600  
  
### 이 오류를 해결하려면  
  
1.  이벤트에 대한 액세스 한정자를 수정하거나 현재 컨텍스트에서 액세스할 수 있는 이벤트를 선택합니다.
