# MSBuild 오류 MSB4137
**MSB4137: 구성 파일의 "\<x\>"에 잘못된 값이 지정되었습니다.  속성 이름 또는 ToolsVersion이 빈 문자열이 아닙니다.**  
  
 도구 집합 정의에 잘못된 속성 이름이나 값이 포함되어 있습니다.  
  
### 이 오류를 해결하려면  
  
-   도구 집합 정의에 잘못되었거나 비어 있는 속성 이름이나 값이 포함되어 있지 않은지 확인합니다.
