# MSBuild 오류 MSB3122
**MSB3122: 파일 연결을 사용하려면 완전 신뢰가 필요합니다.**  
  
 파일 연결을 구성하는 응용 프로그램을 게시하려면 해당 응용 프로그램이 완전 신뢰 응용 프로그램이어야 합니다.  
  
### 이 오류를 해결하려면  
  
-   ClickOnce 보안 설정을 활성화하고 응용 프로그램을 완전 신뢰 응용 프로그램으로 설정합니다.  자세한 내용은 [방법: ClickOnce 보안 설정 사용](../Topic/How%20to:%20Enable%20ClickOnce%20Security%20Settings.md)을 참조하십시오.
