# MSBuild 오류 MSB8004
*\<디렉터리\>* 디렉터리가 슬래시로 끝나지 않습니다.*\\{1\\}* 디렉터리의 적절한 평가를 허용하는 데 필요하므로 이 빌드 인스턴스에서 슬래시를 추가합니다.
