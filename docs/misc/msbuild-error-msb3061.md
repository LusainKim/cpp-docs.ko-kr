# MSBuild 오류 MSB3061
**"'\<file\>'" 파일을 삭제할 수 없습니다. '  \<error\>'**  
  
 `Delete` 작업에서 지정된 파일을 삭제할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
