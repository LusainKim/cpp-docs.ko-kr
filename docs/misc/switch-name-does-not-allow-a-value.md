# Switch &lt;name&gt; does not allow a value.
이 오류는 값을 허용하지 않는 스위치에 값을 할당하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  스위치 값을 생략하고 명령을 다시 입력하십시오.
