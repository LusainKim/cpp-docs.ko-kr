# 리소스 컴파일러 오류 RC2154
제어 문자가 범위\[^A \- ^Z\]를 벗어났습니다.  
  
 **ACCELERATORS** 문의 제어 문자가 잘못되었습니다. 캐럿\(**^**\) 뒤의 문자는 A에서 Z\(포함\) 사이여야 합니다.
