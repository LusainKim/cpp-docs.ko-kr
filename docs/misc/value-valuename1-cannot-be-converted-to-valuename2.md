# &#39;&lt;valuename1&gt;&#39;값을 &#39;&lt;valuename2&gt;&#39;로 변환할 수 없습니다.
호환되지 않는 두 형식 간에 형식을 변환하려고 했습니다.  
  
 **오류 ID:** BC30742  
  
### 이 오류를 해결하려면  
  
-   캐스트 중인 형식을 확인합니다.
