# 예외 문제 해결: Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException
코드에서 문서에 없는 컨트롤에 액세스하려고 하면 <xref:Microsoft.VisualStudio.Tools.Applications.Runtime.ControlNotFoundException> 예외가 throw됩니다. 컨트롤이가 최종 사용자나 다른 코드에 의해 제거되었을 수 있습니다. 컨트롤에 액세스해 보기 전에는 컨트롤을 확인할 수 있는 방법이 없습니다. 컨트롤이 제거되었으면 솔루션이 작동하지 않습니다.
