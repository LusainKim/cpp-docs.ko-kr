# &#39;Namespace&#39; 문은 짝이 되는 &#39;End Namespace&#39;로 끝나야 합니다.
`Namespace` 블록은 `End Namespace` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC30626  
  
### 이 오류를 해결하려면  
  
-   `Namespace` 블록 끝에 `End Namespace` 문을 추가합니다.
