# .NET Framework 디렉터리를 찾을 수 없습니다. &lt;error&gt;
[!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)] 어셈블리가 저장된 디렉터리를 찾을 수 없습니다.  
  
 **오류 ID:** BC31508  
  
### 이 오류를 해결하려면  
  
1.  프로그램을 다시 컴파일하여 오류가 다시 발생하는지 확인합니다.  
  
2.  오류가 계속되면 작업을 저장하고 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]를 다시 시작합니다.  
  
3.  그래도 오류가 다시 발생하면 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]을 다시 설치합니다.  
  
4.  다시 설치 후에도 오류가 계속 발생하면 Microsoft 기술 지원 서비스에 알립니다.
