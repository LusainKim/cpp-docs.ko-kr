# &#39;&lt;variablename&gt;&#39; 지역 변수를 참조하려면 이 변수를 먼저 선언해야 합니다.
프로시저 내의 변수가 선언 전에 사용됩니다.  
  
 **오류 ID:** BC32000  
  
### 이 오류를 해결하려면  
  
-   변수의 첫 번째 참조 앞으로 선언을 이동합니다.
