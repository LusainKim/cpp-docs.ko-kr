# 리소스 컴파일러 오류 RC2018
'hexnumber' 문자를 인식할 수 없습니다.  
  
 지정된 16진수 숫자에 해당하는 ASCII 문자가 원본 파일에 나타나지만 잘못된 문자입니다.  
  
 이 오류는 원본 파일 손상으로 발생할 수 있습니다.
