# Menu captions cannot contain separators.
이 오류는 메뉴 캡션 이름에 대시 문자\(\-\)를 사용하는 경우 발생합니다.  대시 문자는 메뉴 구분 기호로 지정되어 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  대시 문자를 사용하지 않는 새 캡션 이름을 입력하십시오.
