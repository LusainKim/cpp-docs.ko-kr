# XML 태그에 시작 &#39;&lt;&#39;가 필요합니다.
XML 리터럴의 XML 태그에 필요한 시작 '\<' 문자가 없습니다.  
  
 **오류 ID:** BC31165  
  
### 이 오류를 해결하려면  
  
-   XML 리터럴의 XML 태그 시작에 '\<' 문자를 추가합니다.
