# 예외 문제 해결: System.IO.DirectoryNotFoundException
<xref:System.IO.DirectoryNotFoundException> 예외는 디렉터리나 파일 경로의 일부를 찾을 수 없는 경우에 throw됩니다.  
  
## 관련 팁  
 **지정한 위치에 해당 디렉터리가 있는지 확인하십시오.**  
 지정된 위치에 디렉터리가 있는지 확인합니다.  
  
 **상대 경로를 사용하는 경우 현재 디렉터리가 올바른지 확인하십시오.**  
 현재 디렉터리를 잘못 지정하면 경로가 올바르지 않을 수 있습니다.
