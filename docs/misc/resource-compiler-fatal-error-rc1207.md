# 리소스 컴파일러 심각한 오류 RC1207
기본 코드 페이지가 잘못되었습니다.  
  
 **RCCODEPAGE** 환경 변수에서 지정한 코드 페이지가 유효하지 않습니다. 자세한 내용은 *런타임 라이브러리 참조*에서 [코드 페이지](../c-runtime-library/code-pages.md)를 참조하세요.
