# 이 컨텍스트에서는 &#39;New&#39;를 사용할 수 없습니다.
형식 이름이 필요한 위치에 `New` 키워드를 사용했습니다. 컨텍스트에서 `New`와 함께 사용할 수 없는 값 형식이 필요합니다.  
  
 **오류 ID:** BC30200  
  
### 이 오류를 해결하려면  
  
-   `New` 키워드를 제거합니다.
