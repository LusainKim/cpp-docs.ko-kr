# 리소스 컴파일러 오류 RC2168
리소스가 너무 큽니다.  
  
 리소스의 크기 제한을 초과했습니다. 커서, 아이콘, 비트맵 또는 기타 파일 기반 리소스에는 이 제한이 적용되지 않습니다.
