# 대상 디렉터리가 소스 디렉터리 아래에 있기 때문에 작업을 완료할 수 없습니다.
순환 작업이 실패했습니다. 순환 작업이 계속 실행되므로 완료할 수 없습니다. 예를 들어 개체 A는 개체 B에서 상속하려고 하고 개체 B는 개체 A에서 상속하려고 합니다.  
  
### 이 오류를 해결하려면  
  
-   상속 시 순환 참조가 없어야 합니다.
