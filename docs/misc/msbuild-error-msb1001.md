# MSBuild 오류 MSB1001
**알 수 없는 스위치입니다.**  
  
 명령줄에 알 수 없는 스위치를 지정했습니다.  
  
### 이 오류를 해결하려면  
  
1.  스위치의 철자를 확인합니다.  올바른 스위치 목록을 표시하려면 `msbuild /help`를 입력합니다.
