# Command allows at most one argument.
이 오류는 인수를 허용하지 않거나 하나의 인수만 허용하는 명령에 너무 많은 인수를 입력하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  올바른 명령 구문에 대한 설명서를 확인한 다음 다시 시도하십시오.
