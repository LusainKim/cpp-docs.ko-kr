# Solution platform name is not valid.
솔루션 플랫폼 이름에는 다음 문자를 사용할 수 없습니다.  
  
-   파운드\(\#\)  
  
-   백분율\(%\)  
  
-   앰퍼샌드\(&\)  
  
-   별표\(\*\)  
  
-   세로줄\(&#124;\)  
  
-   백슬래시\(\\\)  
  
-   콜론\(:\)  
  
-   큰따옴표\("\)  
  
-   보다 작음\(\<\)  
  
-   보다 큼\(\>\)  
  
-   마침표\(.\)  
  
-   물음표\(?\)  
  
-   슬래시\(\/\)  
  
-   선행 공백 또는 후행 공백\(' '\)  
  
-   Windows 또는 DOS에 예약된 이름\(nul, aux, con, com1, lpt1 등\)  
  
### 이 오류를 해결하려면  
  
1.  위에 나열된 문자가 포함되지 않은 솔루션 플랫폼 이름을 입력합니다.
