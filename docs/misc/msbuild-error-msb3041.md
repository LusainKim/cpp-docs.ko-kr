# MSBuild 오류 MSB3041
**"'\<file\>'"에 대해 매니페스트 리소스 이름을 만들 수 없습니다.**  
  
 `CreateManifestResourceName` 작업에서 지정된 파일에 대한 매니페스트 리소스 이름을 만들 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
