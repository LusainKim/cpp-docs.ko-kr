# 리소스 파일의 형식이 잘못되었습니다.
프로젝트의 리소스 파일이 Windows 리소스 파일의 표준 형식을 따르지 않거나 리소스 파일과 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 프로젝트가 일치하지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  리소스 원본 및 리소스 파일을 다시 만들고 Windows SDK\(소프트웨어 개발 키트\)에 설명되어 있는 구문을 준수합니다.  
  
2.  프로젝트에서 잘못된 리소스 파일을 제거합니다.
