# 솔루션 탐색기에서 사용할 수 없는 프로젝트에 항목을 추가할 수 없습니다.
일반적으로 이 오류는 지정된 프로젝트가 솔루션 탐색기에서 열려 있지 않은 경우에 발생합니다. Visual Studio는 현재 솔루션 탐색기에서 현재 사용할 수 없는 프로젝트에는 항목을 추가할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
1.  지정된 프로젝트를 현재 솔루션에 추가하거나 지정된 프로젝트가 포함된 솔루션을 여세요.
