# 리소스 컴파일러 심각한 오류 RC1106
잘못된 스위치: 옵션  
  
 옵션이 올바른 리소스 컴파일러 옵션이 아닙니다.  
  
 유효한 옵션은 다음과 같습니다.  
  
 \/r  
  
 .res 파일을 내보냅니다.  
  
 \/v  
  
 자세한 정보를 표시합니다\(인쇄 진행 메시지\).  
  
 \/d  
  
 기호를 정의합니다.  
  
 \/fo  
  
 .res 파일의 이름을 바꿉니다.  
  
 \/l  
  
 16진수의 기본 언어 ID입니다.  
  
 \/I  
  
 **INCLUDE** 검색 경로를 추가합니다.  
  
 \/x  
  
 **INCLUDE** 환경 변수를 무시합니다.  
  
 \/c  
  
 NLS 변환에 사용되는 코드 페이지를 정의합니다.
