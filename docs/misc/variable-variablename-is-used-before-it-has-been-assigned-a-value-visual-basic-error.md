# 값이 할당되기 전에 &#39;&lt;variablename&gt;&#39; 변수를 사용했습니다(Visual Basic 오류).
값이 할당되기 전에 변수가 사용되어 <xref:System.NullReferenceException>이 발생했습니다.  
  
 **오류 ID:** BC42109  
  
### 이 오류를 해결하려면  
  
-   변수에 값이 할당되었는지 확인합니다.
