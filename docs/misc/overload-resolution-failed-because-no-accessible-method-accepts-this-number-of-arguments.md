# 지정한 개수의 인수를 허용하는 액세스 가능한 &#39;&lt;method&gt;&#39;가 없으므로 오버로드 확인에 실패했습니다.
오버로드된 메서드를 호출했지만, 컴파일러에서 일치하는 인수 목록이 있는 메서드를 찾을 수 없습니다.  
  
 **오류 ID:** BC30516  
  
### 이 오류를 해결하려면  
  
-   제공한 인수 목록을 확인합니다.
