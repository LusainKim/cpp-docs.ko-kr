# &#39;System.Runtime.InteropServices.DllImportAttribute&#39;는 &#39;Get&#39; 또는 &#39;Set&#39;에 적용할 수 없습니다.
`DllImportAttribute` 특성을 `Get` 또는 `Set` 속성 프로시저에 적용했습니다.  
  
 **오류 ID:** BC31524  
  
### 이 오류를 해결하려면  
  
1.  `Get` 및 `Set` 속성 프로시저에서 `DllImportAttribute`를 제거합니다.
