# 대리자를 구현하는 데 필요한 &#39;&lt;typename&gt;&#39; 형식의 대리 생성자를 찾을 수 없습니다.
형식을 포함하는 어셈블리가 손상되었거나 잘못되었을 수 있습니다.  
  
 **오류 ID:** BC31074  
  
### 이 오류를 해결하려면  
  
1.  이 형식을 포함하는 어셈블리가 유효한지 확인합니다.  
  
2.  프로그램을 다시 컴파일하여 오류가 다시 발생하는지 확인합니다.  
  
3.  오류가 계속되면 작업을 저장하고 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]를 다시 시작합니다.
