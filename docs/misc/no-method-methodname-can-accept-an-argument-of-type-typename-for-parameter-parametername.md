# &#39;&lt;methodname&gt;&#39; 메서드는 &#39;&lt;parametername&gt;&#39; 매개 변수에 대해 &#39;&lt;typename&gt;&#39; 형식의 인수를 사용할 수 없습니다.
메서드 호출에 유효하지 않은 형식의 인수가 포함되어 있습니다.  
  
### 이 오류를 해결하려면  
  
1.  인수의 형식을 확인합니다.  
  
2.  메서드의 매개 변수 형식과 인수 형식을 일치시켜 계약 내에 있는지 확인합니다. 필요에 따라 형식을 변경합니다.
