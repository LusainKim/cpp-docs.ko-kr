# The &lt;name&gt; switch requires a value.
이 오류는 필요한 값 없이 스위치를 지정하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  명령을 다시 입력하고 스위치 값을 지정하십시오.  
  
     — 또는 —  
  
     명령을 다시 입력하고 스위치를 생략하십시오.
