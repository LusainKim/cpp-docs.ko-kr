# MSBuild 오류 MSB3114
**MSB3114: '\<assembly\>' 어셈블리에서 참조하는 '\<file\>' 파일을 찾을 수 없습니다.**  
  
 이 오류는 기존 매니페스트에서 확인할 수 없는 참조를 발견한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   매니페스트에서 파일 참조가 충돌하는지 확인합니다.
