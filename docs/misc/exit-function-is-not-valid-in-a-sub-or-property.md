# Sub 또는 Property에서는 &#39;Exit Function&#39;을 사용할 수 없습니다.
`Exit Function`는 `Sub` 프로시저 또는 `Property` 프로시저에 나타납니다.`Exit` 문은 해당 문이 발생한 블록과 일치해야 합니다.  
  
 **오류 ID:** BC30067  
  
### 이 오류를 해결하려면  
  
-   `Exit Function`를 `Exit Sub` 또는 `Exit Property` 문으로 적절하게 바꿉니다.
