# 리소스 컴파일러 경고 RC4203
VIRTKEY 없이 SHIFT 또는 CONTROL이 사용되었습니다.  
  
 액셀러레이터 키 테이블 리소스에서 SHIFT 또는 CONTROL에 VIRTKEY가 필요합니다. SHIFT와 CONTROL은 VIRTKEY 형식 액셀러레이터에서 플래그 비트로 표시되므로 VIRTKEY와 독립적으로 존재할 수 없습니다.
