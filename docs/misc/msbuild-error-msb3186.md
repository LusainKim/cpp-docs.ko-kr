# MSBuild 오류 MSB3186
**MSB3186: 생성된 매니페스트의 어셈블리 ID를 작업 입력 매개 변수에서 유추할 수 없습니다.**  
  
 이 오류는 응용 프로그램 또는 배포 매니페스트에 대한 어셈블리 이름을 빌드 프로세스에서 유추할 수 없는 경우 발생합니다.  어셈블리 이름이 명시적으로 지정되지 않았습니다. 기본 매니페스트에 ID가 없으며 진입점 ID도 지정되지 않았습니다.
