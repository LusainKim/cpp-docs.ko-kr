# 예외 문제 해결: System.ServiceModel.PoisonMessageException
메시지가 "포이즌 메시지"로 간주되면 <xref:System.ServiceModel.PoisonMessageException> 예외가 throw됩니다. 메시지를 전송하려는 반복적인 시도가 실패한 경우 메시지가 포이즌 메시지로 간주됩니다.
