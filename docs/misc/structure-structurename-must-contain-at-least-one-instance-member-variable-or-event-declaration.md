# &#39;&lt;structurename&gt;&#39; 구조체에는 적어도 하나의 인스턴스 멤버 변수 또는 이벤트 선언이 있어야 합니다.
기능적으로 빈 구조체가 발생합니다. 하나 이상의 변수 멤버가 `Structure` 문과 `End Structure` 문 사이에 선언되어야 합니다.  
  
 **오류 ID:** BC30281  
  
### 이 오류를 해결하려면  
  
-   `Dim` 문 또는 `Event` 문을 구조체에 추가합니다.
