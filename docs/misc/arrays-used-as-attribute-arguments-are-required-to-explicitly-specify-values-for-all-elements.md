# 모든 요소에 대한 값을 명시적으로 지정하려면 특성 인수로 사용되는 배열이 필요합니다.
특성 인수로 사용되는 배열에 지정되지 않은 값을 가진 요소가 있습니다.  
  
 **오류 ID:** BC31110  
  
### 이 오류를 해결하려면  
  
-   모든 요소 값을 지정합니다.
