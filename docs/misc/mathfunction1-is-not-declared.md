# &#39;&lt;mathfunction1&gt;&#39;이 선언되지 않았습니다.
'\<mathfunction1\>'이 선언되지 않았습니다. 함수는 'System.Math' 클래스로 이동되어 '\<mathfunction2\>'로 이름이 변경됩니다.  
  
 이전 버전의 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]에 내장된 여러 함수가 <xref:System.Math?displayProperty=fullName> 네임스페이스로 이동되었습니다. 그 결과 모든 프로그래밍 언어에서 더 일반적으로 기능을 사용할 수 있게 되었습니다.  
  
 **오류 ID:** BC30819  
  
### 이 오류를 해결하려면  
  
-   <xref:System.Math?displayProperty=fullName>에 선언된 메서드를 사용합니다.
