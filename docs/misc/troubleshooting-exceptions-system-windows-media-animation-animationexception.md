# 예외 문제 해결: System.Windows.Media.Animation.AnimationException
속성에 애니메이션을 적용하는 동안 오류가 발생하면 <xref:System.Windows.Media.Animation.AnimationException> 예외가 throw됩니다.  
  
## 설명  
 이 클래스는 음의 <xref:System.Windows.Media.Animation.Timeline.Duration%2A> 또는 잘못된 <xref:System.Windows.Media.Animation.Timeline.AccelerationRatio%2A> 및 <xref:System.Windows.Media.Animation.Timeline.DecelerationRatio%2A> 설정을 사용하여 애니메이션을 만드는 등의 타이밍 관련 오류에는 사용되지 않습니다.
