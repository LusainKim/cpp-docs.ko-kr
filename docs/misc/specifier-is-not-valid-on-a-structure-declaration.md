# &#39;&lt;specifier&gt;&#39;는 구조체 선언에서 사용할 수 없습니다.
`Structure` 선언에서 유효하지 않은 `Dim` 또는 `Override`와 같은 지정자를 사용했습니다.  
  
 **오류 ID:** BC30395  
  
### 이 오류를 해결하려면  
  
-   지정자를 제거합니다.
