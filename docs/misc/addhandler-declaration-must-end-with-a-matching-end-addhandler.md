# &#39;AddHandler&#39; 선언은 짝이 되는 &#39;End AddHandler&#39;로 끝나야 합니다.
`AddHandler` 선언은 `End AddHandler` 문으로 끝나야 합니다.  
  
 **오류 ID:** BC31115  
  
### 이 오류를 해결하려면  
  
-   `AddHandler` 선언이 `End AddHandler` 문으로 끝나는지 확인합니다.
