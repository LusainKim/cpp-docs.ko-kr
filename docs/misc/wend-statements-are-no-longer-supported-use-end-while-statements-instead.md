# &#39;Wend&#39; 문은 더 이상 지원되지 않습니다. 대신 &#39;End While&#39; 문을 사용하세요.
`End While` 구문이 `Wend` 키워드를 대체합니다.  
  
 **오류 ID:** BC30809  
  
### 이 오류를 해결하려면  
  
-   `End While`을 사용하여 `While` 블록을 종료합니다.
