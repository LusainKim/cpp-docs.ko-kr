# &#39;Finally&#39; 밖으로는 분기할 수 없습니다.
`Finally` 블록 내의 `GoTo` 문이 블록 밖으로 분기합니다.`Catch` 또는 `Finally` 블록 안이나 밖으로 분기할 수 없습니다.  
  
 **오류 ID:** BC30101  
  
### 이 오류를 해결하려면  
  
-   `GoTo` 문을 제거하고 의사 결정 또는 루프 제어 구조를 통해 프로그램 논리를 구현합니다.
