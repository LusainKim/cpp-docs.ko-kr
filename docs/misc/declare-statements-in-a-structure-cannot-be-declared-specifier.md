# 구조체의 &#39;Declare&#39; 문은 &#39;&lt;specifier&gt;&#39;로 선언할 수 없습니다.
`Declare` 선언에 `Structure` 선언에서 유효하지 않은 지정자가 있습니다.  
  
 **오류 ID:** BC30791  
  
### 이 오류를 해결하려면  
  
-   지정자를 제거합니다.
