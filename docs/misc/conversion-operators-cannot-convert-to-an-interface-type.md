# 변환 연산자는 인터페이스 형식으로 변환할 수 없습니다.
반환 형식으로 인터페이스 형식을 사용하여 변환 연산자를 선언했습니다.  
  
 컴파일 시간에 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]은 참조 형식에서 인터페이스로 미리 정의된 변환이 있는 것으로 간주합니다. 이러한 변환은 런타임에 실패할 수 있지만 컴파일러에서 런타임 결과를 예측할 수 없으므로 이러한 변환의 컴파일을 허용합니다.  
  
 컴파일러에서는 이 변환이 이미 정의되어 있다고 간주하므로 다시 정의할 수 없습니다.  
  
 **오류 ID:** BC33025  
  
### 이 오류를 해결하려면  
  
-   이 연산자 정의를 완전히 제거합니다. 이미 미리 정의되어 있습니다.
