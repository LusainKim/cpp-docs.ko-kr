-   [특성이 같은 XML 주석 태그 '\<tag\>'가 동일한 XML 주석 블록에서 두 번 이상 나타납니다.](../misc/xml-comment-tag-tag-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block.md)
