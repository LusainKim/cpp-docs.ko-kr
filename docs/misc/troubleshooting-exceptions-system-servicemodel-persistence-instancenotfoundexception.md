# 예외 문제 해결: System.ServiceModel.Persistence.InstanceNotFoundException
다음과 같은 경우에 <xref:System.ServiceModel.Persistence.InstanceNotFoundException> 예외가 throw됩니다.  
  
-   완료된 것으로 표시된 영속적인 서비스 인스턴스에 대해 작업이 수행되는 경우  
  
-   <xref:System.ServiceModel.Persistence.SqlPersistenceProviderFactory>로 생성된 지속성 공급자가 데이터베이스에 없는 상태 데이터의 잠금, 잠금 해제 또는 처리를 시도하는 경우
