# 이 시스템은 지정한 로캘을 지원하지 않습니다.
`VbStrConv` 열거형을 사용하려고 하지만 시스템이 지정된 로캘을 지원하지 않습니다.  
  
### 이 오류를 해결하려면  
  
1.  로캘을 지원하도록 컴퓨터를 업그레이드하거나, 로캘을 지원하는 컴퓨터에서 응용 프로그램을 실행합니다.
