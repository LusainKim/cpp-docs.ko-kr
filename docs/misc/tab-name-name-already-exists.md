# Tab name &lt;name&gt; already exists.
도구 상자에 있는 각 탭의 이름은 고유해야 합니다.  이 오류는 도구 상자 탭에 입력한 이름이 도구 상자에 있는 기존 탭의 이름과 같을 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  탭에 고유한 이름을 입력하십시오.
