# &#39;&lt;membername&gt;&#39;이 선언되지 않았거나 이를 포함하는 모듈이 디버깅 세션에서 로드되지 않았습니다.
지정된 멤버를 찾을 수 없습니다.  
  
 **오류 ID:** BC30699  
  
### 이 오류를 해결하려면  
  
1.  멤버가 정의되었으며 철자가 맞는지 확인합니다.  
  
2.  모듈에 선언된 멤버 중 하나에 액세스합니다. 선언된 모듈이 아직 로드되지 않아 디버깅 환경에서 멤버를 찾을 수 없는 경우도 있습니다.
