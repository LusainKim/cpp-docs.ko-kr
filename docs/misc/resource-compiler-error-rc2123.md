# 리소스 컴파일러 오류 RC2123
메뉴에 END가 필요합니다.  
  
 **MENU** 문의 끝에 **END** 키워드가 와야 합니다. 열려 있는 따옴표나 일치하지 않는 **BEGIN** 및 **END** 문 쌍이 없는지 확인합니다.
