# 리소스 컴파일러 오류 RC2174
비트맵 파일의 파일 이름이 2.03 형식이 아닙니다.  
  
 비트맵이 버전 2.03 이전의 형식을 사용했습니다. 버전 3.00 이상의 형식을 사용하여 비트맵을 변환하거나 다시 그려야 합니다.
