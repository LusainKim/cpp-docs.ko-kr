# 리소스 컴파일러 오류 RC2126
메뉴 문자열이 필요합니다.  
  
 각 **MENUITEM** 및 **POPUP** 문은 메뉴 항목 또는 팝업 메뉴의 이름을 지정하는 큰따옴표로 묶인 문자열인 *text* 필드를 포함해야 합니다.**MENUITEM SEPARATOR** 문에는 따옴표 붙은 문자열이 필요하지 않습니다.
