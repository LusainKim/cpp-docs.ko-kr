# 리소스 컴파일러 오류 RC2176
**파일 이름에 이전 DIB가 있습니다. SDKPAINT를 통과하세요.**  
  
 지정된 파일에 이전 형식의 장치 독립적 비트맵이 있습니다. 파일을 현재 형식으로 변환해야 합니다. 이 작업을 위해 Windows 3.0 SDK에 제공된 SDKPAINT 응용 프로그램 또는 동등한 응용 프로그램을 사용할 수 있습니다.
