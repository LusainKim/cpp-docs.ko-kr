# File name or default path is too long.
이 오류는 코드 주석 웹 보고서를 만들려고 했지만 주석 웹 페이지 빌드 대화 상자의 웹 페이지 저장 위치 상자에 지정된 파일 이름 또는 경로가 너무 긴 긴 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  260자 미만의 파일 이름 또는 경로를 지정합니다.
