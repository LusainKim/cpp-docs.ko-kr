# 리소스 컴파일러 심각한 오류 RC1122
파일을 쓰는 동안 I\/O 오류가 발생했습니다.  
  
 리소스 컴파일러가 파일에 쓸 수 없습니다.  
  
### 다음과 같은 가능한 원인을 확인하여 수정하려면  
  
1.  디스크 공간이 부족합니다. 여유 공간의 크기는 만들려는 실행 파일 크기의 두 배와 같아야 합니다.  
  
2.  볼륨이 읽기 전용입니다.  
  
3.  불량 섹터입니다.  
  
4.  공유 위반입니다.
