# 현재 FieldWidths를 사용하여 줄 &lt;number&gt;의 구문을 분석할 수 없습니다.
해당 필드의 너비가 지정된 것과 다르기 때문에 지정한 줄을 구문 분석할 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   줄을 올바르게 구문 분석할 수 있도록 `FieldWidths`를 조정하거나 줄을 처리하도록 예외 처리 코드를 삽입합니다.
