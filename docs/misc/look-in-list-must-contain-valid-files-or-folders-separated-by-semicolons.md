# Look In list must contain valid files or folders, separated by semicolons.
이 오류는 **찾는 위치** 목록에 나열된 디렉터리 또는 파일에 잘못된 문자가 포함되어 있는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  경로나 파일 이름을 입력하는 대신 **찾는 위치** 대화 상자를 사용하여 검색할 디렉터리 또는 파일 이름을 찾아봅니다.  **찾는 위치** 목록 옆에 있는 줄임표 단추를 선택하여 **찾는 위치** 대화 상자에 액세스할 수 있습니다.
