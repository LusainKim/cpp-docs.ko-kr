# &#39;GoSub&#39; 문은 더 이상 지원되지 않습니다.
`GoSub`를 사용하여 프로시저를 호출할 수는 없습니다.  
  
 **오류 ID:** BC30814  
  
### 이 오류를 해결하려면  
  
-   `GoSub`를 사용하지 않고 프로시저를 직접 호출합니다. 예를 들면 다음과 같습니다.  
  
    ```  
    CalculateInterest(Amount, Rate, Time)  
    ```
