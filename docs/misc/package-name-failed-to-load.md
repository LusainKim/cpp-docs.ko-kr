# Package &lt;name&gt; failed to load.
패키지를 열 수 없습니다.  이 오류는 패키지가 제대로 등록되지 않았거나, 패키지에 DLL이 없거나, DLL이 손상되었거나 또는 패키지에 다른 구성 요소로부터 요청하는 DLL이 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  해당 패키지를 제공하는 제품을 다시 설치하십시오.
