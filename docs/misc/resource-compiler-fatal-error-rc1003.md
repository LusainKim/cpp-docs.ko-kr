# 리소스 컴파일러 심각한 오류 RC1003
오류 수가 number개를 초과했습니다. 컴파일을 중지합니다.  
  
 오류 수가 너무 많아 컴파일을 계속 진행할 수 없습니다. 일부 오류를 수정한 다음 다시 컴파일하세요.
