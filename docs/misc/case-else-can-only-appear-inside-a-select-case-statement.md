# &#39;Case Else&#39;는 &#39;Select Case&#39; 문 내부에만 사용할 수 있습니다.
`Case Else` 문이 `Select` 블록 외부에서 발생합니다.`Case Else` 문은 `Select` 또는 `Select Case` 문과 해당 `End Select` 문 사이에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30071  
  
### 이 오류를 해결하려면  
  
-   `Case Else` 문을 제거하거나 `Select` 블록 내로 이동합니다.
