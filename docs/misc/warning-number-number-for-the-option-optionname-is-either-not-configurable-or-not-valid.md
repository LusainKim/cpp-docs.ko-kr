# &#39;&lt;optionname&gt;&#39; 옵션의 경고 번호 &#39;&lt;number&gt;&#39;가 구성 가능하거나 올바르지 않습니다.
경고 ID가 잘못되었습니다. ID는 0보다 큰 정수 값이어야 합니다.  
  
 **오류 ID:** BC2026  
  
### 이 오류를 해결하려면  
  
-   제공된 경고 ID를 검사하여 유효한지 확인합니다.
