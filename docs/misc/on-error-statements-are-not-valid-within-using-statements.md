# &#39;On Error&#39; 문은 &#39;Using&#39; 문 안에서 사용할 수 없습니다.
`On Error` 문이 `Using` 문 내에 나타나지만 해당 컨텍스트에서 사용할 수 없습니다.  
  
 **오류 ID:** BC36013  
  
### 이 오류를 해결하려면  
  
-   `On Error` 문 대신 `Try…Catch` 블록과 같은 구조적 오류 처리를 사용합니다.
