# &#39;&lt;modifier&gt;&#39;는 지역 상수 선언에서 사용할 수 없습니다.
프로시저 내의 `Const` 문에 `Overloads`와 같은 잘못된 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30246  
  
### 이 오류를 해결하려면  
  
1.  `Const` 문에서 잘못된 키워드를 제거합니다.
