# 메서드 본문 안에는 문을 사용할 수 없습니다.
프로시저에 다른 프로시저 선언과 같은 잘못된 문이 포함되어 있습니다. 프로시저 선언은 중첩할 수 없습니다.  
  
 **오류 ID:** BC30289  
  
### 이 오류를 해결하려면  
  
-   프로시저 선언을 제거하거나 바깥쪽 프로시저 밖으로 이동합니다.
