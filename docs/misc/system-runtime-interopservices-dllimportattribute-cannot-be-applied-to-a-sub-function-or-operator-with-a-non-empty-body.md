# &#39;System.Runtime.InteropServices.DllImportAttribute&#39;는 본문이 비어 있는 Sub, Function 또는 Operator에만 적용할 수 있습니다.
`DllImportAttribute` 특성이 비어 있는 `Sub`, `Function` 또는 `Operator`에 적용되었습니다.  
  
 **오류 ID:** BC31522  
  
### 이 오류를 해결하려면  
  
-   이 특성을 사용하려면 `Sub`, `Function` 또는 `Operator`에서 모든 코드를 제거합니다.
