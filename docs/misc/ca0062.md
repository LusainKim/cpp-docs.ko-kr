# CA0062
규칙 집합 '*RuleSetName*'에서 참조된 '*RuleId*' 규칙을 찾을 수 없습니다.  
  
 이 오류는 지정한 규칙을 찾을 수 없음을 나타냅니다.  
  
 대개 이 오류는 규칙 집합 파일을 수동으로 편집했기 때문에 발생합니다.  이 오류를 피하려면 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] 규칙 집합을 사용하여 규칙을 지정합니다.  자세한 내용은 [사용자 지정 규칙 집합 만들기](../Topic/Creating%20Custom%20Code%20Analysis%20Rule%20Sets.md)을 참조하십시오.  
  
 이 문제를 해결 하려면 확인 id 규칙 집합 파일에 지정 된 모든 유효한 지 확인 합니다. 비표준 규칙 파일을 사용 하는 확인 하 고 적절 한 규칙 힌트 경로가 있는 규칙 집합 파일에 지정 됩니다.
