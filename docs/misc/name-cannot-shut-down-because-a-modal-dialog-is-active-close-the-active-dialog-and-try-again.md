# &lt;name&gt; cannot shut down because a modal dialog is active. Close the active dialog and try again.
이 오류는 운영 체제를 종료할 때 응용 프로그램에 모달 대화 상자가 열려 있어 응용 프로그램을 닫지 못하는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  모달 대화 상자를 닫은 다음 응용 프로그램을 닫거나 운영 체제를 종료하십시오.
