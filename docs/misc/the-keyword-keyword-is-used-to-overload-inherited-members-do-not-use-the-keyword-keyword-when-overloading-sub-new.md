# &#39;&lt;keyword&gt;&#39; 키워드를 사용하여 상속된 멤버를 오버로드했습니다. &#39;Sub New&#39;를 오버로드할 때는 &#39;&lt;keyword&gt;&#39; 키워드를 사용하지 마세요.
생성자가 `Overloads` 키워드를 사용하여 선언되었습니다.  
  
 Visual Basic은 생성자 상속 또는 오버로드를 지원하지 않습니다.  
  
 **오류 ID:** BC32040  
  
### 이 오류를 해결하려면  
  
-   모든 생성자 선언에서 `Overloads` 키워드를 제거합니다.
