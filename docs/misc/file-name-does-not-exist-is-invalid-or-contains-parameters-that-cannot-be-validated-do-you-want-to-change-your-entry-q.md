# 파일 이름이 없거나 잘못되었거나 확인할 수 없는 매개 변수를 포함하고 있습니다. 입력한 내용을 변경하시겠습니까?
이 오류는 일반적으로 **프로그램추가** 대화 상자에 잘못된 경로 또는 파일 이름을 입력한 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  **찾아보기**를 선택하여 올바른 파일을 탐색합니다.  
  
     또는  
  
2.  경로 및 파일 이름을 다시 입력합니다.
