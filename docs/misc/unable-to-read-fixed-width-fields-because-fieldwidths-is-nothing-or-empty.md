# FieldWidths가 Nothing이거나 비어 있기 때문에 고정 폭 필드를 읽을 수 없습니다.
`FieldWidths` 속성이 `Nothing`으로 설정되거나 비어 있기 때문에 `TextFieldParser`가 고정 폭 필드를 읽을 수 없습니다.  
  
### 이 오류를 해결하려면  
  
-   `FieldWidths`를 유효한 값으로 설정합니다.
