# 조건부 컴파일 식에는 내장 형식 이름만 사용할 수 있습니다.
조건부 컴파일 식이 내장 형식만 사용할 수 있는데 비내장 형식을 참조합니다.  
  
 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)]의 내장 형식에 대한 자세한 내용은 [Data Types Summary](../Topic/Data%20Types%20Summary%20\(Visual%20Basic\).md)을 참조하세요.  
  
 **오류 ID:** BC31426  
  
### 이 오류를 해결하려면  
  
-   비내장 형식을 내장 형식으로 바꿉니다.
