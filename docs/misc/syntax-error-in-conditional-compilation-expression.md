# 조건부 컴파일 식에 구문 오류가 있습니다.
조건부 컴파일 식에서 일반 구문 오류를 생성합니다.  
  
 **오류 ID:** BC31427  
  
### 이 오류를 해결하려면  
  
1.  조건부 컴파일 및 식에서 사용되는 키워드에 대한 문서를 검사합니다. 설명서의 예제 코드를 이 오류를 생성한 소스 줄과 비교합니다.  
  
2.  오류의 원인을 식별할 수 없으면 해당 상황에 대한 정보를 수집하여 Microsoft 기술 지원 서비스에 알립니다.
