# &#39;&lt;modulename&gt;&#39;은 모듈이며 어셈블리로 참조할 수 없습니다.

 **오류 ID:** BC31076  
  
### 이 오류를 해결하려면  
  
-   어셈블리에 대한 참조를 변경합니다.
