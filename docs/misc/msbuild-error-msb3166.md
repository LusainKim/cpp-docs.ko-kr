# MSBuild 오류 MSB3166
**MSB3166: '\<package\>' 항목에 필요한 '\<file\>' 파일을 찾을 수 없습니다.**  
  
 이 오류는 EULA\(최종 사용자 사용권 계약\) 또는 외부 검사 파일이 없는 경우 발생합니다.  디스크에 파일이 있는지 확인하십시오.  
  
### 이 오류를 해결하려면  
  
-   Visual Studio SDK를 다시 설치합니다.  
  
     \-또는\-  
  
-   필요한 파일을 적절한 위치에 복사합니다.
