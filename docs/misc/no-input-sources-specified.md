# 입력 소스를 지정하지 않았습니다.
컴파일 시 입력 소스를 지정하지 않았습니다.  
  
 **오류 ID:** BC2008  
  
### 이 오류를 해결하려면  
  
-   소스 코드 파일을 포함했는지 확인합니다.
