# Error on line &lt;line&gt;. Expected &#39;expected&#39; but found &#39;found&#39;
프로젝트 파일을 읽을 때 발생하는 일반적인 XML 구문 분석 오류입니다.  이 오류는 사용자 단위 설정 파일\(.vbproj.user\/.csproj.user\)뿐 아니라 프로젝트 파일\(.vbproj\/.csproj\)에도 발생할 수 있습니다.  자세한 정보는 프로젝트 시스템의 `expected` 매개 변수와 `found` 매개 변수에 제공됩니다.  
  
 이 오류는 주로 프로젝트 파일이나 사용자 파일을 직접 편집했을 때 발생합니다.  
  
 **이 오류를 해결하려면**  
  
-   프로젝트 파일을 다시 편집하여 적절한 조건으로 복원합니다.  
  
     오류 심각도에 따라 프로젝트를 열 수 없는 경우도 있습니다.
