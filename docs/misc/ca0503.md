# CA0503
이 속성은 더 이상 사용되지 않습니다.  대체 속성을 사용하십시오.  
  
 CA0503 경고는 더 이상 사용되지 않는 속성에 빌드 또는 프로젝트 구성 파일의 값이 포함된 경우에 생성됩니다.  이 경고를 해결하려면 이 속성을 대체하는 지정된 속성을 사용합니다.
