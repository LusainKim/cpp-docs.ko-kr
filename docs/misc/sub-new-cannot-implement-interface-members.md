# &#39;Sub New&#39;는 인터페이스 멤버를 구현할 수 없습니다.
`Sub New`는 생성자이며 멤버를 구현할 수 없습니다.  
  
 **오류 ID:** BC31042  
  
### 이 오류를 해결하려면  
  
-   `Sub New` 프로시저에서 `Implements` 문을 제거합니다.
