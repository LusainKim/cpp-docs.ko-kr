# MSBuild 오류 MSB3171
**MSB3171: 매니페스트를 생성하는 동안 문제가 발생했습니다. '  \<error\>'**  
  
 이는 참조된 파일이나 어셈블리에 대한 정보를 가져오는 동안 발생하는 예기치 않은 오류입니다.  이 메시지는 응용 프로그램 매니페스트나 배포 매니페스트를 생성하는 동안 표시될 수 있습니다.  
  
### 이 오류를 해결하려면  
  
-   MSBuild 오류의 원인과 가능한 해결 방법에 대해서는 [추가 리소스](../Topic/Additional%20MSBuild%20Resources.md)를 참조하십시오.
