# CA0505
대체 속성이 정의되었으므로 사용되지 않는 속성은 무시됩니다.  
  
 CA505 경고는 지정한 속성이 더 이상 사용되지 않으며 이 속성과 대체 속성 모두 프로젝트 또는 빌드 구성 파일의 값을 포함하는 경우에 생성됩니다.  이 문제를 해결하려면 사용되지 않는 속성을 제거합니다.
