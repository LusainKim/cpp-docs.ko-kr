# 리소스 컴파일러 심각한 오류 RC1206
명령줄에서 지정한 코드 페이지가 레지스트리에 없습니다.  
  
 코드 페이지 지정\(\/c\) 옵션이 위치가 레지스트리에서 지정되지 않은 코드 페이지 뒤에 나왔습니다. 설치를 실행하여 지정한 코드 페이지를 설치합니다.
