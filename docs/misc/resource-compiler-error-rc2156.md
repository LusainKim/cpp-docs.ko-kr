# 리소스 컴파일러 오류 RC2156
문자열 또는 상수 액셀러레이터 키 명령이 필요합니다.  
  
 리소스 컴파일러에서 액셀러레이터 키에 설정되는 키 종류를 확인할 수 없습니다.**ACCELERATORS** 문의 `event` 필드가 잘못되었을 수 있습니다.
