# Clipboard cannot be opened.
이 오류는 항목을 잘라내거나 복사하거나 또는 붙여넣으려고 하지만 클립보드에 액세스할 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
1.  클립보드를 사용 중인 다른 프로그램을 닫고 다시 시도하십시오.
