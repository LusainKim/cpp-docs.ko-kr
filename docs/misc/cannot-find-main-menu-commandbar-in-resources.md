# Cannot find main menu commandbar in resources.
이 오류는 통합 개발 환경에서 시작 시 메뉴 모음 명령을 찾을 수 없는 경우 발생합니다.  .dll이 삭제, 이동 또는 손상되었습니다.  
  
### 이 오류를 해결하려면  
  
-   **프로그램 추가\/제거**를 사용하여 제품을 복구하거나 다시 설치합니다.
