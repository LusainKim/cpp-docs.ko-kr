# 컴파일러 오류 CS1583
'file'은 올바른 Win32 리소스 파일이 아닙니다.  
  
 이 오류는 [\/win32res](../Topic/-win32res%20\(C%23%20Compiler%20Options\).md) 컴파일러 옵션과 함께 유효하지 않거나 잘못된 형식의 Win32 리소스 파일의 이름을 지정하는 경우 발생합니다. Visual Studio에서는 프로젝트 디자이너의 응용 프로그램 창에서 파일 이름을 지정합니다.
