# 시스템 이벤트 로그를 삭제할 수 없습니다.
삭제할 수 없는 시스템 이벤트 로그를 삭제하려고 했습니다. 시스템 로그는 시스템 시작 및 하드웨어 오류와 같은 시스템 이벤트를 추적합니다.  
  
### 이 오류를 해결하려면  
  
-   응용 프로그램이 시스템 이벤트 로그 대신 응용 프로그램 또는 사용자 지정 로그에 작성하게 하는 것이 좋습니다.  
  
-   시스템 이벤트 로그를 삭제하려고 하지 마세요.
