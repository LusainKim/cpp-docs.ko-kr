# &#39;&lt;typename&gt;&#39;은 지원되지 않는 형식입니다.
지정된 형식을 [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)]로 나타낼 수 없습니다.  
  
 **오류 ID:** BC30649  
  
### 이 오류를 해결하려면  
  
-   지원되는 형식을 사용합니다.  
  
-   새 형식을 개발합니다.
