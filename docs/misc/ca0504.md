# CA0504
규칙 디렉터리가 없으므로 무시되었습니다.  
  
 CA504 경고는 프로젝트 또는 빌드 구성 파일의 **CodeAnalysisRuleDirectories** 속성에 지정된 디렉터리를 찾을 수 없는 경우에 생성됩니다.  이 문제를 해결하려면 유효한 디렉터리 경로를 입력합니다.
