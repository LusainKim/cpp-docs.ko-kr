# MSBuild 오류 MSB3189
**MSB3189: 이 응용 프로그램의 업데이트 위치가 로컬 경로입니다.**  
  
 이 경고는 배포 매니페스트에만 해당합니다.  
  
### 이 오류를 해결하려면  
  
-   업데이트 위치가 유효한 URL인지 확인합니다.
