# 프로젝트 로드 오류 대화 상자
프로젝트 파일을 로드할 수 없는 경우 이 대화 상자가 나타납니다.  
  
 **지정한 오류를 수정한 후 프로젝트를 다시 로드하십시오.**  
 오류와 관련된 정보를 제공합니다.  이 대화 상자의 정보를 사용하여 프로젝트 파일에서 오류를 찾아 해결합니다.  프로젝트를 저장한 다음 IDE에서 다시 로드합니다.  
  
 **이 세션 동안 이 대화 상자를 다시 표시 안 함**  
 이 옵션을 선택하면 **프로젝트 로드 오류** 대화 상자는 다음에 IDE가 열릴 때까지 더 이상 표시되지 않습니다.
