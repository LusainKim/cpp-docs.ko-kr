# 개체가 이 작업을 지원하지 않습니다(Visual Basic).
이 개체에서 지원되지 않는 메서드 또는 속성을 참조했습니다.  
  
### 이 오류를 해결하려면  
  
1.  개체에 대한 자세한 내용은 개체 설명서를 참조하세요.  
  
2.  속성 및 메서드 이름의 철자가 정확한지 확인합니다.
