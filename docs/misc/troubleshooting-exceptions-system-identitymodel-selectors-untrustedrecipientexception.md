# 예외 문제 해결: System.IdentityModel.Selectors.UntrustedRecipientException
사용자가 토큰을 요청하는 엔터티의 정보를 검토하여 해당 엔터티를 신뢰하지 않기로 결정한 경우 <xref:System.IdentityModel.Selectors.UntrustedRecipientException> 예외가 throw됩니다. 사용자가 CardSpace 사용자 인터페이스에서 **Windows CardSpace 사용 안 함**을 클릭합니다.
