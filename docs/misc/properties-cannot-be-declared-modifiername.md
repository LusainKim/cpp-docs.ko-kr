# 속성은 &#39;&lt;modifiername&gt;&#39;으로 선언할 수 없습니다.
`Property`에 사용된 구문에 오류가 있습니다. 이 오류는 일반적으로 `WithEvents`와 같은 지정된 키워드를 `Property` 선언에서 사용할 수 없음을 나타냅니다.  
  
 **오류 ID:** BC30639  
  
### 이 오류를 해결하려면  
  
-   `Property` 선언의 구문을 확인합니다.
