# 리소스 컴파일러 오류 RC2118
예상된 글꼴 이름  
  
 **DIALOG** 문의 FONT 옵션에 있는 *typeface* 필드는 큰따옴표로 묶인 ASCII 문자열이어야 합니다. 이 필드는 글꼴의 이름을 지정합니다.
