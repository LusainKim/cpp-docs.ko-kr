# Imports 별칭에는 형식 문자를 사용할 수 없습니다.
`Imports` 문의 가져오기 별칭에 하나 이상의 식별자 형식 문자가 포함되어 있습니다.  
  
 가져오기 별칭은 유효한 Visual Basic 이름이어야 합니다. 허용되는 문자에 식별자 형식 문자\(`%`, `&`, `@`, `!`, `#` 및 `$`\)는 포함되지 않습니다.[Declared Element Names](../Topic/Declared%20Element%20Names%20\(Visual%20Basic\).md)을 참조하세요.  
  
 **오류 ID:** BC31398  
  
### 이 오류를 해결하려면  
  
-   가져오기 별칭에서 식별자 형식 문자를 제거합니다.
