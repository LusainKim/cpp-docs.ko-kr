# 예외 문제 해결: System.ComponentModel.Win32Exception
Win32 오류 코드에 대해 throw되는 예외입니다.  
  
## 설명  
 Win32 오류 코드는 숫자 값 대신 시스템 메시지로 변환되어 화면에 표시됩니다.<xref:System.ComponentModel.Win32Exception.NativeErrorCode%2A> 속성을 사용하면 오류 코드의 숫자 값에 액세스할 수 있습니다.
