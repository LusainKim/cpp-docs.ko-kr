# ‘End Property’는 짝이 되는 ‘Property’ 뒤에 와야 합니다.
`End Property` 문이 앞에서 일치하는 `Property` 선언 없이 코드에 나타납니다.  
  
 **오류 ID:** BC30431  
  
### 이 오류를 해결하려면  
  
-   `End Property` 문이 중복되는 경우 제거합니다.  
  
-   `Property` 프로시저가 누락된 경우 제공합니다.  
  
-   `End Property`를 코드의 적절한 위치로 이동합니다.
