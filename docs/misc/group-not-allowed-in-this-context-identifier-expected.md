# 이 컨텍스트에서는 &#39;Group&#39;을 사용할 수 없습니다. 식별자가 필요합니다.
`Group` 키워드가 `Aggregate` 절의 `Into` 섹션에 포함되어 있습니다.`Group` 키워드는 `Group By` 또는 `Group Join` 절에서만 유효합니다.  
  
 **오류 ID:** BC36618  
  
### 이 오류를 해결하려면  
  
-   `Aggregate` 절에서 `Group` 키워드를 제거합니다.`Group By` 절을 그룹 결과에 대한 쿼리에 추가할 수 있습니다.
