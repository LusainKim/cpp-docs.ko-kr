# &#39;&lt;property1&gt;&#39;과 &#39;&lt;property2&gt;&#39; 중 하나만 &#39;Default&#39;로 선언되었으므로 서로 오버로드할 수 없습니다.
속성이 `Default`를 지정하면 해당 이름에 오버로드된 모든 속성이 `Default`를 지정해야 합니다.  
  
 **오류 ID:** BC30361  
  
### 이 오류를 해결하려면  
  
-   오버로드된 모든 속성이 `Default`로 선언되었는지 확인합니다.
