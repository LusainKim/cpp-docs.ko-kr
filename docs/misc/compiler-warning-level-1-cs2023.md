# 컴파일러 경고(수준 1) CS2023
지시 파일에 지정되었기 때문에 \/noconfig 옵션을 무시합니다.  
  
 [\/noconfig](../Topic/-noconfig%20\(C%23%20Compiler%20Options\).md) 컴파일러 옵션이 지시 파일에서 지정되었으며 이는 허용되지 않습니다.  
  
 이 경고는 [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md) 옵션으로 표시하지 않을 수 없습니다.
