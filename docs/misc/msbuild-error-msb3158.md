# MSBuild 오류 MSB3158
**MSB3158: '\<culture\>' culture의 리소스를 찾을 수 없습니다.  '\<culture\>' culture를 대신 사용합니다.**  
  
 이 경고는 지정된 culture를 사용하는 패키지 매니페스트 파일\(package.xml\)을 지정된 제품이 찾을 수 없는 경우 발생합니다.  
  
### 이 오류를 해결하려면  
  
-   적절한 패키지 매니페스트 파일을 제공합니다.
