# _FREEENTRY, _USEDENTRY
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 구문  
  
```  
#include <malloc.h>  
```  
  
## 설명  
 이러한 상수는 **\_HEAPINFO** 구조체의 **\_useflag** 요소에 대한 `_heapwalk` 루틴에서 지정한 값을 나타냅니다.  힙 항목의 상태를 나타냅니다.
