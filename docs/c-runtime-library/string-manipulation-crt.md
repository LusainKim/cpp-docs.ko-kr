# 문자열 조작(CRT)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

이러한 루틴은 null로 끝나는 싱글 바이트 문자, 와이드 문자 및 멀티 바이트 문자 문자열에서 작동합니다.  [버퍼 조작](../c-runtime-library/buffer-manipulation.md)에서 설명된 버퍼 조작 루틴을 사용하여, 문자 배열이 null 문자로 끝나지 않는 작업을 할 수 있습니다.
