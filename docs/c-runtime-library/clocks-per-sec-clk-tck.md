# <a name="clockspersec-clktck"></a>CLOCKS_PER_SEC, CLK_TCK
## <a name="syntax"></a>구문  
  
```  
  
#include <time.h>  
```  
  
## <a name="remarks"></a>설명  
 초 단위 시간은 `clock` 함수에 의해 반환된 값을 `CLOCKS_PER_SEC`로 나눈 값입니다. `CLK_TCK`는 동일하지만 사용되지 않습니다.
