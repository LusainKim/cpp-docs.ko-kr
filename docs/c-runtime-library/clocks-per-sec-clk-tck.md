# CLOCKS_PER_SEC, CLK_TCK
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 구문  
  
```  
  
#include <time.h>  
```  
  
## 설명  
 초에 대한 시간은 `CLOCKS_PER_SEC` 의해 나눠진, `clock` 로 반환되는 값입니다.  `CLK_TCK` 은 동일하지만, 사용되지 않습니다.
