# <a name="putenv"></a>putenv
이 POSIX 함수는 사용되지 않습니다. 대신 ISO C++와 호환되는 [_putenv](../../c-runtime-library/reference/putenv-wputenv.md) 또는 보안이 강화된 [_putenv_s](../../c-runtime-library/reference/putenv-s-wputenv-s.md)를 사용하세요.  
  
> [!IMPORTANT]
>  이 API는 Windows 런타임에서 실행되는 응용 프로그램에서 사용할 수 없습니다. 자세한 내용은 [/ZW에서 지원하지 않는 CRT 함수](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx)를 참조하세요.
