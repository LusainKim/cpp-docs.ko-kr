# ungetch
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 POSIX 함수는 사용되지 않습니다.  ISO c \+ \+ 규칙은 [\_ungetch](../../c-runtime-library/reference/ungetch-ungetwch-ungetch-nolock-ungetwch-nolock.md) 대신에 사용합니다.  
  
> [!IMPORTANT]
>  이 API는 Windows 런타임에서 실행되는 응용 프로그램에서 사용할 수 없습니다.  자세한 내용은 [\/ZW에서 지원하지 않는 CRT 함수](http://msdn.microsoft.com/library/windows/apps/jj606124.aspx)를 참조하십시오.
