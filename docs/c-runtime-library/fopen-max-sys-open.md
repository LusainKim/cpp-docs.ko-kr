# FOPEN_MAX, _SYS_OPEN
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 구문  
  
```  
  
#include <stdio.h>  
  
```  
  
## 설명  
 동시에 열 수 있는 파일의 최대 수입니다.  `FOPEN_MAX` 는 ANSI와 호환가능한 이름입니다.  `_SYS_OPEN` 는 기존 코드와의 호환성을 제공합니다.
