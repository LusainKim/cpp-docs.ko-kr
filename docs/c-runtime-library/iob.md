# <a name="iob"></a>_iob
stdio 컨트롤 구조체의 배열입니다.  
  
## <a name="syntax"></a>구문  
  
```  
FILE _iob[_IOB_ENTRIES];  
```  
  
## <a name="remarks"></a>설명  
 `IOB_ENTRIES`는 stdio.h에 20으로 정의됩니다.
