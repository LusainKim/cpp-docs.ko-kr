# TMP_MAX, L_tmpnam
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 구문  
  
```  
#include <stdio.h>  
```  
  
## 설명  
 `TMP_MAX` `tmpnam` 함수가 생성할 수 있는 고유한 파일 이름의 최대 수입니다.  `L_tmpnam` 는 `tmpnam` 에 의해 생성된 임시 파일이름의 길이입니다.
