# RAND_MAX
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

## 구문  
  
```  
  
#include <stdlib.h>  
  
```  
  
## 설명  
 상수 `RAND_MAX` 는 `rand` 함수로부터 반환될 수 있는 최대값입니다.  `RAND_MAX` 는 0x7fff 값으로 정의됩니다.
