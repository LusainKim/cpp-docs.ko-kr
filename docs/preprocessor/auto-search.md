# auto_search
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**C\+\+ 전용**  
  
 형식 라이브러리를 \#import로 참조하고 형식 라이브러리 자체가 다른 형식 라이브러리를 참조할 때 컴파일러가 다른 형식 라이브러리에 대해 암시적 \#import를 수행할 수 있도록 지정합니다.  
  
## 구문  
  
```

```
