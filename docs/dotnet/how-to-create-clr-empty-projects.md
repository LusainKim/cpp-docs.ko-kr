# 방법: CLR 빈 프로젝트 만들기
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

To create a CLR empty project, use the **CLR Empty Project** template, which is available from the **New Project** dialog box.  
  
> [!NOTE]
>  실제 설정이나 버전에 따라서 IDE에 나타나는 기능의 모양이 도움말의 설명과 다를 수 있습니다.  설정을 변경하려면 **도구** 메뉴에서 **설정 가져오기 및 내보내기**를 선택합니다.  자세한 내용은 [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ko-kr/22c4debb-4e31-47a8-8f19-16f328d7dcd3)을 참조하십시오.  
  
### CLR 빈 프로젝트를 만들려면  
  
1.  **파일** 메뉴에서 **새로 만들기**를 클릭한 다음 **프로젝트**를 클릭합니다.  
  
     **새 프로젝트** 대화 상자가 나타납니다.  
  
2.  **설치된 템플릿**에서 **Visual C\+\+** 노드를 클릭한 다음 **CLR** 노드를 클릭합니다.  **CLR 빈 프로젝트** 아이콘을 선택합니다.  
  
3.  **이름** 상자에 응용 프로그램의 고유 이름을 입력합니다.  
  
    > [!NOTE]
    >  **새 프로젝트** 대화 상자에서 다른 프로젝트 및 솔루션 설정을 지정할 수도 있지만 이러한 설정은 필요하지 않습니다.  
  
4.  **확인**을 클릭합니다.
