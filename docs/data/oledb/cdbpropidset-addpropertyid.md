# CDBPropIDSet::AddPropertyID
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Adds a property ID to the property ID set.  
  
## 구문  
  
```  
  
      bool AddPropertyID(   
   DBPROPID propid    
) throw( );  
```  
  
#### 매개 변수  
 `propid`  
 \[in\] The property ID to be added to the property ID set.
