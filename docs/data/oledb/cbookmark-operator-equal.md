# <a name="cbookmarkoperator-"></a>CBookmark::operator =

## <a name="syntax"></a>구문  
  
```  
  
      CBookmark& operator =(   
   const CBookmark& bookmark    
) throw( );  
```  
  
## <a name="remarks"></a>설명  
 이 연산자는에 필요 **CBookmark\<0 >**합니다.
