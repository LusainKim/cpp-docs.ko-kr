# COLUMN_NAME_LENGTH
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Represents a binding on the rowset to the specific column in the rowset.  Similar to [COLUMN\_NAME](../../data/oledb/column-name.md), except that this macro also takes column length.  
  
## 구문  
  
```  
  
COLUMN_NAME_LENGTH(  
pszName  
,   
data  
,   
length  
 )  
  
```  
  
#### 매개 변수  
 `pszName`  
 \[in\] A pointer to the column name.  The name must be a Unicode string.  You can accomplish this by putting an 'L' in front of the name, for example: `L"MyColumn"`.  
  
 `data`  
 \[in\] The corresponding data member in the user record.  
  
 *length*  
 \[in\] The variable to be bound to the column length.  
  
## 설명  
 See [COLUMN\_NAME](../../data/oledb/column-name.md) for information on where the **COLUMN\_NAME\_\*** macros are used.
