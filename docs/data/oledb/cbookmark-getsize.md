# <a name="cbookmarkgetsize"></a>CBookmark::GetSize
책갈피 버퍼의 크기를 검색합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
virtual DBLENGTH GetSize( ) const throw( );  
  
```  
  
## <a name="return-value"></a>반환 값  
 버퍼의 크기(바이트)입니다.  
  
## <a name="requirements"></a>요구 사항  
 **헤더:** atldbcli.h
