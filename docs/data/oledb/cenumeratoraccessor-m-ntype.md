# CEnumeratorAccessor::m_nType
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

A variable indicating whether the row describes a data source or an enumerator.  
  
## 구문  
  
```  
  
USHORT m_nType;  
  
```
