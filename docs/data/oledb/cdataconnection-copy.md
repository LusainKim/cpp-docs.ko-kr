# <a name="cdataconnectioncopy"></a>CDataConnection::Copy
기존 데이터 연결의 복사본을 만듭니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      CDataConnection& Copy(   
   const CDataConnection & ds    
) throw( );  
```  
  
#### <a name="parameters"></a>매개 변수  
 `ds`  
 [in] 복사할 기존 데이터 연결에 대 한 참조입니다.
