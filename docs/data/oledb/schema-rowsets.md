# <a name="schema-rowsets"></a>스키마 행 집합
스키마 (또는 메타 데이터)은 데이터베이스의 구조 나 공급자, 행 집합, 테이블, 열에 대 한 정보 등 조직에 대 한 정보 또는 실제 데이터베이스 내용 외의 기타 정보입니다. OLE DB 스키마 행 집합을 사용 하 여 행 집합을 열지 않고도 이러한 정보를 가져올 수 있습니다.  
  
 스키마 행 집합을 사용 하는 방법에 대 한 정보를 참조 하십시오. [스키마 행 집합 메타 데이터 구하기](../../data/oledb/obtaining-metadata-with-schema-rowsets.md)합니다.
