# 데이터베이스 클래스를 사용하는 시기는 언제입니까?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

대부분의 응용 프로그램에서는 데이터베이스에 데이터 저장소가 있어야 하고, 많은 다른 응용 프로그램은 데이터베이스를 사용함으로써 이점을 얻을 수 있습니다.  데이터베이스를 사용하면 대부분의 경우 여러 사용자와 여러 응용 프로그램이 액세스할 수 있는 융통성 있는 데이터 리포지토리를 사용할 수 있습니다.  데이터베이스를 사용하면 대용량 데이터를 저장할 수 있고 쿼리와 업데이트를 수행할 때 고속 데이터 액세스가 가능합니다.
