# 데이터 소스: 데이터 소스의 스키마 확인(ODBC)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 항목은 MFC ODBC 클래스에 적용됩니다.  
  
 `CRecordset` 개체에서 데이터 멤버를 설정하려면 연결할 데이터 소스의 스키마를 알아야 합니다.  데이터 소스의 스키마를 확인하려면 데이터 소스의 테이블 목록, 각 테이블의 열 목록, 각 열의 데이터 형식, 인덱스의 존재 여부 등에 대한 정보를 가져와야 합니다.
