# DAO나 ODBC를 직접 호출할 수 있습니까?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

일반적으로 MFC에서는 보다 정교한 제어가 필요할 경우 클래스를 통해 DAO나 ODBC에 액세스할 뿐만 아니라 직접 호출할 수도 있습니다.  MFC는 Windows용 프로그래밍을 단순화하려고 하지만 사용자가 내부 API에 액세스하는 경우에는 불가능합니다.
