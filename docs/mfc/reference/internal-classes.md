# 내부 클래스
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

The following classes are used internally in MFC.  완벽을 기하기 위해 이 단원에서는 이러한 내부 클래스에 대해 설명하지만 이러한 내부 클래스는 코드에서 직접 사용할 수 없습니다.
