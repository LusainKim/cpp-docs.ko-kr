# DHTML 이벤트 맵
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

The following macros can be used to handle DHTML events.  
  
## DHTML Event Map Macros  
 The following macros can be used to handle DHTML events in [CDHtmlDialog](../../mfc/reference/cdhtmldialog-class.md)\-derived classes.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_EVENT\_MAP](../Topic/BEGIN_DHTML_EVENT_MAP.md)|Marks the start of the DHTML event map.|  
|[BEGIN\_DHTML\_EVENT\_MAP\_INLINE](../Topic/BEGIN_DHTML_EVENT_MAP_INLINE.md)|Marks the start of the DHTML event map.|  
|[DECLARE\_DHTML\_EVENT\_MAP](../Topic/DECLARE_DHTML_EVENT_MAP.md)|Declares the DHTML event map.|  
|[DHTML\_EVENT](../Topic/DHTML_EVENT.md)|Used to handle an event at the document level for a single HTML element.|  
|[DHTML\_EVENT\_AXCONTROL](../Topic/DHTML_EVENT_AXCONTROL.md)|Used to handle an event fired by an ActiveX control.|  
|[DHTML\_EVENT\_CLASS](../Topic/DHTML_EVENT_CLASS.md)|Used to handle an event at the document level for all HTML elements with a particular CSS class.|  
|[DHTML\_EVENT\_ELEMENT](../Topic/DHTML_EVENT_ELEMENT.md)|Used to handle an event at the element level.|  
|[DHTML\_EVENT\_ONAFTERUPDATE](../Topic/DHTML_EVENT_ONAFTERUPDATE.md)|Used to handle the **onafterupdate** event from an HTML element.|  
|[DHTML\_EVENT\_ONBEFOREUPDATE](../Topic/DHTML_EVENT_ONBEFOREUPDATE.md)|Used to handle the **onbeforeupdate** event from an HTML element.|  
|[DHTML\_EVENT\_ONBLUR](../Topic/DHTML_EVENT_ONBLUR.md)|Used to handle the **onblur** event from an HTML element.|  
|[DHTML\_EVENT\_ONCHANGE](../Topic/DHTML_EVENT_ONCHANGE.md)|Used to handle the `onchange` event from an HTML element.|  
|[DHTML\_EVENT\_ONCLICK](../Topic/DHTML_EVENT_ONCLICK.md)|Used to handle the **onclick** event from an HTML element.|  
|[DHTML\_EVENT\_ONDATAAVAILABLE](../Topic/DHTML_EVENT_ONDATAAVAILABLE.md)|Used to handle the **ondataavailable** event from an HTML element.|  
|[DHTML\_EVENT\_ONDATASETCHANGED](../Topic/DHTML_EVENT_ONDATASETCHANGED.md)|Used to handle the **ondatasetchanged** event from an HTML element.|  
|[DHTML\_EVENT\_ONDATASETCOMPLETE](../Topic/DHTML_EVENT_ONDATASETCOMPLETE.md)|Used to handle the **ondatasetcomplete** event from an HTML element.|  
|[DHTML\_EVENT\_ONDBLCLICK](../Topic/DHTML_EVENT_ONDBLCLICK.md)|Used to handle the **ondblclick** event from an HTML element.|  
|[DHTML\_EVENT\_ONDRAGSTART](../Topic/DHTML_EVENT_ONDRAGSTART.md)|Used to handle the **ondragstart** event from an HTML element.|  
|[DHTML\_EVENT\_ONERRORUPDATE](../Topic/DHTML_EVENT_ONERRORUPDATE.md)|Used to handle the **onerrorupdate** event from an HTML element.|  
|[DHTML\_EVENT\_ONFILTERCHANGE](../Topic/DHTML_EVENT_ONFILTERCHANGE.md)|Used to handle the **onfilterchange** event from an HTML element.|  
|[DHTML\_EVENT\_ONFOCUS](../Topic/DHTML_EVENT_ONFOCUS.md)|Used to handle the **onfocus** event from an HTML element.|  
|[DHTML\_EVENT\_ONHELP](../Topic/DHTML_EVENT_ONHELP.md)|Used to handle the `onhelp` event from an HTML element.|  
|[DHTML\_EVENT\_ONKEYDOWN](../Topic/DHTML_EVENT_ONKEYDOWN.md)|Used to handle the **onkeydown** event from an HTML element.|  
|[DHTML\_EVENT\_ONKEYPRESS](../Topic/DHTML_EVENT_ONKEYPRESS.md)|Used to handle the **onkeypress** event from an HTML element.|  
|[DHTML\_EVENT\_ONKEYUP](../Topic/DHTML_EVENT_ONKEYUP.md)|Used to handle the **onkeyup** event from an HTML element.|  
|[DHTML\_EVENT\_ONMOUSEDOWN](../Topic/DHTML_EVENT_ONMOUSEDOWN.md)|Used to handle the **onmousedown** event from an HTML element.|  
|[DHTML\_EVENT\_ONMOUSEMOVE](../Topic/DHTML_EVENT_ONMOUSEMOVE.md)|Used to handle the `onmousemove` event from an HTML element.|  
|[DHTML\_EVENT\_ONMOUSEOUT](../Topic/DHTML_EVENT_ONMOUSEOUT.md)|Used to handle the **onmouseout** event from an HTML element.|  
|[DHTML\_EVENT\_ONMOUSEOVER](../Topic/DHTML_EVENT_ONMOUSEOVER.md)|Used to handle the **onmouseover** event from an HTML element.|  
|[DHTML\_EVENT\_ONMOUSEUP](../Topic/DHTML_EVENT_ONMOUSEUP.md)|Used to handle the **onmouseup** event from an HTML element.|  
|[DHTML\_EVENT\_ONRESIZE](../Topic/DHTML_EVENT_ONRESIZE.md)|Used to handle the **onresize** event from an HTML element.|  
|[DHTML\_EVENT\_ONROWENTER](../Topic/DHTML_EVENT_ONROWENTER.md)|Used to handle the **onrowenter** event from an HTML element.|  
|[DHTML\_EVENT\_ONROWEXIT](../Topic/DHTML_EVENT_ONROWEXIT.md)|Used to handle the **onrowexit** event from an HTML element.|

|[DHTML\_EVENT\_TAG](../Topic/DHTML_EVENT_TAG.md)|Used to handle an event at the document level for all elements with a particular HTML tag.|

## URL Event Map Macros  
 The following macros can be used to handle DHTML events in [CMultiPageDHtmlDialog](../../mfc/reference/cmultipagedhtmldialog-class.md)\-derived classes.  
  
|||  
|-|-|  
|[BEGIN\_DHTML\_URL\_EVENT\_MAP](../Topic/BEGIN_DHTML_URL_EVENT_MAP.md)|Marks the start of the multipage DHTML and URL event map.|  
|[BEGIN\_EMBED\_DHTML\_EVENT\_MAP](../Topic/BEGIN_EMBED_DHTML_EVENT_MAP.md)|Marks the start of an embedded DHTML event map.|  
|[BEGIN\_URL\_ENTRIES](../Topic/BEGIN_URL_ENTRIES.md)|Marks the start of a URL event entry map.|  
|[DECLARE\_DHTML\_URL\_EVENT\_MAP](../Topic/DECLARE_DHTML_URL_EVENT_MAP.md)|Declares the multipage DHTML and URL event map.|  
|[END\_DHTML\_URL\_EVENT\_MAP](../Topic/END_DHTML_URL_EVENT_MAP.md)|Marks the end of the multipage DHTML and URL event map.|  
|[END\_EMBED\_DHTML\_EVENT\_MAP](../Topic/END_EMBED_DHTML_EVENT_MAP.md)|Marks the end of an embedded DHTML event map.|  
|[END\_URL\_ENTRIES](../Topic/END_URL_ENTRIES.md)|Marks the end of a URL event entry map.|  
|[URL\_EVENT\_ENTRY](../Topic/URL_EVENT_ENTRY.md)|Maps a URL or HTML resource to a page in a multipage dialog.|
