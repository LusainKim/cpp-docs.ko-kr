# 인터넷 URL 전역 구문 분석
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

When a client sends a query to the Internet server, you can use one of the URL parsing globals to extract information about the client.  
  
### Internet URL Parsing Globals  
  
|||  
|-|-|  
|[AfxParseURL](../Topic/AfxParseURL.md)|Parses a URL string and returns the type of service and its components.|  
|[AfxParseURLEx](../Topic/AfxParseURLEx.md)|Parses a URL string and returns the type of service and its components, as well as providing the user name and password.|
