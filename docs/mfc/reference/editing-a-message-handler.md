# <a name="editing-a-message-handler"></a>메시지 처리기 편집
메시지 처리기를 정의 하면 코드를 추가 하거나 수정할 멤버 함수의 정의로 이동할 수 있습니다.  
  
 멤버 함수 정의로 이동 하 여 [대화 상자 편집기](../../windows/dialog-editor.md), 처리기가 이미 정의 된 컨트롤을 두 번 클릭 합니다. 이 정의 되어 있는 선택한 컨트롤의 메시지 처리기 파일을 이동 합니다.  
  
 멤버 함수 정의로 이동 하려면 [클래스 뷰](http://msdn.microsoft.com/en-us/8d7430a9-3e33-454c-a9e1-a85e3d2db925), 클래스 뷰에서 함수 이름을 두 번 클릭 합니다.
