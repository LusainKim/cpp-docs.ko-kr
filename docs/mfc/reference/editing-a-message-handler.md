# 메시지 처리기 편집
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

메시지 처리기를 정의했으면 멤버 함수 정의로 이동하여 코드를 추가하거나 수정할 수 있습니다.  
  
 [대화 상자 편집기](../../mfc/dialog-editor.md)에서 멤버 함수 정의로 이동하려면 이미 처리기가 정의된 컨트롤을 두 번 클릭합니다.  그러면 선택된 컨트롤의 메시지 처리기가 정의되어 있는 파일로 이동됩니다.  
  
 [클래스 뷰](http://msdn.microsoft.com/ko-kr/8d7430a9-3e33-454c-a9e1-a85e3d2db925)에서 멤버 함수 정의로 이동하려면 클래스 뷰에서 함수 이름을 두 번 클릭합니다.
