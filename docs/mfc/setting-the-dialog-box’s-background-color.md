# 대화 상자의 배경색 설정
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

대화 상자 창에 대해 `WM_CTLCOLOR` 메세지를 처리하여 대화 상자의 배경 색을 변경할 수 있습니다.  설정한 색은 지정된 대화 상자에만 적용 됩니다.
