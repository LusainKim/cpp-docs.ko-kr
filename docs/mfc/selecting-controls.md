# Selecting Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

크기 조정, 맞춤, 이동, 복사, 삭제 등의 작업을 수행할 컨트롤을 선택한 다음, 원하는 작업을 수행합니다.  대부분의 경우, 둘 이상의 컨트롤을 선택해야 [대화 상자 편집기 도구 모음](../mfc/showing-or-hiding-the-dialog-editor-toolbar.md)에 있는 크기 조정 및 맞춤 도구를 사용할 수 있습니다.  
  
 컨트롤을 선택하면 주변에 회색 테두리가 생기고 이 선택 테두리에는 작은 사각형 모양의 실선\(활성\) 또는 비어 있는\(비활성\) "크기 조정 핸들"이 표시됩니다.  여러 개의 컨트롤을 선택하면 기준 컨트롤에는 실선 크기 조정 핸들이 생기고 선택된 나머지 컨트롤에는 모두 비어 있는 크기 조정 핸들이 생깁니다.  
  
 여러 개의 컨트롤을 크기 조정하거나 맞출 때 대화 상자 편집기에서는 "기준 컨트롤"을 사용하여 다른 컨트롤의 크기와 맞춤 방법을 결정합니다.  기본적으로 처음 선택한 컨트롤이 기준 컨트롤이 됩니다.  
  
-   [여러 개의 컨트롤 선택](../mfc/selecting-multiple-controls.md)  
  
-   [기준 컨트롤 지정](../mfc/specifying-the-dominant-control.md)  
  
 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하십시오. 관리되는 프로젝트에 리소스 파일 추가, 리소스 액세스, 정적 리소스 표시, 속성에 리소스 문자열 할당 등의 작업을 수동으로 수행하는 방법에 대한 자세한 내용은 [연습: Windows Forms 지역화](http://msdn.microsoft.com/ko-kr/9a96220d-a19b-4de0-9f48-01e5d82679e5) 및 [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)을 참조하십시오.
