# Sizing Individual Controls
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

크기 조정 핸들을 사용하여 컨트롤 크기를 조정할 수 있습니다.  크기 조정 핸들에 포인터를 두면, 포인터의 모양은 컨트롤 크기를 조정할 수 있는 방향을 가리키도록 바뀝니다.  활성 크기 조정 핸들은 실선입니다. 크기 조정 핸들이 비어 있는 선이면 이 선을 기준으로 하여 컨트롤 크기를 조정할 수 없습니다.  
  
 안내선이나 여백에 컨트롤을 맞추거나 맞춘 컨트롤과 안내선을 멀리 이동하여 컨트롤 크기를 조정할 수도 있습니다.  
  
### 컨트롤 크기를 조정하려면  
  
1.  컨트롤을 선택합니다.  
  
2.  크기 조정 핸들을 끌어 컨트롤의 크기를 변경합니다.  
  
    -   위와 옆에 있는 크기 조정 핸들은 가로나 세로 크기를 변경합니다.  
  
    -   모퉁이에 있는 크기 조정 핸들은 가로와 세로의 크기를 모두 변경합니다.  
  
    > [!TIP]
    >  Shift 키를 누른 채로 오른쪽과 아래쪽 화살표 키를 사용하여 한 번에 하나의 DLU\(대화 상자 단위\)씩 컨트롤 크기를 조정할 수 있습니다.  
  
### 컨트롤 크기를 텍스트에 맞게 자동으로 조정하려면  
  
1.  **서식** 메뉴에서 **크기를 내용에 맞춤**을 선택합니다.  
  
 \-또는\-  
  
-   마우스 오른쪽 단추로 컨트롤을 클릭하고 바로 가기 메뉴에서 **크기를 내용에 맞춤**을 선택합니다.  
  
 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하십시오. 관리되는 프로젝트에 리소스 파일 추가, 리소스 액세스, 정적 리소스 표시, 속성에 리소스 문자열 할당 등의 작업을 수동으로 수행하는 방법에 대한 자세한 내용은 [연습: Windows Forms 지역화](http://msdn.microsoft.com/ko-kr/9a96220d-a19b-4de0-9f48-01e5d82679e5) 및 [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)을 참조하십시오.
