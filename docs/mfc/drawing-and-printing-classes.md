# 클래스 그리기 및 인쇄
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Windows에서 모든 그래픽은 출력 장치 컨텍스트 \(DC\) 라는 가상 그리기 영역에 그려집니다.  MFC는 비트 맵, 브러시, 팔레트, 그리고 펜으로 윈도우 그리기 도구뿐만 아니라 다양한 DC 형식을 캡슐화하는 클래스를 제공합니다.
