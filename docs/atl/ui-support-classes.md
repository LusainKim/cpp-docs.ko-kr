# UI Support Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

일반 UI 지원 다음 클래스를 제공합니다.  
  
-   [IDocHostUIHandlerDispatch](../atl/reference/idochostuihandlerdispatch-interface.md) Microsoft HTML 구문 분석 및 렌더링 엔진 인터페이스.  
  
-   [IOleObjectImpl](../atl/reference/ioleobjectimpl-class.md) 주 메서드를 통해 컨테이너와 통신 컨트롤을 제공 합니다.  활성화 및 비활성화의 현재 위치에서 컨트롤을 관리합니다.  
  
-   [IOleInPlaceObjectWindowlessImpl](../atl/reference/ioleinplaceobjectwindowlessimpl-class.md) 정품의 현재 위치에서 컨트롤을 관리 합니다.  창 없는 컨트롤을 메시지를으로 끌어서 놓기 작업에 참여할 수 있습니다.  
  
-   [IOleInPlaceActiveObjectImpl](../atl/reference/ioleinplaceactiveobjectimpl-class.md) 전체 컨트롤과 컨테이너 사이의 통신을 지원 합니다.  
  
-   [IViewObjectExImpl](../atl/reference/iviewobjecteximpl-class.md) 컨트롤 자체를 직접 표시 하 고 컨테이너 표시에서 변경 내용을 알릴 수 있습니다.  깜빡임 없는 드로잉, 투명 한 사각형이 아닌 컨트롤 및 적중 테스트를 지원합니다.
