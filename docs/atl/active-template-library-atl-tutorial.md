# ATL(액티브 템플릿 라이브러리) 자습서
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

ATL 효율적인 융통성 있고 간단한 컨트롤을 만드는 과정을 단순화 하도록 설계 되었습니다.  이 자습서에서는 여러 ATL 및 COM 기본 개념을 보여 주는 ActiveX 컨트롤을 만드는 과정을 안내 합니다.  
  
 이 자습서를 수행 하면 원과 채워진된 다각형을 그리는 ATL 프로젝트에 컨트롤을 추가 하는 방법에 설명 합니다.  다음 다각형 수 있고 속성이 변경 될 때 컨트롤을 업데이트에 대 한 그리기 코드를 만드는 변의 수를 나타내는 속성을 추가 합니다.  다음 몇 가지 Vbscript를 사용 하 여 이벤트에 응답 하는 웹 페이지에 컨트롤이 표시 됩니다.  
  
 자습서는 7 단계로 구분 됩니다.  이후 단계 이전에 완료 된 작업에 따라 순서로 각 단계를 수행 해야 합니다.  시작 하기 전에 특정 컴퓨터에서 ActiveX 구성 요소를 등록 하는 데 필요한 권한이 있는지 확인 해야 합니다.  .Net Visual Studio 터미널 서비스 연결을 통해 실행 하는 경우 일반적으로 문제가입니다.
