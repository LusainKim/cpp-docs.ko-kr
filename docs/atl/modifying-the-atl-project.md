# Modifying the ATL Project
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

이 시점에서 합성 컨트롤 프로젝트 합성 컨트롤에 필요한 개체를 구현합니다.  복합 컨트롤을 포함 하 고 필요한 이벤트를 처리 하는 컨트롤을 추가 하려면 다음 단계를입니다.  
  
 다른 ActiveX 또는 Windows 컨트롤을 추가 하려면 새 리소스 스크립트를 추가 하 고 대화 상자 편집기를 사용 합니다.  추가 컨트롤 \(및 관련된 작업\)에 대 한 자세한 내용은  [대화 상자 편집기](../mfc/dialog-editor.md).  
  
 ActiveX 컨트롤에서 필요한 이벤트 처리를 참조 하십시오.  [복합 컨트롤 기능 추가](../atl/adding-functionality-to-the-composite-control.md).
