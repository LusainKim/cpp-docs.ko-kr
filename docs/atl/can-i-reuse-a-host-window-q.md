# Can I Reuse a Host Window?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

호스트 창은 다시 사용 하는 권장 되지 않습니다.  코드의 견고성을 확인 하 여 호스트 창의 수명 단일 컨트롤의 수명에 연결 해야 합니다.
