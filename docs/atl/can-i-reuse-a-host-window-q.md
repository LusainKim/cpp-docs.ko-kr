# <a name="can-i-reuse-a-host-window"></a>호스트 창 다시 사용할 수 있습니까?
호스트 창을 다시 사용 하는 권장 되지 않습니다. 코드의 견고성을 보장 하려면 단일 컨트롤의 수명과 호스트 창의 수명을 제한 해야 합니다.
