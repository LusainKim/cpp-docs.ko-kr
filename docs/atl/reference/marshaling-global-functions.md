# Marshaling Global Functions
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

|||

|[AtlUnmarshalPtr](../Topic/AtlUnmarshalPtr.md)|마샬링 데이터를 스트림 인터페이스 포인터로 변환합니다.|
