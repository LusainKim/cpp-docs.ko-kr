# Service Map Macros
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이러한 매크로 서비스 맵 및 항목을 정의합니다.  
  
|||  
|-|-|  
|[BEGIN\_SERVICE\_MAP](../Topic/BEGIN_SERVICE_MAP.md)|ATL 서비스 맵의 시작 부분을 표시 합니다.|  
|[END\_SERVICE\_MAP](../Topic/END_SERVICE_MAP.md)|ATL 서비스 맵의 끝을 표시 합니다.|  
|[SERVICE\_ENTRY](../Topic/SERVICE_ENTRY.md)|개체는 특정 서비스 ID를 지원 하도록 나타냅니다.|  
|[SERVICE\_ENTRY\_CHAIN](../Topic/SERVICE_ENTRY_CHAIN.md)|지시  [IServiceProviderImpl::QueryService](../Topic/IServiceProviderImpl::QueryService.md) 에 지정 된 개체의 체인.|
