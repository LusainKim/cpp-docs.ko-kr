# <a name="com-10-support-in-atl-projects"></a>COM + 1.0 ATL 프로젝트에서 지원 합니다.
사용할 수는 [ATL 프로젝트 마법사](../../atl/reference/creating-an-atl-project.md) 프로젝트를 만들려면 COM + 1.0 구성 요소에 대 한 기본 지원 합니다.  
  
 분산된 구성 요소 기반 응용 프로그램을 개발 하기 위한 COM + 1.0 사용할 수 있습니다. 또한 이러한 응용 프로그램 관리 및 배포에 런타임 인프라를 제공 합니다.  
  
 선택 하는 경우는 **COM + 1.0 지원** 확인란을 마법사 링크 단계에서 빌드 스크립트를 수정 합니다. 특히, COM + 1.0 프로젝트 링크 다음 라이브러리에 있습니다.  
  
-   comsvcs.lib  
  
-   Mtxguid.lib  
  
 선택 하는 경우는 **COM + 1.0 지원** 확인란을 선택할 수도 있습니다 **지원 구성 요소 등록자**합니다. 구성 요소 관리자 사용 구성 요소 목록을 가져오려면, 구성 요소를 등록 또는 (개별적으로 또는 한 번에) 구성 요소 등록을 취소 하면 COM + 1.0 개체를 수입니다.
