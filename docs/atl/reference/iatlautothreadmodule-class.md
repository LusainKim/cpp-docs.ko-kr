# IAtlAutoThreadModule Class
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 클래스는 인터페이스를 나타내는 한 `CreateInstance` 메서드.  
  
> [!IMPORTANT]
>  이 클래스와 해당 멤버를 실행 하는 응용 프로그램에서 사용할 수 있는 [!INCLUDE[wrt](../../atl/reference/includes/wrt_md.md)].  
  
## 구문  
  
```  
  
__interface IAtlAutoThreadModule  
  
```  
  
## 설명  
 클래스  [CAtlAutoThreadModuleT](../../atl/reference/catlautothreadmodulet-class.md) 에서 파생 된 `IAtlAutoThreadModule`를 사용 하 여 개체를 만들고 인터페이스 포인터를 검색 하는 코드를 제공 합니다.
