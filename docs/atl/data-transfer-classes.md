# Data Transfer Classes
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

다양 한 유형의 데이터 전송은 다음과 같은 클래스를 지원합니다.  
  
-   [IDataObjectImpl](../atl/reference/idataobjectimpl-class.md) 지원 일정 데이터를 표준 형식을 검색 하 고 데이터 집합을 사용 하 여 전송 합니다.  데이터 핸들이 변경 알림을 관리 어드바이스를 연결 하 여.  
  
-   [CBindStatusCallback](../atl/reference/cbindstatuscallback-class.md) 비동기 모니커의 개체의 비동기 데이터 전송에 대 한 정보를 주고받을 수 있습니다.
