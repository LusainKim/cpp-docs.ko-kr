# What Is AtlAxWin100?
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

`AtlAxWin100`ATL의 컨트롤 호스팅 기능을 제공 하는 창 클래스의 이름이입니다.  이 클래스의 인스턴스를 만들 때 창 프로시저는 자동 컨트롤 호스팅 API는 창에 연결 된 호스트 개체를 만들어 창의 제목으로 지정 된 컨트롤을 로드 하려면 사용 됩니다.
