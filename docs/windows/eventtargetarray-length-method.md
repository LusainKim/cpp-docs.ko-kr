# EventTargetArray::Length 메서드
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라구조체를 지원하고 코드에서 직접적으로 사용되는 것을 의도하지 않습니다.  
  
## 구문  
  
```  
size_t Length();  
```  
  
## 반환 값  
 현재 이벤트 처리기의 내부 배열에 있는 요소의 수입니다.  
  
## 설명  
 현재 이벤트 처리기의 내부 배열에 있는 요소의 수를 가져옵니다.
