# Editing Managed Resource Files
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

관리되는 리소스 파일\(.resx\)은 XML 파일입니다.  **새 항목 추가** 대화 상자에서 프로젝트에 관리되는 리소스 파일을 추가할 때는 기본적으로 **관리되는 리소스 편집기**가 열립니다.  
  
 자세한 내용은 [.Resx 파일 형식의 리소스](http://msdn.microsoft.com/ko-kr/0c476133-87e4-47e8-b0ef-4b88f4ef3dc5)를 참조하십시오.
