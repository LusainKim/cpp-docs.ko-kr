# HString::Detach 메서드
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

지정된 HString 개체를 기본 값에서 해제합니다.  
  
## 구문  
  
```  
HSTRING Detach() throw()  
```  
  
## 반환 값  
 분리 작업을 시작하기 전에 기본 HString 값입니다.
