# ComPtr::Reset
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

이 ComPtr과 연관된 인터페이스의 포인터에 대한 모든 참조를 해제합니다.  
  
## 구문  
  
```  
unsigned long Reset();  
```  
  
## 반환 값  
 해제된 참조 수입니다\(있는 경우\).  
  
## 요구 사항  
 **헤더:** client.h  
  
 **네임스페이스:** Microsoft::WRL
