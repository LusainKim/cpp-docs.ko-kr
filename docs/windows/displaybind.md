# displaybind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

사용자에 게 바인딩할 수 있다고 표시 되어야 하는 속성을 나타냅니다.  
  
## 구문  
  
```  
  
[displaybind]  
  
```  
  
## 설명  
 **Displaybind** C\+\+ 특성을 동일한 기능을가지고 있는  [displaybind](http://msdn.microsoft.com/library/windows/desktop/aa366804) MIDL 속성입니다.  
  
## 예제  
 예제를 보려면  [바인딩할 수 있는](../windows/bindable.md) 예를 사용 하는 방법에 대 한  **displaybind**.
