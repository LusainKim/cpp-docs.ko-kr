# hidden
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

항목이 존재 하지만 사용자 기반 브라우저에 표시 되어서는 안 있음을 나타냅니다.  
  
## 구문  
  
```  
  
[hidden]  
  
```  
  
## 설명  
 **숨겨진** C\+\+ 특성을 동일한 기능을가지고 있는  [숨겨진](http://msdn.microsoft.com/library/windows/desktop/aa366861) MIDL 속성입니다.  
  
## 예제  
 예제를 보려면  [바인딩할 수 있는](../windows/bindable.md) 예를 사용 하는 방법에 대 한  **숨겨진**.  
  
## 요구 사항  
  
### 컨텍스트 특성  
  
|||  
|-|-|  
|**적용 대상**|`interface` **클래스**, `struct`, 메서드, 속성|  
|**반복 가능**|아니요|  
|**필수 특성**|**coclass** \(적용 될 때  **클래스** 또는 `struct`\)|  
|**잘못 된 특성**|없음|  
  
 자세한 내용은  [컨텍스트 특성](../windows/attribute-contexts.md).
