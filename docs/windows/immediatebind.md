# immediatebind
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

데이터베이스가 즉시 데이터 바인딩된 개체의 속성 변경 알림을 받을 수 있도록 나타냅니다.  
  
## 구문  
  
```  
  
[immediatebind]  
  
```  
  
## 설명  
 **Immediatebind** C\+\+ 특성을 동일한 기능을가지고 있는  [immediatebind](http://msdn.microsoft.com/library/windows/desktop/aa367045) MIDL 속성입니다.  
  
## 예제  
 참조 하십시오  [바인딩할 수 있는](../windows/bindable.md) 예를 사용 하는 방법에 대 한  **immediatebind**.
