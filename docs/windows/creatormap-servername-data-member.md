# CreatorMap::serverName 데이터 멤버
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.  
  
## 구문  
  
```  
const wchar_t* serverName;  
```  
  
## 설명  
 CreatorMap에 대한 서버 이름을 저장합니다.
