# Creating and Setting Guides and Margins
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

컨트롤을 이동하거나 추가하거나 현재 레이아웃을 다시 배치할 때 안내선을 사용하면 컨트롤을 정확하게 대화 상자에 맞출 수 있습니다.  안내선은 편집기에 표시된 대화 상자에 파란색 점선으로 나타나고 눈금자\(대화 상자 편집기의 왼쪽 맨 위\)에는 대응하는 화살표가 표시됩니다.  
  
 대화 상자를 만들 경우, 네 가지 여백이 있습니다.  여백은 파란색 점선으로 표시되는 수정된 안내선입니다.  
  
### 안내선을 만들려면  
  
1.  눈금자 내부를 한 번 클릭하여 안내선을 만듭니다.  한 번 클릭하면 새 안내선이 만들어지고 두 번 클릭하면 안내선 설정값을 지정할 수 있는 [안내선 설정 대화 상자](../mfc/guide-settings-dialog-box.md)가 시작됩니다.  
  
### 안내선을 설정하려면  
  
1.  대화 상자에서 안내선을 클릭하여 새 위치로 끕니다.  눈금자의 화살표를 클릭하여 관련 안내선을 끌 수도 있습니다.  
  
     안내선의 좌표는 창 아래의 상태 표시줄과 눈금자에 표시됩니다.  포인터를 눈금자의 화살표 위로 이동하면 안내선의 정확한 위치가 표시됩니다.  
  
### 안내선을 삭제하려면  
  
1.  대화 상자 밖으로 안내선을 끕니다  
  
 \-또는\-  
  
-   대응하는 화살표를 눈금자 밖으로 끕니다.  
  
#### 여백을 이동하려면  
  
1.  새 위치로 여백을 끕니다.  
  
     여백을 표시하지 않으려면 0의 위치로 여백을 이동합니다.  여백을 이전 위치로 되돌리려면 여백의 0 위치에 포인터를 두고 여백을 원래의 위치로 이동합니다.  
  
 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하십시오. 관리되는 프로젝트에 리소스 파일 추가, 리소스 액세스, 정적 리소스 표시, 속성에 리소스 문자열 할당 등의 작업을 수동으로 수행하는 방법에 대한 자세한 내용은 [연습: Windows Forms 지역화](http://msdn.microsoft.com/ko-kr/9a96220d-a19b-4de0-9f48-01e5d82679e5) 및 [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)을 참조하십시오.
