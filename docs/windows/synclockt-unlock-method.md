# SyncLockT::Unlock 메서드
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라구조체를 지원하고 코드에서 직접적으로 사용되는 것을 의도하지 않습니다.  
  
## 구문  
  
```  
void Unlock();  
```  
  
## 설명  
 모든 경우에, 현재 syncLockT 개체에 의해 보유된 리소스의 컨트롤을 릴리스합니다.
