# oleautomation
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

인터페이스를 자동화 호환입니다.  
  
## 구문  
  
```  
  
[oleautomation]  
  
```  
  
## 설명  
 **Oleautomation**  C\+\+ 특성을 동일한 기능을가지고 있는  [oleautomation](http://msdn.microsoft.com/library/windows/desktop/aa367129) MIDL 속성입니다.  
  
## 예제  
 에 대 한 예제를 참조 하십시오.  [defaultvalue](../windows/defaultvalue.md) 및  [nonextensible](../windows/nonextensible.md) 의 샘플 사용에 대 한  **oleautomation**.
