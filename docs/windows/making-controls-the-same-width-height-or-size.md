# Making Controls the Same Width, Height, or Size
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

기준 컨트롤의 크기로 컨트롤 그룹의 크기를 조정할 수 있습니다.  또한 [각 캡션 텍스트의 크기에 따라 컨트롤의 크기를 조정](../mfc/sizing-individual-controls.md)할 수도 있습니다.  
  
### 컨트롤의 너비, 높이 또는 크기를 동일하게 만들려면  
  
1.  크기를 조정할 [컨트롤을 여러 개 선택](../mfc/selecting-multiple-controls.md)합니다.  
  
     그룹에서 처음으로 선택한 컨트롤이 기준 컨트롤이 됩니다.  그룹에 있는 컨트롤의 최종 크기는 기준 컨트롤의 크기에 따라 결정됩니다.  기준 컨트롤 선택에 대한 자세한 내용은 [기준 컨트롤 지정](../mfc/specifying-the-dominant-control.md)을 참조하십시오.  
  
2.  **서식** 메뉴에서 **같은 크기로**를 선택한 후, 다음 명령 중 하나를 선택합니다.  
  
    -   **Both**  
  
    -   **높이**  
  
    -   **너비**  
  
 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하십시오. 관리되는 프로젝트에 리소스 파일 추가, 리소스 액세스, 정적 리소스 표시, 속성에 리소스 문자열 할당 등의 작업을 수동으로 수행하는 방법에 대한 자세한 내용은 [연습: Windows Forms 지역화](http://msdn.microsoft.com/ko-kr/9a96220d-a19b-4de0-9f48-01e5d82679e5) 및 [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)을 참조하십시오.
