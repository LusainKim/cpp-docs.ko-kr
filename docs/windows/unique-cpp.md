# unique (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

고유한 포인터가 지정합니다.  
  
## 구문  
  
```  
  
[unique]  
  
```  
  
## 설명  
 **고유** C\+\+ 특성을 동일한 기능을가지고 있는  [고유](http://msdn.microsoft.com/library/windows/desktop/aa367294) MIDL 속성.  
  
## 예제  
 참조는  [ref](../windows/ref-cpp.md) 샘플 사용을 예를 들어  **고유**.
