# Nil 구조체
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라구조체를 지원하고 코드에서 직접적으로 사용되는 것을 의도하지 않습니다.  
  
## 구문  
  
```  
struct Nil;  
```  
  
## 설명  
 사용된 서식 옵션을 지정하지 않은 매개 변수를 나타냅니다.  
  
 Nil은 비어 있는 구조입니다.  
  
## 상속 계층  
 `Nil`

 **헤더:** implements.h  
  
 **네임스페이스:** Microsoft::WRL::Details
