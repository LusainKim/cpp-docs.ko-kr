# &lt;Device&gt; 이미지 열기 대화 상자(아이콘에 대한 이미지 편집기)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

이 대화 상자를 사용하면 장치 이미지를 열 수 있습니다.  현재 리소스에 포함된 기존 장치 이미지가 표시됩니다.  
  
 **현재 이미지**  
 리소스에 포함된 이미지가 표시됩니다.  열 이미지 형식을 선택합니다.
