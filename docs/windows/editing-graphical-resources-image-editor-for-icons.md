# Editing Graphical Resources (Image Editor for Icons)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

그래픽 리소스는 응용 프로그램에 사용하기 위해 정의하는 이미지입니다.  자유롭게 그릴 수도 있고 도형을 사용하여 그릴 수도 있습니다.  이미지의 일부를 선택하여 편집, 대칭 이동 또는 크기 조정 작업을 할 수도 있고, 선택한 이미지 부분에서 사용자 지정 브러시를 만들어 그릴 수도 있습니다.  이미지 속성을 정의하고 다른 형식으로 이미지를 저장한 후에 다른 형식으로 이미지를 변환할 수 있습니다.  
  
 새 그래픽 리소스를 만들 수도 있고 [기존 리소스를 가져와](../windows/how-to-import-and-export-resources.md) 편집한 다음 프로젝트에 추가할 수도 있습니다.  또한 [프로젝트 외부의 이미지 편집](../mfc/editing-an-image-outside-of-a-project-image-editor-for-icons.md)을 위해 프로젝트에 포함되지 않은 이미지를 열고 편집할 수도 있습니다.

 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하십시오. 관리되는 프로젝트에 리소스 파일 추가, 리소스 액세스, 정적 리소스 표시, 속성에 리소스 문자열 할당 등의 작업을 수동으로 수행하는 방법에 대한 자세한 내용은 [연습: Windows Forms 지역화](http://msdn.microsoft.com/ko-kr/9a96220d-a19b-4de0-9f48-01e5d82679e5) 및 [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)을 참조하십시오.  
  
> [!NOTE]
>  이미지 편집기를 사용하여 32비트 이미지를 볼 수는 있지만 편집할 수는 없습니다.
