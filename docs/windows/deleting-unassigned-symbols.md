# <a name="deleting-unassigned-symbols"></a>할당되지 않은 기호 삭제
### <a name="to-delete-an-unassigned-unused-symbol"></a>할당되지 않은(사용되지 않은) 기호를 삭제하려면  
  
1.  에 [리소스 기호 대화 상자](../windows/resource-symbols-dialog-box.md)를 삭제 하 고 클릭 하려는 기호 선택 **삭제**합니다.  
  
    > [!NOTE]
    >  리소스 파일에서 사용되지 않은 기호를 삭제하려면 먼저 프로그램의 다른 위치나 컴파일 시간에 포함된 리소스 파일에서 사용되지 않는지 확인합니다.  
  

  
 **Requirements**  
  
 Win32
