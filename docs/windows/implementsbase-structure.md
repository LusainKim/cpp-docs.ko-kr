# ImplementsBase 구조체
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라구조체를 지원하고 코드에서 직접적으로 사용되는 것을 의도하지 않습니다.  
  
## 구문  
  
```  
struct ImplementsBase;  
```  
  
## 설명  
 [Implements 구조체](../windows/implements-structure.md) 에서 유효한 템플릿 매개변수를 사용합니다.  
  
 설계에서 ImplementsBase 구조는 비어 있습니다.  
  
## 상속 계층  
 `ImplementsBase`

 **헤더:** implements.h
