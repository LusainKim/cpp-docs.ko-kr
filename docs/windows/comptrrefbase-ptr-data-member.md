# ComPtrRefBase::ptr_ 데이터 멤버
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.  
  
## 구문  
  
```  
T* ptr_;  
```  
  
## 설명  
 현재 템플릿 매개 변수로 지정된 형식에대한 포인터입니다.  
  
## 설명  
 `ptr_`보호 된 데이터 멤버입니다.
