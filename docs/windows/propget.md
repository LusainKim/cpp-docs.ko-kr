# propget
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

속성 접근자 함수를 지정합니다.  
  
## 구문  
  
```  
  
[propget]  
  
```  
  
## 설명  
 **Propget** C\+\+ 특성을 동일한 기능을가지고 있는  [propget](http://msdn.microsoft.com/library/windows/desktop/aa367145) MIDL 속성입니다.  
  
## 예제  
 예제를 보려면  [바인딩할 수 있는](../windows/bindable.md) 의 샘플 사용에 대 한  **propget**.
