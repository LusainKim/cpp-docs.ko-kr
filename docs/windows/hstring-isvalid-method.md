# HString::IsValid 메서드
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

현재 HString 개체가 비어 있는지 여부를 나타냅니다.  
  
## 구문  
  
```  
bool IsValid() const throw()  
```  
  
#### 매개 변수  
 현재 HString 개체가 비어 있으면 `true`, 그렇지 않으면, `false`.
