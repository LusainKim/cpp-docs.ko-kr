# requestedit
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

이 속성을 지원함을 나타내는  **OnRequestEdit** 알림.  
  
## 구문  
  
```  
  
[requestedit]  
  
```  
  
## 설명  
 **Requestedit** C\+\+ 특성을 동일한 기능을가지고 있는  [requestedit](http://msdn.microsoft.com/library/windows/desktop/aa367155) MIDL 속성입니다.  
  
## 예제  
 예제를 보려면  [바인딩할 수 있는](../windows/bindable.md) 의 샘플 사용에 대 한  **requestedit**.
