# Opening the Resource Editor for a Given Symbol
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

[리소스 기호 대화 상자](../windows/resource-symbols-dialog-box.md)에서 기호를 찾는 경우 특정 기호의 사용 방법에 대한 추가 정보를 원할 수 있습니다.  **사용 리소스 보기** 단추를 통해 이 정보를 빠르게 가져올 수 있습니다.  
  
### 기호가 사용되는 리소스 편집기로 이동하려면  
  
1.  **리소스 기호** 대화 상자의 **이름** 상자에서 기호를 선택합니다.  
  
2.  **다음에서 사용** 상자에서 관심 있는 리소스 형식을 선택합니다.  
  
3.  **사용 리소스 보기** 단추를 클릭합니다.  
  
     리소스가 해당 편집기 창에 나타납니다.  
  
 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하세요.  
  
 요구 사항  
  
 Win32
