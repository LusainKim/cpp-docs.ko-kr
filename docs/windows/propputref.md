# propputref
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

값이 아닌 참조를 사용 하 여 속성 설정 함수를 지정 합니다.  
  
## 구문  
  
```  
  
[propputref]  
  
```  
  
## 설명  
 **Propputref** C\+\+ 특성을 동일한 기능을가지고 있는  [propputref](http://msdn.microsoft.com/library/windows/desktop/aa367147) MIDL 속성입니다.  
  
## 예제  
 예제를 보려면  [바인딩할 수 있는](../windows/bindable.md) 의 샘플 사용에 대 한  **propputref**.
