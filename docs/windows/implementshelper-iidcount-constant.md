# ImplementsHelper::IidCount 상수
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라구조체를 지원하고 코드에서 직접적으로 사용되는 것을 의도하지 않습니다.  
  
## 구문  
  
```  
static const unsigned long IidCount;  
```  
  
## 설명  
 현재 구현 개체에 구현된 인터페이스 Id 저장합니다.
