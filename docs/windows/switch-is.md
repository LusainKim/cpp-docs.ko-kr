# switch_is
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

식 또는 공용 구조체 멤버 선택은 통합 분석할으로 역할을 하는 식별자를 지정 합니다.  
  
## 구문  
  
```  
  
[switch_is]  
  
```  
  
## 설명  
 **Switch\_is** C\+\+ 특성을 동일한 기능을가지고 있는  [switch\_is](http://msdn.microsoft.com/library/windows/desktop/aa367275) MIDL 속성입니다.  
  
## 예제  
 참조는  [경우](../windows/case-cpp.md) 샘플 사용을 예를 들어  **switch\_is**.
