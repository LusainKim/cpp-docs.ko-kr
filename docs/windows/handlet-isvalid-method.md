# HandleT::IsValid 메서드
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

현재 HandleT 개체 핸들이 있는지 여부를 나타냅니다.  
  
## 구문  
  
```  
bool IsValid() const;  
```  
  
## 반환 값  
 HandleT의 핸들을 나타내는 경우, `true` 그렇지 않으면 `false`.
