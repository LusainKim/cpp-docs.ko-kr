# Sizing a Control While You Add It
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

### 컨트롤을 추가할 때 크기를 조정하려면  
  
1.  [도구 상자 창](../Topic/Toolbox.md)에서 컨트롤을 선택합니다.  
  
2.  대화 상자에서 새 컨트롤의 왼쪽 위 모퉁이를 둘 위치에 십자 모양의 커서를 놓습니다.  
  
3.  마우스 단추를 클릭한 채로 대화 상자에서 컨트롤의 왼쪽 위 모퉁이를 앵커로 고정한 후 컨트롤이 원하는 크기가 될 때까지 커서를 오른쪽 아래로 끕니다.  
  
    > [!NOTE]
    >  실제로는 컨트롤의 4개 모퉁이 중 어느 곳이나 앵커로 고정하여 그릴 수 있습니다.  이 과정에서는 왼쪽 위 모퉁이를 예로 사용했습니다.  
  
4.  마우스 단추를 놓습니다.  그러면 컨트롤이 지정한 크기로 대화 상자에 고정됩니다.  
  
    > [!TIP]
    >  대화 상자에 컨트롤을 놓은 후 컨트롤 테두리에서 크기 조정 핸들을 이동하여 컨트롤 크기를 조정할 수 있습니다.  컨트롤에 대한 자세한 내용은 [각 컨트롤 크기 조정](../mfc/sizing-individual-controls.md)을 참조하십시오.  
  
 관리되는 프로젝트에 리소스를 추가하는 방법은 .NET Framework 개발자 가이드의 [응용 프로그램의 리소스](../Topic/Resources%20in%20Desktop%20Apps.md)를 참조하십시오. 관리되는 프로젝트에 리소스 파일 추가, 리소스 액세스, 정적 리소스 표시, 속성에 리소스 문자열 할당 등의 작업을 수동으로 수행하는 방법에 대한 자세한 내용은 [연습: Windows Forms 지역화](http://msdn.microsoft.com/ko-kr/9a96220d-a19b-4de0-9f48-01e5d82679e5) 및 [Walkthrough: Using Resources for Localization with ASP.NET](../Topic/Walkthrough:%20Using%20Resources%20for%20Localization%20with%20ASP.NET.md)을 참조하십시오.
