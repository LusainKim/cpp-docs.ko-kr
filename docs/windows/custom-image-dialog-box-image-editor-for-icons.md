# 사용자 지정 이미지 대화 상자(아이콘에 대한 이미지 편집기)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

사용자 지정 크기와 색상 수를 사용하여 새 이미지를 만들 수 있습니다.  Windows에서 사용하는 아이콘 크기에 대한 자세한 내용은 Windows SDK 설명서에서 [아이콘](_win32_Icons_cpp)을 참조하십시오.  
  
 **너비**  
 사용자 지정 이미지의 너비를 픽셀 단위로 입력할 수 있습니다\(1–512, 최대 2048\).  
  
 **높이**  
 사용자 지정 이미지의 높이를 픽셀 단위로 입력할 수 있습니다\(1–512, 최대 2048\).  
  
 **Colors**  
 사용자 지정 이미지의 색상 수\(2, 16 또는 256\)를 선택할 수 있습니다.
