# ModuleBase::~ModuleBase 소멸자
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

WRL 인프라구조체를 지원하고 코드에서 직접적으로 사용되는 것을 의도하지 않습니다.  
  
## 구문  
  
```  
virtual ~ModuleBase();  
```  
  
## 설명  
 ModuleBase 클래스의 현재 인스턴스를 초기화 해제합니다.
