# _bstr_t::Detach
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 전용**  
  
 `_bstr_t`로 래핑된 `BSTR`을 반환하고 `BSTR`을 `_bstr_t`에서 분리합니다.  
  
## 구문  
  
```  
  
BSTR Detach( ) throw;  
  
```  
  
## 반환 값  
 `_bstr_t`로 래핑된 `BSTR`입니다.  
  
## 예제  
 **Detach**를 사용하는 예제를 보려면 [\_bstr\_t::Assign](../cpp/bstr-t-assign.md)을 참조하십시오.
