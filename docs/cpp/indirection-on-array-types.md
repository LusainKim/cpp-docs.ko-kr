# <a name="indirection-on-array-types"></a>배열 형식에 대한 간접 참조
간접 참조 연산자를 사용 (**\***)에  *n* -차원 배열 형식에서 생성 한  *n* -1 차원 배열입니다. 경우  *n*  1 이면 스칼라 (또는 배열 요소)가 생성 됩니다.
