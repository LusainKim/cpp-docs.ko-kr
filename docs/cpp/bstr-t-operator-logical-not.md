# _bstr_t::operator !
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**Microsoft 전용**  
  
 캡슐화된 `BSTR`이 **NULL** 문자열인지 확인합니다.  
  
## 구문  
  
```  
  
bool operator!( ) const throw( );  
  
```  
  
## 반환 값  
 맞으면 **true**를 반환하고, 아니면 **false**를 반환합니다.
