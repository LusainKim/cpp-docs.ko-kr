# <a name="jump-statements-c"></a>점프 문 (C++)
C++ 점프 문은 제어의 즉시 로컬 전송을 수행합니다.  
  
## <a name="syntax"></a>구문  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
