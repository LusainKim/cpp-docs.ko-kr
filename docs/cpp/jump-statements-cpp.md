# 점프 문 (C++)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

C\+\+ 점프 문은 제어의 즉시 로컬 전송을 수행합니다.  
  
## 구문  
  
```  
break;  
continue;  
return [expression];  
goto identifier;  
```
