# <a name="abort-function"></a>abort 함수

**중단** STDLIB 표준 포함 파일에서 선언 된 함수입니다. H, c + + 프로그램을 종료 합니다. 차이 **종료** 및 **중단** 은 **종료** c + + 런타임 종료 처리 과정이 발생 (전역 개체 소멸자 호출)를 사용 하면 반면 **중단** 에서는 프로그램이 즉시 종료 합니다. 자세한 내용은 참조 [중단](../c-runtime-library/reference/abort.md) 에 *런타임 라이브러리 참조*합니다.
