# 컴파일러 오류 C2817
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator delete'의 반환 형식은 'void'이어야 합니다.  
  
 오버로드된 [operator delete](../Topic/operator%20delete%20\(%3Cnew%3E\).md) 함수는 값을 반환할 수 없습니다.
