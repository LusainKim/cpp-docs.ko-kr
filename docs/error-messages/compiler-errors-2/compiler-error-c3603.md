# 컴파일러 오류 C3603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'Symbol' : 'Type' 형식은 아직 지원되지 않습니다.  
  
 관리되는 개체에서 .NET 런타임이 아직 지원하지 않는 데이터 형식을 사용하려고 했습니다.
