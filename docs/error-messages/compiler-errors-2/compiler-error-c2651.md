# <a name="compiler-error-c2651"></a>컴파일러 오류 C2651
데이터 형식: '연산자' 왼쪽은 클래스, 구조체 또는 공용 구조체 이어야 합니다  
  
 템플릿 매개 변수 형식이 클래스 마치를 사용 하려면 정수 계열 형식이 아니라 클래스를 사용 하 여 클래스 템플릿의 특수화 합니다.
