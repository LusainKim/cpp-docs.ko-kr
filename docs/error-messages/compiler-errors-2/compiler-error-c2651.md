# 컴파일러 오류 C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'data type' : 'operator' 왼쪽은 클래스, 구조체 또는 공용 구조체이어야 합니다.  
  
 템플릿 매개 변수를 클래스처럼 사용하려면 정수 계열 형식 대신에 클래스를 사용하여 클래스 템플릿을 특수화하십시오.
