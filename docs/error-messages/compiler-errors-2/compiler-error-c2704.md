# <a name="compiler-error-c2704"></a>컴파일러 오류 C2704
'identifier': __va_start 내장 함수는 varargs에서만 사용할 수 있습니다.  
  
 `__va_start` 내장 함수는 선언에서 고정 개수의 인수와 함께 사용됩니다.
