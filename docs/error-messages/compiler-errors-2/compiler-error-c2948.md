# 컴파일러 오류 C2948
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

명시적 인스턴스화. 특수화에는 'specifier' 저장소 클래스 지정자를 사용할 수 없습니다.  
  
 이전에 명시적으로 인스턴스화한 템플릿 클래스의 특수화에 저장소 클래스 지정자\(예: `extern`\)를 사용할 수 없습니다.
