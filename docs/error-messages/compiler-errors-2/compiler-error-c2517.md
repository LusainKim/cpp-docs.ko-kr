# 컴파일러 오류 C2517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': '::' 오른쪽을 정의하지 않았습니다.  
  
 범위 결정 연산자\(`::`\) 오른쪽에 있는 식별자는 왼쪽에 있는 클래스, 구조체 또는 공용 구조체의 정의된 멤버여야 합니다.  클래스, 구조체 또는 공용 구조체를 명명하지 않은 경우에는 오른쪽에 있는 식별자를 전역 범위에서 선언해야 합니다.
