# <a name="compiler-error-c2517"></a>컴파일러 오류 C2517
'identifier': 오른쪽에 ':: ' 정의 되지 않습니다  
  
 범위 결정 연산자의 오른쪽에 있는 식별자 (`::`)의 클래스, 구조체 또는 공용 구조체의 왼쪽에 정의 된 멤버 여야 합니다. 없는 클래스, 구조체 또는 공용 이름이 인 경우 글로벌 범위를 갖는 오른쪽에 식별자 선언 되어야 합니다.
