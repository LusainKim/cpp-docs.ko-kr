# <a name="compiler-error-c2927"></a>컴파일러 오류 C2927
'function': 인수를 최소한 하나 이상 사용하여 함수 템플릿을 호출해야 합니다.  
  
 인수 없이 템플릿 함수를 호출할 수 없습니다. 템플릿 인수의 형식에 따라 생성할 함수 버전이 결정됩니다.
