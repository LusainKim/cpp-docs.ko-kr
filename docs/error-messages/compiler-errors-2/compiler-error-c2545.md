# 컴파일러 오류 C2545
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator' : 오버로드된 연산자를 찾을 수 없습니다.  
  
 제공된 피연산자에 연산자를 사용할 수 없습니다.  오버로드된 연산자에 필수 피연산자를 제공해야 합니다.  
  
 이 오류는 피연산자의 형식이 잘못된 경우에 발생할 수 있습니다.  
  
 이 오류를 수정하려면 단일 매개 변수를 사용하는 변환 연산자나 생성자를 정의해야 합니다.
