# 컴파일러 오류 C3506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

LIBID 'id'에 대해 등록된 typelib가 없습니다.  
  
 typelib를 올바르게 등록하지 않았습니다.  regsvr32.exe를 사용하여 형식 라이브러리를 등록하십시오.
