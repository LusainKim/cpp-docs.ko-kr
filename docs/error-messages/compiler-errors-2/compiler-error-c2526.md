# 컴파일러 오류 C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' : C 링크 함수는 C\+\+ 클래스 'identifier2'을\(를\) 반환할 수 없습니다.  
  
 C 링크로 정의된 함수는 사용자 정의 형식을 반환할 수 없습니다.
