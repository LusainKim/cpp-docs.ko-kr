# 컴파일러 오류 C2996
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 재귀 함수 템플릿 정의  
  
 함수 정의가 루트 템플릿 클래스를 인스턴스화하려고 합니다. 재귀 템플릿 인스턴스화가 허용되지 않습니다.
