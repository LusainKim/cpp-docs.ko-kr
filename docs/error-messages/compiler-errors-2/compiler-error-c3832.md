# 컴파일러 오류 C3832
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type library': 형식 라이브러리는 32비트 포인터용으로 빌드된 것처럼 보입니다. 'ptrsize' 한정자를 변경하십시오.  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 지시문의 `ptrsize` 특성을 통해 제공한 명시적 정보가 형식 라이브러리에 있는 정보와 일치하지 않습니다.
