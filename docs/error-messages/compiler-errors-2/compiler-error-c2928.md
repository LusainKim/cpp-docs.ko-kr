# <a name="compiler-error-c2928"></a>컴파일러 오류 C2928
명시적 인스턴스화. 'identifier'는 함수 또는 템플릿-클래스 'class'의 정적 데이터 멤버가 아닙니다.  
  
 함수 또는 `class` 변수가 아닌 `static` 멤버를 명시적으로 인스턴스화할 수 없습니다.
