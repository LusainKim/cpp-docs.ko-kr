# 컴파일러 오류 C3180
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type name' : 이름이 메타데이터 한계인 'limit'자를 초과합니다.  
  
 컴파일러가 메타데이터의 관리되는 형식 이름을 잘랐습니다.  이렇게 잘라내면 `#using` 지시문 또는 다른 언어에서 동일한 역할을 하는 문을 통해 형식을 사용할 수 없게 됩니다.  
  
 형식\-이름 한계에는 네임스페이스 한정자도 포함됩니다.
