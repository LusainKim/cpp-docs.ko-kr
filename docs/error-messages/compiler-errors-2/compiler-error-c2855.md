# 컴파일러 오류 C2855
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'option' 명령줄 옵션이 미리 컴파일된 헤더와 일치하지 않습니다.  
  
 지정된 명령줄 옵션이 미리 컴파일된 헤더를 만드는 데 사용되는 옵션과 다릅니다.  미리 컴파일된 헤더를 지정된 명령줄 옵션을 사용하여 다시 생성하십시오.
