# <a name="compiler-error-c3248"></a>컴파일러 오류 C3248
'function1' : '__sealed'로 선언된 함수를 'function2'로 재정의할 수 없습니다.  
  
파생된 클래스에서 재정의 하려고 한 **__sealed** 가상 메서드.  
  
C3248는 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
