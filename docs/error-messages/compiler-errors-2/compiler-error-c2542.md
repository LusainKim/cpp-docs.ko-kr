# 컴파일러 오류 C2542
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 클래스 개체에 초기화를 위한 생성자가 없습니다.  
  
 초기화에 대응하는 매개 변수 목록을 사용하는 생성자가 없습니다.  
  
 이 오류는 초기화 시 올바르지 않은 매개 변수로 인해 발생할 수 있습니다.
