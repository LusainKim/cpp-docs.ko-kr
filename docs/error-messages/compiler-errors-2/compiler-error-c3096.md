# <a name="compiler-error-c3096"></a>컴파일러 오류 C3096
'attribute': 특성은 특성 클래스의 데이터 멤버에만 허용됩니다.  
  
 특성이 올바르게 적용되었습니다.  
  
 자세한 내용은 참조 [사용자 정의 특성](../../windows/user-defined-attributes-cpp-component-extensions.md)합니다.
