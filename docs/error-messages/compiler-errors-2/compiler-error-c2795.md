# 컴파일러 오류 C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'super::function'은\(는\) 멤버 함수가 아닙니다.  
  
 [super](../../cpp/super.md)를 사용하여 멤버 함수 이외의 멤버에 액세스하려고 하면 이 오류 메시지가 나타납니다.
