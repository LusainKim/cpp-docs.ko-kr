# <a name="compiler-error-c3812"></a>컴파일러 오류 C3812
'property' 속성 선언에서 첫 번째 토큰 이어야 합니다.  
  
 속성을 선언 하는 경우는 `__property` 키워드가 줄에서 첫 번째 토큰 이어야 합니다.  
  
 C3812는 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
