# 컴파일러 오류 C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

명령줄 옵션 '\/Yc \(\/Fdfilename\)'이\(가\) 미리 컴파일된 헤더와 일치하지 않습니다. 이 헤더에서는 '\/Fdfilename'을\(를\) 사용했습니다.  
  
 미리 컴파일된 헤더 사용\([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) 옵션에 의해 지정된 프로그램 데이터베이스가 이전의 미리 컴파일된 헤더 만들기\([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\) 옵션에 의해 지정된 프로그램 데이터베이스와 일치하지 않습니다.
