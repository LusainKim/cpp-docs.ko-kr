# 컴파일러 오류 C2667
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : number개 오버로드 모두에 최적의 변환이 없습니다.  
  
 오버로드된 함수 호출이 모호하며 이를 해결할 수 없습니다.  
  
 함수 호출의 실제 매개 변수를 오버로드된 함수 중 하나에 대응시키는 데 필요한 변환은 오버로드된 다른 모든 함수에 필요한 변환보다 뛰어나야 합니다.  
  
 함수 템플릿의 부분 순서 지정에 대한 자세한 내용은 기술 자료 문서 Q240869를 참조하십시오.
