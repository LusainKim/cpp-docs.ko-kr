# 컴파일러 오류 C2873
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'symbol' : using 선언에서 기호를 사용할 수 없습니다.  
  
 `using` 지시문에 [namespace](../../misc/namespace-declaration.md) 키워드가 없습니다.  이로 인해 컴파일러가 코드를 [using 지시문](../../misc/using-directive-cpp.md)이 아니라 [using 선언](../../cpp/using-declaration.md)으로 잘못 해석합니다.
