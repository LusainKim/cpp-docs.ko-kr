# <a name="compiler-error-c3818"></a>컴파일러 오류 C3818
배열 속성 선언을 'property1' '속성&2;' 인덱스 속성 오버 로드 하지 않습니다.  
  
 오버 로드 하나 인덱서 이며 다른 하나는 배열 속성 때 속성에 대 한 가능한있지 않습니다. 
  
 C3818는 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
