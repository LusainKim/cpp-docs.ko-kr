# 컴파일러 오류 C3134
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'value' : 'attribute' 특성 인수의 값이 잘못된 'type' 형식으로 되어 있습니다.  
  
 값이 특성에 할당될 때 구문 오류가 감지되었습니다.
