# <a name="compiler-error-c2830"></a>컴파일러 오류 C2830
'operator new'를 배치 매개 변수에 기본값을 가질 수 있습니다.  
  
에 대 한 표준 형식 매개 변수 [new 연산자](../../standard-library/new-operators.md#op_new) 기본값을 가질 수 없습니다. 사용자 지정 배치 매개 변수에 기본값을 지정할 수 있습니다.
