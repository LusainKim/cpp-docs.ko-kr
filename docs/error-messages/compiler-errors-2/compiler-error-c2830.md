# 컴파일러 오류 C2830
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator new'에 대한 배치 매개 변수에만 기본값을 사용할 수 있습니다.  
  
 [operator new](../Topic/operator%20new%20\(%3Cnew%3E\).md)에 대한 표준 형식 매개 변수는 기본값을 사용할 수 없습니다.  사용자 정의 배치 매개 변수만 기본값을 지정할 수 있습니다.
