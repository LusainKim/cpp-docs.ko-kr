# 컴파일러 오류 C3130
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

내부 컴파일러 오류: 삽입된 코드 블록을 PDB에 쓰지 못했습니다.  
  
 이 오류는 삽입된 코드 블록을 컴파일러가 .pdb 파일에 쓰지 못한 경우에 발생합니다.  이 오류의 주된 원인은 디스크 공간 부족입니다.
