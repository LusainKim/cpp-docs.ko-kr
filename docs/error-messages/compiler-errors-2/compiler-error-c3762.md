# 컴파일러 오류 C3762
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'attribute' 특성을 처리할 수 없습니다.  
  
 `System.Security.Permissions.SecurityAttribute`에서 상속되는 사용자 정의 특성을 사용하여 보안 특성을 정의했습니다.  이러한 특성은 이 특성이 정의된 어셈블리에서 사용할 수 없습니다.
