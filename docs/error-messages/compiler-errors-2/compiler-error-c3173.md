# 컴파일러 오류 C3173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

IDL 병합 시 버전이 일치하지 않습니다.  
  
 이 오류는 이전 버전의 컴파일러에서 생성된 포함 IDL이 개체 파일에 포함되는 경우에 발생합니다.  컴파일러는 .obj 파일에 포함된 IDL 콘텐츠를 생성하는 데 사용되는 컴파일러와 포함 IDL을 병합하는 데 사용되는 컴파일러가 동일하도록 버전 번호를 인코딩합니다.  
  
 설치된 Visual C\+\+를 업데이트하여 모든 도구가 최신 릴리스 버전이 되도록 합니다.
