# 컴파일러 오류 C3157
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ParamArray 특성은 마지막 매개 변수에만 적용할 수 있습니다.  
  
 [ParamArray](https://msdn.microsoft.com/en-us/library/system.paramarrayattribute.aspx) 특성을 잘못된 매개 변수에 적용했습니다.
