# <a name="compiler-error-c3157"></a>컴파일러 오류 C3157
ParamArray 특성은 마지막 매개 변수에 적용할 수 있습니다.  
  
 [ParamArray](https://msdn.microsoft.com/en-us/library/system.paramarrayattribute.aspx) 특성이 잘못 된 매개 변수에 적용 되었습니다.
