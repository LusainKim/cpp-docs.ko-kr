# 컴파일러 오류 C2544
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\(\)' 연산자에는 '\)'가 있어야 합니다.  
  
 함수 호출 연산자에서 왼쪽 괄호가 없습니다.  
  
 이 오류는 매크로 확장으로 인해 발생할 수 있습니다.
