# <a name="compiler-error-c2700"></a>컴파일러 오류 C2700
'identifier': Throw될 수 없습니다. 자세한 내용을 보려면 /W4를 사용하세요.  
  
 개체를 throw할 수 없습니다. 자세한 진단 정보를 보려면 [/W4](../../build/reference/compiler-option-warning-level.md) 를 사용하여 컴파일합니다.
