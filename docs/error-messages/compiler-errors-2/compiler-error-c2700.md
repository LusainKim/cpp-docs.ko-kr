# <a name="compiler-error-c2700"></a>컴파일러 오류 C2700
'identifier': Throw될 수 없습니다. 자세한 내용을 보려면 /W4를 사용하세요.  
  
 개체를 throw할 수 없습니다. 사용 하 여 컴파일 [/W4](../../build/reference/compiler-option-warning-level.md) 자세한 진단 정보에 대 한 합니다.
