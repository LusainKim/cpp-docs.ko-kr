# 컴파일러 오류 C2654
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 멤버 함수 외부에서 멤버에 액세스하려고 했습니다.  
  
 멤버가 선언 부분에서 액세스되었습니다.  멤버 데이터는 멤버 함수에서만 액세스할 수 있습니다.  
  
 이 오류는 선언에서 변수를 초기화할 때 발생할 수 있습니다.  생성자를 이 용도로 사용하십시오.
