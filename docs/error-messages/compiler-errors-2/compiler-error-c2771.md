# 컴파일러 오류 C2771
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import는 전역 범위 또는 네임스페이스 범위에서만 사용할 수 있습니다.  
  
 `#import` 지시문은 함수나 구조체 등에 사용할 수 없습니다.
