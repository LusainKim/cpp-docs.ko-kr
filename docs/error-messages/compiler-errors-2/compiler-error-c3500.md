# 컴파일러 오류 C3500
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ProgID 'progid'이\(가\) 잘못되었습니다.  
  
 `#import` 문에 잘못된 progid가 지정되었습니다.  Windows 레지스트리를 확인하여 올바른 progid를 지정하십시오.
