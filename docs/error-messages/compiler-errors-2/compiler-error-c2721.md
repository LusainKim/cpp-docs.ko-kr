# 컴파일러 오류 C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'specifier' : 연산자 키워드 및 형식 사이에는 저장소 클래스 지정자를 사용할 수 없습니다.  
  
 사용자 정의 형식 변환이 모든 저장소 클래스에 적용되므로 형식 변환에 저장소 클래스를 지정할 수 없습니다.
