# <a name="compiler-error-c2721"></a>컴파일러 오류 C2721
'specifier': 연산자 키워드 및 형식 간의 저장소 클래스 지정자  
  
 사용자 정의 형식 변환 모든 저장소 클래스에 적용 되는 형식 변환에는 저장소 클래스를 지정할 수 없습니다.
