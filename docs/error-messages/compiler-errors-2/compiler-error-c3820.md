# 컴파일러 오류 C3820
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'name': 이니셜라이저는 관리되어야 합니다.  
  
 관리되는 형식에 대한 *name* 이니셜라이저도 관리해야 합니다.  
  
### 이 오류를 해결하려면  
  
1.  관리되는 이니셜라이저 메서드를 지정합니다.
