# 컴파일러 오류 C2859
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename은\(는\) 이 미리 컴파일된 헤더가 만들어졌을 때 사용된 형식 파일이 아닙니다. 미리 컴파일된 헤더를 다시 만드십시오.  
  
 프로젝트 데이터베이스 및 미리 컴파일된 헤더 파일은 정보의 일관성을 위해 함께 만들어야 합니다.  프로젝트를 다시 빌드하여 미리 컴파일된 헤더를 다시 만드십시오.  
  
 미리 컴파일된 헤더에 대한 자세한 내용은 [\/Y\(미리 컴파일된 헤더\)](../../build/reference/y-precompiled-headers.md)를 참조하십시오.
