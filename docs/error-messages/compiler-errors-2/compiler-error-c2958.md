# 컴파일러 오류 C2958
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

왼쪽 구분 기호\('location'에 있는\)이\(가\) 정확하게 짝이 맞지 않습니다.  
  
 구분 기호의 짝이 맞지 않습니다.  
  
 이 오류는 함수 템플릿 선언에서 괄호의 짝이 맞지 않는 경우에 발생할 수 있습니다.
