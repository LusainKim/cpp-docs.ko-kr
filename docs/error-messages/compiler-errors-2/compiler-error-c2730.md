# 컴파일러 오류 C2730
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class': 자체의 기본 클래스일 수 없습니다.  
  
 재귀 기본 클래스를 사용할 수 없습니다. 다른 클래스를 기본 클래스로 지정합니다.
