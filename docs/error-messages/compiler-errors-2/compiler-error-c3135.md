# 컴파일러 오류 C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'property' : 속성에는 'const' 또는 'volatile' 형식을 사용할 수 없습니다.  
  
 속성에는 [const](../../cpp/const-cpp.md) 및 [volatile](../../cpp/volatile-cpp.md) 키워드를 사용할 수 없습니다.
