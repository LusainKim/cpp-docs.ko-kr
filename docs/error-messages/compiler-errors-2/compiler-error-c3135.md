# <a name="compiler-error-c3135"></a>컴파일러 오류 C3135
'property': 속성에는 'const' 여야 합니다. 또는 'volatile' 형식을  
  
 [const](../../cpp/const-cpp.md) 및 [휘발성](../../cpp/volatile-cpp.md) 키워드는 속성에 허용 되지 않습니다.
