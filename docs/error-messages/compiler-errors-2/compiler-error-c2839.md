# 컴파일러 오류 C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

오버로드된 'operator \-\>'의 'type' 반환 형식이 잘못되었습니다.  
  
 `->` 연산자는 클래스, 구조체 또는 공용 구조체나 이에 대한 참조를 반환해야 합니다.
