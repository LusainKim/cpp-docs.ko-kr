# 컴파일러 오류 C2532
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 참조에 대한 한정자가 잘못되었습니다.  
  
 참조가 변경되었습니다.  다른 개체를 참조하도록 참조를 수정할 수 없습니다.  대신에 포인터를 사용하십시오.
