# 컴파일러 오류 C3670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'override' : 액세스할 수 없는 기본 클래스 메서드 'method'을\(를\) 재정의할 수 없습니다.  
  
 액세스 수준에 따라 파생 형식으로 사용할 수 있는 함수만 재정의할 수 있습니다.  자세한 내용은 [명시적으로 재정의](../../windows/explicit-overrides-cpp-component-extensions.md)를 참조하십시오.  
  
 다음 샘플에서는 C3670 오류가 발생하는 경우를 보여 줍니다.  
  
```  
// C3670.cpp  
// compile with: /clr /c  
public ref class C {  
// Uncomment the following line to resolve.  
// public:  
   virtual void g() { }  
};  
  
public ref class D : public C {  
public:  
   virtual void f() new sealed = C::g {};   // C3670  
};  
```
