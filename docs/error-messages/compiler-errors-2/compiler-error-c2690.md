# <a name="compiler-error-c2690"></a>컴파일러 오류 C2690
'operator': WinRT 또는 관리되는 배열에서 포인터 산술 연산을 수행할 수 없습니다.  
  
 포인터 산술 연산은 WinRT 또는 관리되는 배열에서 허용되지 않습니다. 배열 인덱스 표기법을 사용하여 배열을 트래버스합니다.
