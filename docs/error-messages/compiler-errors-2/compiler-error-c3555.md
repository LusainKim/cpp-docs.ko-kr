# 컴파일러 오류 C3555
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'decltype'의 인수가 잘못되었습니다.  
  
 `decltype(` `expression` `)` 형식 지정자에 대한 인수가 유효한 식이 아닙니다.  
  
> [!NOTE]
>  C3555는 발생할 가능성이 거의 없습니다. 컴파일러에서 C3555가 발생하는 경우는 내부 컴파일러 오류 발생이 원인일 수 있습니다.
