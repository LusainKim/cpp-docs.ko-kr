# 컴파일러 오류 C3501
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ProgID 'progid'에 대해 등록된 typelib가 없습니다.  
  
 지정된 progid의 클래스 ID에 관련된 형식 라이브러리가 없습니다.  따라서 `#import` 문에 이 progid를 전달할 수 없습니다.
