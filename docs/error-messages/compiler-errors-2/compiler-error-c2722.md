# <a name="compiler-error-c2722"></a>컴파일러 오류 C2722
':: operator': 다음 연산자 명령을; 잘못 되었습니다. 'operator 연산자'를 사용 하 여  
  
 `operator` 문 가져오거나 다시 정의한 요소 `::new` 또는 `::delete`합니다. `new` 및 `delete` 는 전역 연산자 하므로 범위 결정 연산자 (`::`)은 의미가 없습니다. 제거는 `::` 연산자입니다.
