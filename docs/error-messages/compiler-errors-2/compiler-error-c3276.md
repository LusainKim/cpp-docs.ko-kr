# <a name="compiler-error-c3276"></a>컴파일러 오류 C3276
'keyword': 종료 처리 하는 동안 __finally/finally 블록 밖으로 점프 동작이 정의 되지 않았습니다  
  
 이 오류는 동일는 [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) 경고 합니다. 그러나 /clr을 사용 하는 경우이 조건은 비활성화할 수 없습니다와 [경고](../../preprocessor/warning.md) pragma입니다.
