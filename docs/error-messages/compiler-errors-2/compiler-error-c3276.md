# 컴파일러 오류 C3276
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'keyword' : 종료 처리를 수행하는 동안 \_\_finally\/finally 블록에서의 점프 동작이 정의되지 않았습니다.  
  
 이 오류는 [C4532](../../error-messages/compiler-warnings/compiler-warning-level-1-c4532.md) 경고와 동일합니다.  그러나 \/clr를 사용하는 경우에는 [warning](../../preprocessor/warning.md) pragma에서 이 조건을 비활성화할 수 없습니다.
