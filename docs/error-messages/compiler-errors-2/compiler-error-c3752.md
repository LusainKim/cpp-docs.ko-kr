# 컴파일러 오류 C3752
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'attribute class': 특성을 분류할 수 없습니다. 이 컨텍스트에서는 'keyword'을\(를\) 사용하지 않아야 합니다.  
  
 사용자 정의 특성이 잘못 적용되었습니다.
