# <a name="compiler-error-c3166"></a>컴파일러 오류 C3166
'포인터': 'type'의 구성원으로 내부 __gc 포인터에 대 한 포인터를 선언할 수 없습니다  
  
컴파일러에서 잘못 된 포인터 선언 발견 (한 `__nogc` 에 대 한 포인터는 `__gc` 포인터입니다.). 
  
C3166는 사용 되지 않는 컴파일러 옵션을 사용 하 여 연결할 수만 **/clr:oldSyntax**합니다.
