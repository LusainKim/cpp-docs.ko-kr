# 컴파일러 오류 C2945
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

명시적 인스턴스화는 템플릿\-클래스 특수화를 참조하지 않습니다.  
  
 템플릿 기반이 아닌 항목을 명시적으로 인스턴스화할 수 없습니다.
