# 컴파일러 오류 C3805
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token': 예기치 않은 토큰이 있습니다. '}' 또는 ','가 필요합니다.  
  
 속성을 정의할 때 잘못된 토큰을 발견했습니다.  잘못된 토큰을 제거하십시오.
