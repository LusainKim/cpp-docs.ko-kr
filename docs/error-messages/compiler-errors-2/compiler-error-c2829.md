# 컴파일러 오류 C2829
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator operator'에 가변 매개 변수 목록을 사용할 수 없습니다.  
  
 두 연산자만 변수 매개 변수 목록을 사용할 수 있습니다.  
  
-   함수 호출 `()`  
  
-   `new`
