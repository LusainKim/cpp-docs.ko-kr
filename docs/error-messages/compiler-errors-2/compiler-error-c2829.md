# <a name="compiler-error-c2829"></a>컴파일러 오류 C2829
'operator 연산자' 가변 매개 변수 목록을 사용할 수 없습니다.  
  
 두 연산자만 변수 매개 변수 목록을 사용할 수 있습니다.  
  
-   함수 호출`()`  
  
-   `new`
