# 컴파일러 오류 C2656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 함수를 비트 필드로 사용할 수 없습니다.  
  
 함수가 비트 필드의 멤버로 선언되었습니다.  
  
 이 오류는 생성자 이니셜라이저 목록에 구문 오류가 있는 경우에 발생할 수 있습니다.
