# 컴파일러 오류 C2426
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token' : 'context'에 잘못된 연산자가 있습니다.  
  
 이 컨텍스트에서 토큰을 연산자로 사용할 수 없습니다.  예를 들어, 인덱스 연산자를 중첩할 수 없습니다.
