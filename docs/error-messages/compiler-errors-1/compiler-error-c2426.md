# <a name="compiler-error-c2426"></a>컴파일러 오류 C2426
'token': '컨텍스트'에 잘못 된 연산자  
  
 이 컨텍스트에서 연산자로 토큰을 사용할 수 없습니다. 예를 들어 인덱스 연산자를 중첩할 수 없습니다.
