# 심각한 오류 C1351
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'dll' dll을 로드하는 동안 오류가 발생했습니다. 호환되지 않는 버전입니다.  
  
 잘못된 버전의 DLL이 발견되었습니다.  
  
 이 오류는 설치 관련 문제를 나타냅니다. 제품 CD에서 DLL을 다시 설치해야 합니다.
