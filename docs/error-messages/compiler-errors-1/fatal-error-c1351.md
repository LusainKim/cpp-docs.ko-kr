# <a name="fatal-error-c1351"></a>심각한 오류 C1351
dll 'dll' 로드 오류: 호환 되지 않는 버전  
  
 잘못 된 버전의 DLL 찾았습니다.  
  
 설치에 문제가 나타내며 제품 CD에서 DLL을 다시 설치 해야 합니다.
