# 심각한 오류 C1065
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컴파일러 한계 : 태그가 부족합니다.  
  
 소스 파일에 있는 클래스, 구조체, 공용 구조체, 네임스페이스 또는 열거형이 65,523개를 넘습니다.  
  
 파일을 좀 더 작은 여러 개의 파일로 분할하십시오.
