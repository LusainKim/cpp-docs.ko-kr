# 컴파일러 오류 C2145
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구문 오류: 식별자 앞에 'token'이 없습니다.  
  
 컴파일러에서는 `token`을 예상했지만 대신 식별자를 발견했습니다.  
  
 이 오류는 블록의 마지막 선언 뒤에 세미콜론이 누락된 경우 발생할 수 있습니다.
