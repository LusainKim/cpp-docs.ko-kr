# <a name="fatal-error-c1064"></a>심각한 오류 C1064
컴파일러 한계: 토큰 내부 버퍼 오버플로가 발생 했습니다.  
  
 식별자는 식별자에 사용 되는 내부 버퍼의 길이 초과 합니다. 이름을 줄이십시오.
