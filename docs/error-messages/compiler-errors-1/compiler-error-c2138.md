# 컴파일러 오류 C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

멤버 없이 열거를 정의하는 것은 잘못입니다.  
  
 \/Za\(Microsoft 확장 사용 안 함\)를 선택한 경우에는 열거형에 멤버가 하나 이상 있어야 합니다.
