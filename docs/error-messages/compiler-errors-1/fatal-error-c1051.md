# 심각한 오류 C1051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'pdbfile' 프로그램 데이터베이스 파일에 사용되지 않는 형식이 있습니다. 삭제한 후 다시 컴파일하십시오.  
  
 컴파일러는 이전 버전 번호를 사용하는 프로그램 데이터베이스 파일을 업데이트할 수 없습니다.  해당 파일을 삭제한 다음 **\/Zi** 또는 **\/ZI**을 사용하여 프로그램을 다시 컴파일하십시오.  자세한 내용은 [\/Z7, \/Zi, \/ZI\(디버깅 정보 형식\)](../../build/reference/z7-zi-zi-debug-information-format.md)을 참조하십시오.
