# 컴파일러 오류 C2223
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-\>identifier' 왼쪽은 구조체\/공용 구조체를 가리켜야 합니다.  
  
 `->`의 왼쪽 피연산자가 클래스, 구조체 또는 공용 구조체에 대한 포인터가 아닙니다.  
  
 이 오류는 왼쪽 피연산자가 정의되지 않은 변수\(`int` 형식\)인 경우에 발생할 수 있습니다.
