# 심각한 오류 C1107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'file' 어셈블리를 찾을 수 없습니다. \/AI를 사용하거나 LIBPATH 환경 변수를 설정하여 어셈블리 검색 경로를 지정하십시오.  
  
 메타데이터 파일이 컴파일러가 찾을 수 없는 [\#using](../../preprocessor/hash-using-directive-cpp.md) 지시문에 전달되었습니다.  
  
 `#using`에 대한 항목에 설명된 LIBPATH 및 [\/AI](../../build/reference/ai-specify-metadata-directories.md) 컴파일러 옵션을 사용하면 컴파일러가 참조된 메타데이터 파일을 찾을 디렉터리를 지정할 수 있습니다.
