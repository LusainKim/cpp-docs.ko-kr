# 컴파일러 오류 C2302
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'.~identifier' 왼쪽에는 클래스\/구조체\/공용 구조체 형식이 있어야 합니다.  
  
 점\(.\) 연산자 왼쪽에 있는 식이 클래스, 구조체 또는 공용 구조체가 아닙니다.
