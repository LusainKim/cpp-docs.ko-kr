# 컴파일러 오류 C2413
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'token' : 맞춤 크기가 잘못되었습니다.  
  
 `ALIGN` 지시문에 사용된 크기가 누락되었거나 유효한 범위에 속하지 않습니다.
