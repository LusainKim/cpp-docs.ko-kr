# 컴파일러 오류 C2121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#': 잘못된 문자: 매크로를 확장한 결과입니다.  
  
 문자열화 연산자\(\#\) 대신 토큰 전달 연산자\(\#\#\)를 사용하는 잘못된 매크로가 잘못된 \# 문자를 삽입했습니다.
