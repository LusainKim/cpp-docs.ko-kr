# <a name="compiler-error-c2121"></a>컴파일러 오류 C2121
'#': 잘못 된 문자: 매크로 확장의 결과  
  
 잘못 된 매크로가 문자열 화 연산자 대신 토큰 붙여넣기 연산자 (#)를 사용 하 여 잘못 된 # 문자 삽입 (#).
