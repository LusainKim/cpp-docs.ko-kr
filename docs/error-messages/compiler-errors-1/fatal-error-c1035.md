# <a name="fatal-error-c1035"></a>심각한 오류 C1035
식이 너무 복잡합니다. 간단하게 만드세요.  
  
 컴파일러가 복잡한 식에 대한 코드를 생성하지 못했습니다. 식을 더 간단한 식으로 분할하고 다시 컴파일합니다.
