# 심각한 오류 C1074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'IDB'는 PDB 파일 filename에 적합한 확장명이 아닙니다.  
  
 컴파일러에서 프로그램 데이터베이스는 .pdb 확장명을 가져야 합니다.
