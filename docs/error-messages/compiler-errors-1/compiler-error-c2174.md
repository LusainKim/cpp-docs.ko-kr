# 컴파일러 오류 C2174
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 실제 매개 변수의 형식이 'void'입니다. 매개 변수 number1, 매개 변수 목록 number2  
  
 매개 변수 목록 `number2`에 전달된 매개 변수 `number1`은\(는\) `void` 매개 변수입니다.  매개 변수에는 `void` 형식을 사용할 수 없습니다.  대신 `void*`를 사용하십시오.
