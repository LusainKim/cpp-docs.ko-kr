# <a name="fatal-error-c1350"></a>심각한 오류 C1350
dll 'dll' 로드 오류: dll을 찾을 수 없습니다  
  
 시도 된 작업을 지 원하는 DLL은 찾을 수 없습니다.  
  
 설치에 문제가 나타내며 제품 CD에서 DLL을 다시 설치 해야 합니다.
