# 컴파일러 오류 C2154
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type': 컴파일러 내장 형식 특성 '\_\_underlying\_type'에 대한 인수로는 열거형 형식만 허용됩니다.  
  
 열거형 형식의 기본 형식만 가져올 수 있습니다.  
  
 자세한 내용은 [Compiler Support for Type Traits](../../windows/compiler-support-for-type-traits-cpp-component-extensions.md)을 참조하세요.
