# 컴파일러 오류 C2406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 'context'에 이름을 정의하지 않았습니다.  
  
 정의되지 않은 식별자가 `SIZE`, `LENGTH` 또는 멤버 선택\(.\) 연산자와 함께 사용되었습니다.
