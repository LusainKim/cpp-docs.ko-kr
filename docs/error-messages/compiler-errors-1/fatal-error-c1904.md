# <a name="fatal-error-c1904"></a>심각한 오류 C1904

 이 오류는 특성 공급자 오류를 나타냅니다.  
  
 오류에 대 한 환경을 기록, 문제를 파악 하 여 재현 가능한 테스트 사례를 문의 한 다음 [의견 보내기](/visualstudio/ide/talk-to-us)합니다.
