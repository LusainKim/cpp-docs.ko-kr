# 심각한 오류 C1309
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

C2.DLL과 PGODB\<버전\>.DLL의 버전이 일치하지 않습니다.  
  
 [프로필 기반 최적화](../../build/reference/profile-guided-optimizations.md)를 빌드하고 사용하기 위해 사용하는 도구 집합에 일치하지 않는 구성 요소가 포함되어 있습니다.  이 오류를 수동으로 해결할 수 없는 경우에는 Visual C\+\+를 다시 설치하세요.
