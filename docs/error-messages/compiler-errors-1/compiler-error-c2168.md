# 컴파일러 오류 C2168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 내장 함수에 대한 실제 매개 변수가 너무 적습니다.  
  
 `intrinsic` 함수에 대한 참조에 매개 변수가 너무 적습니다.
