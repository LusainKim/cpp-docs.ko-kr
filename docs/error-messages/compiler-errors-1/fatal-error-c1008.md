# 심각한 오류 C1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

입력 파일을 지정하지 않았습니다.  
  
 컴파일러에 컴파일할 C 또는 C\+\+ 원본 파일이 지정되지 않았습니다.**CL** 명령줄 및 환경 변수에서 파일 이름 사용을 확인하세요.
