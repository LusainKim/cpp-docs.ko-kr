# 컴파일러 오류 C2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-': 포인터 빼기에는 정수 계열 또는 포인터 피연산자가 있어야 합니다.  
  
 서로 다른 형식을 가리키는 포인터를 빼려고 했습니다.
