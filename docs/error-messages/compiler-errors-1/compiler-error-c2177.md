# 컴파일러 오류 C2177
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

상수가 너무 큽니다.  
  
 상수 값이 할당된 변수 형식에 대해 너무 큽니다.  
  
 다음 샘플에서는 C2177을 생성합니다.  
  
```  
// C2177.cpp int main() { int a=18446744073709551616;   // C2177 int b=18446744073709551615;   // OK }  
```
