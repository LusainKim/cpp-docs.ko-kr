# 심각한 오류 C1196
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 형식 라이브러리 'typelib'의 이 식별자는 올바른 C\+\+ 식별자가 아닙니다.  
  
 형식 라이브러리의 식별자 중 하나가 올바른 C\+\+ 식별자가 아닙니다. 형식 라이브러리는 [\#import](../../preprocessor/hash-import-directive-cpp.md)와 함께 사용할 수 없습니다.
