# 컴파일러 오류 C2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 특성이 서로 다른 함수에 대한 포인터입니다.  
  
 하나의 호출 규칙\(`__cdecl`, `__stdcall` 또는 `__fastcall`\)을 포함하는 함수에 대한 포인터가 다른 호출 규칙을 포함하는 함수에 대한 포인터에 할당되었습니다.
