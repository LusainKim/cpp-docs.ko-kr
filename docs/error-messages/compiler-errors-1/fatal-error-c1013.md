# <a name="fatal-error-c1013"></a>심각한 오류 C1013
컴파일러 한계 : 여는 괄호가 너무 많습니다.  
  
 단일 식에 너무 많은 괄호 수준이 포함되어 있습니다. 식을 단순화하거나 여러 개의 문으로 분리합니다.  
  
 Visual C++ 6.0 SP3 이전 버전에서는 단일 식의 중첩된 괄호에 대한 제한이 59개였습니다. 현재 중첩된 괄호에 대한 제한은 256개입니다.
