# 컴파일러 오류 C2471
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'file' 프로그램 데이터베이스를 업데이트할 수 없습니다.  
  
 컴파일러가 데이터베이스 파일에 작성할 수 없습니다.  
  
### 이 문제를 해결하려면  
  
-   디스크에 충분한 공간이 있는지 확인 하십시오.  
  
-   파일이 읽기 전용이 아닌지 확인 합니다.  
  
-   공유 위반 되는지 확인 합니다.
