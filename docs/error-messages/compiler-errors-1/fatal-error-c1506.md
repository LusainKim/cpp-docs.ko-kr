# 심각한 오류 C1506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

복구할 수 없는 블록 범위 지정 오류입니다.  
  
 블록이 너무 커서 컴파일하지 못했습니다.  
  
### 문제 해결을 위하여 확인해 볼 수 있는 원인  
  
1.  중괄호의 짝이 맞지 않습니다.  
  
2.  예외적인 대형 함수 또는 클래스가 왔습니다.
