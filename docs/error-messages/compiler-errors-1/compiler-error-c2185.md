# 컴파일러 오류 C2185
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 기반 할당이 잘못되었습니다.  
  
 레지스터 변수나 자동\(또는 자동 지역\) 변수가 `__based`로 선언되었습니다.  전역 변수만 `__based`로 선언할 수 있습니다.
