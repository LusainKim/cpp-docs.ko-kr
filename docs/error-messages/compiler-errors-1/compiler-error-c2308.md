# 컴파일러 오류 C2308
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

일치하지 않는 문자열을 연결하고 있습니다.  
  
 와이드 문자열과 비와이드 문자열을 모두 연결하도록 지정되었습니다.  와이드 문자열과 비와이드 문자열을 연결할 수 없습니다.
