# <a name="compiler-error-c2315"></a>컴파일러 오류 C2315
'type1': 'type2'가 줄 number에서 참조를 catch했습니다.  
  
 예외 형식이 이전 처리기에 의해 처리됩니다. 두 번째 catch에 대한 참조가 첫 번째 catch의 형식입니다.
