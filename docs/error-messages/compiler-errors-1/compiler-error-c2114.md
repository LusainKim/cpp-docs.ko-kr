# 컴파일러 오류 C2114
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator': 왼쪽이 포인터이므로 오른쪽에는 정수 계열 값이 있어야 합니다.  
  
 `operator`의 왼쪽 피연산자가 포인터이므로 오른쪽 피연산자는 정수 값이어야 합니다.
