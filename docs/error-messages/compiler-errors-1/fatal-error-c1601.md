# <a name="fatal-error-c1601"></a>심각한 오류 C1601
지원되지 않는 인라인 어셈블리 opcode입니다.  
  
 이 오류는 컴파일러.exe 파일을 설치 실패로 인해 발생 했을 수 있는 불일치를 나타냅니다. 예를 들어 서비스 팩 하지만 프로세서 팩 하지 설치가 있습니다. 필요한 모든 제품을 설치 합니다.
