# 심각한 오류 C1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

CryptoAPI에 대한 호출이 실패했습니다.  
  
 컴파일러가 CryptoAPI를 호출했지만 이 호출에 실패했습니다.  Visual Studio를 다시 설치하십시오. 운영 체제를 다시 설치해야 할 수도 있습니다.
