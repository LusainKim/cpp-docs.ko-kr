# <a name="compiler-error-c2403"></a>컴파일러 오류 C2403
'identifier': 레지스터는 'context'에서 기본/인덱스이어야 합니다.  
  
 간접 메모리 피연산자에 사용하는 레지스터는 이 컨텍스트에서 기본 또는 인덱스 레지스터이어야 합니다.
