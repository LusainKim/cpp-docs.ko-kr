# 컴파일러 오류 C2175
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'locale' : 로캘이 잘못되었습니다.  
  
 지정한 로캘이 잘못되었습니다.  지원되는 로캘을 알아보려면 *Run\-Time Library Reference*에서 [Language and Country\/Region Strings](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)을 참조하십시오.
