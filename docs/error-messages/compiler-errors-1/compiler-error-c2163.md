# <a name="compiler-error-c2163"></a>컴파일러 오류 C2163
'function': 내장 함수로 사용할 수 없음  
  
 `intrinsic` 또는 `function` pragma 내장 형식에서 사용할 수 없는 함수를 나열 합니다. 예를 들어 특정 내장 함수는 /clr 프로그래밍을 사용 하는 프로그램을 컴파일할 때 사용할 수 없습니다.
