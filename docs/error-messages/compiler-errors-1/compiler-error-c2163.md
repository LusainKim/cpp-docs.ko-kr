# 컴파일러 오류 C2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 내장 함수로 사용할 수 없습니다.  
  
 `intrinsic` 또는 `function` pragma는 내장 형식으로 사용할 수 없는 함수를 표시합니다.  예를 들어, \/clr 프로그래밍을 사용하는 프로그램을 컴파일할 경우에는 특정 내장 형식을 사용할 수 없습니다.
