# 컴파일러 오류 C2405
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

오프셋이 있는 간단한 전방 참조가 잘못되었습니다.  
  
 간단한 전방 참조는 레이블만 참조해야 합니다.  추가 오프셋은 사용할 수 없습니다.
