# 컴파일러 오류 C2077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier'은\(는\) 스칼라 필드 이니셜라이저가 아닙니다.  
  
 비스칼라\(구조체, 공용 구조체, 배열 또는 클래스\)를 사용하여 비트 필드를 초기화하려고 했습니다.  정수 또는 부동 소수점 값을 사용하십시오.
