# 컴파일러 오류 C2421
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'context'의 레지스터와 함께 PTR 연산자를 사용했습니다.  
  
 `register` 피연산자에 `PTR` 연산자를 사용할 수 없습니다.
