# 컴파일러 오류 C2418
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

브라우저 파일을 삭제할 수 없습니다. filename  
  
 컴파일러가 브라우저 파일을 삭제하지 못했습니다.  
  
### 문제 해결을 위하여 확인해 볼 수 있는 원인  
  
1.  읽기 전용 파일입니다.  
  
2.  파일이 다른 프로세스에서 사용되고 있습니다.
