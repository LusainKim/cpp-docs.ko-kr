# 컴파일러 오류 C2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구문 오류: 형식 한정자는 '\*' 뒤에 있어야 합니다.  
  
 형식 한정자\(`const` 또는 `volatile`\)가 허용되지 않은 위치에 표시됩니다.
