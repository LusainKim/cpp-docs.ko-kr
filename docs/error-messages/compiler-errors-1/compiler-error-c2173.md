# <a name="compiler-error-c2173"></a>컴파일러 오류 C2173
'function': 실제 매개 변수가 포인터가 아닙니다. 매개 변수 number1, 매개 변수 목록 number2  
  
 `number1` 매개 변수 목록에 전달된 `number2` 매개 변수는 포인터가 아닙니다. 함수에는 포인터가 필요합니다.
