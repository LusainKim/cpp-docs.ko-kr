# 컴파일러 오류 C2173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 실제 매개 변수가 포인터가 아닙니다. 매개 변수 number1, 매개 변수 목록 number2  
  
 `number2` 매개 변수 목록에 전달된 `number1` 매개 변수는 포인터가 아닙니다. 함수에는 포인터가 필요합니다.
