# 컴파일러 오류 C2018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'hexnumber' 문자를 인식할 수 없습니다.  
  
 소스 파일에 16진수로 식별되는 예기치 않은 ASCII 문자가 있습니다.  이 오류를 해결하려면 해당 문자를 제거하십시오.
