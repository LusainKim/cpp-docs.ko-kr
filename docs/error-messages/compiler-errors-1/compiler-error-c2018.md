# <a name="compiler-error-c2018"></a>컴파일러 오류 C2018
'hexnumber' 문자를 인식할 수 없습니다.  
  
 소스 파일에는 해당 16 진수 번호로 식별 되는 예기치 않은 ASCII 문자를 포함 합니다. 이 오류를 해결 하려면 문자를 제거 합니다.
