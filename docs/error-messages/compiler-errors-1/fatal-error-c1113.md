# 심각한 오류 C1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#using이 'file'에서 실패했습니다.  
  
 MSIL\(Microsoft Intermediate Language\) 형식의 파일만 [\#using](../../preprocessor/hash-using-directive-cpp.md) 지시문에 전달할 수 있습니다.  [\/clr](../../build/reference/clr-common-language-runtime-compilation.md) 컴파일러 옵션을 사용하면 MSIL 출력 파일을 만들 수 있습니다.  다른 [!INCLUDE[vsprvs](../../assembler/masm/includes/vsprvs_md.md)] 언어도 MSIL 파일을 만듭니다.
