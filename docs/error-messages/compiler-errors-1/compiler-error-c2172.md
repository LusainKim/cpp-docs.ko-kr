# 컴파일러 오류 C2172
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 실제 매개 변수가 포인터가 아닙니다. 매개 변수 number  
  
 매개 변수 `number`가 포인터가 아닙니다. 함수에는 포인터가 필요합니다.
