# 컴파일러 오류 C2205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 블록 범위를 사용하여 extern 변수를 초기화할 수 없습니다.  
  
 `extern` 변수는 함수에서 초기화할 수 없습니다.
