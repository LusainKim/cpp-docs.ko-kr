# <a name="fatal-error-c1313"></a>심각한 오류 C1313
컴파일러 한계: 형식 블록 수 number 보다 깊은 수준으로 중첩 될 수 없습니다  
  
 예외 처리 등의 구조적된 예외 처리 블록이 너무 많이 중첩 되었습니다.  코드를 단순화하세요.
