# 심각한 오류 C1313
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컴파일러 한도 : 형식 블록은 number보다 깊은 수준으로 중첩될 수 없습니다.  
  
 예외 처리\(또는 구조적 예외 처리\) 블록이 너무 깊이 중첩되었습니다.  코드를 단순하게 변경해야 합니다.
