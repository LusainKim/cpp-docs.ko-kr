# 컴파일러 오류 C2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

함수 형식으로 잘못 캐스팅했습니다.  
  
 ANSI C에서는 함수 포인터와 데이터 포인터 간에 캐스팅하지 못합니다.
