# <a name="compiler-error-c2066"></a>컴파일러 오류 C2066
함수 유형으로 캐스팅 올바르지 않습니다.  
  
 ANSI c에서는 함수 포인터와 데이터 포인터 간의 캐스팅 없습니다.
