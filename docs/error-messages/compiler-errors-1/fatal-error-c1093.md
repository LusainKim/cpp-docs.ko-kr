# 심각한 오류 C1093
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function name' API 호출이 'location of call'에서 실패했습니다 : 'text from run\-time'  
  
 .NET 함수를 호출하지 못했습니다.  ***text from run\-time*** 부분은 COM 런타임에서 제공되는 경우도 있고 그렇지 않은 경우도 있습니다.  
  
 시스템 오류 메시지에 대한 자세한 내용은 winerror.h 시스템 파일과 [FormatMessage](http://msdn.microsoft.com/library/windows/desktop/ms679351)를 참조하십시오.
