# 컴파일러 오류 C2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-': 포인터는 다른 포인터에서만 뺄 수 있습니다.  
  
 빼기 연산의 오른쪽 피연산자는 포인터지만 왼쪽 연산자는 포인터가 아닙니다.
