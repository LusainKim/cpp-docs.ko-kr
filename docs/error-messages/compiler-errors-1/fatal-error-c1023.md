# <a name="fatal-error-c1023"></a>심각한 오류 C1023
'file': pch에서 예기치 못한 오류가 발생했습니다. pch를 다시 빌드해 보세요.  
  
 C1023은 여러 가지 문제 중 하나로 인해 발생할 수 있으며, 미리 컴파일된 헤더 파일을 다시 빌드하면 해결할 수 있습니다.  
  
 C1023은 빌드된 컴퓨터가 아닌 다른 컴퓨터에서 미리 컴파일된 헤더 파일을 사용하려는 경우에 발생할 수 있습니다.
