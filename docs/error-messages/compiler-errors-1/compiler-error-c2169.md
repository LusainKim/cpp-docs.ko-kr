# 컴파일러 오류 C2169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function': 내장 함수이므로 정의할 수 없습니다.  
  
 함수에 대해 표시되는 함수 정의가 이미 `intrinsic` 함수로 선언되었습니다.
