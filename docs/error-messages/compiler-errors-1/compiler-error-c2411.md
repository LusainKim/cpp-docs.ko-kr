# <a name="compiler-error-c2411"></a>컴파일러 오류 C2411
'identifier': 'context'에 잘못 된 구조체/공용 구조체 멤버  
  
### <a name="to-fix-by-checking-the-following-possible-causes"></a>다음과 같은 가능한 원인을 확인하여 수정하려면  
  
1.  식별자는 표시 구조 또는 union이 컨텍스트에서 구성원이 아닙니다.  
  
2.  식별자는 구조체 또는 공용 구조체 멤버 선택 (.) 연산자와 함께 지정 된의 구성원이 아닙니다.
