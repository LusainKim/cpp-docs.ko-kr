# 심각한 오류 C1201
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

클래스 템플릿 정의에 구문 오류가 있어서 계속할 수 없습니다.  
  
 클래스 템플릿 정의를 구문 분석하는 동안 예기치 않은 오류가 발생했습니다.  
  
 다른 오류를 모두 수정하고 다시 컴파일합니다. 실패하는 경우 오류 상황을 확인하고 문제를 파악하여 재현 가능한 테스트 사례를 만든 다음 [Microsoft 기술 지원 서비스](../Topic/Talk%20to%20Us.md)에 문의합니다.
