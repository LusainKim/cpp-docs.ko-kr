# 컴파일러 오류 C2155
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?' : 왼쪽 피연산자가 잘못되었습니다. 산술 형식 또는 포인터 형식이 필요합니다.  
  
 `?` 왼쪽에 있는 식은 0과 비교할 수 없습니다.  0과 비교할 수 있는 산술 또는 포인터 식을 사용해야 합니다.
