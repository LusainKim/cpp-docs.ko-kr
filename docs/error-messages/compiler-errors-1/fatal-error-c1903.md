# <a name="fatal-error-c1903"></a>심각한 오류 C1903
이전 오류를 복구할 수 없습니다. 컴파일이 중지됩니다.  
  
 컴파일러가 너무 많은 오류를 발견하여 계속할 수 없습니다. 오류를 수정한 다음 다시 컴파일하세요.
