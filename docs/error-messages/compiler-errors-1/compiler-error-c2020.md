# 컴파일러 오류 C2020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member' : 'class' 멤버 재정의  
  
 기본 클래스나 구조체에서 상속된 멤버가 재정의됩니다.  상속된 멤버 중 기본 클래스에서 `virtual`로 선언되지 않은 멤버는 재정의할 수 없습니다.
