# 심각한 오류 C1057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

매크로 확장에서 예기치 않은 파일의 끝이 나타났습니다.  
  
 매크로 호출에서 오른쪽 괄호가 빠졌으므로 매크로 호출 인수를 모으는 중에 컴파일러가 소스 파일의 끝에 도달했습니다.
