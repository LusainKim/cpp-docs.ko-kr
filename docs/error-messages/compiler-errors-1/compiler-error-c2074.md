# 컴파일러 오류 C2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 'class\-key' 초기화에는 중괄호가 있어야 합니다.  
  
 지정된 클래스, 구조체 또는 공용 구조체 이니셜라이저 주변에 중괄호가 없습니다.
