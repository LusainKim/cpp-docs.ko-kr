# <a name="compiler-error-c2322"></a>컴파일러 오류 C2322
'identifier': 'dllimport' dllimport의 주소가 고정 주소가 아닙니다.  
  
 비고정 값이 `dllimport`로 선언된 함수의 주소로 제공되었습니다.
