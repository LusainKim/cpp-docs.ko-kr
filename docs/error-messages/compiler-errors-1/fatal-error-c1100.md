# 심각한 오류 C1100
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

OLE를 초기화할 수 없습니다 : system error message  
  
 컴파일러가 COM\(Component Object Model\) 라이브러리를 초기화할 수 없습니다.  [CoInitialize](http://msdn.microsoft.com/library/windows/desktop/ms678543)를 참조하십시오.
