# 컴파일러 오류 C2446
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator' : 'type1'에서 'type2'\(으\)로 변환되지 않았습니다.  
  
 컴파일러는 `type1`을 `type2`로 변환할 수 없습니다.  C\/C\+\+ 의미를 위반했으므로 이 변환은 잘못되었습니다.
