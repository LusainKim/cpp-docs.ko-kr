# <a name="compiler-error-c2412"></a>컴파일러 오류 C2412
'label': 대/소문자 구분 레이블을 재정의 했습니다  
  
 레이블이 현재 함수에 두 번 이상 정의 됩니다. 레이블 및 해당 참조의 맞춤법을 변경 합니다.
