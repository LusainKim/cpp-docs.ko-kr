# 컴파일러 오류 C2412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'label' : 대\/소문자를 구분하지 않는 레이블을 재정의했습니다.  
  
 현재 함수에서 레이블이 여러 번 정의되었습니다.  레이블의 철자와 해당 참조를 변경하십시오.
