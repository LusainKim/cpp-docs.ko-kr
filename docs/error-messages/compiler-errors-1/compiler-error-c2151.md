# 컴파일러 오류 C2151
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

언어 특성이 두 개 이상입니다.  
  
 함수에는 호출 규칙을 지정하는 키워드가 두 개 이상 있습니다\(`__cdecl`, `__stdcall` 또는 `__fastcall`\).
