# <a name="fatal-error-c1098"></a>심각한 오류 C1098
버전이 편집하며 계속하기 엔진과 일치하지 않습니다.  
  
 사용 중인 디버거 버전이 실행 파일을 만드는 데 사용된 컴파일러와 일치하지 않습니다. 다시 컴파일해도 문제가 해결되지 않는 경우 Visual C++를 다시 설치하여 적절한 버전의 디버거와 컴파일러를 사용해야 할 수도 있습니다.
