# 컴파일러 경고 (수준 1) C4031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

둘째 형식 매개 변수 목록이 첫째 목록보다 깁니다.  
  
 함수가 다른 형식 매개 변수로 다시 선언되었습니다.  컴파일러는 첫째 선언에 지정된 형식 매개 변수를 사용합니다.
