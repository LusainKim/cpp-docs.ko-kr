# 컴파일러 경고 (수준 1) C4403
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

PTR 연산자가 잘못되었습니다.  
  
 PTR 를 인라인 어셈블러 코드에 잘못 사용했습니다.  
  
 컴파일러에서 PTR 연산자를 무시했습니다.
