# 컴파일러 경고 (수준 1) C4344
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

동작 변경: 명시적 템플릿 인수를 사용하면 'function'이\(가\) 호출됩니다.  
  
 명시적 템플릿 인수를 사용하여 함수를 호출하면 명시적 인수를 지정하지 않았을 때와는 다른 함수가 호출됩니다.
