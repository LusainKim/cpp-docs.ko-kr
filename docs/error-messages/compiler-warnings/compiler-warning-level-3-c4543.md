# 컴파일러 경고 (수준 3) C4543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

삽입된 텍스트는 'no\_injected\_text' 특성 때문에 표시되지 않습니다.  
  
 소스 코드에 [no\_injected\_text](../../windows/no-injected-text.md) 특성이 표시되어 있으므로 컴파일러에서는 특성을 사용하여 코드를 삽입할 수 없습니다.  
  
 이 경고는 특성으로 인해 코드를 삽입할 수 없음을 나타냅니다.
