# 컴파일러 경고 (수준 1) C4113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1'은\(는\) 'identifier2'과\(와\) 매개 변수 목록이 다릅니다.  
  
 함수 포인터가 다른 함수 포인터에 할당되었지만 함수의 형식 매개 변수 목록이 일치하지 않습니다.  이 할당은 수정되지 않은 상태로 컴파일됩니다.
