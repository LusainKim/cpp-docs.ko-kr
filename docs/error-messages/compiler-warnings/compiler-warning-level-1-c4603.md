# 컴파일러 경고(수준 1) C4603
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\<identifier\>': 매크로가 정의되지 않았거나 미리 컴파일된 헤더 사용 후와 정의가 다릅니다.  
  
 *식별자* 자리 표시자에서 지정된 매크로가 미리 컴파일된 헤더가 사용된 후 다르거나 더 이상 정의되지 않습니다.
