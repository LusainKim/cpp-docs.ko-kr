# 컴파일러 경고(수준 1) C4168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컴파일러 한계: 디버거 형식을 벗어났습니다. 'database' 프로그램 데이터베이스를 삭제하고 다시 빌드합니다.
