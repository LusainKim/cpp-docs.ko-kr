# 컴파일러 경고(수준 1) C4187
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\#import 특성 'attribute1' 및 'attribute2'는 호환되지 않으므로 모두 무시됩니다.  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 문이 `no_implementation` 및 `implementation_only` 특성을 지정했습니다. 모두 무시됩니다.
