# <a name="compiler-warning-level-3-c4306"></a>컴파일러 경고(수준 3) C4306
**'**   
 ***식별자* ': 변환할 '**   
 ***type1* 'to'**   
 ***유형 2* ' 더 큰**  
  
 식별자 형식이 큰 포인터로 캐스팅 합니다. 새로운 유형의 채워지지 않은 상위 비트가 0으로 채워진 됩니다.  
  
 이 경고는 의도 하지 않은 변환을 나타낼 수 있습니다. 결과 포인터는 유효 하지 않게 될 수 있습니다.
