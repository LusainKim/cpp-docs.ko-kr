# <a name="compiler-warning-level-3-c4622"></a>컴파일러 경고(수준 3) C4622
미리 컴파일된 헤더를 'file' 개체 파일에 만드는 동안 생성된 디버그 정보를 덮어쓰고 있습니다.  
  
 으로 컴파일할 때 지정된 된 파일의 CodeView 정보가 손실 되었습니다는 [/Yu](../../build/reference/yu-use-precompiled-header-file.md) (미리 컴파일된 헤더 사용) 옵션입니다.  
  
 개체 파일 이름 바꾸기 (사용 하 여 [/Fo](../../build/reference/fo-object-file-name.md))를 만들거나 때 미리 컴파일된 헤더를 사용 하 여 파일을 선택한 다음 새 개체 파일을 사용 하 여 연결 합니다.
