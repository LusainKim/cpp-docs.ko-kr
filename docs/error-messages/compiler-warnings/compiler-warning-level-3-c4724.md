# 컴파일러 경고(수준 3) C4724
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0의 나머지 연산이 발생할 수 있습니다.  
  
 나머지 연산의 두 번째 피연산자가 컴파일 시간에 0으로 계산되어 정의되지 않은 결과를 제공합니다.
