# 컴파일러 경고 (수준 4) C4206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**비표준 확장을 사용했습니다 : 번역 단위가 비어 있습니다.**  
  
 파일이 전처리 후 빈 상태가 되었습니다.  
  
 이러한 확장을 사용하면 코드를 다른 컴파일러로 옮길 수 없습니다.  이는 ANSI 규격\([\/Za](../../build/reference/za-ze-disable-language-extensions.md)\)에서 오류를 발생시키며 C 소스 코드에만 적용됩니다.
