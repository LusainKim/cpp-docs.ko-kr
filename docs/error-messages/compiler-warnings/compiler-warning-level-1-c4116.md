# 컴파일러 경고 (수준 1) C4116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

괄호 안에 명명되지 않은 형식 정의가 있습니다.  
  
 명명되지 않은 구조체, 공용 구조체 또는 열거 형식이 괄호 식에 정의되어 있습니다.  이러한 형식 정의는 의미가 없습니다.  
  
 정의는 C 함수 호출에서 전역 범위를 사용하며,  C\+\+ 함수 호출에서는 함수가 호출될 때 사용되는 것과 같은 범위를 사용합니다.
