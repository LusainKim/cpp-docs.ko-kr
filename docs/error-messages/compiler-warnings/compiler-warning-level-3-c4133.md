# 컴파일러 경고 (수준 3) C4133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type' : 'type1'과\(와\) 'type2' 사이의 형식이 호환되지 않습니다.  
  
 이 경고는 형식이 서로 다른 두 포인터 사이에 뺄셈을 하려는 경우에 발생할 수 있습니다.  
  
 이 경고를 방지하려면 적절한 형식 변환을 제공하십시오.
