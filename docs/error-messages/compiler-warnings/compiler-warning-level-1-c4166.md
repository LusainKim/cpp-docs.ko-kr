# 컴파일러 경고(수준 1) C4166
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**생성자\/소멸자에 대한 호출 규칙이 잘못되었습니다.**  
  
 생성자 및 소멸자에는 플랫폼에 대한 기본값 이외의 다른 호출 규칙을 사용할 수 없습니다\(명시적으로 **\_\_clrcall**을 지정하는 경우 제외\).
