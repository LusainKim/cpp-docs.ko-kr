# 컴파일러 경고 (수준 2) C4007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 'attribute'이어야 합니다.  
  
 함수에 대한 필수 특성이 명시적으로 언급되지 않았습니다.  예를 들어, **main** 함수는 `__cdecl` 특성을 사용해야 합니다.  컴파일러는 특성을 강제로 지정합니다.
