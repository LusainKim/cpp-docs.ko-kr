# 컴파일러 경고 (수준 3) C4282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

그 다음 'type' 형식을 통해 'operator \-\>' 재귀가 발생했습니다.  
  
 C4281 경고 다음에 이 경고가 발생하면 **operator–\>가** `type`을 통해 자신을 호출한다는 것을 나타냅니다.
