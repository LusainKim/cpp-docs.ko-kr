# 컴파일러 경고 (수준 1) C4024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 형식 및 실제 매개 변수 'number'의 형식이 서로 다릅니다.  
  
 해당 형식 매개 변수와 실제 매개 변수의 형식이 서로 다릅니다.  컴파일러는 실제 매개 변수를 변경하지 않은 상태로 전달하며  수신 함수가 이 매개 변수 형식을 필요한 형식으로 변환합니다.
