# 컴파일러 경고 (수준 1) C4375
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

public이 아닌 메서드 'method2'이\(가\) 'method1'을\(를\) 재정의하지 않습니다.  
  
 다른 형식을 구현하는 형식이 재정의 메서드를 정의했으나 재정의 메서드가 public이 아니었습니다.  따라서 메서드가 기본 형식 메서드를 재정의하지 않습니다.
