# 컴파일러 경고 (수준 1) C4402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

PTR 연산자를 사용해야 합니다.  
  
 인라인 어셈블리 코드의 형식으로 참조하거나 캐스팅할 때 형식에 PTR 연산자 없이 피연산자를 사용했습니다.  
  
 컴파일러에서는 PTR 연산자를 사용한 것으로 간주합니다.
