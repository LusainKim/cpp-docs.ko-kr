# 컴파일러 경고 (수준 1) C4411
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 치환 레지스터가 되는 기호입니다.  
  
 식별자는 치환 레지스터가 되는 지역 기호이므로 다른 기호 없이 피연산자에 사용됩니다.
