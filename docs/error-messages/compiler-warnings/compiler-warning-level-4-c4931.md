# 컴파일러 경고 (수준 4) C4931
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

number비트 포인터에 대한 형식 라이브러리를 빌드했다고 간주합니다.  
  
 [\#import](../../preprocessor/hash-import-directive-cpp.md) 지시문의 **ptrsize** 특성을 통해 명시적 정보를 제공하지 않았으므로 컴파일러에서 형식 라이브러리의 포인터 크기를 ***number***로 결정했습니다.  
  
 이 경고는 기본적으로 해제되어 있습니다.  자세한 내용은 [기본적으로 해제되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md)를 참조하십시오.
