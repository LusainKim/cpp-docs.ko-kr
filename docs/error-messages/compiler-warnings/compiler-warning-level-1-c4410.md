# 컴파일러 경고 (수준 1) C4410
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

피연산자 크기가 잘못되었습니다.  
  
 명령에 대한 피연산자 중 하나의 크기가 잘못되었으므로  피연산자에 대해 유효한 최소 크기를 사용했습니다.
