# 컴파일러 경고 (수준 1) C4098
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 'void' 함수에서 값을 반환하고 있습니다.  
  
 반환 형식이 [void](../../cpp/void-cpp.md)로 선언된 함수가 값을 반환하는 `return` 문을 사용합니다.  컴파일러는 함수가 `int` 형식의 값을 반환한다고 가정합니다.
