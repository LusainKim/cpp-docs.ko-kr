# 컴파일러 경고 (수준 1) C4502
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'linkage specification'에는 'extern' 키워드를 사용해야 하고 이 사양은 다른 모든 지정자 앞에 와야 합니다.  
  
 링크를 `extern` 키워드 없이 지정했습니다.  링크는 extern 형식에만 관련이 있으므로  
  
 컴파일러에서 `extern` 키워드를 사용한 것으로 간주합니다.
