# 컴파일러 경고 C4961
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

프로필 데이터가 '.pgd file'에 병합되지 않았습니다. 프로필 기반 최적화를 사용하지 않습니다.  
  
 사용할 수 있는 프로필 데이터\(.pgc 파일\)가 없으므로 프로필 기반 최적화가 수행될 수 없습니다.
