# 컴파일러 경고 (수준 1) C4034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sizeof가 0을 반환합니다.  
  
 `sizeof` 연산자가 크기가 0인 피연산자\(빈 구조체, 공용 구조체, 클래스 또는 열거 형식이나 `void` 형식\)에 적용되었습니다.
