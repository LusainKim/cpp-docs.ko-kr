# <a name="compiler-warning-level-1-c4034"></a>컴파일러 경고 (수준 1) C4034
sizeof 0을 반환합니다.  
  
 `sizeof` 크기가 0 인 피연산자에 연산자가 적용 (빈 구조체, 공용 구조체, 클래스 또는 열거형된 형식 또는 형식 `void`).
