# 컴파일러 경고 (수준 1) C4506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' 인라인 함수에 대한 정의가 없습니다.  
  
 주어진 함수를 선언하여 인라인용으로 표시했지만 정의하지 않았습니다.  
  
 컴파일러에서 함수를 인라인하지 않았습니다.  
  
 인라인될 외부 함수는 `extern` 키워드를 사용하여 선언하십시오.
