# 컴파일러 경고 (수준 1) C4089
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 실제 매개 변수 number, 정식 매개 변수 number에서 형식이 다릅니다.  
  
 해당 형식 매개 변수와 실제 매개 변수의 형식이 서로 다릅니다.  실제 매개 변수는 변경되지 않은 상태로 전달되며  함수는 실제 매개 변수를 함수 정의에 지정된 형식으로 캐스팅합니다.
