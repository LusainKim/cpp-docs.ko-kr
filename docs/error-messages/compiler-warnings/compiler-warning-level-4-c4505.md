# 컴파일러 경고 (수준 4) C4505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 참조되지 않은 지역 함수를 제거했습니다.  
  
 주어진 함수는 지역 함수이고 모듈의 본문에서 참조하지 않았으므로 이는 불필요한 코드입니다.  
  
 컴파일러에서 이 불필요한 함수에 대해 코드를 생성하지 않았습니다.
