# 컴파일러 경고 (수준 1) C4406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

지시문의 피연산자가 무시됩니다.  
  
 지시문에서 피연산자를 가져오지 않지만 피연산자가 지정되어 있습니다.  
  
 컴파일러에서 주어진 피연산자를 무시했습니다.
