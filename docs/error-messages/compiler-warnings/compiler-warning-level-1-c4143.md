# 컴파일러 경고(수준 1) C4143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

pragma 'same\_seg'가 지원되지 않으므로 \_\_based 할당을 사용하세요.  
  
 **\#pragma same\_seg**는 더 이상 지원되지 않습니다. 대신 [\_\_based](../../cpp/based-pointers-cpp.md) 키워드를 사용합니다.
