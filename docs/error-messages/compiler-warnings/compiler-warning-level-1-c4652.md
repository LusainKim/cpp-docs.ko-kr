# 컴파일러 경고 (수준 1) C4652
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'option' 컴파일러 옵션이 미리 컴파일된 헤더와 일치하지 않습니다. 현재 명령줄 옵션이 미리 컴파일된 헤더에 정의된 것보다 우선합니다.  
  
 주어진 명령줄 옵션이 미리 컴파일된 헤더\(.pch\)를 만들 때의 옵션과 다르므로  현재 명령줄에 지정된 옵션이 사용되었습니다.  
  
 이 경고가 발생하지 않도록 하려면 주어진 명령줄 옵션을 사용하여 미리 컴파일된 헤더를 다시 생성하십시오.
