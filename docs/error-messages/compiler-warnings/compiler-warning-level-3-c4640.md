# 컴파일러 경고 (수준 3) C4640
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'instance' : 지역 정적 개체를 생성할 때 스레드로부터 안전하게 보호되지 않습니다.  
  
 개체의 정적 인스턴스는 스레드로부터 안전하지 않습니다.  
  
 이 경고는 기본적으로 해제되어 있습니다.  자세한 내용은 [기본적으로 해제되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md)를 참조하십시오.  
  
 다음 샘플에서는 C4640 오류가 발생하는 경우를 보여 줍니다.  
  
```  
// C4640.cpp  
// compile with: /W3  
#pragma warning(default:4640)  
  
class X {  
public:  
   X() {  
   }  
};  
  
void f() {  
   static X aX;   // C4640  
}  
  
int main() {  
   f();  
}  
```
