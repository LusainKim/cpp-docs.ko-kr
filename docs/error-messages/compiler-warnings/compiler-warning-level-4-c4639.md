# 컴파일러 경고 (수준 4) C4639
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

MSXML 오류, XML 문서 주석이 처리되지 않습니다.  
  
 이 경고는 여러 가지 원인으로 인해 발생할 수 있습니다.  
  
 이 경고를 해결하려면  
  
-   다시 컴파일하십시오.  
  
-   공용 언어 런타임을 다시 설치하여 MSXML을 다시 설치하십시오.  
  
-   경고의 원인이 된 문서 주석을 편집하거나 제거하고 다시 컴파일하십시오.  
  
 C4639가 발생하면 이후의 모든 XML 주석 처리가 비활성화되고 .xdc 파일이 생성되지 않습니다.
