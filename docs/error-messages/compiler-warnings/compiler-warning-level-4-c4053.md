# 컴파일러 경고(수준 4) C4053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'?:' 연산에 void 형식이 피연산자로 사용됩니다.  
  
 `?:` 연산자가 형식 `void`의 식에 지정되었습니다.`void` 피연산자의 값이 정의되지 않았습니다.
