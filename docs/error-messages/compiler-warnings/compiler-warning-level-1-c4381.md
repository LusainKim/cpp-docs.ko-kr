# 컴파일러 경고 (수준 1) C4381
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function1': 인터페이스 메서드를 public이 아닌 'function2' 메서드에서 구현하지 않습니다.  
  
 클래스는 인터페이스의 모든 함수를 구현해야 합니다.  해당 기본 클래스 중 하나가 함수를 구현하는 경우 클래스는 이 조건을 충족할 수 있습니다.  그러나 함수는 public 함수로 구현되어야 합니다.
