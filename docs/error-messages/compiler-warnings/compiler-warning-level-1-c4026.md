# <a name="compiler-warning-level-1-c4026"></a>컴파일러 경고(수준 1) C4026
정식 매개 변수 목록을 사용하여 함수를 선언했습니다.  
  
 함수 선언에는 정식 매개 변수가 있지만 함수 정의에는 없습니다. 이 함수에 대한 후속 호출에서는 함수가 매개 변수를 사용하지 않는다고 간주합니다.
