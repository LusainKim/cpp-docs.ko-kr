# 컴파일러 경고(수준 1) C4038
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'modifier': 클래스 한정자가 모호합니다.  
  
 이 한정자는 **dllimport** 또는 [dllexport](../../cpp/dllexport-dllimport.md) 특성을 사용하여 클래스에 대해 사용할 수 없습니다.
