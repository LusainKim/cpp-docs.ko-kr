# 컴파일러 경고 (수준 1) C4029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

선언된 형식 매개 변수 목록이 정의와 다릅니다.  
  
 함수 선언의 형식 매개 변수 형식이 함수 정의의 것과 다릅니다.  컴파일러는 정의에 있는 매개 변수 목록을 사용합니다.
