# 컴파일러 경고 (수준 4) C4536
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'type name' : 형식\-이름이 메타데이터 한계인 'limit'자를 초과합니다.  
  
 관리되는 형식인 경우 형식 이름이 메타데이터에서 잘립니다.  자세한 내용은 [C3180](../../error-messages/compiler-errors-2/compiler-error-c3180.md)을 참조하십시오.  
  
 이 경고는 기본적으로 해제되어 있습니다.  자세한 내용은 [기본적으로 해제되어 있는 컴파일러 경고](../../preprocessor/compiler-warnings-that-are-off-by-default.md)를 참조하십시오.
