# 컴파일러 경고 (수준 3) C4511
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'class' : 복사 생성자를 생성하지 못했습니다.  
  
 컴파일러에서 클래스에 대한 기본 복사 생성자를 생성하지 못했습니다. 기본 클래스에 전용 복사 생성자가 있을 수 있습니다.
