# 컴파일러 경고 (수준 1) C4015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 비트 필드 형식은 정수 계열이어야 합니다.  
  
 비트 필드가 정수 형식으로 선언되지 않았습니다.  컴파일러는 비트 필드의 기본 형식을 부호가 없는 것으로 가정합니다.  비트 필드는 반드시 부호 없는 정수 형식으로 선언해야 합니다.
