# 컴파일러 경고 (수준 1) C4088
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 실제 매개 변수 'number', 정식 매개 변수 'number'의 포인터가 일치하지 않습니다.  
  
 해당 형식 매개 변수와 실제 매개 변수가 각기 다른 간접 참조 수준을 사용합니다.  실제 매개 변수는 변경되지 않은 상태로 전달되며  호출된 함수는 해당 값을 포인터로 해석합니다.
