# 명령줄 경고 D9024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

인식할 수 없는 소스 파일 형식 'filename'입니다. 개체 파일이 선택됩니다.  
  
 지정한 파일의 확장명이 인식되지 않았습니다.  이 파일은 개체 파일로 간주되며 링커로 전달되었습니다.  
  
 다음 확장명이 인식됩니다.  
  
-   .c\(C 소스 파일\)  
  
-   .cxx\(C\+\+ 소스 파일\)  
  
-   .cpp\(C\+\+ 소스 파일\)  
  
-   .obj\(개체 파일\)  
  
-   .lib\(라이브러리 파일\)  
  
-   .def\(모듈 정의 파일\)  
  
-   .exp\(링커 내보내기 파일, LINK \/LIB에서 생성됨\)
