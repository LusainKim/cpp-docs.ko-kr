# NMAKE 심각한 오류 U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'target' 대상 매크로가 확장되지 않습니다.  
  
 지정한 대상은 아직 정의되지 않았거나 null 값을 갖고 있는 매크로의 호출입니다.  NMAKE는 null 대상을 처리할 수 없습니다.
