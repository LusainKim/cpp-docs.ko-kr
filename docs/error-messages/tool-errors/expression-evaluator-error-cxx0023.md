# 식 계산기 오류 CXX0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

기수가 잘못되었습니다.  
  
 C 식 계산기가 지정한 기수를 인식하지 못했습니다.  십진수, 16진수 및 8진수 기수만 유효합니다.  
  
 이 오류는 CAN0023과 동일합니다.
