# 프로젝트 빌드 오류 PRJ0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'file' 임시 파일에 사용자의 ANSI 코드 페이지로 변환할 수 없는 유니코드 내용이 있습니다.  
  
 시스템 코드 페이지에서 확인할 수 없는 [\/MIDL\(MIDL 명령줄 옵션 지정\)](../../build/reference/midl-specify-midl-command-line-options.md) 링커 옵션으로 값을 지정했습니다.  
  
 MIDL 명령\(입력 코드 페이지\)을 지정할 때 사용한 코드 페이지는 시스템 코드 페이지와 같아야 합니다.
