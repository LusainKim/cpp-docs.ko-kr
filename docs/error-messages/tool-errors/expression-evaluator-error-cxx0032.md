# 식 계산기 오류 CXX0032
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

0으로 나누었습니다.  
  
 식에는 0인 나눗수가 포함되어 있습니다. 나눗수는 0이 될 수 없습니다.  이 나눗수가 리터럴 숫자 0이거나 또는 0으로 계산되는 식입니다.  
  
 이 오류는 CAN0032와 동일합니다.
