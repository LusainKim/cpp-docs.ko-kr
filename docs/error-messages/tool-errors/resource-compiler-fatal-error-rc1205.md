# 리소스 컴파일러 심각한 오류 RC1205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

코드 페이지가 잘못되었습니다.  
  
 코드 페이지 지정 옵션\(\/c\) 다음에 잘못된 코드 페이지가 있습니다.  자세한 내용은 *Run\-Time Library Reference*의 [Code Pages](../../c-runtime-library/code-pages.md)를 참조하십시오.
