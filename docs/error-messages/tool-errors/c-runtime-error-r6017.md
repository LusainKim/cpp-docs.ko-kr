# C 런타임 오류 R6017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

예기치 못한 다중 스레드 잠금 오류가 발생했습니다.  
  
 프로세스가 시스템 리소스의 C 런타임 다중 스레드 잠금에 액세스하는 동안 예기치 못한 오류가 발생했습니다.  
  
 일반적으로 이 오류는 프로세스에서 런타임 힙 데이터를 부주의하게 변경할 때 발생합니다.  또한 런타임 코드나 운영 체제 코드의 내부 오류 때문에 발생할 수 있습니다.
