# 링커 도구 오류 LNK1282
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/REBASE file을\(를\) 사용할 수 없습니다. 서명되었습니다.  
  
 서명된 어셈블리의 기본 주소를 [editbin](../../build/reference/editbin-reference.md)의 \/REBASE 옵션을 사용하여 변경하려고 했습니다.  이 오류를 해결하려면 먼저 기본 주소를 변경한 다음 어셈블리에 서명하십시오.
