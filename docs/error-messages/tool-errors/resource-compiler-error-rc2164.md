# 리소스 컴파일러 오류 RC2164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

RCDATA에 예기치 않은 값이 있습니다.  
  
 **RCDATA** 문의 raw\-data 값은 각각 쉼표로 분리된 정수 또는 문자열이어야 합니다.  쉼표 또는 문자열 주위의 따옴표를 제거하지 않았는지 확인하십시오.
