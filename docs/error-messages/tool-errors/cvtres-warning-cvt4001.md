# CVTRES 경고 CVT4001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**컴퓨터 종류가 지정되지 않았습니다.**   
 ***type*\(으\)로 간주됩니다.**  
  
 CVTRES가 컴퓨터 사양을 찾지 못했습니다.  지정된 컴퓨터 형식으로 간주되었습니다.  기본값이 잘못되었으면 \/MACHINE 옵션을 사용하여 CVTRES를 다시 실행합니다.
