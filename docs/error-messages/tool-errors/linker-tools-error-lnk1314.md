# 링커 도구 오류 LNK1314
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

손상되었거나 잘못된 COFF 기호 테이블\(정의되지 않은 정적 또는 레이블 기호\)입니다.  
  
 지정된 개체에서 섹션에 대해 컴파일러가 생성한 내용이 손상된 것 같습니다.  
  
 이 오류는 컴파일러의 오류를 나타내기도 합니다.
