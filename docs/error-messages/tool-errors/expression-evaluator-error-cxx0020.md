# 식 계산기 오류 CXX0020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

피연산자 형식이 이 연산에 맞지 않습니다.  
  
 이 연산자에 맞지 않는 형식으로 식에 적용되었습니다.  
  
 예를 들어, 레지스터의 주소를 가져오거나 부동 소수점 식을 배열의 첨자로 사용할 수 없습니다.  
  
 이 오류는 CAN0020과 동일합니다.
