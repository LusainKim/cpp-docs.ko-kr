# 리소스 컴파일러 심각한 오류 RC1020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

예기치 못한 '\#endif'입니다.  
  
 일치하는 `#if`, **\#ifdef** 또는 **\#ifndef** 지시문 없이 `#endif` 지시문이 표시되었습니다.  
  
 각 `#if`, **\#ifdef** 및 **\#ifndef** 문에 일치하는 `#endif`가 있는지 확인하십시오.
