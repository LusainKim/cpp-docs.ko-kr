# 프로젝트 빌드 오류 PRJ0040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

빌드에 내부 오류가 있습니다.계속할 수 없습니다.프로젝트를 다시 로드한 후 다시 시도하십시오.  
  
 프로젝트 시스템에서 사용자의 프로젝트를 처리할 수 없습니다.  
  
 프로젝트를 다시 로드해 보십시오.
