# 식 계산기 오류 CXX0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

인수 목록이 함수와 일치하지 않습니다.  
  
 식이 프로그램에 정의된 동일한 함수 이름의 형식 매개 변수 목록과 일치하지 않는 실제 매개 변수 목록으로 함수를 호출했습니다.  
  
 정확한 매개 변수 일치 또는 개체 생성이 필요 없는 일치가 있는 경우에만 오버로드된 함수를 호출할 수 있습니다.  
  
 이 오류는 CAN0047과 동일합니다.
