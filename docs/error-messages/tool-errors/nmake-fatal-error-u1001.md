# NMAKE 심각한 오류 U1001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구문 오류: 매크로에 'character' 문자가 잘못되었습니다.  
  
 지정한 문자가 매크로에 들어 있지만 문자, 숫자 또는 밑줄이 아닙니다.  
  
 이 오류는 매크로 확장에 콜론을 사용하지 않은 경우에 발생할 수 있습니다.  
  
```  
syntax error : illegal character '=' in macro  
```
