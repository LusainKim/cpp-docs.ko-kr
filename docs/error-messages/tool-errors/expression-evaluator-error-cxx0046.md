# 식 계산기 오류 CXX0046
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

멤버 함수에 인수 목록이 필요합니다.  
  
 식이 멤버 함수를 호출했지만 실제 매개 변수를 지정하지 않았습니다.  
  
 이 오류는 CAN0046과 동일합니다.
