# 명령줄 경고 D9028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

최소 다시 빌드에 실패했습니다. 정상 빌드로 되돌립니다.  
  
 프로젝트 .idb 파일이 손상되었습니다.  이 파일을 삭제하고 다시 빌드하십시오.  
  
 최소 다시 빌드와 .idb 파일에 대한 자세한 내용은 [\/Gm 컴파일러 옵션](../../build/reference/gm-enable-minimal-rebuild.md)을 참조하십시오.
