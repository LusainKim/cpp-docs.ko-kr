# 링커 도구 경고 LNK4220
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'linker option' 값 'value'이\(가\) 잘못되었습니다. 기본값이 선택됩니다.  
  
 범위를 벗어나는 값이 [\/TLBID](../../build/reference/tlbid-specify-resource-id-for-typelib.md) 옵션에 지정되었습니다.  **\/TLBID**의 기본값은 1입니다.
