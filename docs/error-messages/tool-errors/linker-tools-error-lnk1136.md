# 링커 도구 오류 LNK1136
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일이 잘못되었거나 손상되었습니다.  
  
 입력 파일의 헤더가 손상되었습니다. 또는 입력 파일의 크기가 0이거나 비정상적으로 작습니다.  
  
 LNK1136은 파일을 추가 종속성으로 잘못 지정한 경우에도 발생할 수 있습니다.
