# 링커 도구 오류 LNK1240
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

IDL 내용을 컴파일하지 못했습니다.  
  
 링커가 MIDL을 생성하여 포함된 IDL을 컴파일했지만 문제가 발생했습니다.  MIDL에 의해 지정된 오류를 확인하십시오.
