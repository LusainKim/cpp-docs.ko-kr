# 식 계산기 오류 CXX0066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

정적 멤버가 없습니다.  
  
 클래스의 정적 멤버를 찾을 수 없거나 정의되지 않았습니다.  정적 클래스 멤버가 선언되었으나 정의되지 않은 경우 또는 디버그 정보가 들어 있지 않은 모듈에만 정의되고 참조된 경우에 이 오류가 발생합니다.  
  
 이 오류는 CAN0066과 동일합니다.
