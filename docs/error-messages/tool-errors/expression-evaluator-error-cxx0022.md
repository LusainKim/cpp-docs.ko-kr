# 식 계산기 오류 CXX0022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_main 이전의 함수 호출입니다.  
  
 C 식 계산기는 디버거가 **\_main** 함수에 들어가기 전에 함수를 계산할 수 없습니다.  프로그램은 **\_main** 함수가 호출되기 전에는 제대로 초기화되지 않습니다.  
  
 이 오류는 CAN0022와 동일합니다.
