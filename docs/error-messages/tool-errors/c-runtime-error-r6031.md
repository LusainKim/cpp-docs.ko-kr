# C 런타임 오류 R6031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

 이 진단 메시지는 로드 잠금 과정에서 MSIL 명령을 실행했음을 의미합니다.  자세한 내용은 [혼합형 어셈블리 초기화](../../dotnet/initialization-of-mixed-assemblies.md)을 참조하십시오.
