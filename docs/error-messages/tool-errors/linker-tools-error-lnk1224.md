# 링커 도구 오류 LNK1224
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이미지 기준 주소이\(가\) 잘못되었습니다.  
  
 잘못된 이미지 기준 주소를 지정했습니다.  기준 주소는 64KB로 맞춰야 하고 마지막 네 개의 16진수 숫자는 0이어야 합니다. 이미지 기준은 32비트 부호 있는 값 또는 부호 없는 값 안에 맞춰야 합니다.
