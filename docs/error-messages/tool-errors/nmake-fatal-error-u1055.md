# NMAKE 심각한 오류 U1055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

환경 공간이 부족합니다.  
  
 운영 체제에 환경 변수를 위한 공간이 부족합니다.  
  
 환경 공간을 늘이거나 환경 변수의 개수를 줄이십시오.
