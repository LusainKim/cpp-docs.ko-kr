# 링커 도구 오류 LNK1141
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

내보내기 파일을 빌드하는 동안 오류가 발생했습니다.  
  
 [LINK](../../build/reference/linker-command-line-syntax.md)가 내보내기\(.exp\) 파일을 빌드하지 못했습니다.  
  
 이 오류의 원인으로는 메모리 부족과 같은 시스템 문제와 모듈 정의 문 또는 옵션의 구문 오류 등이 있습니다.  
  
 이 오류는 추가 정보를 제공하는 다른 오류가 발생한 다음에 발생합니다.
