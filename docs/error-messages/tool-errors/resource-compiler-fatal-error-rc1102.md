# 리소스 컴파일러 심각한 오류 RC1102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

내부 오류: RCPP에 대한 인수가 너무 많습니다.  
  
 리소스 컴파일러 전처리기에 너무 많은 인수가 전달되었습니다.  소스 파일에서 기호 정의\(\/d\) 옵션으로 정의한 기호의 개수를 줄이십시오.  포함 검색 경로\(\/i\) 옵션을 사용하여 포함 파일 검색 경로를 너무 많이 지정할 경우 이 오류가 발생합니다.
