# NMAKE 심각한 오류 U1052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일 'filename'을\(를\) 찾을 수 없습니다.  
  
 NMAKE는 다음 중 하나로 지정한 파일을 찾을 수 없습니다.  
  
-   **\/F** 옵션  
  
-   **\!INCLUDE** 전처리 지시문  
  
-   지시 파일의 **@** 기호 지정자  
  
 파일이 있는지 및 파일 이름의 맞춤법이 올바른지 여부를 확인하십시오.
