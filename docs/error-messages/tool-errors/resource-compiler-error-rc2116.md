# 리소스 컴파일러 오류 RC2116
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ID에는 숫자가 있어야 합니다.  
  
 **DIALOG** 문에서 컨트롤 문의 `id` 필드에는 숫자가 있어야 합니다.  컨트롤 ID에 숫자나 `#define` 문을 사용했는지 확인하십시오.
