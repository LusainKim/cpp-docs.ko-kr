# 식 계산기 오류 CXX0001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

사용자 함수를 실행하는 동안 예외가 발생했습니다.  
  
 실행 중인 코드로 인해 일반 보호 오류가 발생했습니다.  이 오류는 CAN0001과 동일합니다.
