# 리소스 컴파일러 심각한 오류 RC1109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

리소스 이름을 만드는 동안 오류가 발생했습니다.  
  
 지정한 .res 파일을 만들 수 없습니다.  읽기 전용 드라이브에서 파일을 만들고 있지 않은지 확인하십시오.  \/V 옵션을 사용하여 파일을 만드는 중인지 확인하십시오.
