# BSCMAKE 오류 BK1508
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename' 임시 파일을 삭제할 수 없습니다. \[: reason\]  
  
 BSCMAKE에서 임시 파일 중 하나를 삭제할 수 없습니다.  
  
### 문제 해결을 위하여 확인해 볼 수 있는 원인  
  
1.  다른 프로세스에서 해당 파일이 열려 있습니다.  
  
2.  하드웨어 오류입니다.
