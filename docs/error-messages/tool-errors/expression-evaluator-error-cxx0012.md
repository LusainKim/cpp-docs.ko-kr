# 식 계산기 오류 CXX0012
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

왼쪽 중괄호가 없습니다.  
  
 짝이 맞지 않는 중괄호가 식에 포함되어 있습니다.  중괄호의 짝을 맞추어 식을 다시 입력하십시오.  
  
 이 오류는 CAN0012와 동일합니다.
