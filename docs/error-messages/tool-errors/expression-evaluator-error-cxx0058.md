# 식 계산기 오류 CXX0058
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

오버로드된 연산자를 찾을 수 없습니다.  
  
 클래스 형식이 식에서 왼쪽 피연산자로 지정되었으나 오버로드된 연산자가 이 클래스에 대해 정의되지 않았습니다.  
  
 이 오류는 CAN0058과 동일합니다.
