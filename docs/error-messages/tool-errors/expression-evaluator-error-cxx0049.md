# 식 계산기 오류 CXX0049
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구식 OMF \- 프로그램을 다시 링크하십시오.  
  
 프로그램은 구식 OMF\(개체 모듈 형식\)를 사용했습니다.  
  
 현재 링커 버전으로 프로그램을 다시 링크하십시오.  
  
 이 오류는 CAN0049와 동일합니다.
