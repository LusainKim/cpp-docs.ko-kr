# 리소스 컴파일러 오류 RC2015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

상수에 문자가 너무 많습니다.  
  
 문자 상수에 문자가 두 개 이상 들어 있습니다.  
  
 문자 상수는 문자 한 개\(표준 문자 상수\) 또는 문자 두 개\(긴 문자 상수\)로 제한되어 있습니다.  
  
 이스케이프 시퀀스\(예: 탭 문자 \\t\)는 단일 문자로 변환됩니다.
