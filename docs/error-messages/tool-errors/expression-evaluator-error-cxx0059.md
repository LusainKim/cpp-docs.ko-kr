# 식 계산기 오류 CXX0059
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

왼쪽 피연산자는 함수 이름이 아닌 클래스입니다.  
  
 함수 호출의 왼쪽 피연산자가 클래스 이름이며 함수 호출로 확인될 수 없습니다.  이 오류는 식에 멤버 함수의 이름을 생략하면 발생합니다.  
  
 이 오류는 CAN0059와 동일합니다.
