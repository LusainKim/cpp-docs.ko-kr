# 식 계산기 오류 CXX0055
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컨텍스트 연산자의 기호가 정확하지 않습니다.  
  
 컨텍스트 연산자\(**{}**\)의 기호가 프로그램에서 두 개 이상의 기호를 가리킵니다.  
  
 범위 결정 연산자\(`::`\)가 모호성을 해결할 수 있습니다.  
  
 이 오류는 CAN0055와 동일합니다.
