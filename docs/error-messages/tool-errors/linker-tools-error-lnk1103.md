# 링커 도구 오류 LNK1103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

디버깅 정보가 손상되었습니다. 모듈을 다시 컴파일하십시오.  
  
 이 오류는 유효한 개체 파일을 만들기 전에 컴파일이 종료된 경우에 발생할 수 있습니다.  
  
 지정된 개체 파일을 다시 컴파일하십시오.  다시 컴파일해도 문제가 해결되지 않으면 오류 상태를 적어 두고 문제를 격리시킨 다음 오류를 재현시키는 테스트 사례를 작성하십시오.  그런 다음 기술 지원 도움말 파일 또는 설명서의 기술 지원 단원을 참조하십시오.
