# 식 계산기 오류 CXX0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_based\(void\) 포인터를 사용하려면 :\>연산자가 필요합니다.  
  
 `void` 기반 포인터를 직접 사용할 수 없습니다.  **:\>** 연산자를 사용하여 완전한 포인터를 구성해야 합니다.  
  
 이 오류는 CAN0044와 동일합니다.
