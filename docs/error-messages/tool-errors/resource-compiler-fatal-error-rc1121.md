# 리소스 컴파일러 심각한 오류 RC1121
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일을 읽는 동안 I\/O 오류가 발생했습니다.  
  
 리소스 컴파일러에서 파일을 읽을 수 없습니다.  파일이 들어 있는 드라이버가 사용 가능한지 해당 파일이 유효한지 확인하십시오.
