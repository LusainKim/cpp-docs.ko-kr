# 리소스 컴파일러 심각한 오류 RC1022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\#endif'이\(가\) 있어야 합니다.  
  
 `#if`, **\#ifdef** 또는 **\#ifndef** 지시문이 `#endif` 지시문으로 끝나지 않았습니다.  
  
 이 문 앞의 `#if`, **\#ifdef** 또는 **\#ifndef** 문이 유효한 상태인지 확인하십시오.
