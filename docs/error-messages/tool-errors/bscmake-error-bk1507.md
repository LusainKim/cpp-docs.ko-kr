# BSCMAKE 오류 BK1507
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename' 임시 파일을 열 수 없습니다. \[: reason\]  
  
 BSCMAKE에서 임시 파일을 열 수 없습니다.  
  
### 문제 해결을 위하여 확인해 볼 수 있는 원인  
  
1.  **TMP** 환경 변수가 유효한 드라이브 및 디렉터리에 설정되지 않았습니다.  
  
2.  디스크가 꽉 찼습니다.
