# 수학 오류 M6102
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

비정상적입니다.  
  
 연산이 매우 작은 부동 소수점 숫자를 생성했으나 상위값 손실로 인해 잘못되었습니다.  일반적으로 비정상적인 부동 소수점 예외는 마스크되어 트랩 및 연산됩니다.  
  
 프로그램은 종료 코드 130로 종료됩니다.
