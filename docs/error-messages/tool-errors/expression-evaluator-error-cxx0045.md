# 식 계산기 오류 CXX0045
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

함수가 아닙니다.  
  
 프로그램에서 인수 목록이 함수 이름이 아닌 기호에 제공되었습니다.  
  
## 예제  
  
```  
queue( alpha, beta )  
```  
  
 `queue` 는 함수가 아닙니다.  
  
 이 오류는 CAN0045와 동일합니다.
