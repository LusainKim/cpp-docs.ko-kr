# 링커 도구 오류 LNK1164
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

section 섹션 맞춤\(number\)이 \/ALIGN 값보다 큽니다.  
  
 개체 파일에 있는 지정된 섹션의 맞춤 크기가 [\/ALIGN](../../build/reference/align-section-alignment.md) 옵션으로 지정된 값보다 큽니다.  **\/ALIGN** 값은 2의 거듭제곱이어야 하며 개체 파일에 지정된 섹션 맞춤과 같거나 그보다 커야 합니다.  
  
 좀 더 작은 섹션 맞춤으로 다시 컴파일하거나 **\/ALIGN** 값을 늘리십시오.
