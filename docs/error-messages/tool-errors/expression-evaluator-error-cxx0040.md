# 식 계산기 오류 CXX0040
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

함수에 암시적 변환이 필요합니다.  
  
 C 식 계산기는 생성자 호출과 관련된 암시적 변환을 지원하지 않습니다.  
  
 이 오류는 CAN0040과 동일합니다.
