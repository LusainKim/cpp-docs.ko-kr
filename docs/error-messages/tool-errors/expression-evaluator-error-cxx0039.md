# 식 계산기 오류 CXX0039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

기호가 정확하지 않습니다.  
  
 C 식 계산기는 식에 사용할 기호의 인스턴스를 결정할 수 없습니다.  기호는 상속 트리에 두 번 이상 발생했습니다.  
  
 범위 결정 연산자\(`::`\)로 식에 사용할 인스턴스를 명시적으로 지정해야 합니다.  
  
 이 오류는 CAN0039와 동일합니다.
