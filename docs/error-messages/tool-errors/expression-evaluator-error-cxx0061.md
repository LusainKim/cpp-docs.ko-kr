# 식 계산기 오류 CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

함수 중첩 한도를 초과했습니다.  
  
 식에 한계보다 중첩 한도가 더 큰 함수가 들어 있습니다.  식을 수정하여 중첩 한도를 줄이십시오.  
  
 이 오류는 CAN0061과 동일합니다.
