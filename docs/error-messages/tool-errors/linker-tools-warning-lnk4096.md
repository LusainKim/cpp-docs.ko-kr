# 링커 도구 경고 LNK4096
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/BASE 값 "number"은\(는\) Windows 95 및 Windows 98에서 사용할 수 없습니다. 이미지가 실행되지 않을 수도 있습니다.  
  
 지정한 기본 주소가 잘못되었습니다.  Windows 95 및 Windows 98 실행 파일은 0x400000을 넘는 기본 주소를 사용해야 합니다.  기본 주소에 대한 자세한 내용은 [\/BASE](../../build/reference/base-base-address.md) 링커 옵션을 참조하십시오.
