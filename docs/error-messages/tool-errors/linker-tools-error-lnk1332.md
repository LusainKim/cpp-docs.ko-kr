# 링커 도구 오류 LNK1332
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\<count\>u Windows 런타임 형식을 가져온 모듈과 정의한 모듈이 다릅니다.  
  
 현재 대상 생성 될 때 링커가 검색 \<  `count`  \> 모듈에서 가져온 이며 또한 다른 모듈에 정의 된 각각의 Windows 런타임 형식입니다.  
  
### 이 오류를 해결하려면  
  
-   각 오류 메시지에서 제안에 따라 빌드를 LNK2039 오류를 수정 합니다.
