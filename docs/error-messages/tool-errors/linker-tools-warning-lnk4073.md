# 링커 도구 경고 LNK4073
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

.ilk 파일에 대한 맵을 만들 수 없습니다. 비증분 링크합니다.  
  
 공유 메모리에 대형 연속 공간이 부족하여 LINK가 증분 상태\(.ilk\) 파일을 만들지 못했습니다.  LINK가 비증분 빌드를 수행했습니다.
