# NMAKE 심각한 오류 U1071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'targetname' 대상을 위한 종속 트리에서 순환됩니다.  
  
 지정한 대상에 대한 종속 트리에 순환 종속성이 있습니다.  지정한 대상은 종속 파일 중 하나에 의존합니다.  순환 종속성이 잘못되었습니다.
