# 링커 도구 경고 LNK4010
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

하위 시스템 버전 번호 number이\(가\) 잘못되었습니다. 기본 하위 시스템 버전이 선택됩니다.  
  
 이미지의 하위 시스템\([\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)\)에 대한 버전을 지정할 수 있습니다.  각 하위 시스템에는 최소 버전 요구 사항이 있습니다.  지정된 버전이 최소 버전보다 낮으면 이 경고가 발생하고 링커가 기본 하위 시스템만 사용합니다.
