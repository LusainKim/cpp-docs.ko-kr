# 리소스 컴파일러 오류 RC4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' 매크로의 실제 매개 변수가 너무 많습니다.  
  
 이 식별자로 지정된 실제 매개 변수의 개수는 해당 식별자의 매크로 정의에 지정한 형식 매개 변수의 개수보다 큽니다.  
  
 추가 실제 매개 변수는 수집되었지만 매크로를 확장하는 동안 무시되었습니다.
