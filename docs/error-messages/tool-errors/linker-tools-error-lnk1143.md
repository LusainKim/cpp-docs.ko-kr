# 링커 도구 오류 LNK1143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일이 잘못되었거나 손상되었습니다. number COMDAT 섹션에 대한 기호가 없습니다.  
  
 이 오류는 개체 파일이 손상된 경우에 발생할 수 있습니다.  
  
 파일을 다시 빌드하십시오.
