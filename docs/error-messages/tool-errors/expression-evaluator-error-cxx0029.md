# 식 계산기 오류 CXX0029
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구조체 포인터가 아닙니다.  
  
 멤버 선택 연산자\(**\-\>**\)가 구조체에 대한 포인터가 아닌 식에 적용되었습니다.  
  
 전체 식이 괄호로 올바르게 묶여 있는지 확인하거나 주소 식을 적절한 구조체 포인터 형식으로 형식 변환하십시오.  
  
 이 오류는 CAN0029와 동일합니다.
