# 리소스 컴파일러 심각한 오류 RC1019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

예기치 못한 '\#else'입니다.  
  
 `#else` 지시문이 `#if`, **\#ifdef** 또는 **\#ifndef** 구문에 표시되지 않았습니다.  
  
 이 문 앞의 `#if`, **\#ifdef** 또는 **\#ifndef** 문이 유효한 상태인지 확인하십시오.
