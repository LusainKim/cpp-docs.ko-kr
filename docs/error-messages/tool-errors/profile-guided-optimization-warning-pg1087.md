# 프로필 기반 최적화 경고 PG1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

.PGC 파일 'file'이\(가\) 잘못되었습니다.  
  
 .pgc 파일이 손상되었습니다. 프로파일링 세션이 갑자기 중단되었기 때문일 수 있습니다.  프로파일링 시나리오를 다시 실행하고 세션을 정상적으로 마쳐야 합니다.
