# 링커 도구 오류 LNK1221
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

하위 시스템을 유추할 수 없으므로 정의해야 합니다.  
  
 링커에 정보가 부족하므로 대상으로 지정할 하위 시스템을 유추할 수 없습니다.  
  
 이 오류를 해결하려면 [\/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)을 사용하십시오.
