# 리소스 컴파일러 심각한 오류 RW1030
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

출력 오류  
  
 이 오류는 리소스 컴파일러에서 대상 파일을 만들 수 없는 경우에 발생할 수 있습니다.  디스크 공간이 충분한지와 볼륨에 대한 쓰기 사용 권한이 있는지 확인하십시오.
