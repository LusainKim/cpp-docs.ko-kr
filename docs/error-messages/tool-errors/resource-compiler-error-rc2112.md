# 리소스 컴파일러 오류 RC2112
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

대화 상자에는 BEGIN이 있어야 합니다.  
  
 **BEGIN** 키워드는 **DIALOG** 키워드 바로 뒤에 와야 합니다.
