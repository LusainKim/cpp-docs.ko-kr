# 리소스 컴파일러 오류 RC2109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

숫자 대화 상자 상수가 있어야 합니다.  
  
 **DIALOG** 문에는 *x, y, width* 및 *height* 필드에 정수 값이 있어야 합니다.  이 값이 **DIALOG** 키워드 뒤에 포함되어 있으며 음수가 아닌지 확인하십시오.
