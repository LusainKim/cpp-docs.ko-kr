# 식 계산기 오류 CXX0067
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

지원되지 않는 함수 계산입니다.  
  
 식에 함수 호출이 포함되어 있습니다.  일부 식 계산기는 함수 호출을 지원하지 않습니다.  
  
 이 오류는 CAN0067과 동일합니다.
