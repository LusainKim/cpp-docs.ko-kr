# 식 계산기 오류 CXX0063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

오버로드된 연산자 \-\> 가 지원되지 않습니다.  
  
 식은 오버로드된 클래스 멤버 액세스 연산자\(**\-\>**\)를 사용했습니다.  
  
 이 오류는 CAN0063과 동일합니다.
