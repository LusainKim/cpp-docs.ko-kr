# 리소스 컴파일러 오류 RC2127
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

쉼표로 구분된 버전 WORD가 있어야 합니다.  
  
 버전 리소스에 들어 있는 버전 번호는 형식이 **WORD**여야 하며 쉼표로 구분됩니다.
