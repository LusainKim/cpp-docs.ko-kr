# 식 계산기 오류 CXX0072
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

오류: 형식 정보가 없거나 알 수 없는 것입니다.  
  
 .pch 파일이 링크되지 않았거나 \/Zi 옵션으로 컴파일되지 않은 모듈에 있는 형식에 대한 참조가 코드에 있습니다.  
  
 이 오류는 CAN0067과 동일합니다.
