# 리소스 컴파일러 심각한 오류 RC1203
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

16진수 기본 언어 ID가 잘못 지정되었습니다.  
  
 기본 언어 지정 옵션\(\/l\) 다음에 잘못된 16진수 언어 ID가 있습니다.  유효한 언어 ID 목록은 *Run\-Time Library Reference*의 [Language and Country\/Region Strings](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)를 참조하십시오.
