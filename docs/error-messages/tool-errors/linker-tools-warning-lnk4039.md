# 링커 도구 경고 LNK4039
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/SECTION 옵션으로 지정된 'name' 섹션이 없습니다.  
  
 [DUMPBIN](../../build/reference/dumpbin-reference.md) 또는 [EDITBIN](../../build/reference/editbin-reference.md)이 입력 파일에서 `name`이라는 섹션을 찾지 못했습니다.  
  
 파일에서 섹션을 확인하려면 DUMPBIN을 [\/HEADERS](../../build/reference/headers.md) 옵션과 함께 실행하십시오.
