# 식 계산기 오류 CXX0054
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

중단점 식의 레지스터가 잘못되었습니다.  
  
 중단점 식에 레지스터를 사용할 수 없습니다.  
  
 이 오류는 CAN0054와 동일합니다.
