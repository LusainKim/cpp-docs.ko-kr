# 리소스 컴파일러 오류 RC2124
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

빈 메뉴는 사용할 수 없습니다.  
  
 **MENU** 문에서 메뉴 항목이 정의되기 전에 **END** 키워드가 있습니다.  리소스 컴파일러에는 빈 메뉴를 사용할 수 없습니다.  **MENU** 문 안에 여는 따옴표가 없는지 확인하십시오.
