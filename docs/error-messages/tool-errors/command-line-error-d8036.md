# 명령줄 오류 D8036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\/option'을\(를\) 여러 소스 파일에서 사용할 수 없습니다.  
  
 다음 컴파일러 옵션은 여러 소스 파일에서 사용할 수 없습니다.  
  
-   어셈블리 파일 목록 이름 지정\(\/Fa\)  
  
-   개체 파일 이름 바꾸기\(\/Fo\)  
  
-   지역 변수 없이 소스 브라우저 정보 만들기\(\/Fr\)  
  
-   지역 변수를 포함하는 소스 브라우저 정보 만들기\(\/FR\)
