# 식 계산기 오류 CXX0048
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

지원되지 않는 호출 시퀀스입니다.  
  
 식에 지정된 함수는 C 식 계산기가 지원하지 않는 호출 시퀀스를 사용하고 있습니다.  조사식 창 식에서 이 함수를 호출할 수 없습니다.  
  
 이 오류는 CAN0048과 동일합니다.
