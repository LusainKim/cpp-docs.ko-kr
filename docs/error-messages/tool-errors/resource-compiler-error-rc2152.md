# 리소스 컴파일러 오류 RC2152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

제어 문자가 잘못되었습니다.  
  
 **ACCELERATORS** 문의 제어 문자가 잘못되었습니다.  올바른 제어 문자는 캐럿\(**^**\) 뒤에 한 글자만 붙습니다.
