# 프로젝트 빌드 오류 PRJ0023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'tool' 도구, 알 수 없는 속성에 잘못된 파일 이름 'file'이\(가\) 있습니다.  
  
 ***tool***  도구에 지정된 파일 이름 ***file***이\(가\) 잘못되었습니다.  
  
 알 수 없는 매크로나 잘못된 매크로를 사용한 것 같습니다.  
  
 프로젝트의 속성에 액세스하는 방법에 대한 자세한 내용은 [Visual C\+\+ 프로젝트 속성 설정](../../ide/working-with-project-properties.md)을 참조하십시오.
