# NMAKE 심각한 오류 U1000
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구문 오류: 매크로 호출에 '\)'가 없습니다.  
  
 매크로 호출에 일치하는 오른쪽 괄호 '**\)**' 없이 왼쪽 괄호 '**\(**' 가 있습니다.  올바른 형식은 **$\(***name***\)**입니다. `$`*n*은 한 문자로 된 이름을 사용합니다.
