# 링커 도구 오류 LNK1218
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

경고가 오류로 처리됩니다. 출력 파일이 생성되지 않습니다.  
  
 **\/WX**를 사용하여 링크하는 경우 모든 링커 경고가 오류로 처리되고 출력 파일이 만들어지지 않습니다.  
  
 자세한 내용은 [\/WX\(링커 경고를 오류로 처리\)](../../build/reference/wx-treat-linker-warnings-as-errors.md)를 참조하십시오.
