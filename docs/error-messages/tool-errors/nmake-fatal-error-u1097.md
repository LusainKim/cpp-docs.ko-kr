# NMAKE 심각한 오류 U1097
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일 이름 부분 구문에는 종속 파일이 있어야 합니다.  
  
 현재 종속성에는 명시적 또는 암시적 종속 파일이 없습니다.  백분율\(`%`\) 지정자를 사용하는 파일 이름 부분 구문은 현재 대상의 첫째 종속 파일의 구성 요소를 나타냅니다.
