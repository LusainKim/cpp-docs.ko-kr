# BSCMAKE 오류 BK1510
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename .SBR 파일이 손상되었습니다.  
  
 지정한 .sbr 파일의 형식이 올바르지 않습니다.  다시 컴파일하여 새 .sbr 파일을 만드십시오.
