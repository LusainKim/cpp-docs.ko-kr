# BSCMAKE 오류 BK1504
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename' 파일에 위치를 지정할 수 없습니다. \[: reason\]  
  
 BSCMAKE는 파일에서 특정 위치로 이동할 수 없습니다.  
  
### 문제 해결을 위하여 확인해 볼 수 있는 원인  
  
1.  디스크가 꽉 찼습니다.  
  
2.  하드웨어 오류입니다.  
  
3.  디스크 공간이 부족하거나 .sbr 파일을 만드는 동안 컴파일러 중지로 인해 파일이 잘렸습니다.
