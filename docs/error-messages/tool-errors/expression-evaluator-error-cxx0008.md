# 식 계산기 오류 CXX0008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**문자열 끝에 큰따옴표\("\)가 없습니다.**  
  
 문자열 리터럴 끝에 있어야 할 큰따옴표가 없습니다.  문자열 리터럴을 큰따옴표로 묶어 식을 다시 입력하십시오.  
  
 이 오류는 CAN0008과 동일합니다.
