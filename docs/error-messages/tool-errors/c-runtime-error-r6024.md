# C 런타임 오류 R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\_onexit\/atexit 테이블에 대한 공간이 부족합니다.  
  
 `_onexit` 또는 `atexit` 함수에 사용할 수 있는 메모리가 없습니다.  이 오류는 메모리가 부족하기 때문에 발생합니다.
