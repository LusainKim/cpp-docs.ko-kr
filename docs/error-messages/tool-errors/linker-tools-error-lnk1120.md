# 링커 도구 오류 LNK1120
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

number개의 확인할 수 없는 외부 참조입니다.  
  
 오류 LNK1120은 이 링크에 대한 확인할 수 없는 외부 참조의 개수\(`number`\)를 제공합니다.  확인할 수 없는 외부 참조를 생성한 조건은 이 오류 메시지 이전에 나오는 오류 [LNK2001](../../error-messages/tool-errors/linker-tools-error-lnk2001.md)\(확인할 수 없는 각 외부 참조에 대해 한 번 발생\)과 함께 설명됩니다.  
  
 더 많은 정보를 위해 [http:\/\/support.microsoft.com\/default.aspx?scid\=kb;en\-us;815650](http://support.microsoft.com/default.aspx?scid=kb;en-us;815650)을 참조하십시오.
