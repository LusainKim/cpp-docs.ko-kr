# 식 계산기 오류 CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

중단점을 바인딩하는 동안 비 함수 기호를 발견했습니다.  
  
 중단점으로 사용한 기호는 함수가 아닙니다.  이 오류는 데이터 멤버를 중단점으로 지정한 경우 발생합니다.  
  
 이 오류는 CAN0053과 동일합니다.
