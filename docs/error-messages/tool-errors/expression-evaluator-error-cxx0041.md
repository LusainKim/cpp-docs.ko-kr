# 식 계산기 오류 CXX0041
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

클래스 요소는 정적 멤버 또는 멤버 함수여야 합니다.  
  
 클래스, 구조체 또는 공용 구조체의 비 정적 멤버가 사용할 클래스의 인스턴스화를 지정하지 않고 사용되었습니다.  
  
 정적 데이터 멤버 또는 멤버 함수만 인스턴스화를 지정하지 않고 사용할 수 있습니다.  
  
 이 오류는 CAN0041과 동일합니다.
