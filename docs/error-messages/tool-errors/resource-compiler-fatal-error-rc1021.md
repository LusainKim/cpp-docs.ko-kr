# 리소스 컴파일러 심각한 오류 RC1021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'string' 전처리기 명령이 잘못되었습니다.  
  
 파운드 기호\(**\#**\) 다음의 문자가 올바른 전처리기 지시문을 형성하지 못했습니다.  
  
 파운드 기호를 식별자의 첫 문자로 사용할 수 없습니다.
