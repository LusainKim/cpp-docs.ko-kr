# BSCMAKE 오류 BK1505
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename' 파일에서 읽을 수 없습니다. \[: reason\]  
  
 BSCMAKE에서 파일을 읽을 수 없습니다.  
  
### 문제 해결을 위하여 확인해 볼 수 있는 원인  
  
1.  파일이 손상되었습니다.  
  
2.  디스크 공간이 부족하여 컴파일러를 실행할 수 없거나 .sbr 파일을 만드는 동안 중지로 인해 파일이 잘렸습니다.
