# 수학 오류 M6110
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

스택 오버플로입니다.  
  
 부동 소수점 식 때문에 부동 소수점 스택 오버플로가 발생했습니다.  
  
 스택 오버플로 부동 소수점 예외는 8087\/287\/387 보조 프로세서가 일반적으로 지원하는 8수준 이외에 7수준까지 트랩됩니다.  
  
 프로그램은 종료 코드 138로 종료됩니다.
