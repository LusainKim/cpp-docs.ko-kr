# BSCMAKE 오류 BK1516
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bscfile이\(가\) 손상되었습니다. 증분 업데이트를 할 수 없습니다.  
  
 .bsc 파일이 손상되었습니다. 빌드하는 동안 발생한 시스템 오류 때문일 수 있습니다.  .bsc 파일을 삭제하고 모든 .sbr 파일과 .bsc 파일을 차례로 다시 빌드합니다.
