# 링커 도구 오류 LNK1287
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

관리되는 진입점 함수가 잘못되었습니다.  
  
 관리되는 이미지에 대한 진입점이 유효하지 않습니다.  관리되는 진입점 함수의 반환 형식은 `void` 또는 `int`여야 합니다.  관리되는 진입점 함수의 매개 변수 형식으로는 `void` 또는 `String []`만 허용됩니다.
