# 리소스 컴파일러 심각한 오류 RC1101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

리소스 이진 파일 이름을 지정하지 않았습니다.  
  
 출력 이름 바꾸기\(\/fo\) 옵션 뒤에 파일 이름이 없습니다.  
  
 \/fo 옵션에 다음 구문을 사용하십시오.  
  
 **RC \/fo**   
 ***outfilename infilename.* rc**
