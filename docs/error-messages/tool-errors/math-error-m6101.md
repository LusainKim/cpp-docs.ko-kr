# 수학 오류 M6101
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

잘못되었습니다.  
  
 연산이 잘못되었습니다.  
  
 이 오류는 피연산자가 NaN\(숫자가 아님\) 또는 무한대인 경우에 발생할 수 있습니다.  
  
 프로그램은 종료 코드 129로 종료됩니다.
