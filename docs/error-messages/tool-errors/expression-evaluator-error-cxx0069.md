# 식 계산기 오류 CXX0069
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

변수에 스택 프레임이 필요합니다.  
  
 스택 프레임에서 벗어났기 때문에 식 계산기가 변수를 계산할 수 없습니다.  이 오류는 변수가 인라인 함수의 일부로 선언되었기 때문에 발생합니다.
