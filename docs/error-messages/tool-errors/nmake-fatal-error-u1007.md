# NMAKE 심각한 오류 U1007
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이름에 큰따옴표를 사용할 수 없습니다.  
  
 지정한 대상 이름이나 파일 이름에 큰따옴표\(**"**\)가 들어 있습니다.  
  
 큰따옴표로 파일 이름을 묶을 수 있지만 파일 이름 안에 포함될 수는 없습니다.
