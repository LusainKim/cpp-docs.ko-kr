# NMAKE 심각한 오류 U1076
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이름이 너무 깁니다.  
  
 문자열이 다음 한계 중 하나를 초과했습니다.  
  
-   1024 문자의 매크로 이름  
  
-   256 문자의 대상 경로 이름  
  
-   2048 문자의 단일 명령
