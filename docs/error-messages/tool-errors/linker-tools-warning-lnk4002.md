# 링커 도구 경고 LNK4002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

symbol이\(가\) object에 정의됩니다.  
  
 데코레이팅된 형식으로 표시되는 기호가 데코레이팅되지 않은 형식으로 `object`에 지정되었지만 데코레이팅된 기호에만 일치하는 것이 없었습니다.  항상 이 경고 앞에는 경고 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md)가 오며 이 경고 다음에는 심각한 오류인 [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md)가 옵니다.
