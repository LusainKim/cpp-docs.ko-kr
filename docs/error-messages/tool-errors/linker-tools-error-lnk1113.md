# 링커 도구 오류 LNK1113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컴퓨터 종류 type이\(가\) 잘못되었습니다.  
  
 개체 헤더에 지정된 컴퓨터 종류가 잘못되었습니다.  
  
 이 오류는 파일이 손상된 경우에 발생할 수 있습니다.  개체를 다시 빌드하십시오.
