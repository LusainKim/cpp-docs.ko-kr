# 리소스 컴파일러 오류 RC2111
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컨트롤 형식이 잘못되었습니다.  
  
 **DIALOG** 문에 들어 있는 각 CONTROL 문은 다음 중 하나여야 합니다: 3STATE, AUTO3, AUTOCHECK, AUTORADIO, BEDIT, CHECKBOX, COMBOBOX, CONTROL, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ICON, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, SCROLLBAR, USERBUTTON.  
  
 CONTROL 문을 제대로 입력했는지 확인하십시오.
