# 리소스 컴파일러 오류 RW4003
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

VIRTKEY 없이 SHIFT 또는 CONTROL을 사용했습니다.  
  
 액셀러레이터 키 테이블 리소스에서 SHIFT나 CONTROL을 사용하려면 VIRTKEY가 있어야 합니다.  SHIFT와 CONTROL은 VIRTKEY 형식 액셀러레이터 키에서 플래그 비트로 나타나기 때문에 VIRTKEY로부터 독립적으로 존재할 수 없습니다.
