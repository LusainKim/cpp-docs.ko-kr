# 식 계산기 오류 CXX0009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

문자 상수 다음에 작은따옴표\('\)가 없습니다.  
  
 문자 상수 끝에 있어야 할 작은따옴표가 없습니다.  문자 상수를 작은따옴표로 묶어 식을 다시 입력하십시오.  
  
 이 오류는 CAN0009와 동일합니다.
