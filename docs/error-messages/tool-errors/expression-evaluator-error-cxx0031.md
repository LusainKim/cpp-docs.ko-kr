# 식 계산기 오류 CXX0031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

식을 확장할 수 없습니다.  
  
 식 계산기에 내부 오류가 발생했습니다.  
  
 올바르게 계산되는 동일한 식을 작성할 수 있습니다.  
  
 이 오류는 CAN0031과 동일합니다.
