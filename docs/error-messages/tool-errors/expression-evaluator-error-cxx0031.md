# <a name="expression-evaluator-error-cxx0031"></a>식 계산기 오류 CXX0031
식 확장할 수 없습니다.  
  
 식 계산기에서 내부 오류가 발생했습니다.  
  
 올바르게 계산 될 수 있는 해당 하는 식을 작성할 수 있습니다.  
  
 이 오류는 can0031과 동일 합니다.
