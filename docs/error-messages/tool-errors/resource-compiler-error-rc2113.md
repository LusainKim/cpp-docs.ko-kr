# 리소스 컴파일러 오류 RC2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

대화 상자에는 END가 있어야 합니다.  
  
 **END** 키워드는 **DIALOG** 문 끝에 와야 합니다.  이전 문의 여는 따옴표가 남아 있으면 안 됩니다.
