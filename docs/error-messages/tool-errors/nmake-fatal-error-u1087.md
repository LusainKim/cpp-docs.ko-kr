# NMAKE 심각한 오류 U1087
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

동일한 대상에 :과 ::을 사용할 수 없습니다.  
  
 단일 콜론\(**:**\) 및 이중 콜론\(`::`\) 종속성 모두에 대상을 지정할 수 없습니다.  
  
 대상을 여러 설명 블록에 지정하려면 각 종속성 줄에 `::`을 사용합니다.
