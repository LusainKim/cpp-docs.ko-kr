# 리소스 컴파일러 심각한 오류 RC1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

옵션 스위치가 잘못되었습니다. \/d 스위치가 너무 많습니다.  
  
 기호 정의 옵션\(\/d\)을 사용하여 너무 많은 기호를 정의했습니다.  일부 기호를 소스 파일에 정의한 다음 다시 컴파일하십시오.
