# C 런타임 오류 R6019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

콘솔 장치를 열 수 없습니다.  
  
 CONIO.h에 선언되어 있는 콘솔 함수를 호출했으나 운영 체제에서 콘솔에 대한 액세스를 허용하지 않았습니다.
