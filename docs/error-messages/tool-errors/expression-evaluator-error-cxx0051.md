# 식 계산기 오류 CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

오버로드된 두 개 이상의 기호가 중단점에 지정되었습니다.  
  
 식 계산기는 오버로드된 두 개 이상의 기호 중에서 어떤 것을 중단점으로 사용할지 결정할 수 없습니다.  
  
 이 오류는 CAN0051과 동일합니다.
