# 식 계산기 오류 CXX0060
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

레지스터를 사용할 수 없습니다.  
  
 식이 사용할 수 없는 레지스터를 지정했습니다.  이 오류는 실행 중인 컴퓨터에 없는 레지스터에 액세스할 경우 발생합니다.  
  
 이 오류는 CAN0060과 동일합니다.
