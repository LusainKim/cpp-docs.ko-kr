# 링커 도구 오류 LNK1152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

데코레이팅되지 않은 기호를 하나 이상 확인할 수 없습니다.  
  
 이 오류는 확인할 수 없는 각 데코레이팅되지 않은 기호에 대한 하나의 경고 [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md)와 데코레이팅되지 않은 기호에 대해 발견된 중복 기호에 대한 두 개 이상의 경고 [LNK4002](../../error-messages/tool-errors/linker-tools-warning-lnk4002.md)가 발생한 다음에 발생합니다.
