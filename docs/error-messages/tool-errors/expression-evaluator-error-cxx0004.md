# 식 계산기 오류 CXX0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구문 오류입니다.  
  
 식 구문이 잘못되었습니다.  
  
 올바른 구문으로 식을 다시 입력하십시오.  
  
 이 오류는 CAN0004와 동일합니다.
