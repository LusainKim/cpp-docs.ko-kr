# 리소스 컴파일러 심각한 오류 RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

컴파일러 한계: 'macro' 매크로가 너무 많이 중첩되었습니다.  
  
 파일이 매크로 중첩에 대한 리소스 컴파일러의 한계를 초과했습니다.  지정한 소스 파일을 수정하여 매크로 중첩 한도를 줄이십시오.
