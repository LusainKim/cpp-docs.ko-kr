# 리소스 컴파일러 심각한 오류 RC1208
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

입력 파일의 확장명이 .RES입니다.  
  
 확장명 .RES는 리소스 컴파일러의 출력으로 사용되며  리소스 컴파일러 스크립트에는 .RC 확장명을 사용해야 합니다.
