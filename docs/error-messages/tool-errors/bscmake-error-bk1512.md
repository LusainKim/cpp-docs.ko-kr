# BSCMAKE 오류 BK1512
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename: 용량을 초과했습니다.  
  
 정의 개수, 참조 개수, 모듈 개수 또는 기타 정보가 한계를 초과했기 때문에 BSCMAKE가 찾아보기 정보를 빌드할 수 없습니다.  
  
### 문제를 해결하려면 다음과 같은 해결책을 사용해 보십시오.  
  
1.  \/Em, \/Es 또는 \/Ei를 사용하여 일부 정보를 제외합니다.  
  
2.  \/Iu 옵션을 생략합니다.
