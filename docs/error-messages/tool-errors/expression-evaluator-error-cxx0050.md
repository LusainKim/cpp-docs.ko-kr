# 식 계산기 오류 CXX0050
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

클래스\/구조체\/공용 구조체 형식은 ::의 왼쪽에 나와야 합니다.  
  
 범위 결정 연산자\(`::`\) 왼쪽에 있는 기호가 클래스, 구조체 또는 공용 구조체가 아닙니다.  
  
 이 오류는 CAN0050과 동일합니다.
