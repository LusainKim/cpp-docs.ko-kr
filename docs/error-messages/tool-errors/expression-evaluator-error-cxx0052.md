# 식 계산기 오류 CXX0052
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

멤버 함수가 없습니다.  
  
 멤버 함수가 중단점으로 지정되었지만 찾을 수 없습니다.  이 오류는 인라인 함수에 중단점을 설정한 경우 발생합니다.  
  
 인라인을 강제로 끄고\(\/Ob0\) 다시 컴파일하여 함수에 중단점을 설정합니다.  
  
 식이 정의되지 않은 함수를 호출했습니다.  
  
 이 오류는 CAN0052와 동일합니다.
