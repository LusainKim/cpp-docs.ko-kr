# 명령줄 오류 D8022
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'messagefile'을\(를\) 열 수 없습니다.  
  
 지정한 파일이 현재 디렉터리에 없거나 PATH 환경 변수에 지정한 디렉터리에 없습니다.  메시지 파일에는 컴파일러 명령줄 구문과 옵션에 대한 요약 정보가 들어 있습니다.  
  
 이 파일을 현재 디렉터리나 현재 경로에 있는 디렉터리로 이동합니다.  파일을 찾을 수 없으면 SETUP 프로그램을 실행하여 배포 디스크로부터 복사합니다.
