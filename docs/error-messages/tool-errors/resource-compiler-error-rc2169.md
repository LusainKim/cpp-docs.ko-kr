# 리소스 컴파일러 오류 RC2169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename 리소스 파일이 2.03 형식이 아닙니다.  
  
 지정한 리소스는 버전 2.03 이전 형식을 사용했습니다.  이 리소스 파일을 버전 3.00 이상의 형식으로 변환하거나 다시 만들어야 합니다.
