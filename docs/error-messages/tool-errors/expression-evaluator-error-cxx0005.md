# 식 계산기 오류 CXX0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

지원되지 않는 연산자입니다.  
  
 식에 지원되지 않는 C 연산자가 지정되었습니다.  지원되는 C 연산자를 사용하여 같은 식을 작성하십시오.  
  
 이 오류는 CAN0005와 동일합니다.
