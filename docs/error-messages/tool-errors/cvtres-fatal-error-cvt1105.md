# CVTRES 심각한 오류 CVT1105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일에서 찾을 수 없습니다.  
  
 CVTRES는 파일에서 특정 위치로 이동할 수 없습니다.  
  
 이 오류는 파일이 손상된 경우에 발생할 수 있습니다.
