# 리소스 컴파일러 심각한 오류 RC1047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

"option 옵션이 너무 많습니다. 'string'"  
  
 옵션을 너무 많이 지정했습니다.  지정한 문자열은 오류를 발생시킨 옵션에 대한 인수입니다.
