# 프로젝트 빌드 오류 PRJ0014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

생성된 프로세스를 제어하는 작업 개체가 실패했습니다.  빌드를 계속할 수 없습니다.  
  
 개발 환경에서 오류가 발생했습니다.  
  
 이 오류를 해결하려면 프로젝트를 닫고 다시 로드하십시오.  필요한 경우 Visual Studio를 종료하고 다시 시작하십시오.
