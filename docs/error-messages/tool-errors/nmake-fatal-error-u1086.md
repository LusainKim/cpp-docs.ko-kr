# NMAKE 심각한 오류 U1086
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**유추 규칙은 종속 파일을 사용할 수 없습니다.**  
  
 유추 규칙에 있는 콜론\(**:**\) 뒤에는 다음 중 하나가 와야 합니다.  
  
-   줄 바꿈 문자  
  
-   뒤에 명령이 오는 세미콜론\(**;**\)  
  
-   뒤에 주석이 오는 파운드 기호\(**\#**\)
