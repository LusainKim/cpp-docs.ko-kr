# 리소스 컴파일러 오류 RC2167
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

인식할 수 없는 VERSIONINFO 필드입니다. BEGIN 또는 쉼표가 있어야 합니다.  
  
 **VERSIONINFO** 구조체 선언의 **FIXED** 부분에 인식할 수 없는 필드가 있습니다.  **VERSIONINFO** 필드는 쉼표로 구분된 DWORDS여야 합니다.
