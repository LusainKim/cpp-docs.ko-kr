# 프로젝트 빌드 오류 PRJ0005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'directory' 디렉터리에 임시 파일을 만들 수 없습니다.  
  
 임시 파일을 만들기 위한 호출이 실패했습니다.  호출이 실패한 원인은 다음과 같습니다.  
  
-   임시 파일 이름이 부족합니다.  
  
-   임시 디렉터리가 읽기 전용입니다.  
  
-   임시 디렉터리나 TMP 환경 변수가 없습니다.  
  
-   시스템의 여유 공간이 부족합니다.
