# 식 계산기 오류 CXX0034
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**연산자와 호환될 수 없는 형식입니다.**  
  
 지정된 피연산자 형식은 이 연산에 적합하지 않습니다.  
  
 예를 들어, 어떤 값으로도 포인터를 곱할 수 없습니다.  
  
 피연산자를 형식 변환하면 연산자와 호환될 수 있습니다.
