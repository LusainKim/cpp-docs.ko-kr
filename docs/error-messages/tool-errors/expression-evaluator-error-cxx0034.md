# <a name="expression-evaluator-error-cxx0034"></a>식 계산기 오류 CXX0034
**연산자와 호환 되지 않는 형식**  
  
 지정 된 피연산자 형식을 작업에 적합 하지 않습니다.  
  
 예를 들어, 모든 값에 대 한 포인터를 곱합니다 수 없습니다.  
  
 입력 해야 할 수 피연산자를 연산자와 호환 되는 형식으로 캐스팅 합니다.
