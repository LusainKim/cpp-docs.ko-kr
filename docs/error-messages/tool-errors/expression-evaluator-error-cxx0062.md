# 식 계산기 오류 CXX0062
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

지원되지 않는 생성자 호출입니다.  
  
 식이 생성자를 호출했습니다.  식은 생성자를 명시적으로 호출할 수 없으며 생성자를 호출해야 되는 변환을 수행할 수 없습니다.  
  
 이 오류는 CAN0062와 동일합니다.
