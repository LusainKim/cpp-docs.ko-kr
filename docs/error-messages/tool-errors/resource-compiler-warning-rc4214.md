# 리소스 컴파일러 오류 RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

코드 페이지가 잘못되었습니다. 무시했습니다.  
  
 .rc 파일에 코드 페이지 인수가 들어 있으며 지정된 코드 페이지가 잘못되었습니다.  자세한 내용은 [IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674)를 참조하십시오.
