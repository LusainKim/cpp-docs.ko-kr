# 수학 오류 M6205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : \_TLOSS 오류  
  
 전체 상위값 손실\(정밀도 손실\)이 발생했습니다.  
  
 피연산자는 0과 2\*pi 사이 숫자여야 하는데 sin, cos 또는 tan의 피연산자로 매우 큰 숫자를 지정한 경우 이 오류가 발생합니다.
