# 링커 도구 경고 LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

잘못된 지시문 \/directive이\(가\) 발견되었습니다. 무시됩니다.  
  
 [comment](../../preprocessor/comment-c-cpp.md) pragma를 통해 전달한 지시문이 잘못되었습니다.  링커가 `/directive`를 무시합니다.
