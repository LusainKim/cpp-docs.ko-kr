# 링커 도구 경고 LNK4199
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:dllname을\(를\) 무시합니다. dllname에서 가져오기를 찾을 수 없습니다.  
  
 `dllname`이 내보낸 함수는 필요하지 않으므로 링커가 `dllname`을 무시합니다.
