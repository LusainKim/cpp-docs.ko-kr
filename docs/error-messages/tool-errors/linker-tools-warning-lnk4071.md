# 링커 도구 경고 LNK4071
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

다음 링크에 대해 증분 링크할 수 없습니다.  
  
 LINK가 하나 이상의 기호에 대해 여러 정의를 발견했지만 출력 파일을 만드는 데 오류에 관계 없이 [\/FORCE](../../build/reference/force-force-file-output.md) 또는 **\/FORCE:MULTIPLE**이 사용되었습니다.  LINK가 증분 상태 파일\(.ilk\)을 삭제했습니다.
