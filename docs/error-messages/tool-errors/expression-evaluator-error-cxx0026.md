# <a name="expression-evaluator-error-cxx0026"></a>식 계산기 오류 CXX0026
서식 문자열이 잘못되었습니다.  
  
 서식 문자열을 잘못 지정 되었습니다. 식의 구문을 확인합니다.  
  
 이 오류는 can0026과 동일 합니다.  
  
 자세한 내용은 참조 [c + +의 형식 지정자](/visualstudio/debugger/format-specifiers-in-cpp)합니다.
