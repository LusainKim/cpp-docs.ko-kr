# 수학 오류(M6101~M6205)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이 섹션은 빌드 도구에 의해 생성된 오류에 대한 참조입니다.  특정 오류 메시지에 대한 도움말을 가져오려면 출력 창에 있는 오류 번호에서 마우스를 클릭하고 F1 키를 누르거나 인덱스의 **찾을 대상** 상자에 오류 번호를 입력합니다.
