# 식 계산기 오류 CXX0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구조체\/공용 구조체 요소가 아닙니다.  
  
 'Struct.Member' 또는 'pStruct\-\>Member' 형식의 식이 지정되었지만 \<member\>은\(는\) 구조체의 요소가 아닙니다.  
  
 식이 괄호로 올바르게 묶여 있지 않습니다.  
  
 이 오류는 CAN0028과 동일합니다.
