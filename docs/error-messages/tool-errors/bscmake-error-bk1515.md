# BSCMAKE 오류 BK1515
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

bscfile: 호환되지 않는 버전입니다. 증분 업데이트를 할 수 없습니다.  
  
 .bsc 파일이 다른 버전의 BSCMAKE로 만들어졌습니다.  .bsc 파일을 다시 만드십시오.
