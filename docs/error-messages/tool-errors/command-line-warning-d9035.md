# 명령줄 경고 D9035
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'option' 옵션은 더 이상 사용되지 않으므로 이후 릴리스에서 제거될 예정입니다.  
  
 이후 Visual C\+\+ 컴파일러 릴리스에서 제거될 예정인 컴파일러 옵션을 지정했습니다.  *option*을 대신할 수 있는 제안 방법이 있는 경우 이 경고에 뒤이어 [D9036](../../error-messages/tool-errors/command-line-warning-d9036.md) 경고가 표시됩니다.  
  
 지정한 옵션도 작동하지만 지금 빌드 구성을 업데이트해야 합니다.  이렇게 하면 컴파일러를 업그레이드하는 경우에도 프로젝트를 계속 빌드할 수 있습니다.
