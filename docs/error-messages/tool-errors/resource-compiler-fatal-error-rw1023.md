# 리소스 컴파일러 심각한 오류 RW1023
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

파일에 쓰는 동안 I\/O 오류가 발생했습니다. 드라이브가 꽉 찼습니다.  
  
 사용 가능한 공간의 크기는 만들고 있는 실행 파일 크기의 두 배 이상이 되어야 합니다.
