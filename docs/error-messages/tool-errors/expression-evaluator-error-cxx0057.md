# 식 계산기 오류 CXX0057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

줄 번호에 코드가 없습니다.  
  
 지정한 줄 번호에 대해 생성된 코드가 없습니다.  중단점으로 사용할 수 없습니다.  
  
 이 오류는 CAN0057과 동일합니다.
