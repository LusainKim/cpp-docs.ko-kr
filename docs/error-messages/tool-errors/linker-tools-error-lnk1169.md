# 링커 도구 오류 LNK1169
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

하나 이상의 곱하기 정의 기호가 있습니다.  
  
 하나 이상의 기호에 대해 정의가 여러 개 있으므로 빌드하지 못했습니다.  이 오류는 오류 [LNK2005](../../error-messages/tool-errors/linker-tools-error-lnk2005.md)가 발생한 다음에 발생합니다.  
  
 [\/FORCE](../../build/reference/force-force-file-output.md) 또는 **\/FORCE:MULTIPLE** 옵션은 이 오류를 재정의합니다.
