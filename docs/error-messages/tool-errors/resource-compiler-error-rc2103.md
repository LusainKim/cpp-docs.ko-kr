# 리소스 컴파일러 오류 RC2103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

문자열 리터럴에 예기치 않은 파일의 끝입니다.  
  
 문자열이 끝나기 전에 파일 끝이 있습니다.  문자열에 닫는 큰따옴표\(**"**\)가 없습니다.
