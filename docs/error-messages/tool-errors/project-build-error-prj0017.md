# 프로젝트 빌드 오류 PRJ0017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

현재 작업 디렉터리가 잘못되었습니다.  
  
 현재 작업 디렉터리 경로가 \_MAXPATH보다 깁니다.  이 오류를 해결하려면 프로젝트를 이러한 중첩 수준으로 만들지 마십시오.
