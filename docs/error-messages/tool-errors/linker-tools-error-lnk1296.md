# 링커 도구 오류 LNK1296
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

filename을\(를\) 로드할 수 없습니다.  
  
 지정된 DLL을 사용하지 못했습니다.  
  
 DLL의 예상 위치는 현재 디렉터리, 시스템 디렉터리, Windows 디렉터리 및 PATH 환경 변수에 지정된 디렉터리입니다.
