# 링커 도구 오류 LNK1248
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이미지 크기\('size'\)가 허용하는 최대 크기\(80000000\)를 초과합니다.  
  
 링커는 출력 파일의 크기가 프로그램 이미지가 허용하는 최대 크기를 초과하는지 여부를 확인합니다.  프로그램을 여러 개의 DLL로 만들 수도 있습니다.  PE\(프로그램 실행 파일\)에 대한 자세한 내용은 [MSDN Library](http://go.microsoft.com/fwlink/?linkid=556)의 "Peering Inside the PE: A Tour of the Win32 Portable Executable File Format"을 참조하십시오.
