# 프로젝트 빌드 오류 PRJ0015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

시스템에 NULL 장치가 없습니다.빌드를 시작할 수 없습니다.  
  
 시스템 리소스 부족으로 인해 이 오류가 생길 수 있습니다.  일부 응용 프로그램을 닫거나 다시 부팅합니다.  
  
 컴퓨터의 NULL 장치에 대한 충분한 권한이 없는 경우에도 이 오류가 발생할 수 있습니다.
