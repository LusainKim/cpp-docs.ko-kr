# 리소스 컴파일러 오류 RC2165
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

DLGINCLUDE 문에 문자열이 없습니다.  
  
 해당 문이 유효한 포함 파일을 지정하지 않았습니다.  
  
 **DLGINCLUDE** 문에는 다음 구문을 사용해야 합니다.  
  
 **DLGINCLUDE "*filename*.h"**
