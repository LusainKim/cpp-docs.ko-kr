# <a name="expression-evaluator-error-cxx0002"></a>식 계산기 오류 CXX0002

 식은 디버깅 중인 프로그램에 할당 되지 않은 메모리 참조 하려고 합니다.  
  
 이 오류는 can0002와 동일 합니다.
