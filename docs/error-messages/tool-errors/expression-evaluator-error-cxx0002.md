# 식 계산기 오류 CXX0002
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

사용자 메모리에 액세스하는 동안 오류가 발생했습니다.  
  
 식이 디버깅 중인 프로그램에 할당되지 않은 메모리를 참조하려고 했습니다.  
  
 이 오류는 CAN0002와 동일합니다.
