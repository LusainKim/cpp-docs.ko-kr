# 리소스 컴파일러 심각한 오류 RW1004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

예기치 않은 파일의 끝입니다.  
  
 이 오류는 텍스트 파일의 마지막 줄에 줄 바꿈 및 캐리지 리턴 문자가 없는 경우에 발생할 수 있습니다.
