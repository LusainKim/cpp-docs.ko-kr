# 식 계산기 오류 CXX0016
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

상수가 너무 큽니다.  
  
 C 식 계산기는 4,294,967,295\(0FFFFFFFF 16진수\)보다 큰 부호 없는 정수형 상수나 크기가 약 1.8E\+308보다 큰 부동 소수점 상수를 계산할 수 없습니다.  
  
 이 오류는 CAN0016과 동일합니다.
