# BSCMAKE 오류 BK1517
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

sbrfile의 소스 파일이 \/Yc 및 \/Yu로 컴파일되었습니다.  
  
 .sbr 파일이 자신을 참조합니다.  \/Yc를 사용하여 컴파일한 다음 \/Yu로 다시 컴파일했습니다.  소스 파일의 컴파일러 옵션을 \/Yc로 다시 설정한 다음 **다시 빌드**를 선택하여 새 .sbr 파일을 생성합니다.  소스 파일을 \/Yu로 다시 컴파일하지 마십시오.
