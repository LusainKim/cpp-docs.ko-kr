# 링커 도구 경고 LNK4205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'filename'에 참조 모듈에 대한 현재 디버깅 정보가 없습니다. 디버그 정보가 없는 것처럼 개체를 링크합니다.  
  
 .pdb 파일에 만료된 정보가 있습니다.  링커가 디버그 정보 없이 개체를 계속 링크합니다.  [\/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) 옵션을 사용하여 개체 파일을 다시 컴파일할 수도 있습니다.
