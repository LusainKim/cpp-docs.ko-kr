# 식 계산기 오류 CXX0013
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

연산자가 없습니다.  
  
 식에 있어야 할 연산자를 찾을 수 없습니다.  식의 구문을 확인하십시오.  
  
 이 오류는 CAN0013과 동일합니다.
