# 프로젝트 빌드 오류 PRJ0047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

중단된 프로세스를 계속할 수 없습니다.  빌드하지 못했습니다.  
  
 개발 환경에서 오류가 발생했습니다.  
  
 이 오류를 해결하려면 프로젝트를 닫고 다시 로드하십시오.  필요한 경우 Visual Studio를 종료하고 다시 시작하십시오.
