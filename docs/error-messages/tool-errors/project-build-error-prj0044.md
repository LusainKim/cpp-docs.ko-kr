# 프로젝트 빌드 오류 PRJ0044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'rule' 사용자 지정 빌드 규칙\('file' 파일에 할당됨\)에 대한 '추가 종속성' 속성이 잘못되었습니다.이 속성에 'string'이\(가\) 포함되어 있으며 이것은 'value'\(으\)로 계산됩니다.  
  
 **추가 종속성** 속성이 빈 문자열이거나 잘못된 문자\(파일 또는 디렉터리 이름에 사용할 수 없는 문자\)를 포함하는 문자열입니다.  사용자 지정 빌드 규칙에서는 빌드 작업의 출력이 필요합니다.  
  
 자세한 내용은 [사용자 지정 빌드 도구 지정](../../ide/specifying-custom-build-tools.md)을 참조하십시오.
