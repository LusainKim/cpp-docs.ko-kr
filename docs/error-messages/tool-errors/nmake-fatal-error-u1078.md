# NMAKE 심각한 오류 U1078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'expression'에 상수 오버플로가 발생했습니다.  
  
 지정한 식에 \-2,147,483,648과 2,147,483,647 사이 범위를 초과하는 상수가 들어 있습니다.  상수는 다음 상황에서 나타납니다.  
  
-   전처리 지시문으로 지정된 식  
  
-   대시\(**\-**\) 명령 한정자로 지정된 오류 수준
