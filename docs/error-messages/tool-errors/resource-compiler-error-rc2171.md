# 리소스 컴파일러 오류 RC2171
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

알 수 없는 DIB 헤더 형식입니다.  
  
 해당 비트맵 헤더는 **BITMAPCOREHEADER** 또는 **BITMAPINFOHEADER** 구조가 아닙니다.
