# 링커 도구 오류 LNK1215
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

메타데이터 작업이 실패했습니다\(HRESULT\) : error  
  
 .NET 런타임을 통해 메타데이터를 업데이트하려는 중에 링커가 .NET 런타임에서 오류를 수신했습니다.  
  
 `HRESULT`는 .NET 런타임 메서드로부터의 HRESULT이며,  `error`는 .NET이 제공하는 텍스트입니다.  
  
 일치하지 않는 링커 및 .NET 런타임을 사용하는 것 같습니다. Visual C\+\+를 다시 설치하십시오.
