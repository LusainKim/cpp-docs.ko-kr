# NMAKE 심각한 오류 U1077
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'program': 'value' 반환 코드  
  
 NMAKE가 호출한 지정된 명령 또는 프로그램이 실패하여 지정된 종료 코드를 반환했습니다.  
  
 이 오류를 표시하지 않고 NMAKE 세션을 계속하려면 \/I 옵션과 **.IGNORE** 점 지시문 또는 대시\(**\-**\) 명령 한정자를 사용합니다.  종속 트리의 관련되지 않은 부분에 대해 NMAKE 세션을 계속하려면 \/K 옵션을 사용합니다.
