# 프로젝트 빌드 오류 PRJ0004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'tool' 도구에 대한 명령줄을 생성할 수 없습니다.  
  
 다음과 같이 하나 이상의 속성이 잘못 지정되어 ***tool***에 대한 호출 구문이 잘못되었습니다.  
  
-   잘못된 매크로나 알 수 없는 매크로를 지정했을 수 있습니다.  
  
-   시스템의 여유 공간이 부족할 수 있습니다.  
  
 [명령줄 속성 페이지](../../ide/command-line-property-pages.md)를 검사하여 도구의 설정을 검토할 수 있습니다.
