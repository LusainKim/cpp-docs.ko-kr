# ML Nonfatal Error A2219
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**잘못 된 정렬 오프셋에 대 한 해제 코드**  
  
 피연산자에 대 한 [.ALLOCSTACK](../../assembler/masm/dot-allocstack.md) 및 [.SAVEREG](../../assembler/masm/dot-savereg.md) 8의 배수 여야 합니다.  피연산자에 대 한 [.SAVEXMM128](../../assembler/masm/dot-savexmm128.md) 및 [.SETFRAME](../../assembler/masm/dot-setframe.md) 은 16의 배수 여야 합니다.
