# TEXTEQU
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

할당  *textitem* 에  *이름*.   *Textitem* 리터럴 문자열 앞에 상수가 될 수 있습니다 있는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.  
  
## 구문  
  
```  
  
name TEXTEQU [[textitem]]  
```
