# POPCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

일부 또는 전체를 현재 복원 `context` \(저장 하는  [PUSHCONTEXT](../../assembler/masm/pushcontext.md) 지시문\).  The `context` can be **ASSUMES**, `RADIX`, **LISTING**, **CPU**, or **ALL**.  
  
## 구문  
  
```  
  
POPCONTEXT context  
```
