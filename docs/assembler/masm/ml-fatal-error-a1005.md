# ML Fatal Error A1005
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**어셈블러 제한: 매크로 매개 변수 이름 테이블이 꽉 찼습니다.**  
  
 너무 많은 매개 변수, 지역 또는 레이블 매크로 매크로를 정의 했습니다.  매크로 이름 테이블에 더 이상 공간이 되었습니다.  
  
 짧은 정의 또는 더 적은 이름 또는 불필요 한 매크로 제거 합니다.
