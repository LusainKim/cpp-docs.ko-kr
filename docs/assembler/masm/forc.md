# FORC
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

각 문자에 대해 한 번씩 반복 됩니다 블록 표시 `string`, 현재 문자 대체와 `parameter` 에서 반복 되는.  
  
## 구문  
  
```  
  
   FORC  
parameter, <string> statements  
ENDM  
```
