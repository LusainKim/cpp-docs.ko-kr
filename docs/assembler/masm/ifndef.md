# IFNDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

경우 어셈블리 부여  *이름* 정의 되지 않았습니다.  
  
## 구문  
  
```  
  
IFNDEF name  
```  
  
## 설명  
 참조 하십시오  [IF](../../assembler/masm/if-masm.md) 전체 구문에 대 한.
