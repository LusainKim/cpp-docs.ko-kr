# INCLUDELIB (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

현재 모듈에 연결 해야 하는 링커가 알려 주는  *라이브러리 이름*.  
  
## 구문  
  
```  
  
INCLUDELIB libraryname  
```  
  
## 설명  
 *라이브러리 이름* 세미콜론, 큰 백슬래시 포함 되어 있는 경우 꺽쇠 괄호 안에 묶어야 합니다\-기호, 적은\-기호, 작은따옴표 또는 큰따옴표입니다.
