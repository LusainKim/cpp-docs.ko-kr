# <a name="operator-ptr"></a>연산자 PTR
첫 번째 연산자 강제로 *식* 지정 된 것으로 간주 `type`합니다. 에 대 한 포인터를 지정 하는 두 번째 연산자 `type`합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
