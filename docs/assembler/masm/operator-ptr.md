# operator PTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

첫 번째 연산자 강제로 있는  *식* 를 지정한 것으로 처리 합니다 `type`.  포인터를 두 번째 연산자 지정 `type`.  
  
## 구문  
  
```  
  
      type PTR expression  
[[distance]] PTR type  
```
