# EXITM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

확장은 현재 반복 또는 매크로 블록을 종료 하 고 문 블록 외부의 어셈블리에 시작 됩니다.  
  
## 구문  
  
```  
  
EXITM [[textitem]]   
```  
  
## 설명  
 매크로 함수에서  *textitem* 값이 반환 됩니다.
