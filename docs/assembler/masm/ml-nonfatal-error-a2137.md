# <a name="ml-nonfatal-error-a2137"></a>ML 심각하지 않은 오류 A2137
**INVOKE에 인수가 너무 적습니다.**  
  
 수의 인수를 사용 하 여 전달 되는 [INVOKE](../../assembler/masm/invoke.md) 지시문이 호출 되는 프로시저에 대 한 프로토타입에 지정 하는 필수 매개 변수 수보다 적습니다.
