# .NOCREF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

기호를 기호 테이블 및 브라우저 파일에서 목록이 표시 되지 않습니다.  
  
## 구문  
  
```  
  
.NOCREF [[name[[, name]]...]]  
```  
  
## 설명  
 그런 다음 이름이 지정 된 경우 지정 된 이름은 억제 됩니다.  다른 이름으로 동일한  [.XCREF](../../assembler/masm/dot-xcref.md).
