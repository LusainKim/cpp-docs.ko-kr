# .ERRDEF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

경우 오류를 생성 합니다.  *이름* 미리 정의 된 레이블, 변수 또는 기호입니다.  
  
## 구문  
  
```  
  
.ERRDEF name [[, message]]  
```
