# .386
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

80386 프로세서에 대 한 명령 권한 없는 어셈블리를 사용합니다. 나중에 프로세서를 도입 지침의 어셈블리를 사용할 수 없습니다.  
  
## 구문  
  
```  
  
.386  
  
```  
  
## 설명  
 또한 80387 명령을 사용할 수 있습니다.
