# ML Warning A4014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

지침 및 초기화 된 데이터가 BSS 세그먼트가 없습니다.  
  
 BSS 섹션 내에서 초기화 된 데이터를 정의 하려고 했습니다.  BSS 섹션 이름이 BSS이 클래스로 정의 됩니다.  여기에 단순화 된 세그먼트 `.data?`.
