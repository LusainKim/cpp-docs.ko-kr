# ML Nonfatal Error A2133
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**INVOKE를 덮어쓰기 값 등록**  
  
 레지스터 프로시저에 인수로 전달 된 있지만 코드를 생성 하 여  [호출](../../assembler/masm/invoke.md) 다른 인수를 전달 하는 레지스터의 내용을 파괴 합니다.  
  
 데이터 변환 작업을 수행 하 여 어셈블러 AX, AL, AH, EAX, DX, DL, DH를 및 EDX 레지스터를 사용할 수 있습니다.  
  
 다른 레지스터를 사용 합니다.
