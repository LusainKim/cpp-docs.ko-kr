# <a name="df"></a>DF
할당 및 필요에 따라 각각에 대해 6 바이트의 저장소를 초기화 `initializer`합니다. `DF`동의어 [FWORD](../../assembler/masm/fword.md)합니다.  
  
## <a name="syntax"></a>구문  
  
```  
[[name]] DF initializer [[, initializer]]...  
```
