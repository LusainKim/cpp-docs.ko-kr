# operator NE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

경우 true \(\-1\)를 반환  *식 1* 과 일치 하지 않는  *식 2*, 경우 false \(0\)를 반환 하거나.  
  
## 구문  
  
```  
  
expression1 NE expression2  
```
