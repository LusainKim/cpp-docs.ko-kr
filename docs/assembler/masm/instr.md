# INSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

첫 항목을 찾는  *textitem2* 에서  *textitem1*.  
  
## 구문  
  
```  
  
name  
 INSTR [[position,]] textitem1, textitem2  
```  
  
## 설명  
 시작  *위치* 는 선택 사항입니다.  각 텍스트 항목 앞에 상수 리터럴 문자열 수는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.
