# operator IMAGEREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

이미지 상대 오프셋을 반환 합니다. `expression`.  
  
## 구문  
  
```  
IMAGEREL expression  
```  
  
## 설명  
 결과 값 종종 RVA 또는 상대 가상 주소 라고 합니다.  
  
 IMAGEREL는 COFF 개체 내보내기를 사용할 수 있습니다.
