# ML Nonfatal Error A2019
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**피연산자 형식 레코드 또는 필드가 있어야 합니다.**  
  
 피연산자가 다음의  [폭](../../assembler/masm/operator-width.md) 또는  [마스크](../../assembler/masm/operator-mask.md) 연산자는 유효 하지 않습니다.  
  
 해당  [폭](../../assembler/masm/operator-width.md) 연산자는 레코드 또는 필드의 이름을 나타내는 피연산자를 사용 합니다.  해당  [마스크](../../assembler/masm/operator-mask.md) 연산자 필드 또는 레코드 종류 이름을 나타내는 피연산자를 사용 합니다.
