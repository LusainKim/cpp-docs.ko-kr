# .LISTMACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

시작 코드 또는 데이터 생성의 매크로 확장 문을 나열 합니다.  
  
## 구문  
  
```  
  
.LISTMACRO  
  
```  
  
## 설명  
 이 값이 기본값입니다.  다른 이름으로 동일한  [.XALL](../../assembler/masm/dot-xall.md).
