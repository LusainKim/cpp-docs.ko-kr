# ML Nonfatal Error A2206
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**식에 연산자가 없습니다.**  
  
 연산자가 누락 된 식을 계산할 수 없습니다.  이 오류 메시지 이전 프로그램 오류는 파생 될 수도 있습니다.  
  
 다음 줄이이 오류를 생성 합니다.  
  
```  
value1 = ( 1 + 2 ) 3  
  
```
