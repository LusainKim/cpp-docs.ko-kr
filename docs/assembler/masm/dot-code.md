# .CODE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

사용 하는 경우  [.모델](../../assembler/masm/dot-model.md), 코드 세그먼트의 시작을 나타냅니다.  
  
## 구문  
  
```  
.CODE [[name]]  
```  
  
#### 매개 변수  
  
|Parameter|설명|  
|---------------|--------|  
|`name`|코드 세그먼트 이름을 지정 하는 선택적 매개 변수입니다.  \_TEXT 작은, 소형, 압축, 및 평면에 대 한 기본 이름인  [모델](../../assembler/masm/dot-model.md).  기본 이름은  *modulename*\_TEXT 다른 모델에 대 한.|
