# <a name="ifnb"></a>IFNB
어셈블리를 부여 *textitem* 비어 있지 않은 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
IFNB textitem  
```  
  
## <a name="remarks"></a>설명  
 참조 [IF](../../assembler/masm/if-masm.md) 전체 구문에 대 한 합니다.
