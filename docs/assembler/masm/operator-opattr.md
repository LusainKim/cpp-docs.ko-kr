# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

모드 및 범위를 정의 하는 단어를 반환 합니다.  *식*.  하위 바이트를 반환 하는 바이트와 동일  [.형식](../../assembler/masm/operator-dot-type.md).  높은 바이트 추가 정보가 포함 되어 있습니다.  
  
## 구문  
  
```  
  
OPATTR expression  
```
