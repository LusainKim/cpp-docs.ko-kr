# ML Nonfatal Error A2031
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**인덱스 또는 기본 레지스터 여야 합니다.**  
  
 기본 또는 인덱스 레지스터 메모리 식에 없는 레지스터 사용 하려고 했습니다.  
  
 예를 들어, 다음 식은이 오류가 발생 합니다.  
  
```  
[ax]  
[bl]  
```
