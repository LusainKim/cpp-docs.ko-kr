# ML Fatal Error A1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**줄이 너무 깁니다.**  
  
 소스 파일에서 줄 512 문자 제한을 초과 했습니다.  
  
 다음 여러 실제 줄 줄 연속 문자 \(\\\)와 연결 되 면 결과 논리 줄 여전히 512 자로 제한 됩니다.
