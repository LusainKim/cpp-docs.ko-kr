# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**잘못 된 함께 세그먼트 맞춤**  
  
 지정 된 맞춤은  **맞춤** 또는 `EVEN` 지시문에 지정 된 대로 현재 세그먼트 맞춤 보다 큰 것은  **세그먼트** 지시문입니다.
