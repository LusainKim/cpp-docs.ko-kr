# operator MASK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

반환 비트 마스크의 비트  *recordfieldname* 또는  *레코드* 설정 됩니다, 다른 비트가 모두 선택 취소 합니다.  
  
## 구문  
  
```  
  
MASK {recordfieldname | record}  
```
