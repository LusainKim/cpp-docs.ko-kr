# ML Fatal Error A1017
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**원본 파일 이름이 없습니다.**  
  
 ML 조립 또는 링커로 전달할 파일을 찾을 수 없습니다.  
  
 수행 대상이 되는 파일 이름을 지정 하지 않고 ML 명령줄 옵션을 지정 하면이 오류가 생성 됩니다.  .Asm 확장명 없는 파일을 어셈블할 수 있는  **\/Ta** 명령줄 옵션입니다.  
  
 또한 명령줄 옵션 ML 환경 변수에 포함 되어 있는 경우 매개 변수 없이 ML를 호출 하 여이 오류를 생성할 수 있습니다.
