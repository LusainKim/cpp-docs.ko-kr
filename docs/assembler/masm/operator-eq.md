# operator EQ
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

경우 true \(\-1\)를 반환  *식 1* 인  *식 2*, 또는 그렇지 않은 경우 false \(0\)를 반환 합니다.  
  
## 구문  
  
```  
  
expression1 EQ expression2  
```
