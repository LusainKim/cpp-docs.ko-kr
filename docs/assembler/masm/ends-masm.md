# <a name="ends-masm"></a>ENDS (MASM)
세그먼트, 구조체 또는 공용 구조체의 끝을 표시 *이름* 이전에 시작 된 [세그먼트](../../assembler/masm/segment.md), [구조체](../../assembler/masm/struct-masm.md), [UNION](../../assembler/masm/union.md), 또는 간소화 된 세그먼트 지시문입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name ENDS  
```
