# ENDS (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

세그먼트, 구조체 또는 공용 구조체의 끝을 표시  *이름* 이전에 시작한  [세그먼트](../../assembler/masm/segment.md),  [구조체](../../assembler/masm/struct-masm.md),  [UNION](../../assembler/masm/union.md), 또는 단순화 된 세그먼트 지시문입니다.  
  
## 구문  
  
```  
  
name ENDS  
```
