# CATSTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

텍스트 항목을 연결합니다.  
  
## 구문  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## 설명  
 각 텍스트 항목 앞에 상수 리터럴 문자열 수는 `%`, 또는 매크로 함수에 의해 반환 되는 문자열입니다.
