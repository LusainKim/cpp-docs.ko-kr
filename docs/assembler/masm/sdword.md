# <a name="sdword"></a>SDWORD
할당 및 각각에 대해 저장소의 부호 있는 2 배 워드 (4 바이트)를 선택적으로 초기화 `initializer`합니다. 또한 형식 지정자로 사용할 수 있습니다 형식이 법적 곳입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
