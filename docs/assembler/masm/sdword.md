# SDWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

할당 하 고 필요에 따라 저장소에 서명 된 이중 단어 \(4 바이트\) 각각에 대 한 초기화 `initializer`.  또한 형식 지정자로 사용할 수 있습니다 아무 곳 이나 형식이 잘못 되었습니다.  
  
## 구문  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```
