# .ERRDIF]]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

텍스트 항목에 서로 다른 경우 오류가 발생 합니다.  
  
## 구문  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## 설명  
 경우  **나** 입니다, 비교는 대\/소문자입니다.
