# @InStr
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

첫 항목을 찾는 매크로 함수  *문자열 2* 에서  *문자열 1*시작  *위치* 내  *문자열 1*.  경우  *위치* 표시 되지 않습니다, 검색의 시작 부분에 시작  *문자열 1*.  위치 정수 0 반환 하면  *문자열 2* 찾을 수 없습니다.  
  
## 구문  
  
```  
  
@InStr( [[position]], string1, string2 )  
```
