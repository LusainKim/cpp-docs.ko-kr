# BYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

할당 하 고 필요에 따라 각각에 대 한 바이트 저장소를 초기화 `initializer`.  
  
## 구문  
  
```  
[[name]] BYTE initializer [[, initializer]] ...  
```  
  
## 설명  
 또한 형식 지정자로 사용할 수 있습니다 아무 곳 이나 형식이 잘못 되었습니다.
