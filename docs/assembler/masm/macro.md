# MACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

라는 매크로 블록 표시  *이름* 를 설정 하 고  *매개 변수*  인수에 대 한 개체 틀 매크로가 호출 될 때 전달 된.  
  
## 구문  
  
```  
  
   name MACRO [[parameter [[:REQ | :=default | :VARARG]]]]...  
statements  
ENDM [[value]]  
```  
  
## 설명  
 매크로 함수 반환  *값* 를 호출 하는 문의 합니다.
