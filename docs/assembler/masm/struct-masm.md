# STRUCT (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

구조체 형식에 지정 된 것을 선언  *fielddeclarations*.  각 필드에 올바른 데이터 정의 여야 합니다.  다른 이름으로 동일한  [STRUC](../../assembler/masm/struc.md).  
  
## 구문  
  
```  
  
   name STRUCT [[alignment]] [[, NONUNIQUE]]  
fielddeclarations  
name ENDS  
```
