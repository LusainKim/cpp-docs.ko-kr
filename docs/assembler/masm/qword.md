# QWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

할당 하 고 필요에 따라 각각에 대해 8 바이트의 저장소를 초기화 `initializer`.  또한 형식 지정자로 사용할 수 있습니다 아무 곳 이나 형식이 잘못 되었습니다.  
  
## 구문  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```
