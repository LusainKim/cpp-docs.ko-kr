# .FARDATA
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

사용 하는 경우  [.모델](../../assembler/masm/dot-model.md)에서 초기화 된 데이터에 대 한 먼 데이터 세그먼트의 시작 \(세그먼트 이름을 FAR\_DATA 또는  *이름*\).  
  
## 구문  
  
```  
  
.FARDATA [[name]]  
```
