# ML Nonfatal Error A2078
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**원거리 지침을 허용 하지 않습니다 간접 주소 지정**  
  
 조건부 점프 또는 루프 메모리 피연산자를 사용할 수 없습니다.  상대 주소 또는 레이블을 지정 해야 합니다.
