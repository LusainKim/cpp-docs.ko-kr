# FOR (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

각각에 대해 반복 한 번 블록을 표시 `argument`, 현재와 `argument` 교체 `parameter` 에서 반복 되는.  
  
## 구문  
  
```  
  
   FOR   
   parameter [[:REQ | :=default]] , <argument [[, argument]]...>  
statements  
ENDM  
```
