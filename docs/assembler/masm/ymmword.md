# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

인텔 고급 벡터 확장 \(AVX\) 지시와 256 비트 멀티미디어 피연산자에 사용 됩니다.  
  
## 구문  
  
```  
YMMWORD  
```  
  
## 설명  
 `YMMWORD`동일한 형식으로 나타내기 위한 것입니다 `__m256` AVX 내장.
