# ML Nonfatal Error A2063
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**2 전원에 맞춤 수 있습니다: 식**  
  
 지정 된 식의  [맞춤](../../assembler/masm/align-masm.md) 지시문에 유효 하지 않습니다.  
  
 해당  **맞춤** 식 2와 256, 2의 거듭제곱 이어야 하며 현재 세그먼트, 구조체 또는 공용 구조체의 맞춤 보다 작거나 같아야 합니다.
