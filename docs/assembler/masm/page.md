# PAGE
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

첫 번째 지시문 줄을 설정  *길이* 및 문자  *폭* 프로그램 목록입니다.  인수가 제공 된 경우 페이지 나누기를 생성 합니다.  두 번째 지시문 섹션 수를 증가 하 고 페이지 번호를 1로 다시 설정 합니다.  
  
## 구문  
  
```  
  
      PAGE [[[[length]], width]]  
PAGE +  
```
