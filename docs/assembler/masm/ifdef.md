# <a name="ifdef"></a>IFDEF
어셈블리를 부여 *이름* 이전에 정의 된 레이블, 변수 또는 기호입니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
IFDEF name  
```  
  
## <a name="remarks"></a>설명  
 참조 [IF](../../assembler/masm/if-masm.md) 전체 구문에 대 한 합니다.
