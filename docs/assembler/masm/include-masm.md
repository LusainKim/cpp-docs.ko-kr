# INCLUDE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

삽입 소스 코드에서 주어진 소스 파일  *파일 이름* 어셈블리 중 현재 소스 파일에 있습니다.  
  
## 구문  
  
```  
  
INCLUDE filename  
```  
  
## 설명  
 *파일 이름* 세미콜론, 큰 백슬래시 포함 되어 있는 경우 꺽쇠 괄호 안에 묶어야 합니다\-기호, 더 적은\-기호, 작은따옴표 또는 큰따옴표입니다.
