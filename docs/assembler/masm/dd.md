# <a name="dd"></a>DD
할당 및 필요에 따라 각각에 대해 저장소의 2 배 워드 (4 바이트)을 초기화 `initializer`합니다. `DD`동의어 [DWORD](../../assembler/masm/dword.md)합니다.  
  
## <a name="syntax"></a>구문  
  
```  
[[name]] DD initializer [[, initializer]]...  
```  
  
## <a name="remarks"></a>설명  
 형식 지정자로 사용할 수도 있습니다 형식이 법적 곳입니다.
