# DD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

할당 하 고 필요에 따라 이중 단어 \(4 바이트\) 저장소의 각각에 대 한 초기화 `initializer`.  `DD`동의어입니다  [DWORD](../../assembler/masm/dword.md).  
  
## 구문  
  
```  
[[name]] DD initializer [[, initializer]]...  
```

 또한 형식 지정자로 사용할 수 있습니다 아무 곳 이나 형식이 잘못 되었습니다.
