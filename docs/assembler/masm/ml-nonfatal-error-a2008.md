# ML Nonfatal Error A2008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**구문 오류:**  
  
 현재 위치 하는 토큰 구문 오류가 발생 합니다.  
  
 다음 중 하나가 발생할 수 있습니다.  
  
-   도트 접두사 추가 또는 지시문에서 생략 합니다.  
  
-   예약어 \(같은  **C** 또는  **크기**\)를 식별자로 사용 되었습니다.  
  
-   명령은 현재 프로세서 또는 보조 프로세서 선택에 사용할 수 없었던 사용 되었습니다.  
  
-   런타임 비교 연산자 \(같은 `==`\) 관계형 연산자 대신 조건부 어셈블리 문에서 사용 되었습니다 \(같은  [EQ](../../assembler/masm/operator-eq.md)\).  
  
-   너무 적은 피연산자에 명령 또는 지시문에 지정 되었습니다.  
  
-   사용 되지 않는 지시문을 사용 했습니다.
