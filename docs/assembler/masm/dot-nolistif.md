# .NOLISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

False \(0\)을 조건이 조건부 블록의 목록을 표시 하지 않습니다.  
  
## 구문  
  
```  
  
.NOLISTIF  
  
```  
  
## 설명  
 이 값이 기본값입니다.  다른 이름으로 동일한  [.SFCOND](../../assembler/masm/dot-sfcond.md).
