# operator SECTIONREL
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

식 대상 최종 실행 파일에 포함 된 섹션에 상대적인 섹션 상대 오프셋을 반환 합니다.  
  
## 구문  
  
```  
SECTIONREL expression  
```  
  
## 설명  
 SECTIONREL는 COFF 개체 내보내기를 사용할 수 있습니다.
