# ML Fatal Error A1008
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**일치 하지 않는 매크로 중첩**  
  
 두 매크로 파일 또는 종료 지시문의 끝 전에 종료 되지 않았습니다  [ENDM](../../assembler/masm/endm.md) 매크로 블록 외부에서 찾을 수 있습니다.  
  
 이 오류의 원인 중 하나는 생략을 점 전  [.반복](../../assembler/masm/dot-repeat.md) 또는  [.반면](../../assembler/masm/dot-while.md).
