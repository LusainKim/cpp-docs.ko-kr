# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

상위 16 비트의 하위 32 비트를 반환 합니다.  *식*.  MASM 식이 64 비트 값입니다.  
  
## 구문  
  
```  
  
HIGHWORD expression  
```
