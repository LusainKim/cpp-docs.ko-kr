# ML Nonfatal Error A2107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**암시적 멀리 이동 또는 레이블 근처에 호출을 사용할 수 없습니다.**  
  
 암시적 멀리 점프 또는 다른 세그먼트에 가까운 레이블을 호출 확인 하려고 했습니다.
