# <a name="ml-nonfatal-error-a2107"></a>ML 심각하지 않은 오류 A2107
**암시적 먼 점프 또는 레이블 근처에 호출을 사용할 수 없습니다.**  
  
 암시적까지 이동 하거나 다른 세그먼트에 가까운 레이블에 호출 하려고 했습니다.
