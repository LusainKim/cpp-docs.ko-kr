# operator ||
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Logical `OR`.  내 에서만 사용 되는  [.IF](../../assembler/masm/dot-if.md), [.WHILE](../../assembler/masm/dot-while.md), or [.반복](../../assembler/masm/dot-repeat.md) 차단 되며 런타임에 어셈블리 시간에 평가 됩니다.  
  
## 구문  
  
```  
  
expression1 || expression2  
```
