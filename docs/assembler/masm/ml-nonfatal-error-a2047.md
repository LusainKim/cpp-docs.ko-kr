# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**빈 \(null\) 문자열**  
  
 문자열 구분 쌍 따옴표 및 내 문자 없이 구성 되어 있습니다.  
  
 유효한 문자열 자로 있어야 합니다.
