# LABEL (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

현재 위치 카운터 값을 할당 하 여 새 레이블을 만듭니다 및 특정 `type` 에  *이름*.  
  
## 구문  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
