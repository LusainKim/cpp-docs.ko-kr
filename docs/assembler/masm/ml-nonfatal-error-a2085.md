# ML Nonfatal Error A2085
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**현재 CPU 모드에서 허용 되지 등록 이나 명령**  
  
 명령, 레지스터 또는 현재 프로세서 모드에 대 한 잘못 된 키워드를 사용 하려고 했습니다.  
  
 예를 들어, 32 비트 레지스터를 필요로  [.386](../../assembler/masm/dot-386.md) 이상입니다.  컨트롤 등록 있음이 필요한 권한이 있는 모드 등  [.386P](../../assembler/masm/dot-386p.md) 이상입니다.  에 대해이 오류가 발생할 수 있습니다의  **NEAR32**,  **FAR32**, 및  **플랫** 필요로 하는 키워드입니다. **386** 이상입니다.
