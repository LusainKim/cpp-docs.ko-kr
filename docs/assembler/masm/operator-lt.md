# operator LT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

경우 true \(\-1\)를 반환  *식 1* 입니다 보다 작은  *식 2*, 또는 그렇지 않은 경우 false \(0\)를 반환 합니다.  
  
## 구문  
  
```  
  
expression1 LT expression2  
```
