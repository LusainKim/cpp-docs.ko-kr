# PUSHCONTEXT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

일부 또는 전체를 현재 저장 `context`: 기 수, 플래그에 대 한 cref 이상의 값 프로세서\/보조 프로세서 세그먼트 레지스터가 가정 합니다.  The `context` can be **ASSUMES**, `RADIX`, **LISTING**, **CPU**, or **ALL**.  
  
## 구문  
  
```  
  
PUSHCONTEXT context  
```
