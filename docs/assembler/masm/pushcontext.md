# <a name="pushcontext"></a>PUSHCONTEXT
전체 또는 일부에 현재 저장 `context`: 기 수 값, 목록 및 cref 플래그 또는 프로세서/보조 프로세서 값 세그먼트 레지스터 가정 합니다. `context` 수 **가정**, `RADIX`, **나열**, **CPU**, 또는 **모든**합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
PUSHCONTEXT context  
```
